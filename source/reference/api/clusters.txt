========
Clusters
========

.. default-domain:: mongodb

The ``clusters`` resource provides access to your cluster configurations. The
resource lets you create, edit and delete clusters. The resource requires your
:ref:`group-id`.

.. important::

   Changes to cluster configurations can affect costs. Before making changes,
   please see :doc:`/billing`.

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``

     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/clusters </reference/api/clusters-get-all>`

     - Get all clusters for the specified group.

   * - ``GET``

     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/clusters/{CLUSTER-NAME} </reference/api/clusters-get-one>`

     - Get a cluster in the specified group.

   * - ``POST``

     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/clusters/{CLUSTER-NAME} </reference/api/clusters-create-one>`

     - Create a cluster in the specified group.

   * - ``PATCH``

     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/clusters/{CLUSTER-NAME} </reference/api/clusters-modify-one>`

     - Update a cluster in the specified group.

   * - ``DELETE``

     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/clusters/{CLUSTER-NAME} </reference/api/clusters-delete-one>`

     - Delete a cluster in the specified group.
