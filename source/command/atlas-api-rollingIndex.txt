======================
atlas api rollingIndex
======================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

`Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Creates one index to a database deployment in a rolling manner.

You can't create a rolling index on an M0 free cluster or M2/M5 shared cluster.
