``cluster``
-----------

The ``cluster`` command provides access to your cluster configurations.

- :ref:`List clusters <mcli-atlas-cluster-list-command>`
- :ref:`Describe a cluster <mcli-atlas-cluster-describe-command>`
- :ref:`Create a cluster <mcli-atlas-cluster-create-command>`

- :ref:`Delete a cluster <mcli-atlas-cluster-delete>`

``alerts``
----------

The ``alerts`` command allows you to list |service|
:atlas:`alerts </configure-alerts>`.

- :ref:`List alerts <mcli-atlas-list-alert-config-command>`

``backup``
----------

The ``backup`` command allows you to manage restore jobs.

- :ref:`Start a backup restore job <mcli-atlas-backup-restore-start-cmd>`
- :ref:`List snapshots <mcli-atlas-backup-snapshots-list-cmd>`

``dbuser``
----------

The ``dbuser`` command allows you to list, create, modify, and
delete MongoDB database users.

- :ref:`List database users <mcli-atlas-dbuser-list-command>`
- :ref:`Create a database user <mcli-atlas-dbuser-create-command>`
- :ref:`Modify a database user <mcli-atlas-dbuser-update-command>`
- :ref:`Delete a database user <mcli-atlas-dbuser-delete-command>`
