==============================
Restore Continuous Backup Jobs
==============================

.. default-domain:: mongodb

.. include:: /includes/admonitions/deprecated-continuous-backup.rst

.. include:: /includes/fact-projects-groups-synonyms.rst

Create and retrieve restores jobs for a MongoDB cluster. A restore
job restores a MongoDB cluster to its state from an existing snapshot.

.. include:: /includes/fact-continuous-backup-only-resource.rst

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - GET
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/restoreJobs </reference/api/restore-jobs-get-all>`
     - Retrieve all restore jobs for the specified cluster.

   * - GET
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/restoreJobs/{JOB-ID} </reference/api/restore-jobs-get-one>`
     - Retrieve one restore job for the specified cluster.

   * - POST
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/restoreJobs </reference/api/restore-jobs-create-one>`
     - Create one restore job for the specified cluster.
