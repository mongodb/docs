=================
MMS -- Monitoring
=================

.. only:: saas

   The Monitoring component of MMS (|mms|) is a service created by
   `10gen <http://10gen.com>`_ for monitoring MongoDB deployments. MMS
   Collects statistics on all key server and hardware indicators and
   presents this data through an intuitive web interface.

   This manual describes the installation of the MMS agent and
   operation of the MMS web console. You can find answers to common
   questions in the FAQs, but for all other inquiries please feel free
   to `open a JIRA ticket <https://jira.mongodb.org/secure/CreateIssue.jspa?pid=10582&issuetype=7>`_.
