=========================================
mongocli atlas liveMigrations link delete
=========================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Your API Key must have the Organization Owner role to successfully run this command.
