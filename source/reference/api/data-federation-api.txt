.. _adl-api:

===========================================
{+adf+}
===========================================

.. default-domain:: mongodb

.. meta::
   :keywords: |data-lake|
   :canonical: https://www.mongodb.com/docs/atlas/reference/api-resources-spec/#tag/Data-Federation

The ``dataFederation`` resource provides access to the project's 
{+fdi+} configuration. The resource lets you view, create,
modify, and delete {+fdi+}s. The resource requires your
:ref:`project <projects>`.

.. include:: /includes/fact-projects-groups-synonyms.rst

.. important::

   Changes to {+fdi+} configurations can affect costs. Before 
   making changes, please see :doc:`Billing </billing>`.

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/dataFederation </reference/api/dataFederation-get-all-tenants>`
     - Get all {+fdi+}s for the specified project.

   * - ``GET``
     - :ref:`/api/atlas/v1.0/groups/{GROUP-ID}/privateNetworkSettings/endpointIds <api-pvt-link-get-all>` 
     - Get all the private endpoints for the {+fdi+}s in the specified 
       project.

   * - ``GET``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/dataFederation/{NAME} </reference/api/dataFederation-get-one-tenant>`
     - Get an {+fdi+} in the specified project.

   * - ``GET``
     - :ref:`/api/atlas/v1.0/groups/{GROUP-ID}/privateNetworkSettings/endpointIds/{ENDPOINT-ID} <api-pvt-link-get-one>`
     - Get the specified private endpoint in the specified project.

   * - ``GET``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/dataFederation/{tenantName}/queryLogs.gz </reference/api/dataFederation-download-query-logs>`
     - Download query log for the specified {+fdi+}.

   * - ``POST``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/dataFederation </reference/api/dataFederation-create-one-tenant>`
     - Create a {+fdi+} in the specified project.

   * - ``POST``
     - :ref:`/api/atlas/v1.0/groups/{GROUP-ID}/privateNetworkSettings/endpointIds <api-pvt-link-create-one>`
     - Add one |aws| private endpoint for the {+fdi+}s in the 
       specified project.

   * - ``PATCH``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/dataFederation/{NAME} </reference/api/dataFederation-update-one-tenant>`
     - Update an {+fdi+} in the specified project.

   * - ``DELETE``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/dataFederation/{NAME} </reference/api/dataFederation-delete-one-tenant>`
     - Delete a {+fdi+} in the specified project.

   * - ``DELETE``
     - :ref:`/api/atlas/v1.0/groups/{GROUP-ID}/privateNetworkSettings/endpointIds/{ENDPOINT-ID} <api-pvt-link-delete-one>`
     - Delete one |aws| private endpoint in the specified project.
