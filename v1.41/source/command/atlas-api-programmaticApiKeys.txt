=============================
atlas api programmaticApiKeys
=============================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

`Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Returns, adds, edits, and removes access tokens to use the MongoDB Cloud API.

MongoDB Cloud applies these keys to organizations. These resources can return, assign, or revoke use of these keys within a specified project.
