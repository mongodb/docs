`experimental <https://www.mongodb.com/docs/atlas/cli/current/command/atlas-api/>`_: Updates one Atlas Search index that you identified with its database, collection name, and index name.

Atlas Search indexes define the fields on which to create the index and the analyzers to use when creating the index. To use this resource, the requesting API Key must have the Project Data Access Admin role.

This command is autogenerated and corresponds 1:1 with the Atlas API endpoint https://www.mongodb.com/docs/atlas/reference/api-resources-spec/v2/#tag/Atlas-Search/operation/updateAtlasSearchIndexByName.
