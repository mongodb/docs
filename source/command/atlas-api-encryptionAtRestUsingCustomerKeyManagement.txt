atlas api encryptionAtRestUsingCustomerKeyManagement

`Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Returns and edits the Encryption at Rest using Customer Key Management configuration.

MongoDB Cloud encrypts all storage whether or not you use your own key management.
