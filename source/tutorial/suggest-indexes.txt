===========================
Calculate Suggested Indexes
===========================

.. default-domain:: mongodb

.. only:: cloud

   .. note:: **Available only with Cloud Manager Premium**. This feature is
      available only with Cloud Manager Premium, which comes with certain
      MongoDB subscriptions. `Contact MongoDB <https://www.mongodb.com/contact>`_
      for more information.

Overview
--------

|mms| can analyze the data on query patterns collected by the database
profiler and suggest a set of indexes that could improve query performance.
|mms| scores each suggested index on its expected benefit.

.. _profiling-prerequisites:

Prerequisites
-------------

For |mms| to suggest indexes, the following must be true:

- You must enable database profiling for the MongoDB process. See :doc:`/tutorial/profile-database`.

- The profiler must have data. If profiling is enabled, but no profiling
  data has yet been collected, |mms| cannot suggest indexes.
