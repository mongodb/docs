Manage |mms| Users and Roles
============================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

|mms| manages user authentication and feature authorization.

|mms| requires users to log in to authenticate themselves. Once a user
has logged into |mms|, |mms| reviews which roles the user has been
assigned to authorize what |mms| features the user can use.
