=================
atlas plugin list
=================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Returns a list of all installed plugins.

Syntax
------

.. code-block::
   :caption: Command Syntax

   atlas plugin list [options]

.. Code end marker, please don't delete this comment
