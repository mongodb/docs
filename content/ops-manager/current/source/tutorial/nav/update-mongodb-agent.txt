===========================
Update to the {+mdbagent+}
===========================

.. meta::
   :description: Update your legacy Ops Manager Monitoring, Backup, and Automation Agents to the unified MongoDB Agent by following the appropriate transition scenario.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The {+mdbagent+} replaces the legacy Monitoring, Backup, and
Automation Agents with a single unified agent. If you need to
update your legacy Agents to the {+mdbagent+}, follow one of
the scenarios below to make the transition based on which
legacy agents you have installed.
