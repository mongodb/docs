=================
Send Test Message
=================

.. meta::
   :description: Send test email or SMS messages to users from Ops Manager to verify that notification addresses are valid and alert delivery channels are configured correctly.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol


Use this page in the |onprem| UI to send messages to users to test the validity of
their addresses. Sending a test message helps you confirm that
|onprem| can reach recipients. You can
send test messages by email or |sms|.
