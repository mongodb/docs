.. _om-telemetry:

=========
Telemetry
=========

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

Telemetry collection gathers usage information about your |onprem|
installation and sends it to MongoDB. |onprem|
enables collecting and sending telemetry data by default.

.. important::

   This feature is available starting in |onprem| 8.0.17 and 7.0.20.

To learn more about telemetry behavior and the data that |onprem| 
collects, see :ref:`Telemetry Overview <om-telemetry-overview>`.

Endpoints
-------------------

The following lists the endpoints available for telemetry.

.. list-table::
   :widths: 30 15 55
   :header-rows: 1

   * - Endpoint
     - Method
     - Description

   * - :ref:`Retrieve Telemetry Data <om-get-telemetry-collection-details>`
     - GET
     - Retrieve telemetry collection status and configuration details.

   * - :ref:`Toggle Telemetry Status <om-patch-toggle-telemetry-status>`
     - PATCH
     - Enable or disable telemetry collection for your |onprem| installation.

Required Roles
--------------

All telemetry endpoints require the :authrole:`Global Owner` role.
