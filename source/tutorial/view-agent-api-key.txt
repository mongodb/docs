============================
View a Group's Agent API Key
============================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: twocols

Overview
--------

Each group has an :term:`agent API key` that all group agents use when
communicating with |mms|.

View the Key
------------

To view the key, click :guilabel:`Settings`, then :guilabel:`Group Settings`.
The key appears at the top of the page.
