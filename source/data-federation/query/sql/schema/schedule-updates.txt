Schedule Schema Update Using the |service| UI
---------------------------------------------

Scheduled schema updates sample each namespaces and merge the schema with the existing one.
They are intended to be used for maintaining schema accuracy over time.
For example, schema updates allow {+adf+} to automatically pick-up new fields added to collections.

Use the following procedure to schedule schema updates:

.. procedure::
   :style: normal

   .. include:: /includes/nav/steps-db-deployments-page.rst

   .. step:: Navigate to Manage SQL Schemas page.

             From the Data Federation menu, click :guilabel:`Manage SQL Schemas`.

   .. step:: Enable scheduled schema updates.

      a. Click :guilabel:`Configure schema update schedule`.
      b. Select a frequency.
      c. Click :guilabel:`Save`.
