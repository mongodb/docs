.. _mcli-cm-cluster-describe-command:

=======================================
mongocli cloud-manager cluster describe
=======================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``cluster describe`` command retrieves details about 
the specified cluster in the specified |cloud-short| 
project. You can also retrieve details about a cluster through 
the |cloud-short| :cloudmgr:`API </reference/api/clusters/>`.

.. _cm-cluster-describe-syntax: 

Syntax 
------

.. code-block:: text 

   mongocli cloud-manager|cm cluster describe <cluster-name> 
        [ --profile|-P <profile-name> ]
        [ --projectId <project-ID> ]

.. include:: /includes/fact-command-line-help.rst

.. _cm-cluster-describe-options: 

Options 
-------

.. list-table:: 
   :header-rows: 1
   :widths: 20 10 60 10 

   * - Option 
     - Type 
     - Description 
     - Required?

   * - ``<cluster-name>``
     - string 
     - Name of the cluster to retrieve.
     - yes

   * - ``--profile``, ``-P``
     - string
     - Name of the profile where the public and private 
       keys for the project are saved. If omitted, uses the 
       ``default`` profile. To learn more about creating a 
       profile, see :ref:`mcli-configure`.
     - no

   * - ``--projectId``
     - string
     - Unique identifier of the project that contains the 
       cluster to update. If omitted, uses the project ID in 
       the profile or the :ref:`environment variable <mcli-env-var>`.
     - no

.. _cm-cluster-describe-output:

Output 
------

The command prints the following fields in |json| format to 
the terminal if the command succeeds. If the command returns 
errors, see :ref:`Troubleshooting <troubleshooting>` for 
recommended solutions.

.. include:: /includes/mms-config-props.rst

.. _cm-cluster-describe-examples:

Examples 
--------

The following examples show two ways to run the command.

.. _cm-cluster-describe-example1:

Example 1
~~~~~~~~~

The following ``mongocli cm cluster describe`` command 
retrieves the details about the cluster named ``myCluster`` in the 
specified project. It uses the default profile to access the 
specified project.

.. code-block:: sh
   :copyable: false

   mongocli cm cluster describe myCluster --projectId 5e338925f10fab0a7846874b

.. _cm-cluster-describe-example2:

Example 2
~~~~~~~~~

The following ``mongocli cm cluster describe`` command 
retrieves details about the cluster named ``myCluster``. It 
uses the profile named ``egCMprofile`` where the |api| access keys 
and the project ID are stored.

.. code-block:: sh
   :copyable: false

   mongocli cloud-manager cluster describe myCluster --profile egCMprofile
