=========================
appservices function list
=========================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

List the functions of your app

Displays a list of your app's functions and their directories. Test files will not be taken into account. Navigate to a function's directory to access its source and view its arguments.

Syntax
------

.. code-block::
   :caption: Command Syntax

   appservices function list [options]

.. Code end marker, please don't delete this comment

Options
-------

.. list-table::
   :header-rows: 1
   :widths: 20 10 10 60

   * - Name
     - Type
     - Required
     - Description
   * - --local
     - string
     - false
     - Specify the local filepath of an app to list functions from
   * - -h, --help
     - 
     - false
     - help for list
