.. _mcli-cm-cluster-list-command:

===============================
mcli cloud-manager cluster list
===============================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``cluster list`` command retrieves all the clusters 
in the specified |cloud| project. You can also retrieve 
all the clusters in a project through the |cloud-short| 
:cloudmgr:`UI </tutorial/view-all-clusters/>` or 
:cloudmgr:`API <reference/api/clusters/>`.

.. _cm-cluster-list-syntax:

Syntax 
------

.. code-block:: text 
   
   mcli cm|cloud-manager cluster list 
        [ -p|--profile <profile-name> ]
        [ --projectId projectId ]

.. _cm-cluster-list-options:

Options
-------

.. list-table:: 
   :header-rows: 1
   :widths: 20 10 60 10 

   * - Option 
     - Type 
     - Description 
     - Required?

   * - ``-p``, ``--profile``
     - string
     - The name of the profile where the public and private 
       keys for the project are saved. If omitted, uses the 
       ``default`` profile. To learn more about creating a 
       profile, see :ref:`configure-mcli`.
     - no

   * - ``--projectId``
     - string
     - The unique identifier of the project that contains the 
       cluster to update. If omitted, uses the project ID in 
       the profile or the :ref:`environment variable <mcli-env-var>`.
     - no

.. _cm-cluster-list-output: 

Output 
------

The command prints the following fields in |json| format to 
the terminal if the command succeeds. If the command returns 
errors, see :ref:`Troubleshooting <troubleshooting>` for 
recommended solutions.

.. include:: /includes/mms-config-props.rst

.. _cm-cluster-list-examples:

Examples 
--------

The following examples show two ways to run the command.

.. _cm-cluster-list-example1:

Example 1
~~~~~~~~~

The following ``mcli cm clusters list`` command 
returns the clusters in the specified project in |json| format. 
It uses the profile named ``egCMprofile`` to access the specified 
project.

.. code-block:: text
   :copyable: false

   ~ mcli cm clusters list --projectId 5e2f06127a3e5a5700de8d8d --profile egCMprofile

.. _cm-cluster-list-example2:

Example 2
~~~~~~~~~

The following ``mcli cloud-manager clusters list`` command 
returns the clusters in the specified project in |json| format. 
It uses the ``default`` profile to access the specified project.

.. code-block:: sh
   :copyable: false

   ~ mcli cloud-manager clusters list --projectId 5e2f06127a3e5a5700de8d8d
