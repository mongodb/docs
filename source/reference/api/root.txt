Root
----

This is the starting point (or the homepage, if you will) for the MMS API. From
here, you can traverse the ``links`` to reach all other API resources.

Sample Entity
~~~~~~~~~~~~~

.. code:: javascript

  {
    "throttling": false,
    "links": [ ... ]
  }


Entity Fields
~~~~~~~~~~~~~

.. list-table::
   :widths: 10 10 80
   :header-rows: 1

   * - Name
     - Type
     - Description
   * - throttling
     - boolean
     - Tells whether or not MMS is throttling data. This can be used as a simple
       indicator of the current health of MMS, since throttling is generally
       enabled when MMS is in an unhealthy state.

Links
~~~~~

.. list-table::
   :widths: 10 90
   :header-rows: 1

   * - Relation
     - Description
   * - self
     - Me
   * - groups
     - Groups accessible to the current API user.
   * - user
     - The current API user.

Example
~~~~~~~

Retrieve the root resource:

.. code-block:: sh

   curl -u "<username>:<apiKey>" "https://mms.mongodb.com/api/public/v1.0" --digest -i

.. code-block:: http

   HTTP/1.1 200 OK

   {
     "throttling" : false,
     "links" : [ ... ]
   }
