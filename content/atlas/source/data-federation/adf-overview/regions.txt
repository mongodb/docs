.. note::

   To prevent excessive charges on your bill, create your {+adf+} in
   the same cloud provider region as your data source. You can query
   |aws| |s3| only using {+fdi+}\s created in |aws|, query {+az-bs+}
   only using {+fdi+}\s created in |azure|, and query {+gcs+}
   only using {+fdi+}\s created in |gcp|.

{+adf+} routes your federated database requests through one of the 
following regions:
