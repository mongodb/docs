=============================
|monitoring| Service Tutorial
=============================

.. only:: hosted

   To begin using |monitoring|, :ref:`install the On-Prem Monitoring
   server <tutorial-install-onprem>`, then:

- :ref:`install the Monitoring agent for UNIX/Linux
  <tutorial-install-mms-agent>`, or :ref:`install the Monitoring agent for
  Windows <tutorial-install-windows>`, and

- :ref:`deploy the MMS Monitoring Agent in Production <tutorial-deploying-mms>`

Once you have set up |monitoring|, you can :ref:`start monitoring hosts <tutorial-monitoring>`.
