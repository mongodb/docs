.. procedure:: 
   :style: normal

   .. step:: Go to the :guilabel:`Search Tester` page.

      a. Click the cluster name to view the cluster details.
      #. Click the :guilabel:`Search` tab.
      #. Click the :guilabel:`Query` button to the right of the index to
         query. 

   .. step:: View and edit the query syntax.

      Click :guilabel:`Edit $search Query` to view a default query
      syntax sample in |json| format.
      
   .. step:: Run an |fts| query with the ``embeddedDocument`` operator on the ``schools`` collection.

      Copy and paste the following query into the :guilabel:`Query
      Editor`, and then click the :guilabel:`Search` button in the
      :guilabel:`Query Editor`.

      .. note::

         .. include:: /includes/fact-fts-expand-search-tester-results.rst

      .. tabs:: 

         .. tab:: Basic Search 
            :tabid: basic

            .. include:: /includes/extracts/fts-embedded-document-simple-query-intro.rst
      
            .. io-code-block::
               :copyable: true
      
               .. input:: 
                  :language: json
                  :linenos: 

                  [
                    {
                      "$search": {
                        "index": "embedded-documents-tutorial",
                        "embeddedDocument": {
                          "path": "teachers",
                          "operator": {
                            "compound": {
                              "must": [{
                                "text": {
                                  "path": "teachers.first",
                                  "query": "John"
                                }
                              }],
                              "should":[{
                                "text": {
                                  "path": "teachers.last",
                                  "query": "Smith"
                                }
                              }]
                            }
                          }
                        }
                      }
                    }
                  ]
