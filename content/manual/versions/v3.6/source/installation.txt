===============
Install MongoDB
===============

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

.. index:: tutorials; installation
.. index:: installation tutorials
.. index:: installation guides
.. index:: installation

MongoDB is available in two editions: *Community* and *Enterprise*.

This section of the manual contains information on installing MongoDB.
   
- For instructions on upgrading your current deployment to MongoDB 3.6,
  see :ref:`3.6-upgrade` instead.

- For instructions on upgrading to the latest patch release for your
  current version, see :doc:`/tutorial/upgrade-revision` instead.

Starting in 3.6, each MongoDB installer is packaged with a
platform-specific :ref:`MongoDB Compass <compass-index>` installation
script. This script installs MongoDB Compass as part of the MongoDB
Server installation process.
