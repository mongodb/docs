========================
Create One Agent API Key
========================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none

.. include:: /includes/api/base-api-uri.rst

Resource
--------

.. code-block:: http
   :copyable: false

   POST /groups/{PROJECT-ID}/agentapikeys

This endpoint is restricted to users with the
:authrole:`Project Owner` role.

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :widths: 20 20 60
   :header-rows: 1
   :stub-columns: 1

   * - Name
     - Type
     - Description

   * - PROJECT-ID
     - string
     - Unique identifier of the :cloudmgr:`project </reference/glossary/#std-term-project>` that owns the
       Agent API Key.

Request Query Parameters
~~~~~~~~~~~~~~~~~~~~~~~~

.. include:: /includes/api/no-query-single-parameters.rst

Request Body Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :widths: 20 20 60
   :header-rows: 1
   :stub-columns: 1

   * - Name
     - Type
     - Description

   * - desc
     - string
     - Label for this Agent API Key.

Response
--------

.. include:: /includes/api/list-tables/agent-api-keys-response-documents.rst

Example Request
---------------

.. code-block:: sh

   curl --user "{PUBLIC-KEY}:{PRIVATE-KEY}" --digest \
        --header "Accept: application/json" \
        --header "Content-Type: application/json" \
        --include \
        --request POST "https://cloud.mongodb.com/api/public/v1.0/groups/{PROJECT-ID}/agentapikeys?pretty=true" \
        --data '
          {
            "desc": "Agent API Key for this project"
          }'
