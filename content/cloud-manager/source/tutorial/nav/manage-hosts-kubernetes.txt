=========================================
Deploy MongoDB Resources Using Kubernetes
=========================================

.. meta::
   :description: Deploy MongoDB resources using Kubernetes with Cloud Manager and the Kubernetes Operator for replica sets, sharded clusters, and standalone instances.

.. default-domain:: mongodb

  Use |mms| UI to set up Kubernetes for deploying MongoDB resource.
  
:ref:`Deploy a Replica Set Using Kubernetes <deploy-replica-set>`
  Use |mms| and the |k8s-op-short| to deploy a managed replica set.

`Deploy a Sharded Cluster Using Kubernetes <https://www.mongodb.com/docs/kubernetes/current/tutorial/deploy-sharded-cluster/>`__
  Use |mms| and the |k8s-op-short| to deploy a managed sharded cluster.

:ref:`Deploy a Standalone Using Kubernetes <deploy-standalone>`
  Use |mms| and the |k8s-op-short| to deploy a standalone for testing
  and development.
