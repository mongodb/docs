- Host alerts support these fields:

  - ``HOSTNAME``
  - ``PORT``
  - ``HOSTNAME_AND_PORT``
  - ``REPLICA_SET_NAME``
  - ``TYPE_NAME``

- Replica set alerts support these fields:

  - ``REPLICA_SET_NAME``
  - ``SHARD_NAME``
  - ``CLUSTER_NAME``

- Sharded cluster alerts support these fields:

  - ``CLUSTER_NAME``
  - ``SHARD_NAME``

All other types of alerts do not support matchers.
