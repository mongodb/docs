===================
MongoDB with Cursor
===================

.. meta::
   :description: Learn how to use MongoDB with Cursor.

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

You can use MongoDB with Cursor through the official MongoDB Cursor Plugin.

The MongoDB Cursor Plugin provides:

- The MongoDB MCP Server
- Pre-built agent skills for common MongoDB tasks. See the :ref:`MongoDB Agent Skills documentation <agent-skills>` for more information.

.. _cursor-plugin-install:

Install the MongoDB Cursor Plugin
---------------------------------

The MongoDB Cursor Plugin is available in the official Cursor Plugin Marketplace.

Install the MongoDB Cursor Plugin from Cursor
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

To install the MongoDB Cursor Plugin:

1. Open Cursor.
#. Install the plugin: 

   .. code-block:: bash

      /add-plugin mongodb

#. Follow the prompts to complete the installation.

After installing the plugin, you must configure the MCP Server before using
any agent skills or functionality that connects to MongoDB.

Configure the MCP Server
------------------------

The MongoDB Cursor Plugin bundles the MongoDB MCP Server. To configure the MCP
Server, see `MongoDB MCP Server Get Started <https://www.mongodb.com/docs/mcp-server/get-started/>`__.
