=========================
Configure SSL for MongoDB
=========================

.. default-domain:: mongodb

Overview
--------

If a MongoDB deployment uses SSL, then you must enable the SSL setting for
the deployment.

.. only:: onprem

   You also must configure the agents' SSL settings, as described in
   :doc:`/tutorial/configure-monitoring-agent-for-ssl` and
   :doc:`/tutorial/configure-backup-agent-for-ssl`.

Procedures
----------

Enable SSL for a Monitored Host
+++++++++++++++++++++++++++++++

Before enabling SSL for a monitored host, update the Monitoring Agent's configuration
file to use SSL. See :doc:`/tutorial/configure-monitoring-agent-for-ssl`.

To enable SSL for a host:

.. include:: /includes/steps/monitoring-agent-edit-host-ssl.rst

Enable SSL for a Backup
+++++++++++++++++++++++

Before enabling SSL for a backup, update the Backup Agent's configuration
file to use SSL. See :doc:`/tutorial/configure-backup-agent-for-ssl`.

To enable SSL for a backup:

.. include:: /includes/steps/backup-agent-edit-host-ssl.rst
