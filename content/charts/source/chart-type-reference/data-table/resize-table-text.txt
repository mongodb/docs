=================
Resize Table Text
=================

.. meta::
   :description: Resize table chart text by adjusting the pixel size of header, body, and footer text in the Chart Builder's Customize tab.

You can resize :ref:`Table <data-table-example>` chart text. 
Resizing table text changes the pixel size of the following components:

- Header text, including icons
- Body text and the vertical height of rows
- Footer text

Procedure
---------

To resize table text:

.. procedure:: 
   :style: normal

   .. step:: Navigate to the :guilabel:`Customize` tab in the
      :doc:`Chart Builder </build-charts>`.
   
   .. step:: Click the :guilabel:`General` drop-down and toggle 
      :guilabel:`Text size` to :icon-fa5:`toggle-on` :guilabel:`On`.

   .. step:: Set the :guilabel:`Text Size` to the desired number of pixels. 
      You can adjust text size to be between 8 and 24 pixels. 
      The default is 14 pixels.

For a complete list of Chart Customization options, see :doc:`Customize
Charts </customize-charts/>`.
