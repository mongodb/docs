============
Restore Jobs
============

.. default-domain:: mongodb

.. include:: /includes/fact-groups-projects-synonyms.rst

Create and retrieve restores jobs for a MongoDB cluster. A restore 
job restores a MongoDB cluster to its state from an existing snapshot
or a specific point-in-time.

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - GET
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/restoreJobs </reference/api/restore-jobs-get-all>`
     - Retrieve all restore jobs for the specified cluster.

   * - GET
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/restoreJobs/{JOB-ID} </reference/api/restore-jobs-get-one>`
     - Retrieve one restore job for the specified cluster.

   * - POST
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/restoreJobs </reference/api/restore-jobs-create-one>`
     - Create one restore job for the specified cluster.
