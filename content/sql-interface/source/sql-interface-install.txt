=====================================
Install the MongoDB {+sql-interface+}
=====================================

.. meta::
   :keywords: connect, sql, mongodb
   :description: Installing the MongoDB SQL Interface is a three-step process that includes configuring your database to accept SQL connections, installing the correct driver, and connecting your BI tool.

.. default-domain:: mongodb

.. facet::
   :name: genre
   :values: tutorial

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

The MongoDB {+sql-interface+} lets you query your MongoDB data using standard SQL-based |bi| tools 
like Tableau, Power BI, and DBeaver.

To get started, use the following process:

.. procedure::

   .. step:: Setup the server and schema.

      Follow the instructions in :ref:`Server Setup <connect-with-sql-composable>` to configure your database to accept SQL connections. The setup process depends on your deployment type:

      - MongoDB {+service+}: Connect to your cluster and let {+service+} automatically 
        configure the {+fdi+} in the background.

      - {+ea+}: Configure your self-managed database and generate a SQL schema for 
        your collections using the {+schema-builder+} CLI. To learn more, see `the {+schema-builder+} README <https://translators-connectors-releases.s3.amazonaws.com/mongodb-schema-manager/docs/MongoDB_Schema_Manager-overview.pdf>`__.
   
   .. step:: Setup the driver.

      Follow the instructions in :ref:`Driver Setup <install-sql-interface-driver>` to install the correct driver (|jdbc| or |odbc|) on your client machine.

   .. step:: Connect your |bi| tool.

      Follow the instructions in :ref:`Connect Your BI Tool <sql-connect>` to configure your specific |bi| tool to connect to your MongoDB database.
