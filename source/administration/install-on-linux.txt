Install MongoDB Community Edition on Linux
==========================================

.. contents:: On this page

These documents provide instructions to install MongoDB Community Edition for
various Linux systems.

.. _recommended-linux-install:

Recommended
-----------

For the best installation experience, MongoDB provides packages for popular
Linux distributions. These packages are the preferred way to run
MongoDB. The following guides detail the installation process for these
systems:

.. include:: /includes/toc/dfn-list-spec-installation-linux-packages.rst

Other Systems
-------------

.. include:: /includes/admonition-wsl.rst

For Linux and BSD systems without supported packages, MongoDB works with
the community to ensure that MongoDB packages are available:

- `Alpine Linux <https://pkgs.alpinelinux.org/package/edge/community/x86_64/mongodb>`__
- `Arch Linux <https://www.archlinux.org/packages/community/x86_64/mongodb/>`__
- `Fedora <https://apps.fedoraproject.org/packages/mongodb>`__
- `FreeBSD <https://www.freshports.org/databases/mongodb36/>`__
- `Gentoo <https://packages.gentoo.org/packages/dev-db/mongodb>`__
- `openSUSE <https://software.opensuse.org/package/mongodb>`__
