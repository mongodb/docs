.. _mcli-reference-om:

=================================
Ops Manager ``mongocli`` commands
=================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

``agent``
---------

The ``agents`` command allows you to manage agents installed to
your |cloud-short| deployments.

- :ref:`Upgrade agents to their latest version <mcli-om-agents-upgrade-command>`

``alert``
---------

Lists |mms| :opsmgr:`alerts and alert configurations </tutorial/nav/alerts/>`.

- :ref:`List alert configurations <mcli-om-list-alert-config-command>`
- :ref:`Create one alert configuration <mcli-om-create-alerts-config-command>`
- :ref:`Update one alert configuration <mcli-om-update-alerts-config-command>`
- :ref:`List alert configuration matcher fields <mcli-om-alert-config-fields-type-command>`
- :ref:`List alerts <mcli-om-alert-list-command>`
- :ref:`Describe an alert <mcli-om-alert-describe-command>`
- :ref:`Acknowledge an alert <mcli-om-alert-ack-command>`

``automation``
--------------

Retrieves the latest plan for MongoDB processes in the specified |mms|
project.

- :ref:`Retrieve automation status <mcli-om-automation-status-command>`

``backup``
----------

Manages backup and restore jobs.

- :ref:`Start a backup restore job <mcli-om-backup-restore-start-cmd>`
- :ref:`List snapshots <mcli-om-backup-snapshots-list-cmd>`
- :ref:`List checkpoints <mcli-om-backup-checkpoints-list-cmd>`

``cluster``
-----------

Provides access to your cluster configurations.

- :ref:`List clusters <mcli-om-cluster-list-command>`
- :ref:`Describe a cluster <mcli-om-cluster-describe-command>`
- :ref:`Create a cluster <mcli-om-cluster-create-command>`
- :ref:`Modify a cluster <mcli-om-cluster-update-command>`
- :ref:`Create or modify a cluster <mcli-om-cluster-apply-command>`
- :ref:`Start mongod processes for a cluster <mcli-om-cluster-startup-command>`
- :ref:`Stop mongod processes for a cluster <mcli-om-cluster-shutdown-command>`
- :ref:`Create a rolling index <mcli-om-clusters-indexes-create-cmd>`

``dbuser``
----------

Manages MongoDB database users.

- :ref:`List database users <mcli-om-dbuser-list-command>`
- :ref:`Create a database user <mcli-om-dbuser-create-command>`
- :ref:`Delete a database user <mcli-om-dbuser-delete-command>`

``event``
---------

Lists events for a specified project or organization.

- :ref:`List events <mcli-om-events-list-cmd>`

``iam``
-------

Lists organizations for the specified |onprem|.

- :ref:`List organizations <mcli-iam-org-list>`

``log``
-------

Starts and retrieves log collection jobs for a project.

- :ref:`Start a log collection job <mcli-om-logs-jobs-collect-cmd>`
- :ref:`List log collection jobs <mcli-om-logs-jobs-list-cmd>`
- :ref:`Delete a log collection job <mcli-om-logs-jobs-delete-cmd>`

``measurement``
---------------

Lists measurements for MongoDB processes on a specified host.

- :ref:`List process measurements <mcli-om-measurements-process-cmd>`
- :ref:`Describe available measurements for a specified partition
  <mcli-om-measurements-disks-describe-cmd>`
- :ref:`List available partitions for a process
  <mcli-om-measurements-disks-list-cmd>`

``owner``
---------

Creates the first user with the :opsmgr:`Global Owner
</reference/user-roles/#global-owner-role>` role for an |mms|
installation.

- :ref:`Create owner <mcli-om-owner-create-command>`

``process``
-----------

Retrieves information about MongoDB processes running on a specified
|mms| project.

- :ref:`List processes <mcli-om-process-list-cmd>`
- :ref:`Describe process <mcli-om-process-describe-cmd>`

``security``
------------

Enables an :opsmgr:`authentication
mechanism </tutorial/enable-mongodbcr-authentication-for-group/>`
for the specified |mms| clusters in the given project.

- :ref:`Enable security for a cluster <mcli-om-security-enable-command>`

``server``
----------

Retrieves all hosts that are running an
Automation Agent in the specified |mms| project.

- :ref:`List servers for a project <mcli-om-servers-list-command>`
