=========================================
Delete One Whitelist Entry for an API Key
=========================================

   :backlinks: none

.. include:: /includes/api/base-api-uri.rst

Delete a specified whitelisted address from the specified
:ref:`API Key <mms-prog-api-key>`.

Resource
--------

.. code-block:: http

   DELETE /orgs/{ORG-ID}/apiKeys/{API-KEY-ID}/whitelist/{IP-ADDRESS}

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :widths: 15 10 75
   :header-rows: 1

   * - Name
     - Type
     - Description

   * - ``ORG-ID``
     - string
     - Unique identifier for the :ref:`organization <organizations>`
       to which the target |api| key belongs. Request the
       :doc:`/orgs </reference/api/organizations/organization-get-all>`
       endpoint to retrieve all organizations to which the
       authenticated user has access.

   * - ``API-KEY-ID``
     - string
     - Unique identifier for the |api| key for which you want to
       retrieve whitelist entries. Request the
       :doc:`/orgs/{ORG-ID}/apiKeys </reference/api/api-keys/org/get-all-org-api-key-whitelist>`
       endpoint to retrieve all |api| keys for the specified
       organization to which the authenticated user has access.

   * - ``IP-ADDRESS``
     - string
     - IP address or |cidr| block to delete. If the entry includes a
       subnet mask, use the URL-encoded value ``%2F`` for the forward
       slash ``/``.

Response
--------

This endpoint returns an empty JSON document.

Example Request
---------------

.. code-block:: sh

   curl --user "{PUBLIC-KEY}:{PRIVATE-KEY}" --digest \
     --header "Accept: application/json" \
     --header "Content-Type: application/json" \
     --request DELETE "https://cloud.mongodb.com/api/atlas/v1.0/orgs/{ORG-ID}/apiKeys/{API-KEY-ID}/whitelist/147.58.184.16"

Example Response
----------------

.. code-block:: json
   :copyable: false

   HTTP/1.1 204 No Content
