==================
atlas api clusters
==================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

`Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Returns, adds, edits, and removes database deployments.

Changes to cluster configurations can affect costs. This resource requires your Project ID.

Related Commands
----------------

* :ref:`atlas-api-clusters-autoScalingConfiguration` - `Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Returns the internal configuration of AutoScaling for sharded clusters.
* :ref:`atlas-api-clusters-createCluster` - `Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Creates one cluster in the specified project.
* :ref:`atlas-api-clusters-deleteCluster` - `Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Removes one cluster from the specified project.
* :ref:`atlas-api-clusters-getCluster` - `Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Returns the details for one cluster in the specified project.
* :ref:`atlas-api-clusters-getClusterAdvancedConfiguration` - `Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Returns the advanced configuration details for one cluster in the specified project.
* :ref:`atlas-api-clusters-getClusterStatus` - `Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Returns the status of all changes that you made to the specified cluster in the specified project.
* :ref:`atlas-api-clusters-getSampleDatasetLoadStatus` - `Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Checks the progress of loading the sample dataset into one cluster.
* :ref:`atlas-api-clusters-grantMongoDbEmployeeAccess` - `Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Grants MongoDB employee cluster access for the given duration and at the specified level for one cluster.
* :ref:`atlas-api-clusters-listCloudProviderRegions` - `Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Returns the list of regions available for the specified cloud provider at the specified tier.
* :ref:`atlas-api-clusters-listClusters` - `Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Returns the details for all clusters in the specific project to which you have access.
* :ref:`atlas-api-clusters-listClustersForAllProjects` - `Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Returns the details for all clusters in all projects to which you have access.
* :ref:`atlas-api-clusters-loadSampleDataset` - `Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Requests loading the MongoDB sample dataset into the specified cluster.
* :ref:`atlas-api-clusters-pinFeatureCompatibilityVersion` - `Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Pins the Feature Compatibility Version (FCV) to the current MongoDB version and sets the pin expiration date.
* :ref:`atlas-api-clusters-revokeMongoDbEmployeeAccess` - `Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Revokes a previously granted MongoDB employee cluster access.
* :ref:`atlas-api-clusters-testFailover` - `Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Starts a failover test for the specified cluster in the specified project.
* :ref:`atlas-api-clusters-unpinFeatureCompatibilityVersion` - `Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Unpins the current fixed Feature Compatibility Version (FCV).
* :ref:`atlas-api-clusters-updateCluster` - `Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Updates the details for one cluster in the specified project.
* :ref:`atlas-api-clusters-updateClusterAdvancedConfiguration` - `Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Updates the advanced configuration details for one cluster in the specified project.
* :ref:`atlas-api-clusters-upgradeSharedCluster` - `Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Upgrades a shared-tier cluster to a Flex or Dedicated (M10+) cluster in the specified project.
