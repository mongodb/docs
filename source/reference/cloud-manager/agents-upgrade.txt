.. _mcli-cm-agents-upgrade-command:

=====================================
mongocli cloud-manager agents upgrade
=====================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``agents upgrade`` command upgrades the {+mdbagent+}\s and BI 
Connector in the specified project to their latest version.

.. _cm-agents-upgrade-syntax:

Syntax 
------

.. code-block:: text 

   mongocli cloud-manager|cm agents upgrade
        [ --profile|-P <profile-name> ]
        [ --projectId <id-of-project> ]

.. _cm-agents-upgrade-options:

Options 
-------

.. list-table::
   :widths: 20 10 60 10

   * - Option 
     - Type
     - Description
     - Required?

   * - ``--profile``, ``-P``
     - string
     - Name of the profile where the public and private 
       keys for the project are saved. If omitted, uses the 
       {+default-profile+}. To learn more about creating a 
       profile, see :ref:`mcli-configure`.
     - no

   * - ``--projectId``
     - string
     - Unique identifier of the project that contains the 
       agents you want to update. If omitted, uses the project ID in 
       the profile or :ref:`environment variable <mcli-env-var>`.
     - no

.. _cm-agents-upgrade-output:

Output
------

The command prints the following fields in |json| format 
to the terminal if the command succeeds. If the command returns errors, 
see :ref:`Troubleshooting <troubleshooting>` for recommended solutions.

.. include:: /includes/agents-upgrade-output.rst

Example
-------

The following example uses the ``mongocli cm agents upgrade`` command 
to upgrade the {+mdbagent+}\s and the BI Connector to their latest 
version in the specified project. It uses the profile named 
``myprofile`` for accessing |cloud-short|.

.. code-block:: text

   mongocli cm agents upgrade --projectId 5eac961b0838261259e0302d \ 
   --profile myprofile

The previous command prints the following fields in |json| format to 
the terminal. To learn more about these fields, see 
:ref:`cm-agents-upgrade-output`.

.. code-block:: json
   :copyable: false

   {
	   "automationAgentVersion": "10.15.0.6376-1",
	   "biConnectorVersion": "2.13.4"
   }
