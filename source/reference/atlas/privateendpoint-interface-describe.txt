==================================================
mongocli atlas privateendpoint interface describe
==================================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

.. include:: /includes/styles/corrections.rst

The ``privateendpoint interface describe`` command retrieves information
about one interface endpoint for one private endpoint connection for a
given |service| project.

Syntax 
------

.. code-block:: text 

   mongocli atlas privateendpoint(s)|privateEndpoint(s) interface(s) describe <interface-endpoint-id>
        --privateEndpointId <private-endpoint-id>
        [ --output|-o <output-format> ]
        [ --profile|-P <profile-name> ]
        [ --projectId <project-ID> ]

.. include:: /includes/fact-command-line-help.rst

.. _atlas-privateendpoint-interface-describe-arguments:

Arguments  
---------

.. list-table::
   :header-rows: 1
   :widths: 20 10 60 10

   * - Argument 
     - Type 
     - Description 
     - Required?

   * - ``<interface-endpoint-id>``
     - string
     - Unique identifier of the interface endpoint that you
       want to retrieve.
     - yes

.. _atlas-privateendpoint-interface-describe-options:

Options 
-------

.. list-table::
   :widths: 20 10 60 10
   :header-rows: 1

   * - Option 
     - Type
     - Description
     - Required?

   * - ``--privateEndpointId``
     - string
     - Unique identifier of the private endpoint connection that
       contains the interface endpoint that you want to retrieve.

     - yes

   * - ``--output``, ``-o``
     - string 
     - .. include:: /includes/extracts/fact-basic-options-output.rst
     - no

   * - ``--profile``, ``-P``
     - string
     - Name of the profile where the public and private 
       keys for the project are saved. If omitted, uses the 
       {+default-profile+}. To learn more about creating a 
       profile, see :ref:`mcli-configure`.
     - no

   * - ``--projectId``
     - string
     - Unique identifier of the project for which you want to retrieve
       the interface endpoint. If omitted, uses the
       project ID in the profile or :ref:`environment variable
       <mcli-env-var>`.
     - no

.. _atlas-privateendpoint-interface-describe-output:

Output
------

If the command succeeds, it returns the following output in the default
format. If the command returns errors, see
:ref:`Troubleshooting <troubleshooting>` for recommended solutions.

.. code-block:: sh
   :copyable: false

   ID                       STATUS                           ERROR
   <interface-endpoint-id>   <interface-endpoint-id-status>   <interface-endpoint-id-error>

.. include:: /includes/fact-default-output.rst

- :atlas:`Atlas API </reference/api/private-endpoint-get-one-interface-endpoint/#response-elements>`

Example
-------

.. tabs::

   .. tab:: Default Output
      :tabid: default-output

      The following command retrieves one interface endpoint in one
      private endpoint connection for a project using the
      {+default-profile+}, which contains credentials and the project
      ID. The output is returned in the default format.

      .. code-block:: sh

         mongocli atlas privateendpoint interface describe vpce-05ad08ff76fceb65b \
         --privateEndpointId 5f59240294c39f761dd579c9

      The previous command prints the following to the terminal.

      .. code-block:: sh
         :copyable: false

         ID                       STATUS      ERROR
         vpce-05ad08ff76fceb65b   AVAILABLE

   .. tab:: JSON Output
      :tabid: json-output

      The following command retrieves one interface endpoint in one
      private endpoint connection for a project using the
      {+default-profile+}, which contains credentials and the project
      ID. The output is returned in |json|
      format.

      .. code-block:: sh

         mongocli atlas privateendpoint interface describe vpce-05ad08ff76fceb65b \
         --privateEndpointId 5f59240294c39f761dd579c9 --output json

      The previous command prints the following to the terminal in
      |json| format.

      .. code-block:: json
         :copyable: false

         {
           "interfaceEndpointId": "vpce-05ad08ff76fceb65b",
           "deleteRequested": false,
           "connectionStatus": "AVAILABLE"
         }
