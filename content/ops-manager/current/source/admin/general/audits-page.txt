===============
|onprem| Audits
===============

.. meta::
   :description: View all events tracked by Ops Manager, including group-level, internal, and system events, to review administrative actions and satisfy compliance requirements.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

This page in the |onprem| UI displays all events tracked by |onprem|. This
includes group events and the internal and system events that
aren't tracked at a group level. Use audit events to review
administrative actions, diagnose unexpected behavior, and
satisfy compliance requirements.
