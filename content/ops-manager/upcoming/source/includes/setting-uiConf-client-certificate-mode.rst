.. setting:: Client Certificate Mode

   *Type*: string

   *Default*: None

   
   Specifies if |onprem| requires clients to present valid |tls-ssl|
   client certificates when connecting to it. Accepted values are:
   
   
   - :guilabel:`None`
   - :guilabel:`Required for Agents Only`
   - :guilabel:`Required for All Requests`
   
   
   Corresponds to :setting:`mms.https.ClientCertificateMode`.
