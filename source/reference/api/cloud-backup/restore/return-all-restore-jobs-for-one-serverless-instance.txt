===================================================
Return All Restore Jobs For One Serverless Instance
===================================================

.. default-domain:: mongodb

.. meta::
   :canonical: https://www.mongodb.com/docs/atlas/reference/api-resources-spec/#tag/Cloud-Backups

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

Return all restore jobs of one serverless instance.

Required Roles
--------------

Your API Key must have one of the following roles to
successfully call this resource:

- :authrole:`Organization Owner`
- :authrole:`Project Owner`

.. include:: /includes/api/facts/api-access-list-required.rst

Request
-------

.. include:: /includes/api/base-api-uri.rst

.. code-block:: http

   GET /groups/{groupId}/serverless/{instanceName}/backup/restoreJobs

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :widths: 20 14 11 55
   :header-rows: 1
   :stub-columns: 1

   * - Name
     - Type
     - Necessity
     - Description

   * - groupId
     - string
     - Required
     - Unique 24-hexadecimal digit string that identifies your project.

   * - instanceName
     - string
     - Required
     - Human-readable label that identifies your serverless instance.

Response
--------

.. include:: /includes/get-all-note.rst

.. include:: /includes/api/list-tables/responses/serverless-restore.rst

Example Request
---------------

.. code-block:: sh

   curl --user '{PUBLIC-KEY}:{PRIVATE-KEY}' --digest \ 
        --header 'Content-Type: application/json' \ 
        --include \ 
        --request GET 'https://cloud.mongodb.com/api/atlas/v1.0/groups/{groupId}/serverless/{instanceName}/backup/restoreJobs?pretty=true'
