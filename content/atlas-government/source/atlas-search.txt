====================
MongoDB Search
====================

.. meta::
   :description: Explore advanced text indexing with MongoDB Search, available in Atlas for Government, to enhance search functionality without extra management.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

|fts| offers fine-grained text indexing to enable advanced search functionality without
any additional management. |fts| is available in |cloudgov| with the following
limitations.

See the |service| documentation to configure :ref:`atlas-search` indexes.

|cloudgov-short| Search Limitations
------------------------------------

The following limitations apply to |fts| in |cloudgov|:

**Search Nodes**
   |cloudgov-short| doesn't support dedicated Search Nodes. Search processes
   (``mongot``) run co-located with MongoDB processes on the same nodes. For
   more information about dedicated Search Nodes in commercial |service|, see
   :ref:`configure-search-nodes`.

**Index Management**
   |cloudgov-short| doesn't support driver-native commands for Search index
   management. You can only manage Search indexes through the {+atlas-admin-api+}
   or {+atlas-cli+}. To learn more, see :ref:`create-fts-index`.
