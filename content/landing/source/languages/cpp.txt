Drivers
-------

The C++ Driver is a high-level API for working with MongoDB databases consistent with other MongoDB language drivers.

For instructions on how to getting started, see the `C++ Driver Documentation →
<https://www.mongodb.com/docs/languages/cpp/cpp-driver/current/>`__
