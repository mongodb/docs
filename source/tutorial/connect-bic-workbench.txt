============================
Connect from MySQL Workbench
============================

.. default-domain:: mongodb

For M10+ clusters that have enabled the |bic|, the
:guilabel:`Connect` dialog provides the details to connect via the
|bic|.

.. include:: /images/bic-connection-dialog.rst

Prerequisites
-------------

- |service| cluster with `BI Connector for Atlas <https://docs.atlas.mongodb.com/bi-connection/>`_
  enabled

- `MySQL Workbench 6.3 <https://www.mysql.com/products/workbench/>`_ or later

Procedure
---------

The following tutorial outlines the steps to connect using MySQL
Workbench 6.3. |service| supports both the Community and Commercial
Editions.
