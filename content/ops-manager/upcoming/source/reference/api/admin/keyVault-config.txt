========================
Key Vault Configurations
========================

.. default-domain:: mongodb

.. meta::
   :description: Explore the Ops Manager Administration API endpoints for HashiCorp Vault key vault configurations, including get, create, update, and delete operations.

.. include:: /includes/api/base-admin-keyvault-api-uri.rst

A key vault configuration stores the connection and authentication
details that |onprem| uses to read credentials from HashiCorp Vault.
After you create a key vault configuration, you can configure an
|s3ss| to read its |aws| credentials from that key vault. To learn
more, see :ref:`om-vault-snapshot-store-credentials`.
