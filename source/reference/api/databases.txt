Endpoints
---------

The following endpoints are available for ``databases``.

.. list-table::
   :header-rows: 1
   :widths: 5 75 20

   * - Method
     - Endpoint
     - Description
   
   * - ``GET``
     - :doc:`/api/public/v1.0/groups/{GROUP-ID}/hosts/{HOST-ID}/databases </reference/api/databases-get-all-on-host>`
     - Retrieve all MongodDB processes running on the specified host.

   * - ``GET``
     - :doc:`/api/public/v1.0/groups/{GROUP-ID}/hosts/{HOST-ID}/databases/{DATABASE-NAME} </reference/api/database-get-by-name>`
     - Retrieve a single MongoDB process from the specified host by name.
