==================================================
mongocli cloud-manager measurements disks describe
==================================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

.. include:: /includes/styles/corrections.rst

The ``measurements disks describe`` command lists the available
measurements for a specified partition on a specified |cloud| host.
You can get a list of partition names for a host with the
:ref:`measurements disks list <mcli-cm-measurements-disks-describe-cmd>`
command. You can get the host IDs associated with a particular |cloud|
project with the :ref:`process list <mcli-cm-process-list-cmd>` command.

Syntax 
------

.. code-block:: text 

   mongocli cloud-manager|cm measurements disks describe <hostname>:<port> <partition-name>
        --granularity <granularity>
        --period <period> 
        --start <start>
        --end <end>
        [--type <type>]
        [ --profile|-P <profile-name> ]
        [ --projectId <project-ID> ]

Example
-------

The following example uses the ``mongocli cloud-manager measurements disks describe`` command 
to retrieve available process measurements for the ``data`` partition on a host with ID
``shard-00-02.qmljy.mongodb.net``, listening on port ``27017``. The
granularity is set to one minute, with a period of one minute. The
command uses a profile named ``egCmProfile`` where the project ID and the
|svc-api-key|\s are stored.

.. code-block:: json

   mongocli cloud-manager measurements disks describe \
     shard-00-02.qmljy.mongodb.net:27017 data \
     --granularity PT1M --period PT1M \
     --profile egAtlasProfile
