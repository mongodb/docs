Supported Aggregation Pipeline Expression
=========================================

{+atlas-sp+} supports aggregation pipeline expressions for processing continuous data streams.

The following table lists the aggregation pipeline expressions unique to
{+atlas-sp+} and those which exist in modified form in {+atlas-sp+}.

.. list-table::
   :header-rows: 1
   :widths: 40 60
   
   * - Aggregation Pipeline Expression
     - Purpose
   
   * - ``$convert``
     - Converts binary data in Kafka headers between binData and int, long, 
       double, and string types.
