================================
Get Agents by Type for a Project
================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

Get all agents of a specified type (i.e. Monitoring, Backup, or
Automation) for a group.

.. include:: /includes/fact-groups-projects-synonyms.rst

.. include:: /includes/api/base-api-uri.rst

Resource
--------

.. code-block:: http

   GET /groups/{GROUP-ID}/agents/{TYPE}

**{TYPE}** can be one of the following values:

- ``MONITORING``
- ``BACKUP``
- ``AUTOMATION``

Example Request
---------------

Get All Monitoring Agents for a Group.

.. only:: cloud

   .. code-block:: sh

      curl --user "{username}:{apiKey}" --digest \
       --header "Accept: application/json" \
       --include \
       --request GET "https://cloud.mongodb.com/api/public/v1.0/groups/{GROUP-ID}/agents/MONITORING"

.. only:: onprem

   .. code-block:: sh

      curl --user "{username}:{apiKey}" --digest \
       --header "Accept: application/json" \
       --include \
       --request GET "https://<ops-manager-host>/api/public/v1.0/groups/{GROUP-ID}/agents/MONITORING"
