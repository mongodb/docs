========================
atlas customDbRoles list
========================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

List custom database roles for your project.

Syntax
------

.. code-block::

   atlas customDbRoles list [options]

.. Code end marker, please don't delete this comment
