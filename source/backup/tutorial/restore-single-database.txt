=======================
Restore Single Database
=======================

.. default-domain:: mongodb

.. start-content

Overview
--------

A backup snapshot file contains a complete copy of the contents of your
:program:`mongod` :ref:`dbpath`. To restore a single collection or database or
partial data, retrieve a backup snapshot, expand the snapshot data on a volume,
and then use the :program:`mongodump` and :program:`mongorestore` commands to
build and restore your data.

Procedure
---------

Select and Download a Snapshot
++++++++++++++++++++++++++++++
  
To select and download snapshot, go to the :guilabel:`Backups` page and click
on the :guilabel:`Sharded Cluster Status` tab or :guilabel:`Replica Set Status`
tab. A list of your sharded clusters or replica sets and their backup details
displays.
