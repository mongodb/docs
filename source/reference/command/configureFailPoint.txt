==================
configureFailPoint
==================

.. default-domain:: mongodb

Definition
----------

.. dbcommand:: configureFailPoint

   Configures failure points. Issue the :dbcommand:`configureFailPoint`
   command against the :term:`admin database`. The command takes the
   following form:

   .. code-block:: javascript

      {configureFailPoint: "<failure_point>", mode: <behavior> }

   The command contains the following fields:

   .. include:: /reference/command/configureFailPoint-field.rst

Example
-------

   .. code-block:: javascript

      db.adminCommand( { configureFailPoint: "pingPongPoint", mode: {times: 21} } )

   .. |dbcommand| replace:: :dbcommand:`configureFailPoint`
   .. include:: /includes/note-enabletestcommands.rst
