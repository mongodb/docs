Syntax

.. code-block:: http

   DELETE /groups/{GROUP-ID}/integrations/{INTEGRATION-TYPE}

~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :widths: 15 10 75

   :stub-columns: 1

   * - Parameter
     - Necessity
     - Description

   * - ``{PROJECT-ID}``
     - Required
     - Project identifier.

   * - ``{INTEGRATION-TYPE}``
     - Required
     - Third-party service identifier. Accepted

        --include \
        --request DELETE "https://cloud.mongodb.com/api/public/v1.0/groups/{PROJECT-ID}/integrations/PAGER_DUTY"
