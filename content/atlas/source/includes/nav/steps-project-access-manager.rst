.. step:: In |service|, go to the :guilabel:`Project Access Manager` page.
      
   a. If it's not already displayed, select your desired organization
      from the |ui-org-menu| in the navigation bar.

   #. If it's not already displayed, select your desired project
      from the :guilabel:`Projects` menu in the navigation bar.

   #. Click :guilabel:`Project Identity & Access` in the sidebar under 
      the :guilabel:`Security` section.

   The `Project Access Manager <https://cloud.mongodb.com/go?l=https%3A%2F%2Fcloud.mongodb.com%2Fv2%2F%3Cproject%3E%23%2Faccess>`__ page displays.
