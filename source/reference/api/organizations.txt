=============
Organizations
=============

.. default-domain:: mongodb

The ``orgs`` resource provides access to create or delete |service|
:ref:`organizations <organizations>`.

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``

     - :doc:`/api/atlas/v1.0/orgs </reference/api/organization-get-all>`

     - Get all |service| organizations the authenticated user has access to.

   * - ``GET``

     - :doc:`/api/atlas/v1.0/orgs/{ORG-ID} </reference/api/organization-get-one>`

     - Get information about the |service| organization associated to ``{ORG-ID}``.

   * - ``GET``

     - :doc:`/api/atlas/v1.0/orgs/{ORG-ID}/users </reference/api/organization-users-get-all-users>`

     - Get information about the |service| users associated with ``{ORG-ID}``.

   * - ``POST``

     - :doc:`/api/atlas/v1.0/orgs  </reference/api/organization-create-one>`

     - Create a |service| organization.

   * - ``DELETE``

     - :doc:`/api/atlas/v1.0/orgs/{ORG-ID} </reference/api/organization-delete-one>`

     - Delete the |service| organization associated to ``{ORG-ID}``.

   * - ``GET``

     - :doc:`/api/atlas/v1.0/orgs/{ORG-ID}/invoices/ </reference/api/organization-get-all-invoices>`

     - Get all invoices for the |service| organization associated to ``{ORG-ID}``.

   * - ``GET``

     - :doc:`/api/atlas/v1.0/orgs/{ORG-ID}/invoices/{INVOICE-ID}/csv </reference/api/organization-get-all-invoices>`

     - Get the CSV-formatted invoice identified by ``{INVOICE-ID}`` for the |service| organization associated to ``{ORG-ID}``.
