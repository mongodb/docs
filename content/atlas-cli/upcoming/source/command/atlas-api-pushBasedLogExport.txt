============================
atlas api pushBasedLogExport
============================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

`Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: You can continually push logs from mongod, mongos, and audit logs to an AWS S3 bucket.

Atlas exports logs every 5 minutes.
