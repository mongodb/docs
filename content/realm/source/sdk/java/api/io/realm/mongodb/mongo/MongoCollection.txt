io.realm.mongodb.mongo
======================


 | `java.lang.Object <https://docs.oracle.com/javase/7/docs/api/java/lang/Object.html>`__ 
 | 		io.realm.mongodb.mongo.MongoCollection


The MongoCollection interface provides read and write access to documents.Use :ref:`MongoDatabase.getCollection <io_realm_mongodb_mongo_MongoDatabase_getCollection>`  to get a collection instance.



Before any access is possible, there must be an active, logged-in user.
