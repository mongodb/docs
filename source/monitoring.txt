==========
Monitoring
==========

.. default-domain:: mongodb

|monitoring| is a service for monitoring MongoDB deployments, and an
integral part of of the |mms|. |monitoring| collects statistics on
all key server and hardware indicators and presents this data through
an intuitive web interface.

This manual describes the installation of the Monitoring Agent and
operation of the |monitoring| web console. You can find answers to
common questions in the FAQs, but for all other inquiries please
feel free to `open a JIRA ticket
<https://jira.mongodb.org/secure/CreateIssue.jspa?pid=10582&issuetype=7>`_.
