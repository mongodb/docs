=========================================
Development Release Notes for MongoDB 2.7
=========================================

.. default-domain:: mongodb


This document reflects the current development of MongoDB 2.7, which
will become the next stable release of MongoDB 2.8. These notes are
for development purposes only and do not necessarily reflect the
features, interfaces, or behavior of MongoDB 2.8.

-------------

   ~~~~~~~~~~~~

.. Pluggable Storage Engine API
   ~~~~~~~~~~~~

.. Big Replica Sets

Security Improvements
------------

.. New Authentication via SCRAM-SHA1
   ~~~~~~~~~~~~~~~~~~
   
.. Localbost exception rule tightened 
   ~~~~~~~~~~~~~~~~~~~


Query Engine Improvements
------------

.. Explain() 2.0
   ~~~~~~~~~~~~~~~~~~
   
.. Covered queries now work in sharded clusters
   ~~~~~~~~~~~~~~~~~~~


General Improvements
------------

.. Initial Sync Parallelized
   ~~~~~~~~~~~~~~~~~~
   
.. Ability to Query for "BigPolygons"
   ~~~~~~~~~~~~~~~~~~~

.. Logging enhancements and new log format
   ~~~~~~~~~~~~~~~~~~~

.. Windows Performance Improvements
   ~~~~~~~~~~~~~~~~~~~

.. SecondaryThrottle parameter enhancements for chunk migrations
   ~~~~~~~~~~~~~~~~~~~

.. Parallel index builds with createIndexes cmd
   ~~~~~~~~~~~~~~~~~~~
