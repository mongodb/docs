# Developer Center Redirects

This folder contains redirects for legacy `mongodb.com/developer/*` paths formerly associated with MongoDB's Dev Center. It is linked to the "dummy" `legacy-devcenter-redirects.netlify.app` Netlify site and routed to via b2k.

**Important:** Populate metadata extension is turned off for this Netlify site via environment variable `POPULATE_METADATA_ENABLED` in the netlify.toml. Additionally, do not set any publish directory for this site.
