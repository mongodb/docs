.. _mcli-reference-atlas:

===========================
Atlas ``mongocli`` commands
===========================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

``alert``
----------

Manage |service| :atlas:`alerts and alert configurations </alerts>`.

- :ref:`List alert configurations <mcli-atlas-list-alert-config-command>`
- :ref:`Create one alert configuration <mcli-atlas-create-alerts-config-command>`
- :ref:`Update one alert configuration <mcli-atlas-update-alerts-config-command>`
- :ref:`Delete one alert configuration <mcli-atlas-alerts-configs-delete-cmd>`
- :ref:`List alert configuration matcher fields <mcli-atlas-alert-config-fields-type-command>`
- :ref:`List alerts <mcli-atlas-alert-list-command>`
- :ref:`Describe an alert <mcli-atlas-alert-describe-command>`
- :ref:`Acknowledge an alert <mcli-atlas-alert-ack-command>`

``backup``
----------

Manage backup and restore jobs.

- :ref:`Start a backup restore job <mcli-atlas-backup-restore-start-cmd>`
- :ref:`List snapshots <mcli-atlas-backup-snapshots-list-cmd>`
- :ref:`List checkpoints <mcli-atlas-backup-checkpoints-list-cmd>`

``cluster``
-----------

Manage your |service| cluster configurations.

- :ref:`List clusters <mcli-atlas-cluster-list-command>`
- :ref:`Describe a cluster <mcli-atlas-cluster-describe-command>`
- :ref:`Create a cluster <mcli-atlas-cluster-create-command>`
- :ref:`Modify a cluster <mcli-atlas-cluster-update-command>`
- :ref:`Delete a cluster <mcli-atlas-cluster-delete>`
- :ref:`Pause a cluster <mcli-atlas-cluster-pause-cmd>`
- :ref:`Resume a cluster <mcli-atlas-cluster-start-cmd>`
- :ref:`Create a rolling index <mcli-atlas-clusters-indexes-create-cmd>`

``dbuser``
----------

Manage MongoDB database users.

- :ref:`List database users <mcli-atlas-dbuser-list-command>`
- :ref:`Create a database user <mcli-atlas-dbuser-create-command>`
- :ref:`Modify a database user <mcli-atlas-dbuser-update-command>`
- :ref:`Delete a database user <mcli-atlas-dbuser-delete-command>`

``event``
----------

List events for a specified project or organization.

- :ref:`List events <mcli-atlas-events-list-cmd>`

``log``
--------

Retrieve various log types for a specified host.

- :ref:`Download logs <mcli-atlas-logs-download-cmd>`

``measurement``
----------------

List measurements for MongoDB processes on a specified host.

- :ref:`List process measurements <mcli-atlas-measurements-process-cmd>`
- :ref:`Describe available measurements for a specified partition
  <mcli-atlas-measurements-disks-describe-cmd>`
- :ref:`List available partitions for a process
  <mcli-atlas-measurements-disks-list-cmd>`
- :ref:`List available databases for a process <mcli-atlas-measurements-databases-list-cmd>`

``process``
-----------

Retrieve information about MongoDB processes running on a specified
|service| project.

- :ref:`List processes <mcli-atlas-process-list-cmd>`

``whitelist``
-------------

Manage :atlas:`whitelist </security-whitelist/>` entries.

- :ref:`List whitelist entries <mcli-atlas-whitelist-list-command>`
- :ref:`Describe a whitelist entry <mcli-atlas-whitelist-describe-command>`
- :ref:`Create a new whitelist entry <mcli-atlas-whitelist-create-command>`
- :ref:`Delete a whitelist entry <mcli-atlas-whitelist-delete>`
