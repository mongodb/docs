======================================
Install or Update the Automation Agent
======================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

If a Linux server has an existing MongoDB deployment that was installed
with a package manager, use the same package manager to install the
Automation Agent. If a Linux server has an existing MongoDB deployment
that was installed without a package manager, use the archive install.
