   .. step:: Edit the key.

      a. In the :guilabel:`Actions` column, click :icon-fa5:`pencil-alt`
         corresponding to the key that you want to edit.
      #. In the :guilabel:`Edit Model API Key` page, modify the name of
         the {+model-api-key+}.

         The name must be unique in the project and
         can be up to 250 characters in length.

      #. Click :guilabel:`Save` to apply the changes.
