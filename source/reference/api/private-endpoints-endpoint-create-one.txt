============================================
Create One Private Endpoint for One Provider
============================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

.. include:: /includes/fact-projects-groups-synonyms.rst

Create one of the following: 

- One private endpoint for |aws| or |azure| in an |service|
  project.
- One private endpoint group  for |gcp| in an |service| project. Endpoint groups
  represent a collection of forwarding rules. To learn more, see :doc:`Set up Private
  Service Connect using the API </google-cloud-private-service-connect-api>`.

If the attempt to add an endpoint or endpoint group fails, :doc:`delete it 
</reference/api/private-endpoints-endpoint-delete-one>`, then
try to add a new one.

Required Roles
--------------

You must have at the :authrole:`Project Owner` role for the project to
successfully call this resource.

Request
-------

.. include:: /includes/api-base-url.rst

.. code-block:: http

   POST /groups/{GROUP-ID}/privateEndpoint/{CLOUD-PROVIDER}/endpointService/{ENDPOINT-SERVICE-ID}/endpoint

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :stub-columns: 1
   :widths: 20 14 11 55

   * - Path Parameter
     - Type
     - Necessity
     - Descriptionon

   * - {GROUP-ID}
     - string
     - Required
     - Unique identifier for the :ref:`project <projects>`
       for which you want to create a private endpoint.

   * - {CLOUD-PROVIDER}
     - string
     - Required
     - Cloud provider for which you want to create a private endpoint.
       |service| accepts **AWS**, **AZURE**, or **GCP**.

   * - {ENDPOINT-SERVICE-ID}
     - string
     - Required
     - Unique identifier of the private endpoint service for which you
       want to create a private endpoint.

   .. tab::
      :tabid: azure
   
      .. list-table::
         :header-rows: 1
         :stub-columns: 1
         :widths: 20 14 11 55

         * - Body Parameter
           - Type
           - Necessity
           - Description

         * - id
           - string
           - Required
           - Unique identifier of the private endpoint you :ref:`created
             <create-private-endpoint-api-prerequisites>` in your
             |azure| VNet. 

             Run the `az network private-endpoint list
             <https://docs.microsoft.com/en-us/cli/azure/network/private-endpoint?view=azure-cli-latest#az_network_private_endpoint_list>`__
             |azure| CLI command to retrieve this value:

             .. code-block:: sh

                az network private-endpoint list

             The response looks similar to the following:

             .. literalinclude:: /includes/api/responses/az-network-private-endpoint-list.json
                :language: json
                :linenos:
                :copyable: false
                :emphasize-lines: 5

         * - privateEndpointIPAddress
           - string
           - Required
           - Private IP address of the private endpoint network
             interface you :ref:`created
             <create-private-endpoint-api-prerequisites>` in your
             |azure| VNet.

             To retrieve this value:

             1. Run the `az network private-endpoint list
                <https://docs.microsoft.com/en-us/cli/azure/network/private-endpoint?view=azure-cli-latest#az_network_private_endpoint_list>`__
                |azure| CLI command to retrieve the **networkInterface.id**
                for your private endpoint:

             .. code-block:: sh

                az network private-endpoint list

             The response looks similar to the following:

             .. literalinclude:: /includes/api/responses/az-network-private-endpoint-list.json
                :language: json
                :linenos:
                :copyable: false
                :emphasize-lines: 34

             #. Run the `az network nic show --id {networkInterface.id}
                <https://docs.microsoft.com/en-us/cli/azure/network/nic?view=azure-cli-latest#az_network_nic_show>`__
                |azure| CLI command to retrieve the
                **ipConfigurations.privateIPAddress** for the private
                endpoint network interface. The value of this field is
                your **privateEndpointIPAddress**:

             .. code-block:: sh

                az network nic show --id /subscriptions/00000000-0000-0000-0000-000000000000/resourceGroups/privatelink/providers/Microsoft.Network/networkInterfaces/privatelink.nic.00000000-0000-0000-0000-000000000000

             The response looks similar to the following:

             .. literalinclude:: /includes/api/responses/az-network-nic-show.json
                :language: json
                :linenos:
                :copyable: false
                :emphasize-lines: 25

   .. tab::
      :tabid: gcp
             
      .. list-table::
         :header-rows: 1
         :stub-columns: 1
         :widths: 20 14 11 55
          
         * - Body Parameter
           - Type
           - Necessity
           - Description
          
         * - endpointGroupName
           - string
           - Required
           - Unique identifier of the endpoint group. The endpoint group
             encompasses all of the endpoints that you :ref:`created
             <create-private-endpoint-api-prerequisites>` in |gcp|.

           - string
           - Required
           - Unique identifier of the |gcp| project in which you created your

         * - endpoints
           - object array

             endpoint group. This includes:

             .. list-table::
              :header-rows: 1
              :stub-columns: 1

              * - Body Parameter
                - Type
                - Description
                
              * - ipAddress
                - string
                - Private IP address of the endpoint you created in |gcp|.

              * - endpointName
                - string
                - Forwarding rule that corresponds to the endpoint you created in |gcp|.
