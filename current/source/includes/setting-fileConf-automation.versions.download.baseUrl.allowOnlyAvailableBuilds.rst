.. setting:: automation.versions.download.baseUrl.allowOnlyAvailableBuilds

   *Type*: boolean

   *Default*: True

   
   When set to ``true``, |mms| limits the MongoDB versions you can 
   specify to those available for your deployment. 
   
   This setting is applied only when
   :setting:`automation.versions.download.baseUrl` is set with a 
   custom value.
