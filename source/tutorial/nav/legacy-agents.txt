=============
Legacy Agents
=============

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

.. include:: /includes/legacy-agents/fact-legacy-agents-deprecated.rst

The legacy {+aagent+}, {+magent+}, and {+bagent+} have merged to become
the new :doc:`MongoDB Agent </tutorial/nav/mongodb-agent>`.

You can continue to use the legacy agents on existing clusters, but
MongoDB will not provide bug fixes or add features. New clusters must
use the new MongoDB Agent.
