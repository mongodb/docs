Required Roles 
--------------

.. include:: /includes/fact-fts-required-roles.rst

Resource
--------

.. include:: /includes/api/base-api-uri.rst

.. code-block:: http

   POST /groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/fts/indexes

.. note::

   You cannot create an |fts| index on any collection in the ``admin``,
   ``local``, or ``config`` databases in any cluster.

Request Parameters
------------------

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :stub-columns: 1
   :widths: 15 10 10 65

   * - Path Element
     - Type
     - Necessity
     - Description

   * - ``GROUP-ID``
     - string
     - Required
     - Unique identifier for the :ref:`project <projects>` that
       contains the specified cluster.

   * - ``CLUSTER-NAME``
     - string
     - Required
     - Name of the cluster that contains the collection on which you
       want to create an |fts| index.

Examples 
--------

.. tabs:: 

   .. tab:: Basic Example 
      :tabid: basic

      The following example contains an index definition for an index 
      named ``default``. This is the default definition with 
      :ref:`dynamic mappings <static-dynamic-mappings>` for all fields 
      in the ``fiscalYear2018.orders`` collection.

      Example Request
      ~~~~~~~~~~~~~~~

      .. code-block:: sh
         :linenos:
         :emphasize-lines: 5-12

         curl --user "{PUBLIC-KEY}:{PRIVATE-KEY}" --digest \
              --header "Content-Type: application/json" \
              --include \
              --request POST "https://cloud.mongodb.com/api/atlas/v1.0/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/fts/indexes?pretty=true" \
              --data '{
                  "collectionName": "orders",
                  "database": "fiscalYear2018",
                  "mappings": {
                    "dynamic": true
                  },
                  "name": "default"
                }'

   .. tab:: Synonyms Example 
      :tabid: synonyms

      The following example contains an index definition for an index 
      named ``default``. This index contains a synonym mapping 
      definition to index all the fields in the ``my_test.synonyms`` 
      collection.

      Example Request
      ~~~~~~~~~~~~~~~

      .. code-block:: sh
         :linenos:
         :emphasize-lines: 13-21

         curl --user "{PUBLIC-KEY}:{PRIVATE-KEY}" --digest \
              --header "Content-Type: application/json" \
              --include \
              --request POST "https://cloud.mongodb.com/api/atlas/v1.0/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/fts/indexes?pretty=true" \
              --data '{
                "collectionName": "synonyms",
                "database": "my_test",
                "name": "default",
                "mappings": {
                  "dynamic": true
                },
                "analyzer": "lucene.standard",
                "synonyms": [
                  {
                    "name": "mySynonyms",
                    "source": {
                      "collection": "synonyms"
                    },
                    "analyzer": "lucene.standard"
                  }
                ]
              }'
