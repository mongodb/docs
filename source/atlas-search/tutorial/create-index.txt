You can create an |fts| index using the :ref:`Atlas UI
<fts-tutorial-index-creation-ui>`, :ref:`Atlas Search API
<fts-tutorial-index-creation-api>`, :ref:`{+atlas-cli+}
<fts-tutorial-index-creation-atlas-cli>`, |compass|, or :ref:`a
supported MongoDB Driver in your preferred language
<fts-tutorial-index-creation-programatically>`.

This tutorial uses the ``sample_mflix.movies`` collection from the
:ref:`sample datasets <available-sample-datasets>`.
