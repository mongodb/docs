=======================
Reference Architectures
=======================

.. default-domain:: mongodb

.. meta::
   :description: Use Kubernetes Operator reference architectures as tested, opinionated starting points for deploying MongoDB in single or multi-cluster Kubernetes environments.

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol





Reference architectures provide tested, opinionated starting
points for deploying MongoDB on |k8s|. Use them as a practical
guide when building out your |k8s-op-short| infrastructure.
Each architecture includes all the configuration you need to
stand up a working deployment.
