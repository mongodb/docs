.. note::

   When you upgrade the {+fcv+} to 9.0, MongoDB moves collection
   metadata from the config server to the shards. While the {+fcv+}
   upgrade clones this metadata, :dbcommand:`movePrimary` blocks and
   fails with ``ConflictingOperationInProgress``.

   Retry :dbcommand:`movePrimary` after the {+fcv+} upgrade
   completes.
