=======================================
S3-Compatible Blockstore Configurations
=======================================

.. default-domain:: mongodb

.. meta::
   :description: Explore v1.0 Ops Manager Administration API endpoints for S3-compatible blockstore configurations, including get, create, update, and delete operations.
