=========================================
Enable Authentication for |a-mms| Project
=========================================

.. meta::
   :description: Enable authentication for an Ops Manager project using username/password, LDAP, Kerberos, OIDC, or X.509, and manage users and roles for your deployments.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Enabling authentication for an |mms| project requires all
clients and agents to authenticate with your MongoDB
deployments. |mms| supports many authentication mechanisms
including username/password, |ldap|, Kerberos, |oidc|, and X.509.
