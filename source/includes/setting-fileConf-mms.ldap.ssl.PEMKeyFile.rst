.. setting:: mms.ldap.ssl.PEMKeyFile

   *Type*: string

   
   A file containing a client certificate and private key. Use this
   setting when your |tls-ssl| |ldap| server requires client
   certificates.
   
   .. code-block:: ini
   
      mms.ldap.ssl.PEMKeyFile=/opt/keyFile.pem
   
   
   Corresponds to :setting:`LDAP SSL PEM Key File`.
