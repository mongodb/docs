Definition

.. phpmethod:: MongoDB\Model\IndexInfo::getVersion()

   Return the index version.

   .. code-block:: php

      function getVersion(): integer

Return Values
-------------

The index version.

Examples
--------

.. code-block:: php

   <?php

   $info = new IndexInfo([
       'v' => 1,
   ]);

   var_dump($info->getVersion());

The output would then resemble:

.. code-block:: none

   int(1)
