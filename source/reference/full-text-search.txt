================
Full Text Search
================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Users who require fine-grained text indexing for an |service|
collection can create |fts| indexes. |fts| offers options for
custom analyzers, fuzzy search, and multi-field indexing.
.. note: update with more germane features

.. note::

   |fts| is available only in |service|. Non-|service| MongoDB
   deployments can use :manual:`text indexing </core/index-text/>`.
