{+avs+} supports |ann| search on |service| {+clusters+} running MongoDB
v6.0.11, v7.0.2, or later and |enn| search on {+clusters+} running
MongoDB v6.0.16, v7.0.10, v7.3.2, or later. You can also use {+avs+}
with :ref:`self-managed <install-mdb-community-edition>` or
:atlascli:`local Atlas deployments </atlas-cli-deploy-local>` that you
create with the {+atlas-cli+}.
