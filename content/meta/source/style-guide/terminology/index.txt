======================
Terminology Guidelines
======================

.. default-domain:: mongodb

This section provides guidelines for using terms in MongoDB technical
content.

:doc:`/style-guide/terminology/general-term-guidelines/index`
   Provides some general guidelines for using consistent, short, and
   familiar words.

:doc:`/style-guide/terminology/alphabetical-terms`
   Provides usage guidelines for common words and technical terms that
   are used in MongoDB content.

:doc:`/style-guide/terminology/concise-terms`
   Lists typical wordy terms and phrases and provides shorter
   alternatives that you can use instead.
