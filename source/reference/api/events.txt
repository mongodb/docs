======
Events
======

.. default-domain:: mongodb

The ``events`` resource provides access to retrieve events for the
specified organization or project.

.. include:: /includes/fact-projects-groups-synonyms.rst

.. include:: /includes/api-base-url-new.rst

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :doc:`/orgs/{ORG-ID}/events </reference/api/events-orgs-get-all>`
     - Get all events for the organization associated with ``{ORG-ID}``.

   * - ``GET``
     - :doc:`/orgs/{ORG-ID}/events/{EVENT-ID} </reference/api/events-orgs-get-one>`
     - Get the event with ID ``{EVENT-ID}`` for the organization
       associated with ``{ORG-ID}``.

   * - ``GET``
     - :doc:`/groups/{GROUP-ID}/events </reference/api/events-projects-get-all>`
     - Get all events for the project associated with ``{GROUP-ID}``.

   * - ``GET``
     - :doc:`/groups/{GROUP-ID}/events/{EVENT-ID} </reference/api/events-projects-get-one>`
     - Get the event with ID ``{EVENT-ID}`` for the project associated
       with ``{GROUP-ID}``
