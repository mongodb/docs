==================
$not (aggregation)
==================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

.. expression:: $not

   Returns the boolean opposite value passed to it. When passed a
   ``true`` value, :expression:`$not` returns ``false``; when passed
   a ``false`` value, :expression:`$not` returns ``true``.
