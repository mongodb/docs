   .. step:: Select the database for the collection.

      The main panel and :guilabel:`Namespaces` on the left side list the
      collections in the database.
      
      .. figure:: /images/data-explorer-dbs.png
         :figwidth: 696px
         :alt: Screenshot highlighting databases in the Atlas UI.
      
   .. step:: Select the collection on the left-hand side or in the main panel.

      The main panel displays the :guilabel:`Find`, :guilabel:`Indexes`,
      and :guilabel:`Aggregation` views.
      
   .. step:: Select the :guilabel:`Indexes` view.
      
      The indexes table lists the indexes and associated index information
      for the collection. Index information includes the index definition,
      the size, and the usage frequency.
      
      .. figure:: /images/data-explorer-indexes-view.png
         :figwidth: 696px
         :alt: Screenshot highlighting index view in the Atlas UI.
