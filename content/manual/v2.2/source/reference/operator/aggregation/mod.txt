==================
$mod (aggregation)
==================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

.. expression:: $mod

   Takes an array that contains a pair of numbers and returns the
   *remainder* of the first number divided by the second number.

   .. seealso:: :query:`$mod`
