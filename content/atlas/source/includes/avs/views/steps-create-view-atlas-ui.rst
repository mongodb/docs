   .. step:: Create a ``moviesWithEmbeddings`` View.

      .. code-block:: sh

         db.createView(
           "moviesWithEmbeddings",
           "embedded_movies",
           [
             {
               "$match": {
                 "$expr": {
                   "$ne": [
                     {
                       "$type": "$plot_embedding_voyage_3_large"
                     },
                     "missing"
                   ]
                 }
               }
             }
           ]
         )
