.. _mcli-iam-organization-apikey-delete:

=======================================
mongocli iam organization apikey delete
=======================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``iam organization apikey delete`` command deletes the specified
organization |api| key. You must confirm the operation
when you run this command without the ``--force`` option. 

.. _mcli-iam-organization-apikey-delete-syntax:

Syntax 
------

.. code-block:: text 

   mongocli iam organization|organizations|org|orgs apiKey|apiKeys|apikeys delete|rm <api-key-id>
        [ --force ]
        [ --orgId <organization-id> ]
        [ --output|-o <output-format> ]
        [ --profile|-P <profile-name> ]

.. include:: /includes/fact-command-line-help.rst

.. _mcli-iam-organization-apikey-delete-arguments:

Arguments  
---------

.. list-table::
   :header-rows: 1
   :widths: 20 10 60 10

   * - Argument 
     - Type 
     - Description 
     - Required?

   * - ``<api-key-id>``
     - string
     - Unique identifier for this organization API key.
     - yes

.. _mcli-iam-organization-apikey-delete-options:

Options 
-------

.. list-table:: 
   :header-rows: 1
   :widths: 20 10 60 10

   * - Option
     - Type 
     - Description 
     - Required? 

   * - ``--force`` 
     - 
     - Flag that indicates that the |api| key can 
       be deleted without requiring confirmation.
     - no

   * - ``--orgId``
     - string
     - Unique identifier of the organization of the |api| key you want
       to delete. 
       
       If specified, this value overrides any organization ID
       stored in your :ref:`profile <mcli-profiles>` or
       :ref:`environment variable <mcli-env-var>`.

       If omitted, your profile must contain the
       organization ID.
     - no

   * - ``--output``, ``-o``
     - string 
     - .. include:: /includes/extracts/fact-basic-options-output.rst
     - no

   * - ``--profile``, ``-P`` 
     - string 
     - Name of the profile where your credentials are saved.

       If omitted, uses the {+default-profile+}. To learn more about 
       creating a profile, see :ref:`mcli-configure`.
     - no

.. _mcli-iam-organization-apikey-delete-output:

Output
------

If the command succeeds, it returns the following output in the default
format. If the command returns errors, see :ref:`Troubleshooting
<mcli-troubleshooting>` for recommended solutions. 

.. code-block:: sh
   :copyable: false

   ? Are you sure you want to delete: <api-key-id> Yes
   API Key '<api-key-id>' deleted

.. _mcli-iam-organization-apikey-delete-examples:

Examples
--------

.. tabs::

   .. tab:: Default Output
      :tabid: default-output

      The following command deletes an organization |api| key with
      confirmation. The command uses the {+default-profile+}, which
      contains credentials and the organization ID, and returns the
      output in the default format.

      .. code-block:: sh

         mongocli iam org apikey delete 5f3d8790c9022019903c1234

      The previous command prints the following to the terminal.

      .. code-block:: sh
         :copyable: false

         ? Are you sure you want to delete: 5f3d8790c9022019903c1234 Yes
         API Key '5f3d8790c9022019903c1234' deleted

   .. tab:: JSON Output
      :tabid: json-output

      The following command deletes an organization |api| key without
      confirmation. The command uses the ``myOM`` profile, which 
      contains credentials and specifies the |onprem| service, and 
      returns the output in |json| format.

      .. code-block:: sh

         mongocli iam org apikey delete 5f3d8790c9022019903c1234 --force -P myOM -o json
         
      The previous command updates the |api| key and 
      prints the following to the terminal in |json| format.

      .. code-block:: sh
         :copyable: false

         API Key '5f3d8790c9022019903c1234' deleted
