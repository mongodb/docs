Create One Restore Job for One Cluster

Resource
--------

.. code-block:: http

   POST /groups/{PROJECT-ID}/clusters/{CLUSTER-ID}/restoreJobs

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :widths: 20 20 60
   :header-rows: 1
   :stub-columns: 1

   * - Name
     - Type
     - Description

   * - GROUP-ID
     - string
     - Unique identifier of the project that owns the job.

   * - CLUSTER-ID
     - string
     - Unique identifier of the cluster that the job represents.

     - id: http
       name: HTTP Restore
       content: |

         Create a restore job that transfers a compressed snapshot

               :language: sh

     - id: auto-oplog
       name: Automated Oplog Restore
       content: |

         Create an automated restore from an Oplog timestamp. Valid for
         :term:`replica sets <replica set>` only.

            .. literalinclude:: /includes/api/requests/create-one-restore-job-auto-oplog-cloud.sh

               :language: sh

     - id: auto-pit
       name: Automated PIT Restore
