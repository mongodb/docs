Driver Compatibility
~~~~~~~~~~~~~~~~~~~~

To use {+csfle+} with the :driver:`Node.js </node/current>` driver,
install driver version 3.4.0 or later. To use the Key Rotation API,
install `mongodb-client-encryption
<https://www.npmjs.com/package/mongodb-client-encryption/>`__ version
2.2.0 - 2.x.
