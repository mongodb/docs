.. versionchanged:: 3.0

   Most MongoDB distributions include support for TLS/SSL. See
   :doc:`/tutorial/configure-ssl` and
   :doc:`/tutorial/configure-ssl-clients` for more information about
   TLS/SSL and MongoDB.

.. versionchanged:: 3.4

   If ``--sslCAFile`` is not specified when connecting to an TLS/SSL-enabled
   server, the system-wide CA certificate store will be used.
