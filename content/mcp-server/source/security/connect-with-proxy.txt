The MongoDB MCP server supports connections to MongoDB deployments and
Atlas clusters through proxies.

MCP Environment Variables
-------------------------

To establish a connection with proxy configurations,
use the following variables when :ref:`configuring environment variables
<mcp-server-configuration-environment-variables>`
for the MCP server:

.. include:: /includes/table-proxy-env-variables.rst

.. note::

   The MongoDB MCP Server supports the following proxy types:

   - Socks5 proxies
   - HTTP proxies
   - CONNECT proxies
   - :abbr:`PAC (proxy auto-config)` URLs that resolve to one of the previously
     listed proxies
