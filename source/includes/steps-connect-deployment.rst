.. step:: Check the connection status and connect the deployment.

   If the deployment connection status is :guilabel:`Connected`, the
   deployment is already connected to |charts-short|. The
   :guilabel:`Data Access` section displays all of the collections
   within the selected |service| deployment.
      
   If the deployment connection status is :guilabel:`Disconnected`:
      
   a. Click the connect icon :icon-fa5:`link` next to the deployment
      connection status.
   #. Click :guilabel:`Reconnect` on the modal that appears.
      
   The deployment connects and the :guilabel:`Data Access` section
   displays all of the databases for the selected |service|
   deployment.
