===================
MongoDB with Claude
===================

.. meta::
   :robots: noindex
   :description: Learn how to use MongoDB with Claude Code.

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

You can use MongoDB with Claude Code through the official MongoDB Claude Plugin.

The MongoDB Claude Plugin provides:

- The MongoDB MCP Server
- Pre-built agent skills for common MongoDB tasks

Install the MongoDB Claude Plugin
---------------------------------

The MongoDB Claude Plugin is available in the official Anthropic Plugin Marketplace,
or you can install manually from the `MongoDB GitHub repository <https://github.com/mongodb/agent-skills>`__.

Install the MongoDB Claude Plugin from Claude Code
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

To install the MongoDB Claude Plugin:

1. Start a session with Claude Code.
#. Use the ``/plugin`` command to open the plugins menu.
#. Search for "MongoDB" and select the official MongoDB plugin.
#. Follow the prompts to complete the installation.

After installing the plugin, you must configure the MCP Server before using
any agent skills or functionality that connects to MongoDB.

Install the MongoDB Claude Plugin from the MongoDB GitHub Repository
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

To install the MongoDB Claude Plugin from the MongoDB GitHub repository:

1. Clone the `MongoDB GitHub repository <https://github.com/mongodb/agent-skills>`__.
#. Copy the ``.claude-plugin/`` directory into your project root (as ``.claude-plugin/``).
#. Copy ``mcp.json`` to your project root.
#. Copy the ``skills/`` directory to your project root.
#. Set up and configure the MCP Server.
#. Start a Claude Code session.

Configure the MCP Server
------------------------

The MongoDB Claude Plugin bundles the MongoDB MCP Server. To configure the MCP
Server, see `MongoDB MCP Server Get Started <https://www.mongodb.com/docs/mcp-server/get-started/>`__.
