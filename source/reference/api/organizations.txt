=============
Organizations
=============

.. default-domain:: mongodb

The ``orgs`` resource provides access to create or delete |service|
:ref:`organizations <organizations>`.

.. include:: /includes/api/base-api-uri.rst

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 40 50

   * - Method
     - Endpoint
     - Description

   * - ``GET``

     - :doc:`/orgs </reference/api/organization-get-all>`

     - Get all |service| organizations the authenticated user has access to.

   * - ``GET``

     - :doc:`/orgs/{ORG-ID} </reference/api/organization-get-one>`

     - Get information about the |service| organization identified by ``{ORG-ID}``.

   * - ``GET``

     - :doc:`/orgs/{ORG-ID}/users </reference/api/organization-users-get-all-users>`

     - Get information about the |service| users associated with ``{ORG-ID}``.

   * - ``PATCH``

     - :doc:`/orgs/{ORG-ID} </reference/api/organization-rename>`

     - Rename an |service| organization.

   * - ``DELETE``

     - :doc:`/orgs/{ORG-ID} </reference/api/organization-delete-one>`

     - Delete the |service| organization identified by ``{ORG-ID}``.

   * - ``GET``

     - :doc:`/orgs/{ORG-ID}/groups </reference/api/organization-get-all-projects>`

     - Get all :ref:`projects <projects>` in the organization identified by ``{ORG-ID}``.

   * - ``GET``

     - :doc:`/orgs/{ORG-ID}/invites </reference/api/organization-get-invitations>`

     - Retrieves all unaccepted invitations to the organization identified by 
       ``{ORG-ID}``.

   * - ``DELETE``

     - :doc:`/orgs/{ORG-ID}/invites/{INVITATION-ID} </reference/api/organization-delete-invitation>`

     - Deletes one unaccepted invitation to an organization identified by 
       ``{INVITATION-ID}``.
