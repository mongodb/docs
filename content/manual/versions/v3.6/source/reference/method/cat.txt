Definition
----------

.. method:: cat(filename)

   Returns the contents of the specified file. The method returns with
   output relative to the current shell session and does not impact the
   server.


   .. list-table::
      :header-rows: 1
      :widths: 20 20 80
   
      * - Parameter
   
        - Type
   
        - Description
   
      * - ``filename``
   
        - string
   
        - Specify a path and file name on the local file system.
