The method takes the following field:

.. list-table::
   :header-rows: 1
   :widths: 20 10 10 60

   * - Field
     - Type
     - Necessity
     - Description

   * - ``namespace``
     - String
     - Required
     - Namespace in the format ``"<database>.<collection>"``. Method
       enables automatic chunk merges for the supplied namespace.
