=============================================
mongocli atlas security customerCerts disable
=============================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Disables customer-managed X.509 for a project.

Syntax
------

.. code-block::

   mongocli atlas security customerCerts disable [options]

Options
-------

.. list-table::
   :header-rows: 1
   :widths: 20 10 10 60

   * - Name
     - Type
     - Required
     - Description
   * - -h, --help
     - 
     - false
     - help for disable
   * - --projectId
     - string
     - false
     - Project ID to use. Overrides configuration file or environment variable settings.

-----------------

.. list-table::
   :header-rows: 1
   :widths: 20 10 10 60

   * - Name
     - Type
     - Required
     - Description
   * - -P, --profile
     - string
     - false
     - Profile to use from your configuration file.
