.. procedure::
   :style: normal

   .. include:: /includes/nav/steps-org-access-manager.rst

   .. step:: Click the :guilabel:`Applications` tab.

   .. step:: Click :guilabel:`API Keys`.
      
   .. step:: Click :icon:`trash-alt` to the right of the |api| Key that you want to delete.
      
   .. step:: Click :guilabel:`Delete` to confirm that you want to delete this |api| Key or :guilabel:`Cancel` to leave the key in the Organization.

      Removing an |api| Key from an Organization also removes that key from
      any projects to which the key was granted access.
