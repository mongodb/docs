====================================
Configure Available MongoDB Versions
====================================

.. default-domain:: mongodb

Overview
--------

You can manage the versions that are available to your Automation
Agents to download. MongoDB versions are downloaded by the Automation
Agents "lazily": Automation Agents do not download available versions
unless the version is in use by a MongoDB process on your server.
