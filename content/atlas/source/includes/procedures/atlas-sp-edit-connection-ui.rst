.. procedure::
   :style: normal

   .. include:: /includes/nav/steps-stream-processing.rst

   .. step:: Go to the :guilabel:`Connection Registry`.

      Locate the overview panel of the {+spw+} that 
      contains the connection registry that you want to modify 
      and click :guilabel:`Configure` in the upper right.

   .. step:: Select the :guilabel:`Connection Registry` tab.

   .. step:: Modify the connection.
      
      a. For the connection that you want to modify in the table, 
         click the pencil icon in the :guilabel:`Actions` column.
        
         You can edit a connection only if it is not currently in use 
         by a stream processor. If you want to edit a connection in 
         use, stop all stream processors that use the connection first.

      #. Modify the fields you want to modify.
      
      #. Click :guilabel:`Save changes`.
