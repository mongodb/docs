=================================
Get all Invoices for Organization
=================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

Retrieves all invoices for one organization.

.. include:: /includes/api/base-api-uri.rst

Resource
--------

.. code-block:: http

   GET /orgs/{ORG-ID}/invoices/

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :widths: 20 20 60
   :header-rows: 1

   * - Name
     - Type
     - Description

   * - ``ORG-ID``
     - ObjectId
     - Unique identifier of the organization.

Example Request
---------------

.. only:: cloud

   .. cssclass:: copyable-code
   .. code-block:: sh
      :emphasize-lines: 

      curl --user "{USERNAME}:{APIKEY}" --digest \
       --header "Accept: application/json" \
       --include \
       --request GET "https://cloud.mongodb.com/orgs/{ORG-ID}/invoices/?pretty=true"

.. only:: onprem

   .. cssclass:: copyable-code
   .. code-block:: sh
      :emphasize-lines: 

      curl --user "{USERNAME}:{APIKEY}" --digest \
       --header "Accept: application/json" \
       --include \
       --request GET "https://{OPSMANAGER-HOST}:{PORT}/orgs/{ORG-ID}/invoices/?pretty=true"
