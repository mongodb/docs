.. setting:: AWS Endpoint

   *Type*: string

   *Default*: ``https://email.us-east-1.amazonaws.com``

   
   Sets the `sending API endpoint
   <https://docs.aws.amazon.com/ses/latest/DeveloperGuide/regions.html#region-endpoints>`_
   for the |aws| |ses|.
   
   Corresponds to :setting:`aws.ses.endpoint`.
   
   .. note::
   
      If you are using |aws| |ses| in a region other than US East (N. Virginia),
      you must set this value to the appropriate endpoint for your region.
