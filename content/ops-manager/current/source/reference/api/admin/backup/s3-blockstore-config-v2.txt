==============================================
S3-Compatible Blockstore Configurations (v2.0)
==============================================

.. default-domain:: mongodb

.. meta::
   :description: Explore the v2.0 Ops Manager Administration API endpoints for S3-compatible blockstore configurations, which add HashiCorp Vault support via keyVault.

.. include:: /includes/fact-groups-projects-synonyms.rst

.. include:: /includes/api/base-admin-backup-api-v2-uri.rst

The v2.0 resource for |s3| blockstore configurations adds the
``keyVault`` object. With this object, an |s3| blockstore reads its
|aws| credentials from HashiCorp Vault instead of the |onprem|
application database. To learn more, see
:ref:`om-vault-snapshot-store-credentials`.

To learn more about the v1.0 resource, see
:ref:`S3-Compatible Blockstore Configurations <s3-blockstore-config>`.

configurations.
