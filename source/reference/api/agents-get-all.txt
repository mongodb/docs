========================================
Get Links to Agent Resources for a Group
========================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

Get links to Monitoring, Backup, and Automation agent resources for a
group.

.. only:: cloud

    .. include:: /includes/api-cm-base-url.rst

.. only:: onprem

    .. include:: /includes/api-ops-base-url.rst

Syntax
------

.. code-block:: none

   GET /api/public/v1.0/groups/{GROUP-ID}/agents

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. include:: /includes/api/no-path-parameters.rst

Request Query Parameters
~~~~~~~~~~~~~~~~~~~~~~~~

.. include:: /includes/api/default-query-parameters.rst

Request Body Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. include:: /includes/api/no-body-parameters.rst

Response Elements
-----------------

.. list-table::
   :header-rows: 1

   * - Relation
     - Description
   * - ``self``
     - Me
   * - ``http://mms.mongodb.com/group``
     - The group the agents belong to
   * - ``http://mms.mongodb.com/monitoringAgents``
     - Links to all monitoring agents.
   * - ``http://mms.mongodb.com/backupAgents``
     - Links to all backup agents.
   * - ``http://mms.mongodb.com/automationAgents``
     - Links to all automation agents.

Example
-------

Request
~~~~~~~

.. only:: cloud

   .. code-block:: none

      curl -X GET -u "username:apiKey" --digest "https://cloud.mongodb.com/api/public/v1.0/groups/5421e044e4b09da19248b17f/agents"

.. only:: onprem

   .. code-block:: none

      curl -X GET -u "username:apiKey" --digest "https://<ops-manager-host>/api/public/v1.0/groups/5421e044e4b09da19248b17f/agents"

Response
~~~~~~~~

.. code-block:: none

   {
      "links":[
         {"href":"https://cloud.mongodb.com/api/public/v1.0/groups/5421e044e4b09da19248b17f/agents","rel":"self"},
         {"href":"https://cloud.mongodb.com/api/public/v1.0/groups/5421e044e4b09da19248b17f","rel":"http://mms.mongodb.com/group"},
         {"href":"https://cloud.mongodb.com/api/public/v1.0/groups/5421e044e4b09da19248b17f/agents/MONITORING","rel":"http://mms.mongodb.com/monitoringAgents"},
         {"href":"https://cloud.mongodb.com/api/public/v1.0/groups/5421e044e4b09da19248b17f/agents/BACKUP","rel":"http://mms.mongodb.com/backupAgents"},
         {"href":"https://cloud.mongodb.com/api/public/v1.0/groups/5421e044e4b09da19248b17f/agents/AUTOMATION","rel":"http://mms.mongodb.com/automationAgents"}
      ]
   }
