   :style: normal

   .. step:: Locate the {+database-deployment+} to which to grant access from the list of {+database-deployments+}.

   .. step:: Select the ellipsis icon (:guilabel:`...`) to the right of the {+database-deployment+}.
      
   .. step:: Select :guilabel:`Revoke Temporary Infrastructure Access to MongoDB Support`.

      .. important::
      
         Ensure that all issues requiring MongoDB Support have been 
         addressed before revoking access.
      
   .. step:: Click :guilabel:`Revoke Access`.

      Click :guilabel:`Revoke Access` to revoke temporary 
      infrastructure access.
