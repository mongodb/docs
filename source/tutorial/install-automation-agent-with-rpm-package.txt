Overview
--------

.. include:: /includes/fact-installation-automation-needs-automation-agent.rst

Automation Agents can run only on 64-bit architectures.

Use this procedure to install the agent on RHEL, CentOS, SUSE, Amazon
Linux, and other systems that use ``rpm`` packages.

If you are installing to a server but are not able to install using the ``rpm`` package, 
then instead :doc:`install the agent using an archive
</tutorial/install-automation-agent-from-archive>` and ensure the agent
runs as the same user as the MongoDB process.
