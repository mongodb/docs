Connect from Microsoft Power BI Desktop
=======================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

This tutorial demonstrates how to connect to the |bi| using
`Microsoft Power BI Desktop 
<https://powerbi.microsoft.com/en-us/desktop/>`_.

.. important:: 

   The MongoDB Connector does not support Direct Query in Power BI.
   However, Atlas SQL does support Direct Query in Power BI. To
   connect to Atlas SQL from Power BI, see :ref:`sql-connect-powerbi`.
