===================================
Integrate with OpenTelemetry (OTel)
===================================

.. default-domain:: mongodb

.. meta::
   :keywords: atlas cli, atlas ui
   :description: Configure Atlas to send metric data to OTel endpoints
                 for enhanced monitoring and analysis.

.. facet::
   :name: genre
   :values: tutorial

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

:gold:`IMPORTANT:` This feature is currently available as a Public
Preview. Implementation details are subject to change.

You can configure {+service+} to send metric data about your project
to your OpenTelemetry `collector
<https://opentelemetry.io/docs/collector/>`__, providing a unified
interface to metrics platforms such as `Dynatrace
<https://docs.dynatrace.com/docs>`__ and `New Relic
<https://docs.newrelic.com/>`__.

Required Access
~~~~~~~~~~~~~~~

To integrate {+service+} with OTel metrics, you must have :authrole:`Project
Owner` access to the project. Users with :authrole:`Organization
Owner` access must add themselves to the project as a ``Project
Owner``.

Prerequisites
~~~~~~~~~~~~~

To integrate {+service+} with OTel metrics you must have one of the
following:

- An `OTLP Endpoint
  <https://opentelemetry.io/docs/languages/sdk-configuration/otlp-exporter/>`__,
  if you want to integrate with a custom OTel provider.
- A `Dynatrace API <https://docs.dynatrace.com/docs/dynatrace-api>`__
  token and `Environment ID
  <https://docs.dynatrace.com/docs/discover-dynatrace/get-started/monitoring-environment>`__
  to export to Dynatrace
- A `New Relic API
  <https://docs.newrelic.com/docs/apis/intro-apis/new-relic-api-keys/>`__
  key to export to New Relic.

.. _otel-behavior:

Behavior
~~~~~~~~

Upon creating, updating, or triggering a test of your integration,
{+atlas-sp+} issues a test metric to validate that the integration is
operational.

{+atlas-sp+} can simultaneously support up to:

- One New Relic OTel sink
- One Dynatrace OTel sink
- Two custom OTel sinks

The availability of each sink type is unaffected by the usage of any
other.
