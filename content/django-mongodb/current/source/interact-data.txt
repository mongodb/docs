In this section, you can learn how to use {+django-odm+} to interact with your
MongoDB data.

- :ref:`django-crud`: Learn how to perform create, read, update, and delete
  operations on your data.

- :ref:`django-specify-query`: Learn how to perform complex read operations on 
  your data.

- :ref:`django-raw-queries`: Learn how to use MongoDB's aggregation pipeline syntax
  or the PyMongo driver to query your data.

- :ref:`django-atlas-search`: Learn how to use MongoDB Search to perform advanced text search
  queries on your data.

- :ref:`django-atlas-vector-search`: Learn how to run MongoDB Vector Search queries
  to perform semantic searches on your data.
  
- :ref:`django-transactions`: Learn how to use {+django-odm+}'s transaction API
  to run data operations within a transaction.
