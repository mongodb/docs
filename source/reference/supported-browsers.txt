==================
Supported Browsers
==================

.. meta::
   :description: Ensure optimal use of Ops Manager by setting a display resolution of at least 1280 ✕ 960 pixels and using the latest stable versions of supported browsers.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

To use |mms|:

- Change your display resolution to a minimum of 1280 ✕ 960 pixels
  (:abbr:`SXGA- (Super eXtended Graphics Array)`). |mms| might not
  display correctly if your browser resolution is less than 1000
  pixels.

- Use one of the following supported browsers with Javascript enabled:
