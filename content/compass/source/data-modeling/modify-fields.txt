About this Task 
---------------

In the :guilabel:`Data Modeling` tab, you can:

- Add or remove fields
- Change field data types
- Rename fields

.. include:: /includes/shared/fact-diagram-modification-data.rst

Before You Begin
----------------

You must have an existing diagram in |compass-short| to modify the 
fields. To create a new diagram, see :ref:`compass-create-data-model-diagram`.

Steps
-----

Add a New Field 
~~~~~~~~~~~~~~~

To add a new field to a collection in your |erd|:

.. procedure:: 
   :style: normal 

   .. include:: /includes/shared/steps-data-modeling-open.rst

   .. step:: Add a new field 

      To add a new field, click the :icon-lg:`Plus` icon in the upper-right 
      corner of one of your collection nodes. A new field appears in the 
      collection node.

      Once you add a field, you can edit the field's properties, including: 

      - :ref:`Renaming the field <compass-data-model-rename-field>`.
      - :ref:`Changing the field's data type <compass-data-model-data-type>`.
      - :ref:`Adding relationships <compass-data-modeling-relationships>`.

.. _compass-data-model-rename-field:

Rename a Field 
~~~~~~~~~~~~~~

To rename a field in your |erd|:

.. procedure:: 
   :style: normal

   .. include:: /includes/shared/steps-data-modeling-open.rst
    
   .. include:: /includes/shared/steps-field-side-panel.rst

    .. step:: Rename the field 

       Under the :guilabel:`Name` section, specify a new name for your field. 

.. _compass-data-model-data-type:

Change a Field's Data Type
~~~~~~~~~~~~~~~~~~~~~~~~~~

To change the data type of a field in your |erd|:

.. procedure:: 
   :style: normal 

   .. include:: /includes/shared/steps-data-modeling-open.rst
    
   .. include:: /includes/shared/steps-field-side-panel.rst

   .. step:: Select a new data type

      In the :guilabel:`Datatype` drop-down menu, select the datatypes that you 
      want to assign to the field. Because data in MongoDB has a flexible schema 
      model, you can specify multiple data types for a field. If a field 
      contains multiple data types, |compass-short| labels the field as 
      :guilabel:`(mixed)` in the diagram.

Remove a Field
~~~~~~~~~~~~~~

To delete a field from your diagram:

.. procedure:: 
   :style: normal 

   .. include:: /includes/shared/steps-data-modeling-open.rst
    
   .. include:: /includes/shared/steps-field-side-panel.rst

   .. step:: Delete the field

      In the side panel, click the :icon-lg:`Trash` icon.
