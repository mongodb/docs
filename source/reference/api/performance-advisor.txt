===================
Performance Advisor
===================

.. default-domain:: mongodb

.. include:: /includes/fact-groups-projects-synonyms.rst

.. include:: /includes/api/base-api-uri.rst

The following resources return information from the |mms|
:doc:`Performance Advisor </tutorial/performance-advisor>`,
including existing and suggested indexes, slow query logs,
and specific database namespaces which have experienced
performance issues.

.. _performance-advisor-permissions:

.. include:: /includes/fact-performance-advisor-permissions.rst

Endpoints
---------

.. list-table::
   :header-rows: 1
   :stub-columns: 1
   :widths: 10 50 40

   * - Method
     - Endpoint
     - Description

   * - GET

     - :doc:`/groups/{GROUP-ID}/hosts/{HOST-ID}/performanceAdvisor/slowQueryLogs
       </reference/api/performance-advisor/get-slow-queries>`

     - Get slow queries as determined by the
       :doc:`Performance Advisor </tutorial/performance-advisor>`.

   * - GET

     - :doc:`/groups/{GROUP-ID}/hosts/{HOST-ID}/performanceAdvisor/suggestedIndexes
       </reference/api/performance-advisor/get-suggested-indexes>`

     - Get suggested indexes as determined by the
       :doc:`Performance Advisor </tutorial/performance-advisor>`.
