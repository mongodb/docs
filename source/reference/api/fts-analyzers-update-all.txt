===============================================
Update All User-Defined Analyzers for a Cluster
===============================================

This API request creates new user-defined |fts| :ref:`analyzers
<analyzers-ref>` and replaces existing ones. If you have any
user-defined analyzers for your |fts| :ref:`index <fts-top-ref>`,
they will be replaced by the new analyzer or analyzers specified
in this request.

Resource
--------

.. code-block:: http

   PUT /groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/fts/analyzers

To delete all existing user-defined analyzers from your cluster,
send a ``PUT`` request to this endpoint with an empty array as
the payload.

.. warning::

   Issuing a ``PUT`` request to this API endpoint overwrites any
   existing user-definied |fts| :ref:`analyzers <analyzers-ref>`.

Request Parameters
------------------

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :stub-columns: 1
   :widths: 15 10 10 65

   * - Path Element
     - Type
     - Necessity
     - Description

   * - ``GROUP-ID``
     - string
     - Required
     - Unique identifier for the :ref:`project <projects>`
       that contains the specified cluster.

   * - ``CLUSTER-NAME``
     - string
     - Required
     - Name of the cluster containing the collection with one
       or more |fts| indexes.

.. include:: /includes/api/list-tables/responses/analyzer-definitions.rst

Example Request
---------------

.. code-block:: sh
   :linenos:
   :emphasize-lines: 6-21

   curl --user "{PUBLIC-KEY}:{PRIVATE-KEY}" \
         --digest  \
         --header "Accept: application/json"  \
         --header "Content-Type: application/json"  \
         --request PUT "https://cloud.mongodb.com/api/atlas/v1.0/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/fts/analyzers" \
         --data '[
            {
              "name": "my_new_analyzer",
              "baseAnalyzer": "lucene.standard",
              "maxTokenLength": 48
            },
            {
              "name": "my_other_new_analyzer",
              "baseAnalyzer": "lucene.english",
              "stopwords": [
                "foo",
                "bar",
                "baz"
              ]
            }
          ]'
