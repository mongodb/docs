db.hostInfo()
=============

.. method:: db.hostInfo()

   Returns a document with information about the underlying system
   that the :binary:`~bin.mongod` or :binary:`~bin.mongos` runs on. Some of the
   returned fields are only included on some platforms.

   :method:`db.hostInfo()` provides a helper in the :binary:`~bin.mongo`
   shell around the :dbcommand:`hostInfo` The output of
   :method:`db.hostInfo()` on a Linux system will resemble the
   following:

   .. code-block:: javascript

      {
         "system" : {
                "currentTime" : ISODate("<timestamp>"),
                "hostname" : "<hostname>",
                "cpuAddrSize" : <number>,
                "memSizeMB" : <number>,
                "numCores" : <number>,
                "cpuArch" : "<identifier>",
                "numaEnabled" : <boolean>
         },
         "os" : {
                "type" : "<string>",
                "name" : "<string>",
                "version" : "<string>"
         },
         "extra" : {
                "versionString" : "<string>",
                "libcVersion" : "<string>",
                "kernelVersion" : "<string>",
                "cpuFrequencyMHz" : "<string>",
                "cpuFeatures" : "<string>",
                "pageSize" : <number>,
                "numPages" : <number>,
                "maxOpenFiles" : <number>
         },
         "ok" : <return>
      }

   See :data:`hostInfo` for full documentation of the output of
   :method:`db.hostInfo()`.
