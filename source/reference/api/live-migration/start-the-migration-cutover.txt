===========================
Start the Migration Cutover
===========================

.. default-domain:: mongodb

.. meta::
   :canonical: https://www.mongodb.com/docs/atlas/reference/api-resources-spec/#tag/Cloud-Migration-Service

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

Start the push live migration cutover stage and confirm when the cutover
stage completes. When the cutover stage completes, the push live migration
process stops synchronizing with the source cluster and the live
migration job completes migration.

Required Roles
--------------

Your API Key must have the :authrole:`Organization Owner` role to
successfully call this resource.

Request
--------

.. include:: /includes/api/base-api-uri.rst

.. code-block:: http

   PUT /groups/{groupId}/liveMigrations/{liveMigrationId}/cutover

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :widths: 20 14 11 55
   :header-rows: 1
   :stub-columns: 1

   * - Name
     - Type
     - Necessity
     - Description

   * - groupId
     - string
     - Required
     - Unique 24-hexadecimal digit string that identifies your project.

   * - liveMigrationId
     - string
     - Required
     - Unique 24-hexadecimal digit string that identifies the live
       migration job.
