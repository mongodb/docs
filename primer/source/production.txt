=====================
MongoDB in Production
=====================

.. default-domain:: mongodb

Eventually, after you're comfortable with MongoDB and have developed
applications that store data in MongoDB, you will need to deploy
MongoDB to a production environment to support robust production-grade
applications.

Applications have different set of tolerances and expectations
for its backing MongoDB deployment. However, MongoDB provides features
to automate common operational patterns, and tools to support robust
production deployments.

This section introduces some of the key features of MongoDB that will
support your production deployments.
