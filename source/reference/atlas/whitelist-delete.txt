.. _mcli-atlas-whitelist-delete:

===============================
mongocli atlas whitelist delete
===============================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``whitelist delete`` command deletes the specified 
whitelist entry in the specified |service| project. The command prompts 
you to confirm the operation when you run the command 
without the ``--force`` option. 

You can also delete a whitelist entry by using the |service| 
:atlas:`UI </security-whitelist/#delete-whitelist-entries>` 
or :atlas:`API </reference/api/whitelist-delete-one/>`.

.. _mcli-atlas-whitelist-delete-command-syntax:

Syntax
------

.. code-block:: text
   :copyable: false

   mongocli atlas whitelist delete|rm <whitelist-entry>
        [ --force ]
        [ --profile|-P <profile-name> ]
        [ --projectId <project-ID> ]

.. _mcli-atlas-whitelist-delete-command-options:

Options 
-------

.. list-table::
   :header-rows: 1
   :widths: 20 10 60 10

   * - Option 
     - Type 
     - Description 
     - Required?

   * - ``<whitelist-entry>``
     - string
     - IP address, |aws| security group ID, or |cidr| block of the 
       whitelist entry to delete.
     - yes

   * - ``--force`` 
     - 
     - Flag that indicates that the whitelist entry can 
       be deleted without requiring confirmation.
     - no

   * - ``--profile``, ``-P``
     - string
     - Name of the profile to use for accessing the 
       |service| project. If omitted, uses the ``default`` 
       profile.
     - no

   * - ``--projectId``
     - string
     - Unique identifier of the project containing the 
       whitelist entry to delete.
     - no

.. _mcli-atlas-whitelist-delete-command-output:

Output 
------

The command prints a confirmation message similar to the 
following to the terminal if it succeeds. If the command 
prints an error, see :ref:`Troubleshooting <troubleshooting>` 
for recommended solutions.

.. code-block:: text
   :copyable: false

   Project whitelist entry '<whitelist-entry>' deleted

.. _mcli-atlas-whitelist-delete-command-examples:

Examples 
--------

The following examples use the ``mongocli whitelist cluster delete`` 
command to delete the sample IP address ``192.0.2.0``.

Example 1
~~~~~~~~~

The following ``mongocli atlas whitelist delete`` command deletes 
the IP address ``192.0.2.0`` after prompting for a confirmation. 
It uses the profile named ``egAtlasProfile`` for accessing the 
|service| project. 

.. code-block:: text
   :copyable: false

   mongocli atlas whitelist delete 192.0.2.0 --projectId
   5e2211c17a3e5a48f5497de3 --profile egAtlasProfile
   ? Are you sure you want to delete: 192.0.2.0 Yes

The command prints the following to the terminal:

.. code-block:: text
   :copyable: false

   Project whitelist entry '192.0.2.0' deleted

Example 2
~~~~~~~~~

The following ``mongocli atlas whitelist delete`` command uses the
``--force`` option to delete the IP address ``192.0.2.0`` without
confirmation. The command uses the ``default`` profile for accessing the
|service| project.

.. code-block:: text
   :copyable: false

   mongocli atlas whitelist delete 192.0.2.0 --projectId 5e2211c17a3e5a48f5497de3 --force

The command prints the following to the terminal:

.. code-block:: text
   :copyable: false

   Project whitelist entry '192.0.2.0' deleted
