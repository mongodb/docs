.. _mcli-atlas-private-endpoint-provider-describe-cmd:

==================================================
mongocli atlas privateendpoint <provider> describe
==================================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``privateendpoint <provider> describe`` command retrieves 
information about one private endpoint connection for |aws| or Azure in 
an |service| project. You can also retrieve this information through 
the |service| :atlas:`UI </security-private-endpoint/>` or :atlas:`API 
</reference/api/private-endpoints-service-get-one/>`.

Syntax 
------

.. code-block:: text 

   mongocli atlas privateendpoint(s)|privateEndpoint(s) aws|azure describe|get <private-endpoint-id>
        [ --output|-o <output-format> ]
        [ --profile|-P <profile-name> ]
        [ --projectId <project-ID> ]

.. include:: /includes/fact-command-line-help.rst

.. _atlas-private-endpoint-provider-describe-arguments:

Arguments  
---------

.. list-table::
   :header-rows: 1
   :widths: 20 10 60 10

   * - Argument 
     - Type 
     - Description 
     - Required?

   * - ``<private-endpoint-id>``
     - string
     - Unique identifier of the private endpoint connection that you
       want to retrieve.
     - yes

.. _atlas-privateendpoint-provider-describe-options:

Options 
-------

.. list-table::
   :widths: 20 10 60 10
   :header-rows: 1

   * - Option 
     - Type
     - Description
     - Required?

   * - ``--output``, ``-o``
     - string 
     - .. include:: /includes/extracts/fact-basic-options-output.rst
     - no

   * - ``--profile``, ``-P``
     - string
     - Name of the profile where the public and private 
       keys for the project are saved. If omitted, uses the 
       {+default-profile+}. To learn more about creating a 
       profile, see :ref:`mcli-configure`.
     - no

   * - ``--projectId``
     - string
     - .. include:: /includes/extracts/fact-basic-options-project-id.rst
     - no

.. _atlas-private-endpoint-provider-describe-output:

Output
------

If the command succeeds, it returns the following output in the default
format. If the command returns errors, see
:ref:`Troubleshooting <troubleshooting>` for recommended solutions.

.. code-block:: sh
   :copyable: false

   ID                      ENDPOINT SERVICE             STATUS                       ERROR
   <private-endpoint-id>   <private-endpoint-service>   <private-endpoint-status>    <private-endpoint-error>

.. include:: /includes/fact-default-output.rst

- :atlas:`Atlas API 
  </reference/api/private-endpoints-service-get-one/#response-elements>`

Example
-------

.. tabs::

   .. tab:: Default Output
      :tabid: default-output

      .. tabs::

         .. tab:: |aws|
            :tabid: aws

            The following command retrieves one private endpoint 
            connection for |aws| in a project using the 
            {+default-profile+}, which contains credentials and the 
            project ID. The output is returned in the default format.

            .. code-block:: sh

               mongocli atlas privateendpoint aws describe 5f4fc81c1f03a835c2728ff7

            The previous command prints the following to the terminal.

            .. code-block:: sh
               :copyable: false

               ID                         ENDPOINT SERVICE                                          STATUS             ERROR
