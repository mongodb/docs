======================================
Tutorial: Create and Query a FTS Index
======================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

This tutorial takes you through the steps of setting up and querying
an |service| |fts| index. You will use a collection with movie data
from the |service| :doc:`sample data set </sample-data>`.

Prerequisites
-------------

To complete this tutorial you will need:

- An |service| cluster tier ``M30`` or larger.
- MongoDB version 4.2 or higher.
- The :binary:`mongo <bin.mongo>` shell on your local machine.

.. include:: /includes/steps/fts-tutorial.rst

Summary
-------

In this tutorial you loaded a sample dataset into your |service|
cluster, created a |fts| index, and ran some example queries against
it. More examples can be found througout the |fts| :ref:`documentation
<fts-top-ref>`.
