.. _create-new-cluster:

====================
Create a {+Cluster+}
====================

.. default-domain:: mongodb

In this tutorial, you'll learn how to create a new |service| {+cluster+}.
To learn how to modify an existing |service| {+cluster+}, see
:ref:`scale-cluster`.

:ref:`Clusters <ref-deployment-types>` can be either a :term:`replica set` or a
:term:`sharded {+cluster+}`. This tutorial walks you through creating a replica set.
