=================================
Connect to MongoDBMulti Resources
=================================

.. default-domain:: mongodb

.. meta::
   :keywords: multicluster
   :keywords: multi-cluster

The following section describes how to connect to a 
``MongoDBMulti`` resource that is deployed to |k8s|:

- :ref:`multi-cluster-connect-from-outside-k8s`
  Connect to a ``MongoDBMulti`` resource from outside of the |k8s| clusters.

.. note::

   To connect to a ``MongoDBMulti`` resource from within the |k8s| clusters,
   see :ref:`Connect to a MongoDB Database Resource from Inside Kubernetes <connect-from-inside-k8s>`,
   and select the tab :guilabel:`Using the Kubernetes Secret`. This procedure
   is the same as for single clusters deployed with the |k8s-op-short|.
