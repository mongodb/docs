================
Full Text Search
================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

|service-fullname|'s |fts| feature offers fine-grained text indexing to
enable advanced search functionality without any additional management
required. |fts| provides options for custom :ref:`analyzers
<analyzers-ref>`, fuzzy search, and multi-field indexing.

.. TODO: update with more germane features

.. note::

   |fts| is available only in |service|. Non-|service| MongoDB
   deployments can use :manual:`text indexing </core/index-text/>`.
