===========================================
AuthoritativeShardCatalogCollectionsPresent
===========================================

.. meta::
   :description: Identify authoritative shard catalog collections that remain in the config.shard.catalog namespace after a cluster downgrades below feature compatibility version 9.0.

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1

Description
-----------

.. data:: AuthoritativeShardCatalogCollectionsPresent

   .. include:: /includes/inconsistency-type/AuthoritativeShardCatalogCollectionsPresent

Format
------

.. code-block:: json

   {
      type: "AuthoritativeShardCatalogCollectionsPresent",
      description: "<string>",
      details: {
         namespace: "<string>",
         shard: "<string>"
      }
   }

Example
-------

.. include:: /includes/inconsistency-type/example

.. code-block:: json
   :emphasize-lines: 6-12
   :copyable: false

   {
      cursor: {
         id: Long("0"),
         ns: "test.$cmd.aggregate",
         firstBatch: [
            {
               type: "AuthoritativeShardCatalogCollectionsPresent",
               description: "Found authoritative shard catalog collections on a shard that is fully downgraded",
               details: {
                  namespace: "config.shard.catalog.chunks",
                  shard: "shard-rs0"
               }
            }
         ],
      },
      ok: 1
   }
