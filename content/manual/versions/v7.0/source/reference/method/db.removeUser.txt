Definition
----------

.. method:: db.removeUser(username)

   Removes the specified username from the database.

   The :method:`db.removeUser()` method has the following parameter:


   .. list-table::
      :header-rows: 1
      :widths: 20 20 80
   
      * - Parameter
   
        - Type
   
        - Description
   
      * - ``username``
   
        - string
   
        - The database username.
