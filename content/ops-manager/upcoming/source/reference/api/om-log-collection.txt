.. _api-om-log-collection:

================================
|mms| Server Log Collection Jobs
================================

.. meta::
   :description: Use the Ops Manager API to collect, list, retrieve, download, extend, retry, or delete application server log collection jobs across one or more servers.

.. default-domain:: mongodb

You can use the |mms| |api| to collect application server logs from
one or more |mms| servers. After you submit a request, |mms|
distributes the work to the target servers. Each server collects
its logs in a background task, compresses and encrypts the result,
and stores the archive in the Application Database. Use the
following endpoints to create, monitor, download, extend, retry,
and delete |mms| server log collection jobs.

.. _api-om-log-collection-permissions:

To use the |mms| Server Log Collection resources, you must have one
of the following roles:

- :authrole:`Global Read Only` to list active servers, list jobs,
  get a single job, or download a job archive.
- :authrole:`Global Owner` to create a job, extend a job, retry a
  job, or delete a job.

.. include:: /includes/api/base-api-uri.rst

Endpoints
---------

The following lists the endpoints available for the |mms| Server
Log Collection resource.

.. list-table::
   :header-rows: 1
   :widths: 10 50 40

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :ref:`/admin/servers <api-om-log-collections-list-servers>`
     - Lists active servers eligible for log collection.

   * - ``POST``
     - :ref:`/admin/omLogCollectionJobs <api-om-log-collections-submit>`
     - Creates a new server log collection job.

   * - ``GET``
     - :ref:`/admin/omLogCollectionJobs <api-om-log-collections-get-all>`
     - Retrieves all server log collection jobs.

   * - ``GET``
     - :ref:`/admin/omLogCollectionJobs/{JOB-ID} <api-om-log-collections-get-one>`
     - Retrieves a single server log collection job.

   * - ``GET``
     - :ref:`/admin/omLogCollectionJobs/{JOB-ID}:download <api-om-log-collections-download-job>`
     - Downloads the archive for a completed log collection job.

   * - ``PATCH``
     - :ref:`/admin/omLogCollectionJobs/{JOB-ID} <api-om-log-collections-update-one>`
     - Extends the expiration of a log collection job.

   * - ``POST``
     - :ref:`/admin/omLogCollectionJobs/{JOB-ID}:retry <api-om-log-collections-retry>`
     - Retries the failed tasks in a log collection job.

   * - ``DELETE``
     - :ref:`/admin/omLogCollectionJobs/{JOB-ID} <api-om-log-collections-delete-one>`
     - Deletes a log collection job and its archive.
