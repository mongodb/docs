.. _available-sample-datasets:

============================================
Available Sample Datasets for Atlas Clusters
============================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

This page shows the sample datasets available for |service| clusters.
Click a sample dataset to lean more about it.

For instructions on loading this sample data into your |service|
cluster, see :ref:`load-sample-data`.

.. list-table::
   :header-rows: 1
   :widths: 50 50

   * - Dataset Name
     - Description

   * - :ref:`sample-airbnb`
     -  Contains details on
        `AirBnB <https://www.airbnb.com>`__ listings.

   * - :ref:`sample-geospatial`
     - Contains shipwreck data.

   * - :ref:`sample-mflix`
     - Contains movie data.

   * - :ref:`sample-supplies`
     - Contains data from a mock office supply store.

   * - :ref:`sample-weather`
     - Contains detailed weather reports.

   * - :ref:`sample-training`
     - Contains MongoDB training services dataset.
