==================================================================
Install the {+mcli+}
==================================================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

.. include:: /includes/styles/corrections.rst

You can use the {+mcli-long+} to deploy and manage MongoDB clusters in
|service|, |cloud-short|, and |onprem|.

.. admonition:: Ops Manager Compatibility
   :class: note

   {+mcli+} is compatible with :opsmgr:`Ops Manager v4.2.0
   </release-notes/application/#onprem-server-4-2-0>` and later.

To install the {+mcli+}, choose one of the following methods:

1. Install with Homebrew (MacOS and Linux only).
#. Download and extract the binary.
#. Clone the GitHub repository and install the {+mcli+} with Go.

|

      Prerequisites
      -------------

      To install the {+mcli+} using Homebrew, you must:

      Prerequisites
      -------------

      To install the {+mcli+}, you must install the following tools:

      1. `Git <https://git-scm.com/book/en/v2/Getting-Started-Installing-Git>`_.

      #. `Go v1.14 or newer <https://golang.org/doc/install#install>`_.
