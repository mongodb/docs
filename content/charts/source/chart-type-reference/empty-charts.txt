============
Empty Charts
============

.. meta::
   :description: Explore what causes an empty state to display, with an accompanying visual representation.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

|charts| displays an empty state when no chart data is present. 

Possible Causes of No Data or Documents
---------------------------------------

The following are the possible causes for the absence of data or documents. 

- Empty data source
- Filters excluding all data points
- Charts query returning no result
