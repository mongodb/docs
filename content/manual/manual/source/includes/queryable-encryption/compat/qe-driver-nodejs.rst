Driver Compatibility
~~~~~~~~~~~~~~~~~~~~

To use {+qe+} with the :driver:`Node.js </node/current>` driver,
install driver version 5.5.0 or later and `mongodb-client-encryption
<https://www.npmjs.com/package/mongodb-client-encryption/>`__ version
2.8.0 or later.
