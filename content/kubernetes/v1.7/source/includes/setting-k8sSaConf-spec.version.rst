.. setting:: spec.version

   *Type*: string

   Version of MongoDB that you installed on this |k8s-mdbrsc|.
   
   The format should be ``X.Y.Z`` for the Community edition and
   ``X.Y.Z-ent`` for the Enterprise edition.
