===========
JSON Schema
===========

.. meta::
   :description: Learn how to export your JSON Schema and download sample documents using Relational Migrator.

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

{+rel-mig+} can generate and export a collection's JSON Schema and sample documents 
that align to each collection's schema.

Features
--------

.. list-table::
   :header-rows: 1

   * - Feature
     - Description
   * - JSON Schema
     - Generates JSON Schema files that can be used for :ref:`MongoDB schema validation <schema-validation-json>`.
   * - Sample Documents
     - Generates sample JSON documents that align with each collection's schema.
