.. _delete-project:

================
Delete a Project
================

.. meta::
   :description: Delete a project to remove unnecessary migration workloads, noting that deleted projects cannot be recovered.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Delete a project to remove migration workloads that are no longer needed.

About this Task
---------------

- You cannot recover a deleted project. 

- If the last project on the :guilabel:`Recent Projects` page is deleted, the 
  :guilabel:`New Project` page is displayed.

Steps
-----

1. From the :guilabel:`project view`, click the :guilabel:`Home` icon in the 
   top-left corner.

#. Open the menu next to your project of choice. Select :guilabel:`Delete`.

#. Click :guilabel:`Delete` to confirm.
