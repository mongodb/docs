==========================================
mongocli atlas cluster onlinearchive start
==========================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

.. include:: /includes/fact-online-archive-beta.rst

The ``onlinearchive start`` command starts the archiving process for the 
specified :atlas:`online archive </online-archive/manage-online-archive/>` 
in the specified cluster. 

.. _mcli-atlas-cluster-online-archive-start-syntax:

Syntax 
------

.. code-block:: text 

   mongocli atlas cluster onlinearchive start <online-archive-id>
        --clusterName <cluster-name>  
        [ --output|-o <output-format> ]
        [ --profile|-P <profile-name> ]
        [ --projectId <project-ID> ]

.. _mcli-atlas-cluster-onlinearchive-start-options: 

Options 
-------

.. list-table::
   :header-rows: 1
   :widths: 25 10 55 10

   * - Option
     - Type
     - Description
     - Required?

   * - ``<online-archive-id>``
     - string
     - Unique identifier of the :atlas:`online archive 
       </online-archive/manage-online-archive/>` to start.
     - yes

   * - ``--clusterName``
     - string
     - Name of the cluster. 
     - yes

   * - ``--output``, ``-o``
     - string 
     - .. include:: /includes/extracts/fact-basic-options-output.rst
     - no

   * - ``--profile``, ``-P``
     - string
     - Name of the profile where the public and private 
       keys for the project are saved. If omitted, uses the 
       {+default-profile+}. To learn more about creating a 
       profile, see :ref:`mcli-configure`.
     - no

   * - ``--projectId``
     - string
     - .. include:: /includes/extracts/fact-basic-options-project-id.rst
     - no

Example 
-------

The following example uses the ``mongocli atlas onlinearchive start`` command 
to start the archiving process for an online archive for the cluster named 
``myTestCluster``. The command uses the {+default-profile+} to access the 
project.

.. code-block:: sh 
   
   mongocli atlas cluster onlinearchive start 5f189832e26ec075e10c32d3 --clusterName myTestCluster --output json
