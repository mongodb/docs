.. query:: $nearSphere

   .. versionadded:: 1.8

   The :query:`$nearSphere` operator is the spherical equivalent of
   the :query:`$near` operator. :query:`$nearSphere` returns all
   documents near a point, calculating distances using spherical geometry.

   .. code-block:: javascript

      db.collection.find( { loc: { $nearSphere: [0,0] } } )
