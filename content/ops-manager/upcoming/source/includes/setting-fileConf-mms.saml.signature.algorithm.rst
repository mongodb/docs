.. setting:: mms.saml.signature.algorithm

   *Type*: string

   
   Algorithm to encrypt the signature sent to and from the |idp|.
   
   In the :guilabel:`Select an Algorithm` menu, there are five choices:
   
   - ``rsa-sha1``
   - ``dsa-sha1``
   - ``rsa-sha256``
   - ``rsa-sha384``
   - ``rsa-sha512``
