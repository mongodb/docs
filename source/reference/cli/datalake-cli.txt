Stores 
------

The |data-lake| CLI lets you create, remove, and retrieve stores in 
your {+dl+} storage configuration using the following commands: 

- :ref:`dl-create-store-cmd`
- :ref:`dl-drop-store-cmd`
- :ref:`dl-list-stores-cmd`

Database 
--------

The |data-lake| CLI lets you remove a database from your {+dl+} storage 
configuration using the following commands: 

- :ref:`dl-drop-database-cmd`

Collections and Views
---------------------

The |data-lake| CLI lets you create, remove, and/or rename collections 
and views in your {+dl+} storage configuration using the following 
commands: 

- :ref:`dl-create-collection-views-cmd`
- :ref:`dl-drop-collection-views-cmd`
- :ref:`dl-rename-collection-cmd`

Namespace 
---------

The |data-lake| CLI lets you retrieve :ref:`namespace catalog <manage-ns-catalog-cli>` 
information and update the namespace catalog using the following commands:

- :ref:`catalogInfo-cmd`
- :ref:`updatecatalog-cmd`
