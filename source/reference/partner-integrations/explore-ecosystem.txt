=============================
Explore the MongoDB Ecosystem
=============================

.. default-domain:: mongodb

.. meta::
   :keywords: integration, partner integration, third-party, third-party tools, tools, integration use cases, integration collections, integration idea, integration submission, trending integration, popular integration, new integration, MongoDB developer community, developer community
   :description: Explore a collection of the integrations and partner services. See the top trending integrations. Submit an integration idea.

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

You can explore a collection of the integrations and partner 
services in the {+ecosystem+}. The home page includes 
following information:

- Top three trending integrations that are new and on the rise.
- Top three integrations that the MongoDB developer community uses 
  the most.
- Link to the `Integrations & Tools 
  <https://cloud.mongodb.com/ecosystem/integrations>`__ page where you 
  can explore the full list of integrations. Each Integration has its 
  own page with details about the integration and links to set up the 
  integration.
- Link to submit an integration idea.

Search or Filter the MongoDB Ecosystem
--------------------------------------

To search or filter the full list of available integrations and tools:

.. procedure::
   :style: normal

   .. step:: Go to the {+ecosystem+}.

   .. step:: Click :guilabel:`Integrations & Tools`.

   .. step:: Search or filter the MongoDB Ecosystem.

      Do one or all of the following steps:

      - In the :guilabel:`Search Integrations` search bar, enter a 
        search string.

      - In the :guilabel:`Use Cases` section, select one or more of the 
        use cases to filter by.

      - In the :guilabel:`Collection` section, select one or more of 
        the integration collections to filter by.
