========
Auditing
========

.. default-domain:: mongodb

The following resources provide access to project 
:doc:`auditing </database-auditing>` settings. Each resource requires at 
minimum your :ref:`group-id`. 

.. include:: /includes/fact-log-data-update-interval.rst

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/auditLog </reference/api/auditing-get-auditLog>`
     - Get audit configuration for the project associated with ``{GROUP-ID}``

   * - ``PATCH``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/auditLog </reference/api/auditing-set-auditLog>`
     - Modify the audit configuration for the project associated with ``{GROUP-ID}``
