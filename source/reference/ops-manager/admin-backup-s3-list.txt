.. _mcli-om-admin-backup-s3-list-cmd:

=========================================
mongocli ops-manager admin backup s3 list
=========================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

.. include:: /includes/admonitions/admin-requires-global-owner.rst

The ``admin backup s3 list`` command retrieves the list of 
backup :opsmgr:`S3 blockstore </reference/glossary/#term-s3-snapshot-store>` 
configurations for your |onprem| deployment. You can also list 
|s3| configurations using the
:opsmgr:`API </reference/api/admin/backup/snapshot/s3Configs/get-all-s3-blockstore-configurations/>`, or view |s3| configurations 
in the :opsmgr:`Administration Console </admin-console/>`.

Syntax 
------

.. code-block:: text 

   mongocli ops-manager|om admin backup s3 list|ls
        [ --limit <items-per-page> ]
        [ --output|-o <output-format> ]
        [ --page <page-number> ]
        [ --profile|-P <profile-name> ]

.. include:: /includes/fact-command-line-help.rst

Options
-------

.. list-table::
   :widths: 20 10 60 10
   :header-rows: 1

   * - Option 
     - Type
     - Description
     - Required?

   * - ``--limit``
     - integer
     - Number of items to return per page, up to a maximum of 500.
       Defaults to ``100`` if not specified.
     - no

   * - ``--output``, ``-o``
     - string 
     - .. include:: /includes/extracts/fact-basic-options-output.rst
     - no

   * - ``--page``
     - integer
     - Page number (1-based). Defaults to ``1`` if not specified.
     - no

   * - ``--profile``, ``-P``
     - string
     - Name of the profile where the public and private 
       keys for accessing |onprem| are saved. If omitted, uses the 
       {+default-profile+}. To learn more about creating a 
       profile, see :ref:`mcli-configure`.
     - no

Examples
--------

.. tabs::

   .. tab:: Default Output
      :tabid: default-output

      The following command retrieves the list of |s3| 
      blockstore configurations using the {+default-profile+}, which 
      contains :authrole:`Global Owner` |mms| credentials. The output 
      is returned in the default format.

      .. code-block:: sh

         mongocli ops-manager admin backup s3 list

      The previous command prints the following to the terminal.

      .. code-block:: sh
         :copyable: false

         NAME               URI                        SSL    LOAD FACTOR   AUTH METHOD
         s3BackupExample    mongodb://127.0.0.1:27017  false  1
