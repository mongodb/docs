==================
Encryption at Rest
==================

.. default-domain:: mongodb

The following endpoints allow administrators to enable, disable,
configure, and retrieve the configuration for Encryption at Rest.

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``PATCH``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/encryptionAtRest </reference/api/enable-configure-encryptionatrest>`
     - Enable, disable, and configure Encryption at Rest for an |service| project.

   * - ``GET``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/encryptionAtRest </reference/api/get-configuration-encryptionatrest>`
     - Retrieve the current configuration for Encryption at Rest for an |service| project.
