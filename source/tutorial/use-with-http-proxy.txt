==========================================================================
Configure |onprem| to Pass Outgoing Traffic through an HTTP or HTTPS Proxy
==========================================================================

.. default-domain:: mongodb

Overview
--------

|onprem| can pass all outgoing requests through an HTTP or HTTPS proxy.
This lets |onprem| deployments without direct access to external resources
access outside notification services and the MongoDB version manifest.
