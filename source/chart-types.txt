.. _chart-type-reference:

===========
Chart Types
===========

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

:ref:`Geospatial Charts <geo-spatial>`
--------------------------------------

Geospatial charts display a mark for each unique geographic location.

.. figure:: /images/charts/geospatial-chart-ref.png
   :figwidth: 791px

:ref:`Column and Bar Charts <column-bar-chart-ref>`
---------------------------------------------------

Column and bar charts provide high-level overviews of data
trends by comparing values within a specific category.
   
.. figure:: /images/charts/stacked-bar-chart-reference-small.png
   :figwidth: 550px
   
:ref:`Line and Area Charts <line-area-chart-ref>`
-------------------------------------------------

Line and area charts display information as a series of data
points connected by straight line segments.
   
.. figure:: /images/charts/discrete-area-small.png
   :figwidth: 550px

Grid Charts
-----------

.. container::

   :ref:`Heatmaps <heatmap-chart-ref>`

   Heatmaps represent aggregated data in a tabular format as a range
   of colors.
   
   .. figure:: /images/charts/heatmap-small.png
      :figwidth: 550px
   
   :ref:`Scatter Charts <scatter-chart-ref>`

   Scatter charts plot data with individual data points placed along
   the X and Y axes.
   
   .. figure:: /images/charts/scatter-chart-small.png
      :figwidth: 550px

Text Charts
-----------

.. container::

   :ref:`Data Tables <data-table-ref>`

   Data tables represent data in a tabular view, similar to a
   `spreadsheet <https://en.wikipedia.org/wiki/Spreadsheet>`_.
   
   .. figure:: /images/charts/data-table-small.png
      :figwidth: 550px
   
   :ref:`Number Charts <number-chart-ref>`

   Number charts display a single aggregated value from a data field.
   
   .. figure:: /images/charts/number-chart-small.png
      :figwidth: 550px
