======================================
{+Cloud-Backup+}
======================================

.. default-domain:: mongodb

.. meta::
   :canonical: https://www.mongodb.com/docs/atlas/reference/api-resources-spec/#tag/Cloud-Backups

.. include:: /includes/fact-projects-groups-synonyms.rst

.. important:: {+Cloud-Backup+}s Only

   This resource only applies to clusters using
   :ref:`backup-cloud-provider`. 

The ``/backup/snapshots`` resource enables you to view and delete
{+Cloud-Backup+}s.

Endpoints
---------

.. include:: /includes/api-base-url.rst

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/backup/snapshots </reference/api/cloud-backup/backup/get-all-backups>`
     - Get all {+Cloud-Backup+}s for the specified cluster.

   * - ``GET``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/backup/snapshots/{SNAPSHOT-ID} </reference/api/cloud-backup/backup/get-one-backup>`
     - Get the snapshot associated with ``{SNAPSHOT-ID}``

   * - ``DELETE``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/backup/snapshots/{SNAPSHOT-ID} </reference/api/cloud-backup/backup/delete-one-backup>`
     - Delete the snapshot associated with ``{SNAPSHOT-ID}``

       .. include:: /includes/api/facts/api-access-list-required.rst

   * - ``POST``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/backup/snapshots </reference/api/cloud-backup/backup/take-one-ondemand-backup>`
     - Take an on-demand snapshot.

   * - ``GET``
     - :doc:`/groups/{groupId}/serverless/{instanceName}/backup/snapshots/{snapshotId} </reference/api/cloud-backup/backup/return-one-snapshot-of-one-serverless-instance>`
     - Return one snapshot of one serverless instance from the
       specified project.

   * - ``GET``
     - :doc:`/groups/{groupId}/serverless/{instanceName}/backup/snapshots </reference/api/cloud-backup/backup/return-all-snapshots-of-one-serverless-instance>`
     - Return the last two snapshots of one serverless instance from
       the specified project.
