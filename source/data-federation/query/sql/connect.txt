=======
Connect
=======

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

.. include:: /includes/extracts/fact-adf-sql-preview-message.rst 

After you configure your {+fdi+}, you can connect to it from 
BI tools and applications or from the MongoDB Shell. To connect SQL 
tools, select :guilabel:`Connect using the {+asql+} interface` in the 
{+fdi+} connection modal.

.. figure:: /images/asql-connect-modal.png
   :alt: Screenshot of the Atlas connection modal with "Connect using the {+asql+} Interface" option highlighted.

The following sections provide basic instructions for connecting from 
different tools.

- :ref:`JDBC driver <sql-connect-jdbc>`
- :ref:`DBeaver <sql-connect-dbeaver>`
- :ref:`Tableau <sql-connect-tableau>`
- :ref:`MongoDB Shell <sql-connect-shell>`
