Use this endpoint to delete a specific {+fdi+} associated with an
|service| project. 

Syntax
------

.. code-block:: none

   DELETE /groups/{GROUP-ID}/dataFederation/{NAME}

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :widths: 15 10 75

   * - Path Element
     - Required/Optional
     - Description

   * - ``GROUP-ID``

     - Required.

     - The unique identifier for the :ref:`project <projects>`.

   * - ``NAME``

     - Require
     - The name of the {+fdi+} that you want to delete.

       You can use the :doc:`/reference/api/dataFederation-get-all-tenants`
       endpoint to retrieve all {+fdi+}s associated with the
       project. The ``name`` field in the response of that endpoint
       corresponds to the ``NAME`` parameter here.

Response
--------

This endpoint returns an empty JSON document.

Example
-------

Request
~~~~~~~

.. code-block:: none

   curl -u "username:apiKey" --digest \
    --header "Accept: application/json" \
    --header "Content-Type: application/json" \
    --request DELETE "https://cloud.mongodb.com/api/atlas/v1.0/groups/{GROUP-ID}/dataFederation/{NAME}?pretty=true"

~~~~~~~~

.. code-block:: none
   :copyable: false
