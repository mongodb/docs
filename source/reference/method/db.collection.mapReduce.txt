db.collection.mapReduce()
=========================

.. default-domain:: mongodb

.. method:: db.collection.mapReduce()

   The :method:`db.collection.mapReduce()` method provides a wrapper
   around the :dbcommand:`mapReduce` command.

   .. code-block:: javascript

      db.collection.mapReduce(
                               <map>,
                               <reduce>,
                               {
                                 <out>,
                                 <query>,
                                 <sort>,
                                 <limit>,
                                 <finalize>,
                                 <scope>,
                                 <jsMode>,
                                 <verbose>
                               }
                             )
