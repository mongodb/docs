======================
Version Manifest Page
======================

.. meta::
   :description: View all released MongoDB versions in the Ops Manager version manifest, which is required for running Ops Manager in local mode without internet access.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

This page lists all released MongoDB versions. |onprem| requires
this list to run in local mode, which allows it to deploy and
manage MongoDB without outbound internet access. See the
:ref:`om-modes` tutorial for more
information on configuring local mode.
