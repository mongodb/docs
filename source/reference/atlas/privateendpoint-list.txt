.. _mcli-atlas-privateendpoint-list-cmd:

===================================
mongocli atlas privateendpoint list
===================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``privateendpoint list`` command retrieves information about all
private endpoint connections for a given |service| project.

Syntax 
------

.. code-block:: text 

   mongocli atlas privateendpoint(s)|privateEndpoint(s) list|ls
        [ --output|-o <output-format> ]
        [ --profile|-P <profile-name> ]
        [ --projectId <project-ID> ]

.. include:: /includes/fact-command-line-help.rst

.. _atlas-privateendpoint-list-options:

Options 
-------

.. list-table::
   :widths: 20 10 60 10
   :header-rows: 1

   * - Option 
     - Type
     - Description
     - Required?

   * - ``--output``, ``-o``
     - string 
     - .. include:: /includes/extracts/fact-basic-options-output.rst
     - no

   * - ``--profile``, ``-P``
     - string
     - Name of the profile where the public and private 
       keys for the project are saved. If omitted, uses the 
       {+default-profile+}. To learn more about creating a 
       profile, see :ref:`mcli-configure`.
     - no

   * - ``--projectId``
     - string
     - .. include:: /includes/extracts/fact-basic-options-project-id.rst
     - no

.. _atlas-privateendpoint-list-output:

Output
------

If the command succeeds, it returns the following output in the default
format. If the command returns errors, see
:ref:`Troubleshooting <troubleshooting>` for recommended solutions.

.. code-block:: sh
   :copyable: false

   ID                        ENDPOINT SERVICE               STATUS                         ERROR
   <private-endpoint-1-id>   <private-endpoint-1-service>   <private-endpoint-1-status>    <private-endpoint-1-error>
   <private-endpoint-2-id>   <private-endpoint-2-service>   <private-endpoint-2-status>    <private-endpoint-2-error>

.. include:: /includes/fact-default-output.rst

- :atlas:`Atlas API </reference/api/private-endpoint-get-all-private-endpoint-connections/#response-elements>`

Example
-------

.. tabs::

   .. tab:: Default Output
      :tabid: default-output

      The following command retrieves private endpoint connections for a
      project using the {+default-profile+}, which contains credentials
      and the project ID. The output is returned in the default format.

      .. code-block:: sh

         mongocli atlas privateendpoint list

      The previous command prints the following to the terminal.

      .. code-block:: sh
         :copyable: false

         ID                         ENDPOINT SERVICE                                          STATUS             ERROR
         5f4fc81c1f03a835c2728ff7   com.amazonaws.vpce.us-east-1.vpce-svc-06c32a54aec6cbb44   WAITING_FOR_USER
         5f4fc829a2b47835a58c6d04   com.amazonaws.vpce.us-west-1.vpce-svc-0d950f4cc6b45de33   WAITING_FOR_USER

   .. tab:: JSON Output
      :tabid: json-output

      The following command retrieves private endpoint connections for a
      project using the {+default-profile+}, which contains credentials
      and the project ID. The output is returned in |json| format.

      .. code-block:: sh

         mongocli atlas privateendpoint list --output json

      The previous command prints the following to the terminal in
      |json| format.

      .. code-block:: json
         :copyable: false

         [
