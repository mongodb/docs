======================
Live Migrate Your Data
======================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

You can bring data from existing MongoDB deployments, ``JSON``, or ``CSV`` files
into |service| using one of the following tools managed by |service| within the
|service| user interface. 

.. list-table::
   :header-rows: 1
   :widths: 35 65

   * - Method
     - Description

   * - :doc:`Live migrate a replica set </import/live-import>`

     - Use the |service| user interface that relies on |mongomirror|
       and guides you through a live migration from a MongoDB *replica set*
       into an |service| replica set cluster.

   * - :doc:`Live migrate a sharded cluster </import/live-import-sharded>`

     - Use the |service| user interface that relies on |mongomirror|
       and guides you through a live migration from a MongoDB *sharded cluster* to an
       |service| sharded cluster.

   * - Live migrate a Standalone MongoDB Node

     - :manual:`Convert </tutorial/convert-standalone-to-replica-set>`
       the standalone MongoDB node to a single-node replica set and
       then :doc:`live migrate </import/live-import>` it to |service|.
