|charts| v0.10.0
----------------

*Released August 20, 2018*

New Features
~~~~~~~~~~~~

- :v0.10:`Test Connection </installation/#test-your-connection-to-the-charts-short-metadata-database>`
  tool to help set the correct MongoDB URI during installation.

- Ability to supply connection URI via the ``CHARTS_MONGODB_URI``
  environment variable as an alternative to the Docker Secret.

- New :ref:`Number <number-chart-ref>` chart type, which shows a single
  numeric value.

- Ability to
  :doc:`configure HTTPS </administration/configure-https-deployment>`
  on the |charts-short| web server.

- Ability to :ref:`edit the MongoDB URI <mod-data-source-uri>` of an
  existing :doc:`data source </data-sources>`.

- Longer query timeouts for newly created data sources.

- Ability to filter dashboard and :ref:`data sources <filter-data-sources>`
  lists by the current user’s role.

- :ref:`Customizable “auto refresh” options <configure-auto-refresh>`
  for dashboards.

- Ability to :ref:`edit a dashboard's title and description <dashboard-modify-title-description>`
  from the :guilabel:`Dashboards` tab.

- :ref:`Full-Screen mode <fullscreen-view>` for dashboards.

- :ref:`Duplicate a dashboard <duplicate-dashboard>`.

- :ref:`Duplicate a chart <dashboard-duplicate-chart>`.

- :ref:`Filter <charts-filter-documents>` and
  :ref:`sort <charts-sorting-limiting>` data source dropdown in the
  :doc:`Chart Builder </build-charts>`.

- Allow use of the ``Date()`` operator in the
  :ref:`Chart Builder filter <date-filter>` bar to execute relative date
  queries.

- Improved color contrast for Coloured Bar and Coloured Column charts.

Bug Fixes
~~~~~~~~~

- Ability to add Data Sources from databases with names containing
  special characters.

- Prevent crash when entering a filter prior to choosing a Data Source
  in the Chart Builder.

- Ability to use the multiple fields from an embedded array in the
  same chart.

- Strip out extra quotes in MongoDB URI added by Windows echo command
  during install procedure.

- Miscellaneous styling and usability improvements.
