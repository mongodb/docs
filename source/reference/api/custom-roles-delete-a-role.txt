Syntax
------

.. code-block:: none

   DELETE /groups/{GROUP-ID}/customDBRoles/roles/{ROLE-NAME}

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1

   * - Parameter
     - Required/Optional
     - Description

   * - ``GROUP-ID``

     - Required.

     - The unique identifier for the :ref:`project <group-id>`.

   * - ``ROLE-NAME``

     - Required.

     - The name of the role to update.
