Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/fts/indexes/{DATABASE-NAME}/{COLLECTION-NAME} </reference/api/fts-indexes-get-all>`
     - Get all |fts| indexes for a specified collection.

   * - ``GET``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/fts/indexes/{INDEX-ID} </reference/api/fts-indexes-get-one>`
     - Get one |fts| index by its ``indexId``.

   * - ``GET``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/fts/analyzers </reference/api/fts-analyzers-get-all>`
     - Get all |fts| user-defined analyzers for a specified cluster.

   * - ``POST``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/fts/indexes/ </reference/api/fts-indexes-create-one>`
     - Create an |fts| index.

   * - ``PATCH``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/fts/indexes/{INDEX-ID} </reference/api/fts-indexes-update-one>`
     - Update an |fts| index by its ``indexId``.

   * - ``PUT``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/fts/analyzers
       </reference/api/fts-analyzers-update-all>`
     - Update the |fts| user-defined analyzers for a specified cluster.

   * - ``DELETE``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/fts/indexes/{INDEX-ID}
       </reference/api/fts-indexes-delete-one>`
     - Delete one |fts| index by its ``indexId``.
