=================
MongoDB with Java
=================

.. meta::
   :description: Develop Java applications using Atlas with synchronous and asynchronous drivers, and integrate with Spring Data MongoDB for scalable solutions.

.. include:: /includes/docs-toc.rst

.. introduction::

   Develop resilient and efficient Java applications by using the 
   `MongoDB Atlas cloud database <https://www.mongodb.com/docs/atlas/>`__. 
   With support for both synchronous and asynchronous use cases, the Java 
   Sync and Reactive Streams drivers support Java developers in building 
   scalable, high-performance applications. If you prefer to use an abstraction
   layer, integrations with `Spring Data <https://spring.io/projects/spring-data-mongodb>`__
   and `Hibernate ORM <https://www.mongodb.com/docs/languages/java/mongodb-hibernate/>`__
   allow you to fully utilize the capabilities of MongoDB within your preferred
   Java environment.

   .. button:: Get started with MongoDB Java Sync
      :uri: https://www.mongodb.com/docs/drivers/java/sync/current/get-started/

   .. button:: Get started with MongoDB Java Reactive Streams
      :uri: https://www.mongodb.com/docs/languages/java/reactive-streams-driver/current/getting-started/

Drivers

   .. card::
      :headline: Java Sync
      :url: https://www.mongodb.com/docs/drivers/java/sync/current/
      :icon: /images/icons/java.svg
      :icon-alt: Driver card icon

      Provides a synchronous API for working with MongoDB databases consistent with other MongoDB language drivers.

   .. card::
