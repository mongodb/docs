=======================
Snapshot Export Buckets
=======================

.. default-domain:: mongodb

.. include:: /includes/fact-projects-groups-synonyms.rst 

The ``/backup/exportBuckets`` resource enables you to grant |service| 
access to your |aws| buckets for :ref:`exporting 
<snapshot-export-jobs>` {+Cloud-Backup+} snapshots, retrieve buckets 
that |service| can access, and remove access to buckets for |service|.

Endpoints
---------

.. include:: /includes/api-base-url.rst

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :doc:`/groups/{GROUP-ID}/backup/exportBuckets 
       </reference/api/cloud-backup/export/get-all-export-buckets>`
     - Get all the |aws| buckets that |service| can access for the 
       specified project.

   * - ``GET``
     - :doc:`/groups/{GROUP-ID}/backup/exportBuckets/{BUCKET-ID} 
       </reference/api/cloud-backup/export/get-one-export-bucket>` 
     - Get the |aws| bucket associated with ``{BUCKET-ID}``

   * - ``POST``
     - :doc:`/groups/{GROUP-ID}/backup/exportBuckets 
       </reference/api/cloud-backup/export/create-one-export-bucket>`
     - Grant |service| access to one |aws| bucket for exporting backup 
       snapshot.

   * - ``DELETE``
     - :doc:`/groups/{GROUP-ID}/backup/exportBuckets/{BUCKET-ID} 
       </reference/api/cloud-backup/export/delete-one-export-bucket>` 
     - Remove |service| access to the bucket specified by 
       ``{BUCKET-ID}``
