====================
Serverless Instances
====================

.. default-domain:: mongodb

.. include:: /includes/note-serverless-in-preview-release.rst
  
The ``serverless`` resource provides access to your
{+serverless-instance+} configurations. The resource lets you create,
edit and delete {+serverless-instances+}. The resource requires your
:ref:`project-id`.

.. include:: /includes/fact-projects-groups-synonyms.rst

.. include:: /includes/api/base-api-uri.rst

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 40 50

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :doc:`/groups/{groupId}/serverless/{instanceName} </reference/api/serverless/return-one-serverless-instance>`
     - Returns one {+serverless-instance+} in the specified project.

   * - ``GET``
     - :doc:`/groups/{groupId}/serverless </reference/api/serverless/return-all-serverless-instances>`
     - Returns all {+serverless-instances+} in the specified project.

   * - ``POST``
     - :doc:`/groups/{groupId}/serverless </reference/api/serverless/create-one-serverless-instance>`
     - Creates one {+serverless-instance+} in the specified project.

   * - ``PATCH``
     -  :doc:`/groups/{groupId}/serverless/{instanceName} </reference/api/serverless/update-one-serverless-instance>`
     - Updates one {+serverless-instance+} in the specified project.

   * - ``DELETE``
     - :doc:`/groups/{groupId}/serverless/{instanceName} </reference/api/serverless/remove-one-serverless-instance>`
     - Removes one {+serverless-instance+} in the specified project.
