Definition
----------

.. method:: cd(path)

   .. list-table::
      :header-rows: 1
      :widths: 20 20 80
   
      * - Parameter
        - Type
        - Description

      * - ``path``
        - string
        - A path on the file system local to the :binary:`mongo` shell
          context.


   :method:`cd()` changes the directory context of :binary:`mongo` and
   has no effect on the MongoDB server.
