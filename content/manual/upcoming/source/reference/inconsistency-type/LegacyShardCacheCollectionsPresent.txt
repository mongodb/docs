==================================
LegacyShardCacheCollectionsPresent
==================================

.. meta::
   :description: Identify legacy shard catalog cache collections that remain in the config.cache namespace after a cluster upgrades to feature compatibility version 9.0.

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Description
-----------

.. data:: LegacyShardCacheCollectionsPresent

   .. include:: /includes/inconsistency-type/LegacyShardCacheCollectionsPresent

Format
------

.. code-block:: json

   {
      type: "LegacyShardCacheCollectionsPresent",
      description: "<string>",
      details: {
         namespace: "<string>",
         shard: "<string>"
      }
   }

``LegacyShardCacheCollectionsPresent`` inconsistency documents contain
the following fields:

.. |incon-type| replace:: ``LegacyShardCacheCollectionsPresent``

.. list-table::
   :widths: 30 25 45
   :header-rows: 1

   * - Field
     - Type
     - Description


   * - ``type``
     - string
     - .. include:: /includes/inconsistency-type/type


   * - ``description``
     - string
     - .. include:: /includes/inconsistency-type/descr


   * - ``details``
     - document
     - .. include:: /includes/inconsistency-type/details


   * - ``details.namespace``
     - string
     - .. include:: /includes/inconsistency-type/details.namespace


   * - ``details.shard``
     - string
     - .. include:: /includes/inconsistency-type/details.shard


Example
-------

.. include:: /includes/inconsistency-type/example

.. code-block:: json
   :emphasize-lines: 6-12
   :copyable: false

   {
      cursor: {
         id: Long("0"),
         ns: "test.$cmd.aggregate",
         firstBatch: [
            {
               type: "LegacyShardCacheCollectionsPresent",
               description: "Found legacy non-authoritative shard catalog cache collections on a shard that is fully upgraded",
               details: {
                  namespace: "config.cache.collections",
                  shard: "shard-rs0"
               }
            }
         ],
      },
      ok: 1
   }
