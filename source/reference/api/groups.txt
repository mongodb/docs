Endpoints
---------


.. list-table::
   :header-rows: 1
   :widths: 10 50 40

   * - Method
     - Endpoint
     - Description

   * - GET 
     - :doc:`/api/public/v1.0/groups </reference/api/groups/get-all-groups-for-current-user>`
     - Get All Groups for the Current User

   * - GET
     - :doc:`/api/public/v1.0/groups/{GROUP-ID} </reference/api/groups/get-one-group-by-id>`
     - Get One Group by Its ID

   * - GET
     - :doc:`/api/public/v1.0/groups/byName/{GROUP-NAME} </reference/api/groups/get-one-group-by-name>`
     - Get One Group by Its Name

   * - GET
     - :doc:`/api/public/v1.0/groups/byAgentApiKey/{AGENT-API-KEY} </reference/api/groups/get-one-group-by-agent-api-key>`
     - Get One Group by Its Agent API Key

   * - GET
     - :doc:`/api/public/v1.0/groups/{GROUP-ID}/users </reference/api/groups/get-all-users-in-one-group>`
     - Get All Users in One Group

   * - POST
     - :doc:`/api/public/v1.0/groups </reference/api/groups/create-one-group>`
     - Create One Group

   * - POST
     - :doc:`/api/public/v1.0/groups/{GROUP-ID}/users </reference/api/groups/add-users-to-one-group>`
     - Add Users to One Group

   * - PATCH
     - :doc:`/api/public/v1.0/groups/{GROUP-ID}/ </reference/api/groups/change-one-group-name>`
     - Change One Group's Name

   * - PATCH
     - :doc:`/api/public/v1.0/groups/{GROUP-ID}/ </reference/api/groups/map-ldap-groups-to-manager-roles>`
     - Map LDAP Groups to |onprem| Roles

   * - DELETE
     - :doc:`/api/public/v1.0/groups/{GROUP-ID}/users/{USER-ID} </reference/api/groups/remove-one-user-from-one-group>`
     - Remove One User from One Group

   * - DELETE
     - :doc:`/api/public/v1.0/groups/{GROUP-ID} </reference/api/groups/delete-one-group>`
     - Delete One Group
