Endpoints
---------

The following endpoints are available for agents:

.. list-table::
   :header-rows: 1
   :stub-columns: 1
   :widths: 10 50 40

   * - Method
     - Endpoint
     - Description

   * - GET

     - :doc:`/groups/{GROUP-ID}/agents </reference/api/agents-get-all>`

     - Get links to Monitoring, Backup, and {+aagent+} resources
       for a :term:`project`.

   * - GET

     - :doc:`/groups/{GROUP-ID}/agents/{TYPE} </reference/api/agents-get-by-type>`

     - Get agents of a specified type (Monitoring, Backup, or Automation) for a :term:`project`.

   * - POST

     - :doc:`/groups/{GROUP-ID}/agentapikeys </reference/api/agentapikeys/create-one-agent-api-key>`

     - Create one Agent API Key for a :term:`project`.

   * - GET

     - :doc:`/groups/{GROUP-ID}/agentapikeys </reference/api/agentapikeys/get-all-agent-api-keys-for-project>`

     - Get list of all Agent API Keys for one :term:`project`.

   * - DELETE

     - :doc:`/groups/{GROUP-ID}/agentapikeys/{API-AGENT-KEY-ID} </reference/api/agentapikeys/delete-one-agent-api-key>`

     - Get agents of a specified type (Monitoring, Backup, or Automation) for a group.
