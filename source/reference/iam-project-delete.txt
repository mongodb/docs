.. _mcli-iam-project-delete:

=======================
mcli iam project delete
=======================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``mcli iam project delete`` command deletes a project 
using the specified profile. The specified profile must contain an |api|
key with the ``Project Owner`` role for the project you want to delete.
When you run the command without the ``--force`` option, the command
prompts you to confirm the operation. 

.. _mcli-iam-project-delete-syntax:

Syntax 
------

.. code-block:: sh 

   mcli iam project delete <project-ID>
        [ --force]
        [ --profile|-p <profile-name> ]

.. _mcli-iam-project-delete-options:

Options 
-------

.. list-table:: 
   :header-rows: 1
   :widths: 20 10 60 10

   * - Option 
     - Type 
     - Description 
     - Required? 

   * - ``--force``
     - 
     - Flag that indicates that the project can be deleted without
       requiring confirmation. 
     - no

   * - ``<project-ID>`` 
     - string 
     - Unique identifier of the project you want to delete.
     - yes

   * - ``--profile``, ``-p``
     - string 
     - Name of the profile that contains the access information for
       the project you want to delete. If omitted, uses the ``default``
       profile.  
     - no 

.. _mcli-iam-project-delete-output:

Output 
------

If the command succeeds, it prints a confirmation message similar to
the following to the terminal. If the command 
prints an error, see :ref:`Troubleshooting <troubleshooting>` 
for recommended solutions.

.. code-block:: text
   :copyable: false

   Project '<project-id'> deleted

.. _mcli-iam-project-delete-examples:

Examples 
--------

Example 1
~~~~~~~~~

The following command deletes a project with the Project ID
``5e33545ef10fab20b49c0f3b`` in an |onprem| organization. The command
uses the profile ``myOM``, which contains |api| keys that have the
``Project Owner`` role for this project.  

.. code-block:: text 

   mcli iam project delete 5e33545ef10fab20b49c0f3b --profile myOM

The previous command prints the following to 
the terminal. For more information on these fields, see 
:ref:`Output <mcli-iam-project-list-output>`.

.. code-block:: text
   :copyable: false

   Project '5e33545ef10fab20b49c0f3b' deleted

Example 2
~~~~~~~~~

The following command deletes a project in an |service| organization
without requiring confirmation. The command uses the profile
``myAtlas``, which contains |api| keys that have the ``Project Owner``
role for this project.

.. code-block:: text 

   mcli iam project delete 9dj3545ef10fab20b49c0f3b --force --profile myAtlas

The previous command prints the following to the terminal:

.. code-block:: text
   :copyable: false

   Project '9dj3545ef10fab20b49c0f3b' deleted
