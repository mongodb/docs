==================
View Activity Feed
==================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The |service| :guilabel:`Organization Activity Feed` displays select 
events that occurred for a given |service| 
:ref:`organization <create-organization>`, 
such as a billing or access events. Likewise, the 
|service| :guilabel:`Project Activity Feed` displays select events that 
occurred for a given |service|
:ref:`projects <create-project>`. 
|service| :guilabel:`Project Activity Feed` events typically provide 
more granular detail about the state of your cluster and database.

.. _view-activity-feed:

View the Activity Feed
----------------------

You can access events at the organization or project level. 
You can use the {+atlas-ui+}, {+atlas-cli+}, or the 
{+atlas-admin-api+}. Select the following tabs based on how you would 
like to access events.
