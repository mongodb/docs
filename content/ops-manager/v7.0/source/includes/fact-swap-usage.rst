- :guilabel:`used` displays the total amount of swap space currently in use.

- :guilabel:`free` displays the total amount of swap space that 
  is still available.

- :guilabel:`max used` displays the maximum total amount of swap space 
  in use over the selected time period.

- :guilabel:`max free` displays the maximum amount of 
  available swap space over the selected time period.

Sustained high swap usage can indicate memory pressure on the host.

Monitor swap usage to identify when the operating system starts 
moving memory pages to disk.
