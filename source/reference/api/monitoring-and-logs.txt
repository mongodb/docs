===================
Monitoring and Logs
===================

.. default-domain:: mongodb

The following resources provide access to cluster monitoring and logging
data. Each resource requires at minimum your :ref:`group-id`. 

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/processes </reference/api/processes-get-all>`
     - Get all processes for the specified group.

   * - ``GET``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/processes/{HOST}:{PORT} </reference/api/processes-get-one>`
     - Get information for the specified process in the specified group.

   * - ``GET``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/processes/{HOST}:{PORT}/measurements </reference/api/process-measurements>`
     - Get measurements for the specified host.

   * - ``GET``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/processes/{HOST}:{PORT}/databases </reference/api/process-databases>`
     - Get the list of databases for the specified host.

   * - ``GET``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/processes/{HOST}:{PORT}/databases/{DATABASE-NAME}/measurements </reference/api/process-databases-measurements>`
     - Get measurements of the specified database for the specified host.

   * - ``GET``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/processes/{HOST}:{PORT}/disks </reference/api/process-disks>`
     - Get the list of disks or partitions for the specified host.

   * - ``GET``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/processes/{HOST}:{PORT}/disks/{DISK-NAME}/measurements </reference/api/process-disks-measurements>`
     - Get measurements of specified disk for the specified host.

   * - ``GET``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/clusters/{HOSTNAME}/logs/mongodb.gz </reference/api/logs>`
     - Get the log file for a host in the cluster.
