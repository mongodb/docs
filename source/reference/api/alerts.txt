======
Alerts
======

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: twocols

.. include:: /includes/fact-projects-groups-synonyms.rst

.. include:: /includes/api-base-url.rst

The ``alerts`` resource allows you to retrieve and acknowledge
:doc:`alerts </configure-alerts>`.

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``

     - :doc:`/groups/{GROUP-ID}/alerts </reference/api/alerts-get-all-alerts>`

     - Get all alerts for the project associated to
       ``{GROUP-ID}``.

   * - ``GET``

     - :doc:`/groups/{GROUP-ID}/alerts/{ALERT-ID} </reference/api/alerts-get-alert>`

     - Get the specified alert for the project associated to
       ``{GROUP-ID}``.

   * - ``PATCH``

     - :doc:`/groups/{GROUP-ID}/alerts/{ALERT-ID} </reference/api/alerts-acknowledge-alert>`

     - Acknowledge or unacknowledge the alert specified to
       ``{ALERT-ID}`` for the project associated to
       ``{GROUP-ID}``.
