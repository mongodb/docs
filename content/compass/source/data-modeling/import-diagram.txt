========================================
Import an |erd-title| in |compass-short|
========================================

.. facet::
   :name: genre
   :values: tutorial

.. meta:: 
   :description: Import an entity-relationship diagram into Compass from a .mdm file.

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

You can import an |erd| into |compass-short| from an ``.mdm`` file. Importing 
diagrams can be helpful for when you want to view or edit a diagram that a 
collaborator shares in |compass-short|.

Before You Begin
----------------

To import your |erd|, follow the procedures to 
:ref:`export your diagram <compass-data-modeling-export>` as a 
``.mdm`` file. |compass-short| can import an |erd| only from 
``.mdm`` files and cannot import diagrams from PNG or JSON files.

Steps
-----

.. procedure::
   :style: normal
   
   .. step:: Open the Data Modeling tab 

      In the left sidebar, click :icon-lg:`Diagram` :guilabel:`Data Modeling`.
   
   .. step:: Click Import Diagram

   .. step:: Select your import file

      After you select the ``.mdm`` file you want to import, |compass-short| 
      displays the |erd| in the :guilabel:`Data Modeling` 
      tab.
