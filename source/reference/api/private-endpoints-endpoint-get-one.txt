=========================================
Get One Private Endpoint for One Provider
=========================================

.. default-domain:: mongodb

.. meta::
   :canonical: https://www.mongodb.com/docs/atlas/reference/api-resources-spec/#tag/Private-Endpoint-Services

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

.. include:: /includes/fact-projects-groups-synonyms.rst

Retrieve details for one of the following:

- One private endpoint for |aws| or |azure| in an |service| project.
- One private endpoint group for |gcp| in an |service| project. Endpoint groups
  represent a collection of endpoints. To learn more, see :doc:`Set up a Private
  Endpoint </security-private-endpoint>`.

.. include:: /includes/api-requirements.rst

Required Roles
--------------

You must have at the :authrole:`Project Read Only` role for the project to
successfully call this resource.

Request
-------

.. include:: /includes/api-base-url.rst

.. code-block:: http

   GET /groups/{GROUP-ID}/privateEndpoint/{CLOUD-PROVIDER}/endpointService/{ENDPOINT-SERVICE-ID}/endpoint/{ENDPOINT-ID}

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :stub-columns: 1
   :widths: 20 14 11 55

   * - Path Parameter
     - Type
     - Necessity
     - Description

   * - {GROUP-ID}
     - string
     - Required
     - Unique identifier for the :ref:`project <projects>`
       for which you want to retrieve a private endpoint.

   * - {CLOUD-PROVIDER}
     - string
     - Required
     - Cloud provider for which you want to retrieve a private endpoint.
       |service| accepts **AWS**, **AZURE**, or **GCP**.

   * - {ENDPOINT-SERVICE-ID}
     - string
     - Required
     - Unique identifier of the private endpoint service for which you
       want to retrieve a private endpoint.

   * - {ENDPOINT-ID}
     - string
     - Required
     - Unique identifier of the private endpoint you want to retrieve.

       The format of the **{ENDPOINT-ID}** parameter differs for |aws|, |azure|,
       and |gcp|:

       .. tabs-cloud-providers::

          .. tab::
             :tabid: aws

             .. code-block:: html
                :copyable: false

                vpce-09af09a5ae5826fc3

          .. tab::
             :tabid: azure

             .. code-block:: html
                :copyable: false

                %2Fsubscriptions%2F00000000-0000-0000-0000-000000000000%2FresourceGroups%2Fprivatelink%2Fproviders%2FMicrosoft.Network%2FprivateEndpoints%2Ftest

             .. important::

                You must URL encode the **{ENDPOINT-ID}** for |azure|
                private endpoints.

          .. tab::
             :tabid: gcp
      
             .. code-block:: html
                :copyable: false
      
                0938bccb3948baabb2384059
