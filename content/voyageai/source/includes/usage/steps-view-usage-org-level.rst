.. procedure:: 
   :style: normal 

   .. step:: Log in to `{+service+} <https://cloud.mongodb.com>`__.

   .. step:: Go to the :guilabel:`Usage` page in the {+atlas-ui+}.

      a. If it's not already displayed, select your desired organization
         from the :icon-mms:`office` :guilabel:`Organizations` menu in 
         the navigation bar.
      #. At the organization level, click :guilabel:`Usage`
         under the :guilabel:`AI Models` header in the navigation bar.

   .. step:: Review usage metrics.

      The :guilabel:`Usage` page displays usage metrics for all 
      {+model-api-keys+} in your organization. The page provides 
      the following tabs:

      - :guilabel:`Usage Activity`: Shows detailed usage metrics and 
        charts for your organization.
      - :guilabel:`Free Usage`: Shows free quota consumption by model.

      To learn more, see :ref:`voyage-usage-page-features`.
