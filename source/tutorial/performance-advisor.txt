.. only:: cloud

   .. note::
      
      You must meet the following requirements to view the Performance
      Advisor:

      * Be a Cloud Premium user

      * Run MongoDB version 3.2+ on your cluster

      * Use Automation Agent version 4.5.7+ to manage your cluster

      In addition, users without the :ref:`Data Access Admin <data-access-admin-role>`
      project role cannot view values in example queries. See :doc:`Install
      or Update the Automation Agent</tutorial/nav/install-automation-agent>`
      for upgrade instructions for the Automation Agent.
