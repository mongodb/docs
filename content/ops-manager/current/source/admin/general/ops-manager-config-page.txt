==================
|mms| Config Page
==================

.. meta::
   :description: Update Ops Manager settings including email and SMS notification providers, backup configuration, and other system-level options through the General tab Config page.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

The :guilabel:`General` tab's :guilabel:`Ops Manager Config` page
lets you update your |onprem| settings. You can configure email
and |sms| notification providers, backup settings, and other
system-level options from this page. For information on each
field, see :ref:`conf-mms.properties`.
