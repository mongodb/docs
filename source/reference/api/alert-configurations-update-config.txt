Resource
--------

.. code-block:: none

   PUT /groups/{PROJECT-ID}/alertConfigs/{ALERT-CONFIG-ID}

.. note::

   To update just the enabled/disabled state for the alert
   configuration, see
   :doc:`/reference/api/alert-configurations-enable-disable-config`
   instead.

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

All request path parameters are *required*.

.. list-table::
   :widths: 20 20 60
   :header-rows: 1
   :stub-columns: 1

   * - Parameter
     - Type
     - Description

   * - PROJECT-ID
     - string
     - Unique identifier for this Project.

   * - ALERT-CONFIG-ID
     - string
     - Unique identifier for this alert configuration.

Example Request
---------------

.. only:: cloud

   .. code-block:: sh

      curl --user '{PUBLIC-KEY}:{PRIVATE-KEY}' --digest \
       --header 'Accept: application/json' \
       --header 'Content-Type: application/json' \
       --include \
       --request PUT "https://cloud.mongodb.com/api/public/v1.0/groups/{GROUP-ID}/alertConfigs/{ALERT-CONFIG-ID}" \

         {
           "eventTypeName" : "RESYNC_REQUIRED",
