.. setting:: mms.ldap.user.searchAttribute

   *Type*: string

   
   |ldap| field used for the |ldap| search. This is typically a
   username or an email address. The value of this field is also used
   as the |mms| username.
   
   .. code-block:: ini
   
      mms.ldap.user.searchAttribute=<myAccountName>
   
   
   Corresponds to :setting:`LDAP User Search Attribute`.
