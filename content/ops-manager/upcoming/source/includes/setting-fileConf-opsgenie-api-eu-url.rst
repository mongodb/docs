.. setting:: opsgenie.api.url

   *Type*: string

   *Default*: ``https://api.opsgenie.com/v2/alerts``

   
   |url| that |mms| uses to access the Ops Genie |api| in European countries. 
   
   Enable this custom parameter if your |onprem| instance runs in Europe.
   Then, set its value to ``https://api.eu.opsgenie.com/v2/alerts``.
   
   For more information, see 
   `Opsgenie Alert documentation <https://docs.opsgenie.com/docs/alert-api#:~:text=If%20using%20the%20EU%20instance,for%20requests%20to%20be%20successful.>`__.
