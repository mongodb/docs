========================
Test |onprem| Monitoring
========================

.. default-domain:: mongodb

Overview
--------

The following procedure creates a MongoDB :term:`replica set` and sets up
a database populated with random data
for use in testing |a-mms| installation. Create the replica set on a separate
machine from |mms|.

These instructions create the replica set on a server running **RHEL 6+** or
**Amazon Linux**. The procedure installs all the members to one server.
