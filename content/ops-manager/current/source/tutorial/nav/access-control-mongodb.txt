==============================
Manage MongoDB Users and Roles
==============================

.. meta::
   :description: Enable MongoDB access control and manage database users and custom roles directly from the Ops Manager interface for replica set and sharded cluster deployments.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

You can enable MongoDB access control and manage MongoDB users and
roles directly from the |mms| interface. Enabling access control
requires all clients to authenticate before connecting to your
MongoDB deployments. Use the pages in this section to create and
manage users and custom roles for your projects.
