.. _recent-queries:

===================
View Recent Queries
===================

.. default-domain:: mongodb

You can view up to 20 of the most recent queries for that collection.
Click a query in the list to populate the query bar with that query.

.. figure:: /images/compass/query-history-select.png
   :alt: Compass query history

You can also hover over a query and perform the following actions:

.. list-table::
   :widths: 10 90
   :header-rows: 1

   * - Click
     - Action

   * - :icon-fa5:`star`
     - Save the query as a :ref:`favorite <favorite-queries>`. Specify
       a name for the query and click :guilabel:`Save`.

   * - :icon-fa5:`copy`
     - Copy the query to the clipboard.

   * - :icon-fa5:`trash-alt`
     - Remove the query from the :guilabel:`Recent` list.

.. seealso::

   To learn how to write queries to filter your data, see
   :ref:`query-bar`.
