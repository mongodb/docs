===============================================
Install the MongoDB Agent to Manage Deployments
===============================================

.. default-domain:: mongodb

.. _automation-agent-install-steps:
.. _mongodb-agent-install-steps:
.. _mongodb-agent-install-from-package:
.. _mongodb-agent-install-from-archive:

You have a project and want to install the {+mdbagent+} to manage your
MongoDB deployments. You can also monitor and back up your MongoDB
deployments following this workflow.

.. note:: Review the Prerequisites First

   Please review the :doc:`/tutorial/install-mongodb-agent-prereq`
   before installing the {+mdbagent+}.
