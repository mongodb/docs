==============
Database Users
==============

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: twocols

.. include:: /includes/fact-projects-groups-synonyms.rst

.. include:: /includes/api-base-url.rst

The ``databaseUsers`` resource lets you retrieve, create and modify the 
database users in your cluster. Each user has a set of roles
that provide access to the project's databases. A user's roles apply to
all the clusters in the project: if two clusters have a ``products``
database and a user has a role granting ``read`` access on the
``products`` database, the user has that access on both clusters.

.. note::

   If a database user is assigned a
   :ref:`custom role <atlas-user-privileges>`,
   they cannot be assigned any other roles.

Each user also has a username, password, and
:ref:`authentication database <authentication-database>` used to log
into MongoDB. |service| deployments of MongoDB use ``admin`` as the
authentication database for all users. The authentication database
does not determine a user's roles.

The ``databaseUsers`` resource supports creating temporary database
users that automatically expire within a user-configurable 7-day period.

The ``databaseUsers`` resource requires your :ref:`group-id`.

.. list-table::
   :header-rows: 1
   :widths: 10 70 15

   * - Method
     - Endpoint
     - Description

   * - ``GET``

     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/databaseUsers </reference/api/database-users-get-all-users>`

     - Get all users in the project.

   * - ``GET``

     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/databaseUsers/admin/{USERNAME} </reference/api/database-users-get-single-user>`

     - Get a single user in the project.

   * - ``POST``

     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/databaseUsers </reference/api/database-users-create-a-user>`

     - Create a user for the project.

   * - ``PATCH``

     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/databaseUsers/admin/{USERNAME} </reference/api/database-users-update-a-user>`

     - Update a user for the project.

   * - ``DELETE``

     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/databaseUsers/admin/{USERNAME} </reference/api/database-users-delete-a-user>`

     - Delete a user for the project.
