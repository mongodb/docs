=========
$geometry
=========

.. default-domain:: mongodb

.. operator:: $geometry

   .. versionadded:: 2.4

   The :operator:`$geometry` operator specifies a :term:`GeoJSON` for a
   geospatial query operators. For details on using
   :operator:`$geometry` with an operator, see the operator:

   - :operator:`$geoWithin`

   - :operator:`$geoIntersects`

   - :operator:`$near`
