.. _mcli-atlas-dbuser-list-command:

==========================
mongocli atlas dbuser list
==========================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``dbuser list`` command retrieves the list of MongoDB database users 
on the specified |service| cluster. You can also retrieve the MongoDB 
database users through the |service| :atlas:`API 
<reference/api/database-users-get-all-users/>`.

.. _dbuser-list-syntax:

Syntax
------

.. code-block:: text

   mongocli atlas dbuser list     
        [ --profile|-P <profile-name> ]
        [ --projectId <project-ID> ]

.. include:: /includes/fact-command-line-help.rst

.. _dbuser-list-options:

Options
-------

.. list-table::
   :header-rows: 1
   :widths: 20 10 60 10

   * - Option 
     - Type
     - Description
     - Required?

   * - ``--profile``, ``-P``
     - string
     - Name of the profile where the public and private 
       keys for the project are saved. If omitted, uses the 
       ``default`` profile. To learn more about creating a 
       profile, see :ref:`mcli-configure`.
     - no

   * - ``--projectId``
     - string
     - Unique identifier of the project that contains the 
       cluster to update. If omitted, uses the project ID in 
       the profile or :ref:`environment variable <mcli-env-var>`.
     - no

.. _dbuser-list-examples:

Examples 
--------

The following example uses the ``mongocli atlas dbuser list`` command 
to retrieve the list of MongoDB database users on the |service| cluster. 
The command uses the profile named ``egAtlasProfile``, which contains 
the |svc-api-key|\s and the project ID.

.. code-block:: sh
   :copyable: false

   ~ mongocli atlas dbuser list --profile egAtlasProfile
