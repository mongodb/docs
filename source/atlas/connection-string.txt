Overview
--------

.. short-description::
   
   A brief description or preview of what to expect in this
   guide for creating a MongoDB account.

In this guide, you will learn how to set up your Atlas cluster for
connections from a client by perfomring the following actions:

- Whitelist your IP address to allow access to your Atlas cluster.
- Locate your connection string and copy it.

.. time:: 15

What You'll Need
----------------

- A MongoDB account. See :ref:`create-and-manage-account`.

- An Atlas cluster. See `Create a New Cluster 
  <https://docs.atlas.mongodb.com/tutorial/create-new-cluster/>`__.

- A MongoDB user. See `Configure MongoDB Users
  <https://docs.atlas.mongodb.com/security-add-mongodb-users/>`_
  in the Atlas documentation.
