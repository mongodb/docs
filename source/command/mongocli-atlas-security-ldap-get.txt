Get an LDAP configuration.

Syntax
------

.. code-block::

   mongocli atlas security ldap get [options]

-------

   :header-rows: 1
   :widths: 20 10 10 60

     - Type

     - Description

     - 
     - false

     - string
     - false

       Valid values: json|json-path|go-template|go-template-file

     - false
