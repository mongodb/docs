===============
Network Peering
===============

.. default-domain:: mongodb

.. include:: /includes/fact-projects-groups-synonyms.rst

Base URL: ``https://cloud.mongodb.com/api/atlas/v1.0``
------------------------------------------------------

Endpoints
---------

The following lists the endpoints available for
:doc:`Network Peering </security-vpc-peering>`.

.. list-table::
   :header-rows: 1
   :widths: 10 50 40

   * - Method
     - Endpoint
     - Description

   * - ``GET``

     - :doc:`/groups/{GROUP-ID}/containers </reference/api/vpc-get-containers-list>`

     - Get details for all network peering containers in an |service|
       project for a single cloud provider.

   * - ``GET``

     - :doc:`/groups/{GROUP-ID}/containers/{CONTAINER-ID} </reference/api/vpc-get-container>`

     - Get details for one network peering container in an |service| project.

   * - ``POST``

     - :doc:`/groups/{GROUP-ID}/containers </reference/api/vpc-create-container>`

     - Create one new network peering container in an |service| project.

   * - ``PATCH``

     - :doc:`/groups/{GROUP-ID}/containers/{CONTAINER-ID} </reference/api/vpc-update-container>`

     - Update details for one network peering container in an |service| project.

   * - ``DELETE``

     - :doc:`/groups/{GROUP-ID}/containers/{CONTAINER-ID} </reference/api/vpc-delete-one-container>`

     - Remove one network peering container in an |service| project.

   * - ``GET``

     - :doc:`/groups/{GROUP-ID}/peers </reference/api/vpc-get-connections-list>`

     - Get details for all network peering connections in an |service| project.

   * - ``GET``

     - :doc:`/groups/{GROUP-ID}/peers/{PEER-ID} </reference/api/vpc-get-connection>`

     - Get details for one network peering connection in an |service| project.

   * - ``POST``

     - :doc:`/groups/{GROUP-ID}/peers </reference/api/vpc-create-peering-connection>`

     - Create a new network peering connection in an |service| project.

   * - ``PATCH``

     - :doc:`/groups/{GROUP-ID}/peers/{PEER-ID} </reference/api/vpc-update-peering-connection>`

     - Update a new network peering connection in an |service| project.

   * - ``DELETE``

     - :doc:`/groups/{GROUP-ID}/peers/{PEER-ID} </reference/api/vpc-delete-peering-connection>`

     - Delete one network peering connection in an |service| project.

   * - ``GET``

     - :doc:`/groups/{GROUP-ID}/privateIpMode </reference/api/get-private-ip-mode-for-project>`

     - Verify if an |service| project is in :guilabel:`Connect via Peering Only` mode.

       .. note::

          :guilabel:`Connect via Peering Only` is required to use Network Peering
          with |gcp| and |azure|.

   * - ``PATCH``

     - :doc:`/groups/{GROUP-ID}/privateIpMode </reference/api/set-private-ip-mode-for-project>`

     - Enable or disable :guilabel:`Connect via Peering Only` mode for one |service| project.

       .. note::

          :guilabel:`Connect via Peering Only` is required to use network peering with
          |gcp| and |azure|.
