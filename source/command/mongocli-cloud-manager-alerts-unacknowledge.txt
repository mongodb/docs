===========================================
mongocli cloud-manager alerts unacknowledge
===========================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Unacknowledges one alert for the specified project.

Syntax
------

.. code-block::

   mongocli cloud-manager alerts unacknowledge <alertID> [options]

Arguments
---------

.. list-table::
   :header-rows: 1
   :widths: 20 10 10 60

   * - Name
     - Type
     - Required
     - Description
   * - alertId
     - string
     - false
     - ID of the alert you want to unacknowledge.

Options
-------

.. list-table::
   :header-rows: 1
   :widths: 20 10 10 60

   * - Name
     - Type
     - Required
     - Description
   * - --comment
     - string
     - false
     - Optional description or comment for the entry.
   * - -h, --help
     - 
     - false
     - help for unacknowledge
   * - -o, --output
     - string
     - false
     - Output format.
       Valid values: json|json-path|go-template|go-template-file
   * - --projectId
     - string
     - false
     - Project ID to use. Overrides configuration file or environment variable settings.

Inherited Options
-----------------

   :header-rows: 1

     - Type
     - Required
