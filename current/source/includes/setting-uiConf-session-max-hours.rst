.. setting:: Session Max Hours

   *Type*: number

   
   Number of hours before a session on the |mms| website expires.
   
   .. note::
     
      :guilabel:`Session Max Hours` reflects the total |mms| session time, not
      just idle time. Both active and idle sessions expire when 
      :guilabel:`Session Max Hours` elapses.
   
   Set this value to ``0`` to use browser session cookies only.
   
   Corresponds to :setting:`mms.session.maxHours`.
