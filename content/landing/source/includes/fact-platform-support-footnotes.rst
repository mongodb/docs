
.. [#oracle-linux] 

  MongoDB only supports Oracle Linux running the  Red Hat Compatible
  Kernel (RHCK). MongoDB does **not** support the Unbreakable
  Enterprise Kernel (UEK).

.. [#RockyAlmaLinux-support-install]

  MongoDB on-premises products released for RHEL version 8.0+ are 
  compatible with and supported on Rocky Linux version 8.0+ and 
  AlmaLinux version 8.0+, contingent upon those distributions meeting their 
  obligation to deliver full RHEL compatibility.

.. [#SLES12-SP-info]

  MongoDB versions 5.0 and greater are tested against SLES 12 service 
  pack 5.

.. [#SLES15-SP-info]

  MongoDB versions 7.0 and later are tested against SLES 15 service pack 4. 

.. [#RHEL7-compat]

  MongoDB version 7.0 is built and tested against RHEL 7.9. Earlier versions
  of MongoDB are tested against RHEL 7 and assume forward compatibility.

.. important::

   The platform support matrix above applies to the operating system
   version present at the time of installation. If you upgrade the
   host operating system to a version not listed above after
   installing MongoDB, the resulting configuration is unsupported.
   MongoDB does not guarantee correct functionality or behavior when
   running on an unsupported operating system.
