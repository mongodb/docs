======================
Project IP Access List
======================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: twocols

.. include:: /includes/styles/corrections.rst

The **Access List** endpoint manages a |service| project's
:ref:`IP Access List <access-list>`.

The **Access List** endpoint supports creating temporary Access List
entries that automatically expire within a user-configurable 7-day
period.

Endpoints
---------

.. include:: /includes/api-base-url-new.rst

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``

     - :doc:`/groups/{PROJECT-ID}/accessList </reference/api/ip-access-list/get-all-access-list-entries>`

     - Get all access list entries in the project associated to
       **{PROJECT-ID}**.

   * - ``GET``

     - :doc:`/groups/{PROJECT-ID}/accessList/{ACCESS-LIST-ENTRY} </reference/api/ip-access-list/get-one-access-list-entry>`

     - Get the access list entry specified to **{ACCESS-LIST-ENTRY}**
       from the project associated to **{PROJECT-ID}**.

   * - ``POST``

     - :doc:`/groups/{PROJECT-ID}/accessList </reference/api/ip-access-list/add-entries-to-access-list>`

     - Add one or more access list entries to the project
       associated to **{PROJECT-ID}**.

   * - ``DELETE``

     - :doc:`/groups/{PROJECT-ID}/accessList/{ACCESS-LIST-ENTRY} </reference/api/ip-access-list/delete-one-access-list-entry>`

     - Delete the access list entry specified to
       **{ACCESS-LIST-ENTRY}** from the project associated to
       **{PROJECT-ID}**.
