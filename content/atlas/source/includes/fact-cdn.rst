|service| uses a {+cdn+} to serve content quickly. If your
organization uses a firewall, add the following |service| {+cdn+}
hosts to the firewall's allow list to prevent issues accessing the
{+atlas-ui+}:

- ``https://assets.mongodb-cdn.com/``
- ``https://downloads.mongodb.com/`` (required for the
  {+data-explorer+} feature)
