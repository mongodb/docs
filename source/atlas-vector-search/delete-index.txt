.. _avs-delete-index:

===================================
Delete an {+avs+} Index
===================================

.. default-domain:: mongodb

.. meta:: 
   :keywords: atlas vector search, delete an atlas vector search index, delete my atlas vector search index, remove an atlas vector search index, remove my atlas vector search index, delete an atlas vector search index in the atlas ui
   :description: Learn how to remove your Atlas Vector Search index through the Atlas UI.

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

You can delete an {+avs+} index at any time from the {+atlas-ui+} and
the {+atlas-admin-api+}. 

Required Access
---------------

You must have the :authrole:`Project Search Index Editor` or higher role to 
delete an {+avs+} index.

.. _avs-create-index-supported-clients:

Supported Clients 
-----------------

You can delete an {+avs+} index using the following: 

- {+atlas-ui+}
- {+atlas-admin-api+} :oas-atlas-op:`Remove One Atlas Search Index
  <deleteAtlasSearchIndex>` endpoint
- {+atlas-cli+} v1.14.3 :atlascli:`atlas clusters search indexes delete
  </command/atlas-clusters-search-indexes-delete/>` command on both the
  :atlascli:`cloud deployment </atlas-cli-local-cloud/>` and
  :atlascli:`local deployment </atlas-cli-deploy-local>`
- {+mongosh+} v2.1.2 or later
  :method:`db.collection.createSearchIndex()` method

Procedure
---------

.. tabs:: 

   .. tab:: Atlas UI

   .. tab:: Atlas Administration API 
      :tabid: api 

      To delete an {+avs+} index for a collection using the
      {+atlas-admin-api+}, send a ``DELETE`` request to the |fts|
      ``indexes`` endpoint with either the â‰ˆ ID or name of the
      index to delete. To learn more about the syntax and parameters for
      the endpoint, 
      :oas-atlas-op:`Delete One By Name </deleteAtlasSearchIndexByName>` and
      :oas-atlas-op:`Delete One By ID </deleteAtlasSearchIndex>`.

   .. tab:: Atlas CLI 
      :tabid: atlascli 

      To delete an {+avs+} index for a collection using {+atlas-cli+},
      run the following command: 

      .. code-block:: shell 
         :copyable: true
      
         atlas clusters search indexes delete [indexId] [options]
      
      To learn more about the command syntax and parameters, see the
      {+atlas-cli+} documentation for :atlascli:`atlas clusters search
      indexes delete </command/atlas-clusters-search-indexes-delete/>`
      command. 

   .. tab:: mongosh 
      :tabid: mongosh 

      To delete an {+avs+} index for a collection using {+mongosh+}, do
      the following:
      
      1. :ref:`Connect <connect-mongo-shell>` to the |service|
         {+cluster+} using {+mongosh+}.   
      #. Switch to the database that contains the collection.
      #. Run the :method:`db.collection.dropSearchIndex()` method. 

         The :method:`db.collection.dropSearchIndex()` method has the
         following syntax: 

         .. code-block:: shell 
            :copyable: true 
            :linenos: 

            db.movies.dropSearchIndex( "<index-name>" );
