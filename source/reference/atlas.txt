.. _mcli-reference-atlas:

===========================
Atlas ``mongocli`` Commands
===========================

.. default-domain:: mongodb

:ref:`alert <mcli-reference-atlas-alert>`
  Manage |service| :atlas:`alerts and alert configurations </alerts>`.

:ref:`backup <mcli-reference-atlas-backup>`
  Manage backup and restore jobs.

:ref:`cluster <mcli-reference-atlas-cluster>`
  Manage your |service| cluster configurations.

:ref:`datalake <mcli-reference-atlas-datalake>`
  Manage `Data Lakes <https://docs.mongodb.com/datalake/>`__ associated 
  with an |service| project.

:ref:`dbuser <mcli-reference-atlas-dbuser>`
  Manage MongoDB database users.

:ref:`event <mcli-reference-atlas-event>`
  List events for a specified project or organization.

:ref:`log <mcli-reference-atlas-log>`
  Retrieve various log types for a specified host.

:ref:`metric <mcli-reference-atlas-metric>`
  List metrics for MongoDB processes on a specified host.

:ref:`online archive <mcli-reference-atlas-onlinearchive>`
  Manage :atlas:`online archives </online-archive/manage-online-archive/>` 
  for a cluster.

:ref:`process <mcli-reference-atlas-process>`
  Retrieve information about MongoDB processes running on a specified
  |service| project.

:ref:`whitelist <mcli-reference-atlas-whitelist>`
  Manage :atlas:`whitelist </security-whitelist/>` entries.
