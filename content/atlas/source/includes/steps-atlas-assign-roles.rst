.. procedure::
   :style: normal
      
   .. include:: /includes/nav/steps-project-access-manager.rst
      
   .. step:: Edit |a-service| user or team.

      - To edit |a-service| user, click the :guilabel:`Users` tab.
      - To edit a team, click the :guilabel:`Teams` tab.
      
   .. step:: Update permissions.
      
      In the :guilabel:`Actions` column, click :guilabel:`Edit
      Permissions` next to the |service| user or team whose permissions you
      want to modify.
      
   .. step:: Select the appropriate role or roles for the |service| user or team.
      
   .. step:: Click :icon-fa5:`check` to save.
