.. _mcli-reference-cm:

===================================
Cloud Manager ``mongocli`` Commands
===================================

.. default-domain:: mongodb

:ref:`agent <mcli-reference-cm-agent>`
  Manage agents installed on your |cloud-short| deployments.

:ref:`alert <mcli-reference-cm-alert>`
  Lists |cloud-short| :cloudmgr:`alerts and alert configurations
  </tutorial/nav/alerts/>`.

:ref:`automation <mcli-reference-cm-automation>`
  Retrieves the latest plan for MongoDB processes in the specified
  |cloud-short| project.

:ref:`backup <mcli-reference-cm-backup>`
  Manages backup and restore jobs.

:ref:`cluster <mcli-reference-cm-cluster>`
  Provides access to your cluster configurations.

:ref:`dbuser <mcli-reference-cm-dbuser>`
  Manages MongoDB database users.

:ref:`event <mcli-reference-cm-event>`
  Lists events for a specified project or organization.

:ref:`maintenance window <mcli-reference-om-maintenanceWindow>`
  Manage your |cloud-short| :cloudmgr:`maintenance window
  </reference/api/maintenance-windows>`.

:ref:`metric <mcli-reference-cm-metric>`
  Lists metrics for MongoDB processes on a specified host.

:ref:`process <mcli-reference-cm-process>`
  Retrieves information about MongoDB processes running on a specified
  |cloud| project.

:ref:`security <mcli-reference-cm-security>`
  Enables an :cloudmgr:`authentication mechanism
  </tutorial/enable-mongodbcr-authentication-for-group/>` for the
  specified |cloud-short| clusters in the given project.

:ref:`servers <mcli-reference-cm-server>`
  Retrieves all hosts that are running an Automation Agent in the
  specified |cloud-short| project.
