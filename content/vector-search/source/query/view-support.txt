Use Views with {+avs+}
====================================

.. default-domain:: mongodb

.. meta::
   :description: Use MongoDB Vector Search with views to transform documents and collections, enabling vector search queries on aggregation pipeline results and joined data.

.. |product-name| replace:: {+avs+}

You can create a {+avs+} index on a :ref:`View <views-landing-page>` to transform documents and collections so that you can
partially index a collection, support incompatible data types or
data models, and more.

The following examples use the :ref:`sample_mflix <sample-mflix>`
sample database.

   Example: Filter Documents
   -------------------------

   To create a View, you must have the
   :authaction:`createCollection` privilege.

   You can filter documents to partially index a collection. The
   following example creates a View on the
   ``sample_mflix.embedded_movies`` collection so that only documents
   that have a vector embedding field are indexed.
