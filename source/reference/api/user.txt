===============
|service| Users
===============

.. default-domain:: mongodb

.. include:: /includes/fact-projects-groups-synonyms.rst

The ``users`` resource allows you to create new |service| users and
assign them to any |service| project for which you have the
:authrole:`Project Owner` role.

You can also modify your own |service| profile.
If you are the owner of an |service| organization or project,
you can also update the :ref:`user roles <user-roles>` for any
user with membership in that organization or project.

.. include:: /includes/api-base-url-new.rst

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :doc:`/users/{USER-ID} </reference/api/user-get-by-id>`
     - Retrieves the profile for the specified |service| user with
       the unique user ID ``{USER-ID}``.

   * - ``GET``
     - :doc:`/users/byName/{USER-NAME}
       </reference/api/user-get-one-by-name>`
     - Retrieves the profile for the specified |service| user with
       username ``{USER-NAME}``.

   * - ``POST``
     - :doc:`/users/ </reference/api/user-create>`
     - Creates a |service| user and assigns it to one or more of your
       |service| organizations or projects.

   * - ``PATCH``
     - :doc:`/users/{USER-ID} </reference/api/user-update>`
     - Updates the profile for the specified |service| user with
       the unique ID ``{USER-ID}``.
