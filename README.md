# docs-tutorials

[Tutorials site for MongoDB products and
services](https://docs.mongodb.com/tutorials/).

## Installation Instructions

Before running a local instance of the tutorial site you'll need to install the
following dependencies:

    brew install hugo mmark node

## Building Instructions

After you've installed the necessary dependencies you can start a local
instance of the tutorial site by running:

    make server

You can then point your browser to
[http://localhost:1313/](http://localhost:1313/).
