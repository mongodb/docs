.. include:: /includes/api-requirements.rst

Considerations
~~~~~~~~~~~~~~

-  When you create a private endpoint connection, |service| creates a
   :doc:`network container
   </reference/api/vpc-get-containers-list-all/>` in the project for the
   cloud provider for which you create the private endpoint connection
   if one does not already exist.
