======
logout
======

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

.. dbcommand:: logout

   The :dbcommand:`logout` command terminates the current
   authenticated session:

   .. code-block:: javascript

      { logout: 1 }

   .. note::

      If you're not logged in and using authentication,
      :dbcommand:`logout` has no effect.
