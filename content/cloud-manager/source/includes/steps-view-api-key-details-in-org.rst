   .. step:: Click :guilabel:`API Keys`.

   .. step:: Navigate to :guilabel:`View Details`.
      
      a. Next to the |api| Key, click :icon-mms:`ellipsis`.
      
      b. Click :guilabel:`View Details`.
      
         The :guilabel:`<Public Key> API Key Details` modal displays the:
      
         - Obfuscated :guilabel:`Private Key`.
         - Date the Key was last used.
         - Date the Key was created.
         - |ipv4| addresses on which the key is in the access list.
         - Projects to which the Key has been granted access.
