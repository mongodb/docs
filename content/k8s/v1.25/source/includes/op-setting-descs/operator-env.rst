Label for the |k8s-op-short|\s deployment environment. This value
affects the default timeouts and the logging level and format:

.. list-table:: 
    :header-rows: 1
    :stub-columns: 1

    * - If the value is
      - Log Level is set to
      - Log Format is set to
    * - ``dev``
      - debug
      - text
    * - ``prod``
      - info
      - json

The default value is **prod**.
