.. list-table::
   :widths: 30,70
   :header-rows: 1

   * - Name

     - Description

   * - :dbcommand:`authenticate`

     - Starts an authenticated session using a username and password.

   * - :dbcommand:`authSchemaUpgrade`

     - Supports the upgrade process for user data between version 2.4 and 2.6.

   * - :dbcommand:`copydbgetnonce`

     - This is an internal command to generate a one-time password for use with the :dbcommand:`copydb` command.

   * - :dbcommand:`getnonce`

     - This is an internal command to generate a one-time password for authentication.

   * - :dbcommand:`logout`

     - Terminates the current authenticated session.
