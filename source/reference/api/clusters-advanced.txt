===================
Clusters (Advanced)
===================

.. default-domain:: mongodb

.. include:: /includes/fact-projects-groups-synonyms.rst

The ``clusters`` resource provides access to your cluster
configurations. The resource lets you create, edit and delete clusters.
The resource requires your :ref:`project-id`.

.. important::

   Changes to cluster configurations can affect costs. Before making
   changes, please see :doc:`/billing`.

.. include:: /includes/api/base-api-uri-1-5.rst

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 40 50

   * - Method
     - Endpoint
     - Description

   * - ``GET``

     - :doc:`/groups/{GROUP-ID}/clusters </reference/api/cluster-advanced/get-all-cluster-advanced>`

     - Get all clusters for the specified project.

   * - ``GET``

     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME} </reference/api/cluster-advanced/get-one-cluster-advanced>`

     - Get one cluster in the specified project.

   * - ``POST``

     - :doc:`/groups/{GROUP-ID}/clusters </reference/api/cluster-advanced/create-one-cluster-advanced>`

     - Create one cluster in the specified project.

   * - ``PATCH``

     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME} </reference/api/cluster-advanced/modify-one-cluster-advanced>`

     - Update one cluster in the specified project.
