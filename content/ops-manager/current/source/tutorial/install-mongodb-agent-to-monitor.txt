===============================================================
Install the MongoDB Agent to Only Monitor or Backup Deployments
===============================================================

.. default-domain:: mongodb

.. _automation-agent-install-steps:
.. _automation-agent-install-steps-monitor:
.. _mongodb-agent-install-steps:
.. _mongodb-agent-install-from-package:
.. _mongodb-agent-install-from-archive:

You have a project and want to install the {+mdbagent+} to monitor
and/or back up your MongoDB deployments. You are choosing not to manage
your MongoDB deployments at this time.

.. note:: Review the Prerequisites First

   Please review the :doc:`/tutorial/install-mongodb-agent-prereq`
   before installing the {+mdbagent+}.
