=================
View Agent Status
=================

.. default-domain:: mongodb


Click :guilabel:`Deployment`, then the :guilabel:`Agents` tab, then :guilabel:`All Agents`.
|mms| displays the following information about your agents:

.. list-table::
   :header-rows: 1
   :widths: 30 70

   * - Field

     - Description

   * - :guilabel:`Last Conf Call`

     - The number of seconds since the last time the agent polled |mms|.

   * - :guilabel:`Type`

     - The type of agent.

   * - :guilabel:`Hostname`

     - The hostname for the agent and any warnings, such as that the
       agent is down or out-of-date.

   * - :guilabel:`State`

     - Indicates whether the agent is active.

   * - :guilabel:`Conf Count`

     - The number of times the agent has polled |mms| for the latest
       configuration since the agent was restarted.

   * - :guilabel:`Ping Count`

     - (Monitoring Agent only) The number of times that the agent sent diagnostic
       data about the MongoDB deployment since midnight GMT. Typically, the
       agent sends the data every minute.

   * - :guilabel:`Version`

     - The version of the agent software running on this agent instance.

   * - :guilabel:`Log`

     - Click :guilabel:`view` to view the agent's log.
