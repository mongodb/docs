==========================
replSetAbortPrimaryCatchUp
==========================

.. meta::
   :description: Force the elected primary in a replica set to abort sync and complete the transition to primary using the `replSetAbortPrimaryCatchUp` command.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Definition
----------

.. dbcommand:: replSetAbortPrimaryCatchUp

   The ``replSetAbortPrimaryCatchUp`` command forces the elected
   :term:`primary` member of the replica set to abort sync (catch up)
   then complete the transition to primary.

-------------

This command is available in deployments hosted in the following environments:

.. include:: /includes/fact-environments-atlas-support-all.rst

Syntax
------

The command has the following syntax:

.. code-block:: javascript
   
   db.runCommand(
      { 
        replSetAbortPrimaryCatchUp: 1 
      }
   )
