Resource
--------

.. code-block:: http

   GET /orgs/{ORG-ID}

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :widths: 20 20 60
   :header-rows: 1
   :stub-columns: 1

   * - Path Element
     - Type
     - Description

   * - ORG-ID
     - string
     - *(Required.)* The unique identifier for the :ref:`organization <organizations>` 
       whose information you want to retrieve.

Example Request
---------------

.. only:: cloud

   .. code-block:: sh

     curl --user '{USERNAME}:{APIKEY}' --digest \ 
      --header 'Accept: application/json' \ 
      --include \ 
      --request GET "https://cloud.mongodb.com/api/public/v1.0/orgs/{ORG-ID}/"

.. only:: onprem

   .. code-block:: sh

     curl --user '{USERNAME}:{APIKEY}' --digest \ 
      --header 'Accept: application/json' \ 
      --include \ 
      --request GET "https://{OPSMANAGER-HOST}:{PORT}/api/public/v1.0/orgs/{ORG-ID}/"
