====================================
Optional: Install On-Prem MMS Backup
====================================

.. default-domain:: mongodb

Overview
--------

To install the optional MMS Backup components you must first install the configure the
MMS Backup Daemon Server and the MMS Backup Blockstore Database. [SAM_LINK_TO_HARDWARE_REQUIREMENTS].

Getting Started
---------------

We recommend contacting your MongoDB Account Manager to arrange a guided install of the MMS
Backup components. A MongoDB representative can assist you sizing the components and installing
the software.
