===========================================
Update Regionalized Private Endpoint Status
===========================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

.. include:: /includes/styles/corrections.rst

.. include:: /includes/fact-projects-groups-synonyms.rst

Enable or disable the regionalized private endpoint setting for one
|service| project.

.. include:: /includes/admonitions/warnings/regionalized-pls-change-connection-strings.rst

.. include:: /includes/api-requirements.rst

Required Roles
--------------

You must have the :authrole:`Project Owner` role for the
project to successfully call this resource.

Request
-------

.. include:: /includes/api-base-url.rst

.. code-block:: http

   PATCH /groups/{GROUP-ID}/privateEndpoint/regionalMode

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :stub-columns: 1
   :widths: 20 14 11 55

   * - Path Parameter
     - Type
     - Necessity
     - Description

   * - {GROUP-ID}
     - string
     - Required
     - Unique identifier for the :ref:`project <projects>`
       for which you want to enable or disable the regionalized private
       endpoint setting.

Request Query Parameters
~~~~~~~~~~~~~~~~~~~~~~~~

.. include:: /includes/api/base-query-single-parameters.rst

Request Body Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :stub-columns: 1
   :widths: 20 14 11 55

   * - Path Parameter
     - Type
     - Necessity
     - Description

   * - enabled
     - boolean
     - Required
     - Flag that indicates whether the regionalized private endpoint
       setting is enabled for one |service| project.

       Set this value to **true** to create more than one private
       endpoint in a cloud provider region to connect to
       multi-region and global |service| sharded clusters.

       .. include:: /includes/admonitions/warnings/regionalized-pls-change-connection-strings.rst

       .. include:: /includes/enable-regionalized-privatelink.rst

       Set this value is **false** to disable this setting.
