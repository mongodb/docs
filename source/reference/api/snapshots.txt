===========================
Continuous Backup Snapshots
===========================

.. default-domain:: mongodb

.. include:: /includes/admonitions/deprecated-continuous-backup.rst

.. include:: /includes/fact-projects-groups-synonyms.rst

The ``snapshots`` resource enables you to view snapshot metadata, edit
snapshot expiration dates, and remove existing snapshots. A snapshot
is a backup of your data captured at a specific point in time.

.. include:: /includes/fact-continuous-backup-only-resource.rst

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/snapshots </reference/api/snapshots-get-all>`
     - Get all snapshots for the specified cluster.

   * - ``GET``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/snapshots/{SNAPSHOT-ID} </reference/api/snapshots-get-one>`
     - Get the specified snapshot for the cluster.

   * - ``PATCH``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/snapshots/{SNAPSHOT-ID} </reference/api/snapshots-change-expiration>`
     - Change the expiration date for the specified snapshot.

       .. include:: /includes/fact-api-whitelist-required.rst

   * - ``DELETE``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/snapshots/{SNAPSHOT-ID} </reference/api/snapshots-delete-one>`
     - Delete the specified snapshot.

       .. include:: /includes/fact-api-whitelist-required.rst
