.. _activity-feed:

==================
View Activity Feed
==================

.. meta::
   :keywords: atlas cli, atlas api, atlas ui
   :description: Access and filter activity feeds in Atlas to view organization and project events, including billing, access, and auto-scaling changes, using the Atlas UI, CLI, or Administration API.

.. default-domain:: mongodb

.. facet::
   :name: genre
   :values: tutorial

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The |service| :guilabel:`Organization Activity Feed` displays select events
that occurred for a given |service| :ref:`organization <create-organization>`,
such as billing or access events.

The |service| :guilabel:`Project Activity Feed` provides more granular detail
about the state of your {+cluster+}. The Project Activity Feed
displays select events that occurred for a given |service|
:ref:`project <create-project>`, such as auto-scaling changes to a
minimum or maximum {+cluster+} tier.

Required Access
---------------

To view the Organization Activity Feed, you must have at minimum
:authrole:`Organization Member` access to the organization.

To view the Project Activity Feed, you must have at minimum
:authrole:`Project Read Only` access to the project.
