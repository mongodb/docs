==========================
Add a Custom MongoDB Build
==========================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

You can add custom build versions of MongoDB for your {+aagent+}s
to download. {+aagent+}s download these MongoDB versions
"lazily"; {+aagent+}s do not download binaries for a version of
MongoDB until a MongoDB process uses that version on a managed host.

.. _version-manager:

Add a Custom Build
------------------

The existing versions of MongoDB may not meet your immediate needs.
You can provide other versions of MongoDB through the 
:guilabel:`Custom Builds` functionality.
