Insert Data Into MongoDB Database¶

On this page

  • Prerequisites
  • Procedure
  • Summary
  • What’s Next

You will insert data (in the form of BSON documents) into a collection in MongoDB.

Time required: 15 minutes

Prerequisites¶

1

Find your MongoDB instance.¶

  • windows
  • linux

The Python examples below assume that you have connected to a MongoDB replica set and have accessed a database that contains an inventory collection.

To make sure your MongoDB process is running on linux, run the following command:
ps -ef | grep 'mongod'

If a mongod process is running, you will see something like:

502  6990   540   0  7Feb18 ttys008    8:53.94 ./mongod
2

Access to Mongo Shell¶

If you have properly installed mongo, you should be able to access your mongod instance with:

mongo

If a mongo shell successfully starts, you will see some log output to the console and then a command prompt. Type the following to make sure you are running mongo shell, it will return the version number of the shell.

> db.version()

Procedure¶

NA

1

Start Excel¶

Start Microsoft Excel and open a blank worksheet.

2

Select the Data Tab¶

Select the Data tab to open the Data toolbar.

3

Open a Data Connection Wizard Dialog¶

  1. Click the Get External Data button on the left side of the toolbar.
  2. Click From Other Sources.
  3. Select the From Data Connection Wizard option.
4

Select ODBC DSN¶

Select ODBC DSN from the list of data source options and click Next.

5

Select Your DSN¶

Select the DSN which connects to your BI Connector instance and click Next.

6

Select a Database and Table¶

Select a database from the dropdown list and a collection from which to import data. Click Next when finished.

7

Save the Connection File¶

Save the data connection file and click Finish. If you wish to re-use this connection in the future, you can select it from the Data -> Get External Data -> Existing Connections menu.

8

Specify Worksheet Format¶

In the final dialog window you can specify a format for your worksheet. Click OK when finished.

Summary¶

If you have completed the steps below properly, you should
have successfully added data to your mongodb instance.
