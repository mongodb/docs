Overview
--------

Welcome to the documentation site for the official {+driver-long+}.
You can add the driver to your application to work with MongoDB in Go.
Download it by using `go get <https://pkg.go.dev/cmd/go/internal/get>`__
or set up a runnable project by following the :ref:`Get Started
<go-get-started>` guide.
