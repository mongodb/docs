Deletes the configuration of one oplog store.

Resource
--------

.. code-block:: http

   DELETE /oplog/mongoConfigs/{OPLOG-CONFIG-ID}

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :widths: 20 20 60
   :header-rows: 1
   :stub-columns: 1

   * - Name
     - Type
     - Description

   * - OPLOG-CONFIG-ID
     - string
     - The unique identifier that represents this Oplog configuration.
