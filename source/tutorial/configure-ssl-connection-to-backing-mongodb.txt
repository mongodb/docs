==========================================================
Configure SSL Connections to the Backing MongoDB Instances
==========================================================

.. default-domain:: mongodb

Overview
--------

If you set up your MongoDB backing instances to use access control, then you
must configure the ``conf-mms.properties`` file and ``conf-daemon.properties``
with the necessary certificate information for accessing the MongoDB
instances.

Prerequisites
-------------

Deploy backing MongoDB instances that use access control. For information on
deploying MongoDB with access control, see :manual:`Security Concepts
</core/security>` in the MongoDB manual.
