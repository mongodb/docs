Syntax
------

.. code-block:: none

   DELETE /groups/{GROUP-ID}/whitelist/{WHITELIST-ENTRY}

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :widths: 15 10 75

   * - Path Element
     - Required/Optional
     - Description

   * - ``GROUP-ID``

     - Required.

     - The unique identifier for the :ref:`project <group-id>` for
       which you want to delete a whitelist entry.

   * - ``WHITELIST-ENTRY``

     - Required.

     - Can be either the:
      
       - |aws| security group ID,
       - IP address, or
       - :abbr:`CIDR (Classless Inter-Domain Routing)` address
       
       of the whitelist entry to delete. If the entry includes a subnet
       mask, such as ``192.0.2.0/24``, use the URL-encoded value ``%2F`` 
       for the forward slash ``/``.

       .. important::

          .. include:: /includes/security/delete-ip-access-list-behavior.rst
