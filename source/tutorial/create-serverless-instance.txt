================================
Create a {+Serverless-Instance+}
================================

.. default-domain:: mongodb

This tutorial takes you through the steps to create a new |service|
:ref:`{+serverless-instance+} <ref-deployment-types>`.

To convert your existing shared {+cluster+} to a 
{+serverless-instance+}, see :ref:`convert-to-serverless`.

Required Access
---------------

To create a {+serverless-instance+}, you must have :authrole:`Project Owner` 
access to the project. Users with :authrole:`Organization Owner` access 
must add themselves to the project as a ``Project Owner``.
