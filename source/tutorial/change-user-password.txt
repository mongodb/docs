========================
Change a User's Password
========================

.. versionchanged:: 2.5.3

.. default-domain:: mongodb

Overview
--------

Strong passwords help prevent unauthorized access. Follow a policy for
creating strong passwords.

If appropriate, use a password generator. The ``openssl`` ``rand`` command is
one such generator.
