   .. step:: Go to the :guilabel:`Indexes` tab.

      a. Select the collection with the index that you want to drop.

      #. Click the :guilabel:`Indexes` tab.
      
   .. step:: Drop the index.

      Under the :guilabel:`Action` column, click the 
      :guilabel:`Drop Index` icon for the index that you want to drop.
      
   .. step:: Confirm action.

      In the dialog box, type the name of the index 
      and click :guilabel:`Drop`.
