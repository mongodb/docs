# Voyage AI

This folder contains Jupyter Notebooks and sample applications that demonstrate how to use Voyage AI embedding and reranking models. These notebooks and apps show you how to generate embeddings, perform semantic search, and implement RAG (Retrieval-Augmented Generation) using Voyage AI's best-in-class models.

## Notebooks

| Notebook | Description |
|----------|-------------|
| [quickstart](https://github.com/mongodb/docs-notebooks/blob/main/voyageai/notebooks/quickstart.ipynb) | Generate your first vector embeddings with Voyage AI and build a basic application |
| [semantic-search](https://github.com/mongodb/docs-notebooks/blob/main/voyageai/notebooks/semantic-search.ipynb) | Perform semantic search using Voyage AI embeddings |
| [rag](https://github.com/mongodb/docs-notebooks/blob/main/voyageai/notebooks/rag.ipynb) | Implement Retrieval-Augmented Generation (RAG) with Voyage AI |

## Applications

| Application | Description |
|-------------|-------------|
| [rag-in-memory](https://github.com/mongodb/docs-notebooks/tree/main/voyageai/apps/rag-in-memory) | Simple RAG implementation using in-memory storage with Voyage AI embeddings |
| [rag-with-mongodb](https://github.com/mongodb/docs-notebooks/tree/main/voyageai/apps/rag-with-mongodb) | Production-ready RAG implementation using MongoDB Atlas as the vector store with Voyage AI embeddings |
