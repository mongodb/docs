To create, update, and delete {+model-api-key+}\s:

- At the organization level, you must have :authrole:`Organization
  Owner` access to {+service+}.
- At the project level, you must have :authrole:`Project Model Owner`, 
  :authrole:`Project Owner`, or higher access to {+service+}.

To view {+model-api-key+}\s:

- At the organization and project levels, you must have
  :authrole:`Organization Read Only` or higher access to {+service+}.
- At only the project level, you must have :authrole:`Project Read Only`
  or higher access to {+service+}.
