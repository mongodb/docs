========
Projects
========

.. default-domain:: mongodb

.. include:: /includes/fact-projects-groups-synonyms.rst

The ``groups`` resource provides access to retrieve or create |service|
:ref:`projects <projects>`.

.. include:: /includes/api-base-url-new.rst

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 40 50

   * - Method
     - Endpoint
     - Description

   * - ``GET``

     - :doc:`/groups </reference/api/project-get-all>`

     - Get all projects that the authenticated user can access.

   * - ``GET``

     - :doc:`/groups/{GROUP-ID} </reference/api/project-get-one>`

     - Get information about the project associated with
       ``{GROUP-ID}``.

   * - ``GET``

     - :doc:`/groups/byName/{GROUP-NAME} </reference/api/project-get-one-by-name>`

     - Get information about the project associated with
       ``{GROUP-NAME}``.

   * - ``POST``

     - :doc:`/groups </reference/api/project-create-one>`

     - Create one project.

   * - ``DELETE``

     - :doc:`/groups/{GROUP-ID} </reference/api/project-delete-one>`

     - Delete one project.

   * - ``GET``

     - :doc:`/groups/{GROUP-ID}/teams/ </reference/api/project-get-teams>`

     - Get information about the teams assigned to the project
       associated with ``{GROUP-ID}``.

   * - ``POST``

     - :doc:`/{GROUP-ID}/teams/ </reference/api/project-add-team>`

     - Assign an |service| :ref:`team <manage-teams>` to the
       project associated with ``{GROUP-ID}``.

   * - ``DELETE``

     - :doc:`/groups/{GROUP-ID}/teams/{TEAM-ID} </reference/api/teams-remove-from-project>`

     - Removes the specified |service| team from the specified project.

   * - ``GET``

     - :doc:`/groups/{GROUP-ID}/users </reference/api/user-get-all>`

     - Get information about the |service| users assigned to the project
       associated with ``{GROUP-ID}``.

   * - ``DELETE``

     - :doc:`/groups/{GROUP-ID}/users/{USER-ID} </reference/api/project-remove-user>`

     - Remove the specified |service| user from the project associated
       with ``{GROUP-ID}``.

   * - ``GET``

     - :doc:`/groups/{GROUP-ID}/invites </reference/api/project-get-invitations>`

     - Retrieves all unaccepted invitations to the project associated with 
       ``{GROUP-ID}``.

   * - ``GET``

     - :doc:`/groups/{GROUP-ID}/invites/{INVITATION-ID} </reference/api/project-get-one-invitation>`

     - Retrieves one unaccepted invitation to a project associated with 
       ``{INVITATION-ID}``.
 
   * - ``PATCH``

     - :doc:`/groups/{GROUP-ID}/invites </reference/api/project-update-one-invitation>`

     - Update one unaccepted invitation to the project associated with 
       ``{GROUP-ID}``.

   * - ``PATCH``

     - :doc:`/groups/{GROUP-ID}/invites/{INVITATION-ID} </reference/api/project-update-one-invitation-by-id>`

     - Update one unaccepted invitation by ``{INVITATION-ID}`` to the 
       project associated with ``{GROUP-ID}``.

   * - ``DELETE``

     - :doc:`/groups/{GROUP-ID}/invites/{INVITATION-ID} </reference/api/project-delete-invitation>`

     - Deletes one unaccepted invitation to a project identified by 
       ``{INVITATION-ID}``.
