=========================
How to Index Token Fields 
=========================

.. default-domain:: mongodb

.. meta::
   :keywords: token, token field type, MongoDB search field type, sort MongoDB search results, field type for sorting results by string values, field type for querying string values using in operator, field type for querying string values using equals operator, field type for querying string values using range operator, index string for filtering, index string for sorting
   :description: Use the MongoDB Search token field type to index string values for sorting, faceting, and exact-match querying with the equals, in, and range operators.

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

.. |data-type| replace:: token
.. |fts-ui-field-type| replace:: :guilabel:`Token`
.. |fts-field-type| replace:: ``token``
.. |properties-link| replace:: :ref:`Field Properties <fts-field-types-token-options>`

Use the |fts| ``token`` field type to index string values for
sorting, faceting, and querying. After you index a field as
``token``, you can sort and group results by the indexed field,
and query it using the :ref:`equals <equals-ref>`,
:ref:`in <in-ref>`, and :ref:`range <range-ref>` operators.
