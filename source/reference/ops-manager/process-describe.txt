=====================================
mongocli ops-manager process describe
=====================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``process describe`` command retrieves details about 
the specified MongoDB process in an |mms| project. You can also 
retrieve details about a process through the |mms|
:opsmgr:`API </reference/api/hosts/get-one-host-by-id/index.html>`.

.. _om-process-describe-syntax: 

Syntax
------

.. code-block:: text

   mongocli ops-manager|om process describe|d <process-id> 
        [ --output|-o <output-format> ]
        [ --profile|-P <profile-name> ]
        [ --projectId <project-ID> ]

.. include:: /includes/fact-command-line-help.rst

.. _om-process-describe-options: 

Options 
-------

.. list-table:: 
   :header-rows: 1
   :widths: 20 10 60 10 

   * - Option 
     - Type 
     - Description 
     - Required?

   * - ``--output``, ``-o``
     - string 
     - .. include:: /includes/extracts/fact-basic-options-output.rst
     - no

   * - ``<process-id>``
     - string 
     - Unique identifier of the process you want to describe.

       You can retrieve a list of processes for a project with the 
       ``mongocli`` command 
       :ref:`process list <mcli-om-process-list-cmd>`.
     - yes

   * - ``--profile``, ``-P``
     - string
     - Name of the profile where the public and private 
       keys for the project are saved. If omitted, uses the 
       {+default-profile+}. To learn more about creating a 
       profile, see :ref:`mcli-configure`.
     - no

   * - ``--projectId``
     - string
     - .. include:: /includes/extracts/fact-basic-options-project-id.rst
     - no

.. _om-process-describe-example:

Example
-------

The following example uses the ``mongocli ops-manager describe`` 
command to describe the ``PRIMARY`` member in an |mms| deployment 
replica set. It uses the {+default-profile+} where the |api| 
access keys and the project ID are stored.

.. code-block:: text

   mongocli ops-manager process describe 0ea97615dd78fd94f2b6cfc732af30ce --output json
