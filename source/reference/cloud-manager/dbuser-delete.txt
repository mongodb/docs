====================================
mongocli cloud-manager dbuser delete
====================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``dbuser delete`` command deletes a MongoDB database user 
from the specified |cloud-short| clusters in the given project. You can also 
delete a MongoDB database user through the |cloud-short| :cloudmgr:`UI 
</tutorial/manage-mongodb-users/>`.

``mongocli`` requests a confirmation from the user before sending the
``delete`` command to |cloud-short|.

Syntax
------

.. code-block:: text

   mongocli cloud-manager dbuser delete <username>
        [ --authDB ]
        [ --profile|-P <profile-name> ]
        [ --projectId <project-ID> ]
        
.. include:: /includes/fact-command-line-help.rst

Options
-------

.. list-table::
   :widths: 20 10 60 10

   * - Option 
     - Type
     - Description
     - Required?

   * - ``--authDB``
     - string
     - Name of the authentication database for this user. Defaults to
       ``admin``.
     - no

   * - ``--profile``, ``-P``
     - string
     - Name of the profile where the public and private 
       keys for the project are saved. If omitted, uses the 
       ``default`` profile. To learn more about creating a 
       profile, see :ref:`mcli-configure`.
     - no

   * - ``--projectId``
     - string
     - Unique identifier of the project that contains the 
       cluster to update. If omitted, uses the project ID in 
       the profile or :ref:`environment variable <mcli-env-var>`.
     - no

   * - ``<username>``
     - string
     - Username for authenticating the user to MongoDB.
     - yes

Output 
------

A successful ``dbuser delete`` command returns an in-progress status message
with a link to your |cloud-short| topology page.

Example
-------

The example below uses the ``mongocli cloud-manager dbuser delete`` command to
delete a MongoDB database user with the username ``user1``.

The command uses the profile named ``cmProfile`` for accessing 
|cloud-short|.

.. code-block:: sh

   ~ mongocli cloud-manager dbuser delete user1 --profile cmProfile
