.. setting:: SLO Endpoint URL

   *Type*: string

   
   |url| of the Single Logout endpoint that |onprem| calls when a user
   attempts to log out. If set, when a user attempts to log out of
   |onprem|, they log out of your |idp|. If left blank, logging out of
   |onprem| doesn't log the user out of their |idp| session.
