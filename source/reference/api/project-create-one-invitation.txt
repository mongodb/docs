=============================
Create One Project Invitation
=============================

.. default-domain:: mongodb

.. meta::
   :canonical: https://www.mongodb.com/docs/atlas/reference/api-resources-spec/#tag/Projects

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

Invite one user to the specified |service| project.

.. include:: /includes/fact-projects-groups-synonyms.rst

.. include:: /includes/api-requirements.rst

Required Roles
--------------

You can successfully call this endpoint with the
:authrole:`Project Owner` role.

Resource
--------

.. include:: /includes/api/base-api-uri.rst

.. code-block:: http

   POST /groups/{GROUP-ID}/invites

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :stub-columns: 1
   :widths: 20 14 11 55

   * - Path Element
     - Type
     - Necessity
     - Description

   * - GROUP-ID
     - string
     - Required
     - Unique 24-hexadecimal digit string that identifies the project.

Request Query Parameters
~~~~~~~~~~~~~~~~~~~~~~~~

.. include:: /includes/api/no-query-single-parameters.rst

Request Body Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :stub-columns: 1
   :widths: 20 14 66

   * - Name
     - Type
     - Description

   * - roles
     - array of strings
     - |service| :ref:`roles <project-roles>` to assign to the invited
       user.

       If the user accepts the invitation, |service| assigns these
       roles to them.

   * - username
     - string
     - Email address to which |service| sent the invitation. The user
       uses this email address as their |service| username if they
       accepts this invitation.

Response
--------

.. include:: /includes/api/list-tables/responses/project-invitation-response.rst

Example Request
---------------

.. code-block:: sh

   curl --user "{PUBLIC-KEY}:{PRIVATE-KEY}" --digest --include \
        --header "Accept: application/json" \
        --header "Content-Type: application/json" \
        --request POST "https://cloud.mongodb.com/api/atlas/v1.0/groups/{GROUP-ID}/invites?pretty=true"
        --data '
          { 
             "roles": [ "GROUP_OWNER" ], 
             "username": "jane.smith@example.com" 
          }'
