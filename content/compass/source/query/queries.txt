:guilabel:`My Queries` view. This page explains how to add and view favorite 
queries.

.. _compass-favorite-queries:

Save a Favorite Query
---------------------

You can favorite a query so that you can find it later. If you
load a favorite query, you can change it without
changing the original saved copy.

To add a query to your favorites:

.. _compass-view-favorite-queries:

View Saved Queries
------------------

After you connect to your cluster, you can view your saved queries and 
aggregation pipelines on the :guilabel:`My Queries` tab.

.. note:: 

   To learn how to save and view aggregation pipelines, see: 

   - :ref:`compass-save-agg-pipeline`
   - :ref:`compass-open-saved-pipeline`

.. _compass-saved-query-view:

Use the :guilabel:`My Queries` Tab
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

When you click a saved or favorite query tile, |compass-short| opens the
:guilabel:`Documents` tab with the filter loaded.

In the :guilabel:`My Queries` tab, you can also click a :ref:`saved aggregation 
<compass-view-favorite-queries>` pipeline tile to open the 
:guilabel:`Aggregations` tab with the pipeline loaded.
