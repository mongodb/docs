======================
Backup Oplog is Behind
======================

.. default-domain:: mongodb

Description
-----------

The :alert:`Backup oplog is behind` alert occurs when the |mms| backup service
has not received an oplog from the Backup Agent for at least an hour.

Common Triggers
---------------

This can be caused by the following conditions:

- Backup Agent is down.

- Backup Agent is overloaded.

- Backup Agent can't reach the replica set.

- Backup Agent can't reach or successfully send data to |mms|.

- Backup Agent is continuously restarting.

Action
------

Check the Backup Agent :doc:`logs </tutorial/view-logs>` in |mms| and on the
Backup Agent host to find out which of these occurred.
