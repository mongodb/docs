===================
MongoDB with Claude
===================

.. meta::
   :description: Learn how to use MongoDB with Claude Code.

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

You can use MongoDB with Claude Code through the official MongoDB Claude Plugin.

The MongoDB Claude Plugin provides:

- The MongoDB MCP Server
- Pre-built agent skills for common MongoDB tasks. See the :ref:`MongoDB Agent Skills documentation <agent-skills>` for more information.

.. _claude-plugin-install:

Install the MongoDB Claude Plugin
---------------------------------

The MongoDB Claude Plugin is available in the official Claude Plugin Marketplace. 

Install the MongoDB Claude Plugin
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

To install the MongoDB Claude Plugin:

1. Start a session with Claude Code.
#. Install the MongoDB Claude Plugin: 

   .. code-block:: bash

      /plugin install mongodb

#. Follow the prompts to complete the installation.
#. Run ``/reload-plugins`` to apply the changes. 

After installing the plugin, you must configure the MCP Server before using
any agent skills or functionality that connects to MongoDB.

Configure the MCP Server
------------------------

The MongoDB Claude Plugin bundles the MongoDB MCP Server. To configure the MCP
Server, see `MongoDB MCP Server Get Started <https://www.mongodb.com/docs/mcp-server/get-started/>`__.
