Use this section to find reference information for the {+atlas-cli+},
including how to work with |json| configuration files for commands
that accept the ``--file`` option.

You can pass a ``.json`` configuration file to {+atlas-cli+} commands
that accept the ``--file`` option. Configuration files let you define
complex settings in a reusable format, which is useful for automating
cluster creation, managing alert configurations, and performing other
operations that require many parameters.
