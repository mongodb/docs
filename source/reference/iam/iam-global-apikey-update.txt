================================
mongocli iam globalApiKey update
================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``iam globalApiKey update`` command updates the roles and the description 
of the specified Global |api| Key. You can also modify the Global |api| Key 
roles and description using the |onprem| :opsmgr:`UI 
</admin/general/api-keys-page/#edit-a-global-api-key>` and :opsmgr:`API 
</reference/api/api-keys/global/update-one-global-api-key/>`. You must have 
the :opsmgr:`Global Owner </reference/user-roles/#Global-Owner>` role 
to update a Global |api| key.

.. _mcli-iam-global-apikey-assign-syntax:

Syntax 
------

.. code-block:: text 

   mongocli iam globalApiKey|globalApiKeys update <api-key-id>
         --desc "<description-of-key>" 
        [ --output|-o <output-format> ]
        [ --profile|-P <profile-name> ]
         --role <list-of-roles>

.. _mcli-iam-global-apikey-assign-output:

Output
------

If the command succeeds, it returns the following output in the default
format. If the command returns errors, see :ref:`Troubleshooting
<troubleshooting>` for recommended solutions.

.. code-block:: sh
   :copyable: false

   API Key '<api-key-id>' successfully updated.

The default output is a subset of the fields returned by this command. 
For the complete list of |json| fields returned by the command, see the 
:opsmgr:`API reference 
</reference/api/api-keys/global/update-one-global-api-key/#response>`. 

.. _mcli-iam-global-apikey-assign-examples:

Examples
--------

.. tabs::

   .. tab:: Default Output
      :tabid: default-output

      The following command updates the description and role of a
      Global |api| Key using the {+default-profile+}, which
      contains credentials for accessing the |onprem| instance. The 
      output is returned in the default format.

      .. code-block:: sh

         mongocli iam globalApiKey update 5f3d8790c9022019903c1234 --role GLOBAL_MONITORING_ADMIN --desc "My Sample Global API Key"

      The previous command prints the following to the terminal.

      .. code-block:: json
         :copyable: false

         API Key '5f3d8790c9022019903c1234' successfully updated.

   .. tab:: JSON Output
      :tabid: json-output

      The following command updates the role and description of a Global
      |api| Key using the ``myOM`` profile, which contains the credentials 
      for accessing the |onprem| instance. The output is returned in |json| 
      format.

      .. code-block:: sh

         mongocli iam globalApiKey update 5f3d8790c9022019903c1234 --desc "My Sample Global API Key" --role GLOBAL_MONITORING_ADMIN -P myOM --output json
         
      The previous command updates the |api| key and 
      prints the following to the terminal in |json| format.

      .. code-block:: json
         :copyable: false

           "id": "5f3d8790c9022019903c1234",

               "roleName": "GLOBAL_USER_ADMIN"

             {
               "roleName": "GLOBAL_OWNER"
             }
           ],
