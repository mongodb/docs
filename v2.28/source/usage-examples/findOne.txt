Expected Result
~~~~~~~~~~~~~~~

Running any of the preceding full examples prints results similar to the following:

.. code-block:: none

   { 
      "_id" : ObjectId("5eb3d668b31de5d588f42950"), 
      "name" : "Bagels N Buns", 
      "restaurant_id" : "40363427", 
      "cuisine" : "Delicatessen", 
      "address" : {...}, 
      "borough" : "Staten Island", 
      "grades" : [...] 
   }
