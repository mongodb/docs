Endpoints
---------

The following lists the endpoints available for Backup Daemon
configurations.

   :header-rows: 1
   :widths: 10 50 40

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :doc:`/daemon/configs/{HOST}/{HEAD-ROOT-DIRECTORY} </reference/api/admin/backup/daemonConfigs/get-one-backup-daemon-configuration-by-host>`
     - Get One Backup Daemon Configuration by Host
     
   * - ``GET``
     - :doc:`/daemon/configs </reference/api/admin/backup/daemonConfigs/get-all-backup-daemon-configurations>`
     - Get All Backup Daemon Configurations
     
   * - ``PUT``
     - :doc:`/daemon/configs/{HOST}/{HEAD-ROOT-DIRECTORY} </reference/api/admin/backup/daemonConfigs/update-one-backup-daemon-configuration>`
     - Update One Backup Daemon Configuration
     
   * - ``DELETE``
     - :doc:`/daemon/configs/{HOST}/{HEAD-ROOT-DIRECTORY} </reference/api/admin/backup/daemonConfigs/delete-one-backup-daemon-configuration>`
     - Delete One Backup Daemon Configuration
