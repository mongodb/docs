====================
Remove Legacy Agents
====================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The legacy Backup and Monitoring Agents can be removed after the
{+mdbagent+} has been
:doc:`installed </tutorial/nav/install-mongodb-agent>`.

To remove these Agents, see:

- :doc:`/tutorial/delete-backup-agent`

- :doc:`/tutorial/delete-monitoring-agent`

The MongoDB Agent replaces the Automation Agent, so the Automation Agent does not need to be removed.
