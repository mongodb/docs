.. list-table::
   :widths: 30,70
   :header-rows: 1

   * - Name

     - Description

   * - :dbcommand:`aggregate`

     - Performs :doc:`aggregation tasks </core/aggregation-pipeline>` such as group using the aggregation framework.

   * - :dbcommand:`count`

     - Counts the number of documents in a collection or a view.

   * - :dbcommand:`distinct`

     - Displays the distinct values found for a specified key in a collection or a view.

   * - :dbcommand:`mapReduce`

     - Performs :doc:`map-reduce </core/map-reduce>` aggregation for large data sets.
