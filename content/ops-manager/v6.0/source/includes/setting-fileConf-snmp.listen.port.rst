.. setting:: snmp.listen.port

   *Type*: number

   *Default*: 11611

   
   Applies to SNMPv2c queries that an |snmp| host would send to
   |application|.
   
   Listening |udp| port for |snmp|. Setting to a number less than
   ``1024`` requires running the |application| with elevated privileges.
