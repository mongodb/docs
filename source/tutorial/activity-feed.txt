==================
View Activity Feed
==================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The |service| :guilabel:`Organization Activity Feed` displays select 
events that occurred for a given |service| organization. Likewise, the 
|service| :guilabel:`Project Activity Feed` displays select events that 
occurred for a given |service| project.
