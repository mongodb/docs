Resource
--------

.. code-block:: http
   :copyable: false

   DELETE /groups/{GROUP-ID}/agentapikeys/{API-AGENT-KEY-ID}

This endpoint is restricted to users with the 
:authrole:`Project Owner` role.

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :widths: 20 20 60
   :header-rows: 1

   * - Name
     - Type
     - Description

   * - ``GROUP-ID``
     - string
     - Unique identifier of the :term:`project` that owns the 
       Agent API Key.

   * - ``API-AGENT-KEY-ID``
     - string
     - Unique identifier of Agent API Key.

Example Request
---------------

.. only:: cloud

   .. cssclass:: copyable-code
   .. code-block:: sh

      curl --user "{USERNAME}:{APIKEY}" --digest \
       --include \
       --request DELETE "https://cloud.mongodb.com/api/public/v1.0/groups/{GROUP-ID}/agentapikeys?pretty=true"

.. only:: onprem

   .. cssclass:: copyable-code
   .. code-block:: sh

      curl --user "{USERNAME}:{APIKEY}" --digest \
       --include \
       --request DELETE "https://{OPSMANAGER-HOST}:{PORT}/api/public/v1.0/groups/{GROUP-ID}/agentapikeys?pretty=true"
