==================
atlas teams delete
==================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Delete a team from an organization.

Syntax
------

.. code-block::

   atlas teams delete <teamId> [options]

.. Code end marker, please don't delete this comment
