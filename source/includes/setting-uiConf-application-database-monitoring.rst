.. setting:: Application Database Monitoring

   *Type*: string

   *Default*: false

   
   Flag that indicates that |onprem| can monitor its application
   database.
   
   
   To monitor the application databases, you need to:
   
   1. Click :guilabel:`Enable` on this page.
   2. Install the {+mdbagent+} on the application database hosts.
   3. Add the application database.
   
   .. seealso::
   
      To follow the full procedure, see
      :doc:`/tutorial/enable-appdb-monitoring`.
