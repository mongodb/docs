Resource
--------

.. include:: /includes/api/base-api-uri.rst

.. code-block:: http

   PATCH /groups/{GROUP-ID}

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :widths: 20 14 11 55
   :header-rows: 1
   :stub-columns: 1

   * - Path Element
     - Type
     - Necessity
     - Description

   * - GROUP-ID
     - string
     - Required
     - Unique 24-hexadecimal character string that identifies the
       project.

     - Necessity
     - Description

   * - name
     - string
     - Required
     - Human-readable label that you want to have identify the

Response
--------

.. list-table::
   :widths: 20 14 66
   :header-rows: 1
   :stub-columns: 1

   * - Response Parameter
     - Type
     - Description

   * - id
     - string
     - Unique 24-hexadecimal character string that identifies the
       project.

   * - orgId
     - string
     - Unique 24-hexadecimal character string that identifies the
       |service| organization to which the project belongs.

   * - name
     - string
     - Human-readable label that identifies the project.

   * - publicApiEnabled
     - Boolean
     - Flag that indicates whether someone enabled the API for the
       specified organization.

   * - hostCounts
     - string
     - Quantity of hosts managed in the specified project.

   * - lastActiveAgent
     - string
     - |iso8601-time| when one of the MongoDB Agents sent a ping
       indicating activity.

   * - activeAgentCount
     - integer
     - Quantity of MongoDB Agents currently running in the specified
       project.

   * - replicaSetCount
     - integer
     - Quantity of replica sets included in the specified project.

   * - shardCount
     - integer
     - Quantity of shards included in the specified project.

   * - agentApiKey
     - string
     - Human-readable label that identifies the MongoDB Agents |api|
       key.

   * - ldapGroupMappings
     - array
     - List of |ldap| groups mapped to the specified project.

   * - tags
     - array
     - List of labels applied to the specified project.


Example Request
---------------

.. code-block:: shell

   curl --user "{PUBLIC-KEY}:{PRIVATE-KEY}" --digest --include \
        --header "Accept: application/json" \
        --header "Content-Type: application/json" \
        --request PATCH "https://cloud.mongodb.com/api/atlas/v1.0/groups/{GROUP-ID}" \
        --data '{
          "name" : "updatedProject"
        }'
