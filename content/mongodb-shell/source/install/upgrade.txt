Upgrade mongosh
===============

.. composable-tutorial::
   :options: operating-system, windows-installation-method, macos-installation-method, linux-distribution
   :defaults: windows, msiexec, None, None

   To upgrade to the latest version of mongosh, select your operating
   system and installation method at the top of the page and follow the
   corresponding instructions.

   .. selected-content::
      :selections: windows, msiexec, None, None

      This section describes how to upgrade ``mongosh`` on Windows
      through the MSI installer.

      Steps
      -----

      .. procedure::
         :style: normal

         .. step:: (Optional) Remove any existing mongosh binaries from your system

            If you skip this step, overwrite the existing ``mongosh``
            binary when you install the new version.

         .. step:: Open the MongoDB Shell download page

            Open the `MongoDB Download Center
            <https://www.mongodb.com/try/download/shell>`__.

         .. step:: Select your platform and package
            
            a. In the :guilabel:`Platform` dropdown, select :guilabel:`Windows x64`.
            b. In the :guilabel:`Package` dropdown, select :guilabel:`msi`.

         .. step:: Click :guilabel:`Download`

         .. step:: Double-click the installer file

         .. step:: Follow the installation prompts

         .. step:: Ensure that the new mongosh binary is in your system PATH

            .. include:: /includes/mongosh-verify-build-info.rst

   .. selected-content::
      :selections: macos, None, homebrew, None

      This section describes how to upgrade ``mongosh`` on macOS with
      Homebrew.

      Steps
      -----

      .. procedure::
         :style: normal

         .. step:: Update package information

            .. code-block:: sh

               brew update

         .. step:: Upgrade mongosh
            
            .. code-block:: sh

               brew upgrade mongosh

         .. step:: Verify the installation

            .. include:: /includes/mongosh-verify-build-info.rst
