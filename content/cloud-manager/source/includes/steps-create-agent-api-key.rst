.. procedure::
   :style: normal

   .. include:: /includes/nav/steps-agents.rst
      
   .. step:: Click the :guilabel:`Agent API Keys` Tab.
   
   .. step:: Click :guilabel:`Generate`.
      
   .. step:: Enter a description.

      Enter a description for the key. The description acts as an identifier for the key as
      the key is shown only once upon generation.
      
      The description has a limit of 1000 characters.
      
   .. step:: Copy and record the key.
      
      Copy and record the key when it is generated.
      
      .. include:: /includes/extracts/agent-api-key-admonition.rst
      
      After you have  recorded the key, click :guilabel:`Close`.
