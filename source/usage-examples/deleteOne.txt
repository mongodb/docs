=================
Delete a Document
=================

.. default-domain:: mongodb

You can delete a document in a collection by using the ``DeleteOne()``
method.

in the ``movies`` collection:

   { "_id": ObjectId("573a13bff29313caabd5e06b"), ..., "title": "Twilight", ... }
