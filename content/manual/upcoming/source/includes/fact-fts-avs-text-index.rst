.. note:: 
   
   MongoDB offers an improved full-text search solution, :atlas:`{+fts+} 
   </atlas-search/>`, and vector search solution, :atlas:`{+avs+}

   :ref:`{+fts+} indexes <fts-manage-indexes>` or :ref:`{+avs+} indexes
   <avs-types-vector-search>`  instead of text indexes.
