.. _snapshot-export-jobs:

====================
Snapshot Export Jobs
====================

.. default-domain:: mongodb

.. include:: /includes/fact-projects-groups-synonyms.rst

.. important:: {+Cloud-Backup+}s Only

   This resource only applies to clusters using
   :ref:`backup-cloud-provider`. 

The ``/backup/exports`` resource enables you to export backup snapshots 
to an |aws| :ref:`bucket <snapshot-export-buckets>` and retrieve 
information on export jobs.

Endpoints
---------

.. include:: /includes/api-base-url.rst

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/backup/exports </reference/api/cloud-backup/export/get-all-export-jobs>`
     - Get all {+Cloud-Backup+} snasphot export jobs for the specified 
       cluster.

   * - ``GET``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/backup/restoreJobs/{JOB-ID} </reference/api/cloud-backup/export/get-one-export-job>`
     - Get one {+Cloud-Backup+} snasphot export job specified by the 
       export job ID.

   * - ``POST``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/backup/restoreJobs </reference/api/cloud-backup/export/create-one-export-job>`
     - Export one {+Cloud-Backup+} snapshot specified by the snapshot 
       ID to the bucket specified by the bucket ID.
