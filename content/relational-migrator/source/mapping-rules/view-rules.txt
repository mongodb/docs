==========
View Rules
==========

.. meta::
   :description: View mapping rules in the Schema model pane by selecting a table or collection name under the Relational or MongoDB headings.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

You can view mapping rules in the Schema model pane. Select a table name under 
the ``Relational`` heading or a collection name under the ``MongoDB`` heading.

Steps
-----

1. Open a project.

#. In the :guilabel:`Schema model` pane, click a table name under the 
   :guilabel:`Relational` header or a collection name under the ``MongoDB`` header.

   You can view mapping rules in the ``Mappings`` pane on the right of the screen.
   If you applied a table filter, it is also visible under the :guilabel:`Table filter` 
   header.
