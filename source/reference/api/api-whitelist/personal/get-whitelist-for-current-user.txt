Resource
--------

.. include:: /includes/api/base-api-uri.rst

.. code-block:: http

   GET /users/{USER-ID}/whitelist

Request Parameters
------------------

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :widths: 20 14 11 55
   :header-rows: 1
   :stub-columns: 1

   * - Path Parameter
     - Type
     - Necessity
     - Description

   * - USER-ID
     - string
     - Required
     - Unique 24-hexdecimal-digit string that identifies your |service|
       user.

       To retrieve user IDs, see :ref:`api-get-all-users-in-group`.
       Your user ID corresponds to the **id** field returned in the
       response from that endpoint.
