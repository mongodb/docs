====================
Upgrade the |k8s-op|
====================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The following procedure outlines how to upgrade the |k8s-op-short|
to its latest version. This upgrade process parallels the |k8s-op-short| 
installation process.

.. note:: 

   The upgrade process for |k8s-op-short| requires that you upgrade incrementally 
   through each minor version to your target version. For example, if you are 
   upgrading from version 1.2.x to 1.4.x, you must first upgrade to 1.3.x before 
   proceeding to 1.4.x.
