Platform Support
~~~~~~~~~~~~~~~~

Enterprise Edition
``````````````````

- Starting in MongoDB 4.0.28 the following platforms are no longer supported:

- Ubuntu 16.04 PPC64LE
- RHEL 7.1 PPC64LE
- RHEL 7.2 s390x
