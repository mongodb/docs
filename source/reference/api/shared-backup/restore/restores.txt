========================
Shared-Tier Restore Jobs
========================

.. default-domain:: mongodb

.. include:: /includes/fact-projects-groups-synonyms.rst

.. include:: /includes/fact-only-m2-m5-clusters.rst

The ``/backup/tenant/restores`` resource enables you to view
and create restore jobs from ``M2`` and ``M5`` clusters.

Endpoints
---------

.. include:: /includes/api-base-url.rst

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/backup/tenant/restores </reference/api/shared-backup/restore/get-all-restore-jobs>`
     - Get all restore jobs for the specified ``M2`` or ``M5`` cluster.

   * - ``GET``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/backup/tenant/restores/{RESTORE-ID} </reference/api/shared-backup/restore/get-one-restore-job>`
     - Get the restore job associated to ``{RESTORE-ID}``.

   * - ``POST``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/backup/tenant/restore </reference/api/shared-backup/restore/create-one-restore-job>`
     - Create a new restore job from an ``M2`` or ``M5`` cluster
       snapshot to the specified cluster.
