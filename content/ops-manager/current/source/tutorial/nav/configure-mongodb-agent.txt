==============================================
Configure the {+mdbagent+} for Access Control
==============================================

.. meta::
   :description: Configure the MongoDB Agent to authenticate with the necessary roles and privileges for access control.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

If your MongoDB deployment enforces access control, the {+mdbagent+}
must authenticate to your MongoDB databases as a user with the proper
:manual:`roles </reference/built-in-roles>` and
:manual:`privileges </reference/privilege-actions>`.
