============================
{+mcli+} Quick Start
============================

.. meta::
   :description: Get started with the MongoDB CLI by creating and managing clusters in Cloud Manager or Ops Manager using step-by-step quick start guides for new users.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

{+mcli+} provides a command-line interface for managing your |cloud| and
|mms-full| projects and clusters. Use the quick start guides to
create your first cluster and verify that it is running.

The quick start guides walk you through the following tasks:

- Install and configure the {+mcli+}.
- Create a cluster in your |cloud-short| or |onprem| project.
- Retrieve cluster details to confirm that the cluster is running.
- Watch the automation configuration status.

Prerequisites
-------------

Before you begin, complete the following tasks:

- Ensure that your environment is :ref:`compatible <mcli-compatibility>`.
- :ref:`mcli-install`.
- :ref:`mcli-configure`.
