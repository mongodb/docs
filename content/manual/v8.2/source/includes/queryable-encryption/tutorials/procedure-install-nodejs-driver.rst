.. procedure::

   .. step:: Install a compatible driver version
            
      To use {+qe+} with the :driver:`Node.js </node/current>` driver, install version 5.5.0 or later.

   .. step:: Install the ``mongodb-client-encryption`` package

      Install `mongodb-client-encryption
      <https://www.npmjs.com/package/mongodb-client-encryption/>`__ version
      2.8.0 or later. If you're using Node.js driver version 6.0 or later, you
      must also use ``mongodb-client-encryption`` 6.0 or later.
