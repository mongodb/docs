===============
``mongoexport``
===============

.. default-domain:: mongodb

.. role:: red(strong)
   :class: text-danger

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

:binary:`~bin.mongoexport` is a command-line tool that produces a JSON
or CSV export of data stored in a MongoDB instance.

.. note:: ``mongoexport`` documentation migrated to `MongoDB Database Tools <https://docs.mongodb.com/database-tools>`__


   Starting in MongoDB 4.4, the documentation for ``mongoexport``
   has migrated to:

   - https://docs.mongodb.com/database-tools/mongoexport
