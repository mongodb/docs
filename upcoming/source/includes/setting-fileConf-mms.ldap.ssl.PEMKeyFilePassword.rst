.. setting:: mms.ldap.ssl.PEMKeyFilePassword

   *Type*: string

   
   Password for :setting:`LDAP SSL PEM Key File`. Use this setting
   if the ``PEMKeyFile`` is encrypted.
   
   .. code-block:: ini
   
      mms.ldap.ssl.PEMKeyFilePassword=<password>
   
   
   Corresponds to :setting:`LDAP SSL PEM Key File Password`.
