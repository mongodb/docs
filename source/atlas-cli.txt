=========
Atlas CLI
=========

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

The `{+atlas-cli+} <https://www.mongodb.com/docs/atlas/cli/stable>`__
is a command line interface built specifically for MongoDB |service|.
Use the {+atlas-cli+} to easily manage |service| from the terminal.

.. button:: Install the {+atlas-cli+}
      :uri: https://www.mongodb.com/docs/atlas/cli/stable/install-atlas-cli/

Manage |service|
~~~~~~~~~~~~~~~~

To perform tasks in |service| using the {+atlas-cli+}, see the
:guilabel:`{+atlas-cli+}` tab on a task's page in this documentation. For example, to set up private endpoints using the {+atlas-cli+}, see
the :guilabel:`{+atlas-cli+}` tab on :ref:`<cluster-private-endpoint>`.

All {+atlas-cli+} commands in this documentation reflect the stable
version of the {+atlas-cli+}. If you use an earlier version of
the {+atlas-cli+} and you want to use the commands in this
documentation, you can:

- Refer to the `command reference
  <https://www.mongodb.com/docs/atlas/cli/stable/command/atlas/>`__ 
  and select your {+atlas-cli+} version from the dropdown.
- `Upgrade the {+atlas-cli+} 
  <https://www.mongodb.com/docs/atlas/cli/stable/install-atlas-cli/#update-the-atlas-cli>`__ 
  to the stable version.

To view all parameters and syntax for {+atlas-cli+} commands, see the
`command reference
<https://www.mongodb.com/docs/atlas/cli/stable/command/atlas/>`__.
