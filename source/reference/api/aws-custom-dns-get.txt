Retrieve the custom DNS configuration of an |service| project's clusters 
deployed to |aws|.

.. include:: /includes/api-requirements.rst

Required Roles
--------------

You can successfully call this endpoint with any assigned role.

Request
-------
.. include:: /includes/api-base-url.rst

.. code-block:: none

   GET /groups/{GROUP-ID}/awsCustomDNS

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :widths: 15 10 75

   * - Parameter
     - Required/Optional
     - Description

   * - ``GROUP-ID``
     - Required
     - Unique identifier for the project.
