================================
{+Cloud-Backup+} Snapshot Export
================================

.. default-domain:: mongodb

.. include:: /includes/fact-projects-groups-synonyms.rst

.. important:: {+Cloud-Backup+}s Only

   This resource only applies to clusters using
   :ref:`backup-cloud-provider`. 

The following resources enable you to configure |aws| buckets for 
exporting {+Cloud-Backup+} snasphots and create and retrieve 
information export jobs.

.. list-table::
   :header-rows: 1
   :widths: 40 60

   * - Endpoint
     - Description

   * - :doc:`/backup/exportBuckets </reference/api/cloud-backup/export/export-buckets/>`
     - Create, retrieve, and delete {+Cloud-Backup+} snasphot export 
       buckets.

   * - :doc:`/backup/exports </reference/api/cloud-backup/export/export-jobs/>` 
     - Create and retrieve {+Cloud-Backup+} snasphot export jobs.
