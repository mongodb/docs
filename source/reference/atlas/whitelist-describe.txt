=================================
mongocli atlas whitelist describe
=================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``whitelist describe`` command retrieves details about the 
specified whitelist entry in the specified |service| project. You can 
also retrieve details about a whitelist entry through the |service| 
:atlas:`UI </security-whitelist/#view-whitelist-entries>` or 
:atlas:`API </reference/api/whitelist-get-one-entry/>`.

.. _mcli-atlas-whitelist-describe-syntax:

Syntax 
------

.. code-block:: text 
   :copyable: false

   mongocli atlas cluster describe <whitelist-entry>
        [ --profile|-P <profile-name> ]
        [ --projectId <project-ID> ]

.. _mcli-atlas-whitelist-describe-options:

Options 
-------

.. list-table::
   :header-rows: 1
   :widths: 20 10 60 10

   * - Option
     - Type
     - Description  
     - Required?

   * - ``<whitelist-entry>``
     - string
     - IP address, |cidr| block, or |aws| security group ID of the    
       whitelist entry to retrieve.
     - yes

   * - ``--profile``, ``-P``
     - string
     - Name of the profile where the public and private 
       keys for the project are set. If omitted, uses the 
       ``default`` profile. To learn more about creating a 
       profile, see :ref:`mcli-configure`.
     - no

   * - ``--projectId``
     - string
     - Unique identifier of the project that contains the 
       cluster to update. If omitted, uses the project ID in 
       the profile or :ref:`environment variable <mcli-env-var>`.
     - no

.. _mcli-atlas-whitelist-describe-examples:

Example
-------

The following example uses the ``mongocli atlas whitelist describe`` 
command to retrieve details about the whitelist entry ``192.0.2.0/24``.
It uses the ``default`` profile to access the |service| project that
contains the whitelist.

.. code-block:: text
   :copyable: false

   mongocli atlas whitelist describe 192.0.2.0/24 --projectId 5e1234c17a3e5a48f5497de3

The example command prints the following fields to the terminal. To
learn more about these fields, see :ref:`Output
<mcli-atlas-whitelist-describe-output>`.

.. code-block:: json
   :copyable: false

   {
    "groupId": "5e1234c17a3e5a48f5497de3",
    "cidrBlock": "192.0.2.0/24",
    "ipAddress": "192.0.2.0",
    "comment": "Sample"
   }
