============
$maxDistance
============

.. default-domain:: mongodb

.. operator:: $maxDistance

   The :operator:`$maxDistance` operator specifies a maximum distance
   for which to return the results of a geospatial :operator:`$near` or
   :operator:`$nearSphere` query. Specify the maximum distance in the
   units specified by the coordinate system. For
   `GeoJSON <http://geojson.org/geojson-spec.html>`_ ``Point`` objects,
   specify the distance in meters, not radians.

   The ``2d`` and ``2dsphere`` geospatial indexes both support
   :operator:`$maxDistance`.

   The following example query returns documents with location values
   that are ``10`` or fewer units from the point ``[ 100 , 100 ]``.

   .. code-block:: javascript

      db.collection.find( { loc : { $near : [ 100 , 100 ] ,
                                    $maxDistance: 10 }
                          } )

   MongoDB orders the results by their distance from ``[ 100 , 100 ]``.
   The operation returns the first 100 results, unless you modify the
   query with the :method:`cursor.limit()` method.
