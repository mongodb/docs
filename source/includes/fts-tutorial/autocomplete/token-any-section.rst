      Copy and paste the following query into the 
      :guilabel:`Query Editor`, and then click the :guilabel:`Search` 
      button in the :guilabel:`Query Editor`.

      .. code-block:: json

         [
           {
             $search: {
               autocomplete: {
                 path: "title",
                 query: "men with",
                 tokenOrder: "any"
               }
             }
           }
         ]
