Prerequisites
-------------

.. tabs-platforms::

   tabs:
     - id: windows
       content: |

         - |service| cluster with `BI Connector for Atlas <https://www.mongodb.com/docs/atlas/bi-connection/>`_
           enabled

         - Create a system :ref:`Data Source Name (DSN) <atlas-create-system-dsn>`
           that uses the MongoDB ODBC driver

         - Tableau version 10.3 or later

     - id: macos
       content: |

         - |service| cluster with `BI Connector for Atlas <https://www.mongodb.com/docs/atlas/bi-connection/>`_
           enabled

         - Create a system :ref:`Data Source Name (DSN) <atlas-create-system-dsn>`
           that uses the MongoDB ODBC driver

         - Tableau version 10.4 or later
