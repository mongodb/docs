.. list-table:: 
   :header-rows: 2
   :widths: 30 70 

   * - Column Name 
     - Column Value Description 

   * - :guilabel:`Name`
     - Name of the {+model-api-key+}.

   * - :guilabel:`Secret Key`
     - Redacted {+model-api-key+} associated with the {+model-api-key+}
       name.

   * - :guilabel:`Created On (UTC)`
     - Date in UTC when the user created the {+model-api-key+}.

   * - :guilabel:`Last Used (UTC)`
     - Date in UTC when the {+model-api-key+} was last used. Value is
       ``Never`` if the key is new or has never been used.

   * - :guilabel:`Project`
     - Name of the project that is linked with the {+model-api-key+}.

   * - :guilabel:`Created By`
     - Name of the user who created the key.

   * - :guilabel:`Actions`
     - Actions you can take on the key. You can click:

       - :icon-fa5:`pencil-alt` to edit the {+model-api-key+}.
       - :icon-fa5:`trash-alt` to delete the {+model-api-key+}.
