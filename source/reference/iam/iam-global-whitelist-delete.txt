.. _mcli-iam-global-whitelist-delete:

===================================
mongocli iam globalWhitelist delete
===================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``iam globalWhitelist delete`` command deletes a global whitelist 
entry for an |onprem| instance. You must confirm the operation when 
you run this command without the ``--force`` option.

You must have credentials with the :opsmgr:`Global Owner 
</reference/user-roles/#Global-Owner>` role to delete a global 
whitelist entry. 

.. _mcli-iam-global-whitelist-delete-syntax:

Syntax 
------

.. code-block:: sh 

   mongocli iam globalWhitelist|ipwhitelist delete|rm <whitelist-entry-id>
        [ --force ]
        [ --profile|-P <profile-name> ]

.. include:: /includes/fact-command-line-help.rst

.. _mcli-iam-global-whitelist-delete-args:

Arguments 
---------

.. list-table:: 
   :header-rows: 1
   :widths: 20 10 60 10

   * - Option
     - Type 
     - Description 
     - Required?

   * - ``<whitelist-entry-id>`` 
     - string
     - Unique identifier of the global whitelist entry to delete.
     - yes

.. _mcli-iam-global-whitelist-delete-options:

Options 
-------

.. list-table:: 
   :header-rows: 1
   :widths: 20 10 60 10

   * - Option
     - Type 
     - Description 
     - Required?

   * - ``--force`` 
     - 
     - Flag that indicates that the global whitelist entry can 
       be deleted without requiring confirmation.
     - no

   * - ``--profile``, ``-P`` 
     - string 
     - Name of the profile where your credentials are saved. If omitted, 
       uses the {+default-profile+}. To learn more about creating a 
       profile, see :ref:`mcli-configure`.
       
       You must have credentials with the :opsmgr:`Global Owner 
       </reference/user-roles/#Global-Owner>` role to delete a global 
       whitelist entry. 
     - no

.. _mcli-iam-global-whitelist-delete-output:

Output
------

If the command succeeds, it prints the following output to the terminal.
If the command returns errors, see :ref:`Troubleshooting
<troubleshooting>` for recommended solutions.

.. code-block:: sh
   :copyable: false

   Whitelist entry '<whitelist-entry-id>' deleted

.. _mcli-iam-global-whitelist-delete-example:

Example
-------

The following command deletes the specified whitelist entry for an 
|onprem| instance using the {+default-profile+}, which contains 
the credentials for accessing the instance. The command does 
not prompt for confirmation because the ``--force`` option is 
specified with the command.

.. code-block:: sh

   mongocli iam globalWhitelists delete 5f5bad7a57aef32b04ed0210 --force 

The previous command prints the following output to the terminal.

.. code-block:: json
   :copyable: false

   Whitelist entry '5f5bad7a57aef32b04ed0210' deleted
