.. _mcli-cm-maintenanceWindow-list-command:

=============================================
mongocli cloud-manager maintenanceWindow list
=============================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``maintenanceWindow list`` command retrieves the configuration of
all |cloud-short| :cloudmgr:`maintenance windows </reference/api/maintenance-windows>`
for the specified project. You can also retrieve the configuration of all
maintenance windows through the :cloudmgr:`API 
</reference/api/maintenance-windows-get-all/>`.

.. _mcli-cm-maintenanceWindow-list-syntax:

Syntax 
------

.. code-block:: text 

   mongocli cloud-manager|cm maintenanceWindow list|ls
        [ --output|-o <output-format> ]
        [ --profile|-P <profile-name> ]
        [ --projectId <project-ID> ]

.. _mcli-cm-maintenanceWindow-list-options: 

Options 
-------

.. list-table::
   :header-rows: 1
   :widths: 20 10 60 10

   * - Option
     - Type
     - Description
     - Required?

   * - ``--output``, ``-o``
     - string 
     - .. include:: /includes/extracts/fact-basic-options-output.rst
     - no

   * - ``--profile``, ``-P``
     - string
     - Name of the profile where the public and private 
       keys for the project are saved. If omitted, uses the 
       {+default-profile+}. To learn more about creating a 
       profile, see :ref:`mcli-configure`.
     - no

   * - ``--projectId``
     - string
     - Unique identifier of the project that contains the 
       cluster. If omitted, uses the project ID in the profile or 
       :ref:`environment variable <mcli-env-var>`.
     - no

Example 
-------

The following example uses the ``mongocli cm maintenanceWindow
list`` command to retrieve the configurations of all the maintenance windows
for the project specified in the profile called ``myCm``.

.. code-block:: none
   
   mongocli cm maintenanceWindow list -P myCm

The above command produces the following output. For more information about
these fields, see :ref:`Output <mcli-cm-maintenanceWindow-list-output>`.

.. code-block:: json
   :copyable: false
