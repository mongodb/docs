Example Request
---------------

.. code-block:: shell

   curl --include --user "{username:apiKey}" --digest \
     --header "Accept: application/json" \
     --header "Content-Type: application/json" \
     "https://cloud.mongodb.com/api/atlas/v1.0/orgs?pretty=true"
