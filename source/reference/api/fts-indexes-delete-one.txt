Request Parameters
------------------

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :stub-columns: 1
   :widths: 15 10 10 65

   * - Path Element
     - Type
     - Necessity
     - Description

   * - ``GROUP-ID``
     - string
     - Required
     - Unique identifier for the :ref:`project <projects>`
       that contains the specified cluster.

   * - ``CLUSTER-NAME``
     - string
     - Required
     - Name of the cluster containing the database and collection
       with one or more |fts| indexes.

   * - ``INDEX-ID``
     - string
     - Required
     - Unique identifier of the |fts| index. Use the
       :ref:`fts-indexes-get-all` |api| endpoint to find the IDs
       of all |fts| indexes.
