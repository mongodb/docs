============================================
atlas organizations apiKeys accessLists list
============================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

List IP access list entries for your API Key.

Syntax
------

.. code-block::

   atlas organizations apiKeys accessLists list <apiKeyID> [options]
