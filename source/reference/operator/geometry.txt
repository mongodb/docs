=========
$geometry
=========

.. default-domain:: mongodb

.. operator:: $geometry

   .. versionadded:: 2.4

   The :operator:`$geometry` operator specifies a `GeoJSON object
   <http://geojson.org/geojson-spec.html>`_ for a geospatial query
   operators. For details on using :operator:`$geometry` with an
   operator, see the operator:

   - :operator:`$near`

   - :operator:`$geoWithin`

   - :operator:`$geoIntersects`
