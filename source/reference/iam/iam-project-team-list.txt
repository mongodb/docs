.. _iam-project-team-list:

==============================
mongocli iam project team list
==============================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``mongocli iam project team list`` command lists teams with access 
to the specified project using the specified profile.

.. _mcli-iam-project-team-list-syntax:

Syntax 
------

.. code-block:: text 

   mongocli iam project|s team|s list|ls 
        [ --projectId <project-id>]
        [ --output|-o <output-format> ]
        [ --profile|-P <profile-name> ]

.. include:: /includes/fact-command-line-help.rst

.. _mcli-iam-team-list-options:

Options 
-------

.. list-table:: 
   :header-rows: 1
   :widths: 20 10 60 10

   * - Option 
     - Type 
     - Description 
     - Required? 

   * - ``--projectId`` 
     - string 
     - Unique identifier of the project for which to list teams with 
       access. If omitted, uses the project ID in 
       your profile or :ref:`environment variable <mcli-env-var>`.
     - no

   * - ``--output``, ``-o``
     - string 
     - .. include:: /includes/extracts/fact-basic-options-output.rst
     - no

   * - ``--profile``, ``-P`` 
     - string 
     - Name of the profile that contains the access information. If
       omitted, uses the {+default-profile+}. 
     - no 

.. _mcli-iam-team-list-output:

Output 
------

.. include:: /includes/command-output-intro.rst

.. tabs::

   .. tab:: Default
      :tabid: default

      .. code-block:: none

         ID
         5f6a61ad7840ac2890424ce2
         5f6a5c924166e52c48b212f3

   .. tab:: JSON
      :tabid: json

      The JSON response returns an array of teams with access to the 
      specified project, with the following details about each team:

      .. list-table::
         :widths: 20 80
         :header-rows: 1

         * - Field
           - Description

         * - ``roleNames``
           - Array of roles granted to users on the team for this 
             project.

         * - ``teamId``
           - The unique identifier of the team.

.. _mcli-iam-team-list-examples:

Example
-------

The following command lists teams with access to an |service| project. 
It specifies a project and uses the {+default-profile+} for 
accessing |service|.

.. code-block:: text 

   mongocli iam team list --projectId 5dd6faab74979f10f39ffb78 -o json

The previous command prints the following in the specified format to 
the terminal. For more information on these fields, see 
:ref:`Output <mcli-iam-team-list-output>`.

.. code-block:: json 
   :copyable: false 

   {
     "links": [...],
     "roleNames": [
       "GROUP_READ_ONLY"
     ],
     "teamId": "5f6a61ad7840ac2890424ce2",
   },
   {
     "links": [...],
     "roleNames": [
       "GROUP_CLUSTER_MANAGER",
       "GROUP_DATA_ACCESS_ADMIN"
     ],
     "teamId": "5f6a5c924166e52c48b212f3",
   }
