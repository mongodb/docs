Resource
--------

.. code-block:: http

   POST /groups/{GROUP-ID}/clusters/{CLUSTER-ID}/restoreJobs

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :widths: 20 20 60
   :header-rows: 1
   :stub-columns: 1

   * - Name
     - Type
     - Description

   * - GROUP-ID
     - string
     - Unique identifier of the :term:`group` that owns the job.

   * - CLUSTER-ID
     - string
     - Unique identifier of the cluster that the job represents.

Example Request
---------------

Create a restore job that transfers a compressed snapshot using
:abbr:`SCP (secure copy)`.

.. only:: cloud

   .. cssclass:: copyable-code
   .. code-block:: sh
      :emphasize-lines: 6-19

      curl --user "{USERNAME}:{APIKEY}" --digest \
       --header "Accept: application/json" \
       --header "Content-Type: application/json" \
       --include \
       --request POST "https://cloud.mongodb.com/api/public/v1.0/groups/{GROUP-ID}/clusters/{CLUSTER-ID}/restoreJobs?pretty=true" \
       --data '
        {
          "delivery" : {
            "formatName" : "ARCHIVE",
            "hostname" : "restore.example.com",
            "methodName" : "SCP",
            "password" : "{SCP-PASSWORD}",
            "passwordTypeName" : "TEXT",
            "port" : "8443",
            "targetDirectory" : "/data/restore",
            "username" : "{SCP-USERNAME}"
          },
          "snapshotId" : "{SNAPSHOT-ID}"
        }'
