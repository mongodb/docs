.. _sample-data:

==============================
Sample Data for Atlas Clusters
==============================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

|service| provides sample datasets you can load into your |service|
clusters. You can use these datasets to quickly get started
experimenting with data interactivity tools such as the
|service| :ref:`data-explorer` and :charts:`MongoDB Charts </>`.

For a full list of the available sample datasets see
:ref:`available-sample-datasets`. Each dataset page contains
information on the databases, collections, and indexes in the dataset.

For instructions on loading sample data into your |service| cluster,
see :ref:`load-sample-data`.

Tutorials Using Sample Data Sets
--------------------------------

Once you have loaded sample data into your |service| cluster, the
following tutorials provide some practical examples of utilizing
the sample data.

MongoDB Atlas Tutorials
~~~~~~~~~~~~~~~~~~~~~~~

The :ref:`atlas-getting-started` tutorial walks through setting up
an |service| cluster and populating that cluster with sample data.

MongoDB Charts Tutorials
~~~~~~~~~~~~~~~~~~~~~~~~

These tutorials walk through visualizing your sample data using
:charts:`MongoDB Charts </>`:

- :charts:`Visualize Sales Order Data
  </tutorial/order-data/order-data-tutorial-overview/>`

- :charts:`Visualize Movie Data
  </tutorial/movie-details/movie-details-tutorial-overview/>`

MongoDB University Courses that Use Sample Data Sets
----------------------------------------------------

The following `MongoDB University <https://university.mongodb.com/>`__
courses and trainings utilize the sample datasets available in
|service|.

`M001: MongoDB Basics <https://university.mongodb.com/courses/M001/about>`__
  MongoDB Basics is designed for learners brand new to MongoDB.

`M121: The MongoDB Aggregation Framework <https://university.mongodb.com/courses/M121/about>`__
  This course provides you with the knowledge to use MongoDB
  Aggregation Framework with confidence in your application development
  practices.

`M220J: MongoDB for Java Developers <https://university.mongodb.com/courses/M220J/about>`__
  This course teaches you how to use MongoDB as the database for a 
  Java application.

`M220JS: MongoDB for Javascript Developers <https://university.mongodb.com/courses/M220JS/about>`__
  This course teaches you how to use MongoDB as the database for a
  Node.js application.

`M220P: MongoDB for Python Developers <https://university.mongodb.com/courses/M220P/about>`__
  This course will teach you how to use MongoDB as the database for a
  Python application.

All `In-Person Training <https://www.mongodb.com/products/training/private>`__
  Get quickly ramped on MongoDB with comprehensive private training
  programs for developers and operations teams.
