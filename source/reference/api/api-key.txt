Endpoints
---------

Use the following endpoints to manage personal |api| keys.

.. list-table::
   :header-rows: 1
   :stub-columns: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :doc:`/users/{USER-ID}/keys </reference/api/api-key/api-key-get-all>`
     - Retrieve one user's personal |api| keys.

   * - ``PATCH``
     - :doc:`/users/{USER-ID}/keys/{KEY-ID} </reference/api/api-key/enable-disable-api-key>`
     - Enable or disable one personal |api| key.

   * - ``DELETE``
     - :doc:`/users/{USER-ID}/keys/{KEY-ID} </reference/api/api-key/delete-one-api-key>`
     - Remove one personal |api| key.
