Resource
--------

.. code-block:: http

   DELETE /groups/{GROUP-ID}/maintenanceWindow

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :widths: 20 20 60
   :header-rows: 1

   * - Name
     - Type
     - Description

   * - ``GROUP-ID``
     - string
     - Unique identifier of the project for which you want to reset
       the Maintenance Window.
