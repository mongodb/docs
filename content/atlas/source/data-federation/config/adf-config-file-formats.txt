==========================================
Configuration File Formats for Data Stores
==========================================

.. meta::
   :keywords: atlas data federation, data store configuration
   :description: Configure data store providers in Atlas Data Federation, including AWS S3, Azure Blob Storage, GCP, HTTP endpoints, Atlas clusters, and online archives.

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

Atlas Data Federation supports various data store providers as
federated database instance stores. You must define mappings in
your federated database instance to data stores to run queries
against your data.

.. important::

    Information in your storage configuration is visible at MongoDB
    and stored as operational data to improve the performance of
    Atlas Data Federation. Avoid using personally
    identifiable information (PII) in your configurations.

Select your provider from the dropdown to configure your data store:
