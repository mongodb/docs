.. step:: In |cloud|, go to the :guilabel:`Servers` page for your project.

   a. If it's not already displayed, select the organization that 
      contains your desired project from the |ui-org-menu| in the navigation bar.

   #. If it's not already displayed, select your desired project from 
      the :guilabel:`Projects` menu in the navigation bar.

   #. In the sidebar, click :guilabel:`Servers` under the 
      :guilabel:`Database` heading.

   The `Servers <https://cloud.mongodb.com/go?l=https%3A%2F%2Fcloud.mongodb.com%2Fv2%2F%3Cproject%3E%23%2Fdeployment%2Fservers>`__ page 
   displays.
