Use this endpoint to get a specific {+data-lake+} associated to a
|service| project. 

Syntax
------

.. code-block:: none

   GET /groups/{GROUP-ID}/dataLakes/{NAME}

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :widths: 15 10 75

   * - Path Element
     - Required/Optional
     - Description

   * - ``GROUP-ID``

     - Required.

     - The unique identifier for the :atlas:`project </api/#group-id>`.

   * - ``NAME``

     - Required

     - The name of the {+data-lake+} that you want to retrieve. 
       
       You can use the :doc:`/reference/api/dataLakes-get-all-tenants`
       endpoint to retrieve all {+data-lake+}s associated with the
       project. The ``name`` field in the response of that endpoint
       corresponds to the ``NAME`` parameter here.
