============================
Serverless Private Endpoints
============================

.. default-domain:: mongodb

.. include:: /includes/fact-projects-groups-synonyms.rst

Resources
---------

.. include:: /includes/api-base-url.rst

The following lists the endpoints available for managing 
private endpoints for {+serverless-instances+}:

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - GET
     - :doc:`/groups/{GROUP-ID}/privateEndpoint/serverless/instance/
       {INSTANCE-NAME}/endpoint 
       </reference/api/serverless-private-endpoints-get-all>`
     - Returns all private endpoints for one |service| 
       {+serverless-instance+}.

   * - GET
     - :doc:`/groups/{GROUP-ID}/privateEndpoint/serverless/instance/
       {INSTANCE-NAME}/endpoint/{ENDPOINT-ID} 
       </reference/api/serverless-private-endpoints-get-one>`
     - Returns one private endpoint for one |service| 
       {+serverless-instance+}.

   * - POST
     - :doc:`/groups/{GROUP-ID}/privateEndpoint/serverless/instance/
       {INSTANCE-NAME}/endpoint 
       </reference/api/serverless-private-endpoints-create-one>`
     - Creates one private endpoint for one |service| 
       {+serverless-instance+}.

   * - DELETE
     - :doc:`/groups/{GROUP-ID}/privateEndpoint/serverless/instance/
       {INSTANCE-NAME}/endpoint/{ENDPOINT-ID} 
       </reference/api/serverless-private-endpoints-delete-one>`
     - Deletes one private endpoint for one |service|
       {+serverless-instance+}. 

   * - PATCH
     - :doc:`/groups/{GROUP-ID}/privateEndpoint/serverless/instance/
       {INSTANCE-NAME}/endpoint/{ENDPOINT-ID} 
       </reference/api/serverless-private-endpoints-update-one>`
     - Updates one private endpoint for one |service| 
       {+serverless-instance+}.
