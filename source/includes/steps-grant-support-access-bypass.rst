   .. step:: Grant temporary access.
      
      a. Locate the {+database-deployment+} to which to grant access 
         from the list of {+database-deployments+}.
      
      #. Select the ellipsis icon (:guilabel:`...`) to the right of the 
         {+database-deployment+}.
      
      #. Select :guilabel:`Grant Temporary Infrastructure Access to MongoDB Support`.

      #. Select :guilabel:`Atlas cluster infrastructure and all cluster logs`.
      
      #. Click :guilabel:`Grant Access`.
