===================================
Administration Console : Backup Tab
===================================

.. meta::
   :description: Access and manage Ops Manager Backup resources, including jobs, daemons, blockstores, oplog stores, and snapshot storage, from the Admin interface Backup tab.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

The :guilabel:`Admin` interface's :guilabel:`Backup` tab provides
information on Backup resources, including :term:`jobs <backup job>`,
:opsmgr:`daemons </reference/glossary/#term-Backup-Daemon>`, and
:opsmgr:`blockstores </reference/glossary/#std-term-backup-blockstore-database>`.
For an overview of Backup and Backup resources, see
:doc:`/core/backup-overview`. Use this tab to manage all
aspects of |onprem| Backup, from monitoring active jobs and
daemons to configuring snapshot storage and oplog stores.
