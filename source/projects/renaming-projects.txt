================
Rename a Project
================

.. meta::
   :description: Update the name of an existing project using the Rename option, ensuring the new name is unique and 1-50 characters long.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

To update the name of an existing project, use the :guilabel:`Rename` option.

About this Task
---------------

The new name must be 1-50 characters long and can't be in use by another project.

Steps
-----

1. From the project view, click the :guilabel:`Home` icon in the top-left corner.

#. Open the menu next to your project of choice. Select :guilabel:`Rename`.

#. Enter a new project name.

#. Click :guilabel:`Save`.
