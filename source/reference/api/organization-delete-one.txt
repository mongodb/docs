Syntax
------

.. code-block:: none

   DELETE /api/atlas/v1.0/orgs/{ORG-ID}

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :widths: 15 10 75

   * - Path Element
     - Required/Optional
     - Description

   * - ``ORG-ID``

     - Required.

     - The unique identifier for the |service|
       :ref:`organization <organizations>` to delete.

Request Query Parameters
~~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :widths: 30 10 60

   * - Field
     - Required/Optional
     - Description

   * - ``envelope``

     - Optional.

     - A boolean that specifies whether or not to wrap the response in an
       :ref:`envelope <api-envelope>`.

       Defaults to ``false``.

   * - ``pretty``

     - Optional.

     - A boolean that specifies whether or not to return a "pretty-printed"
       JSON document.

       Defaults to ``false``.

Example
-------

Request
~~~~~~~

.. class:: copyable-code
.. code-block:: none

   curl -u "username:apiKey" --digest -X DELETE "https://cloud.mongodb.com/api/atlas/v1.0/orgs/5a38289880eef5072af2f1f1"
