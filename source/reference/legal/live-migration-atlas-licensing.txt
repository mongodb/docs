
===============================
Ops Manager Migration Licensing
===============================

.. meta::
   :description: Understand the licensing terms for using the Live Migration feature in Ops Manager to migrate MongoDB to Atlas.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

This :doc:`/reference/legal/live-migration-atlas-license`
covers MongoDB Community customers who wish to use the :ref:`Live Migration <migrate-to-atlas>`
feature, available in |onprem| 5.0 and later, to :ref:`migrate MongoDB to Atlas <migrate-community-to-atlas>`.

The use of the |onprem| software with this license agreement is restricted
to the use of the :ref:`Live Migration <migrate-to-atlas>` feature only.

:doc:`/reference/legal/live-migration-atlas-license`
  Provides the full text of the Migration License agreement.
