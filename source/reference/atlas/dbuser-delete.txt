.. _mcli-atlas-dbuser-delete-command:

============================
mongocli atlas dbuser delete
============================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``dbuser delete`` command deletes the specified MongoDB 
database user on the specified |service| cluster. The command prompts 
you to confirm the operation when you run the command without the 
``--force`` option.

You can also delete a MongoDB database user through the |service| 
:atlas:`UI 
</security-add-mongodb-users/index.html#delete-database-users>` or 
:atlas:`API </reference/api/database-users-delete-a-user/>`.

.. _dbuser-delete-syntax:

Syntax
------

.. code-block:: text

   mongocli atlas dbuser delete <username>   
        [ --force ]
        [ --authDB ]
        [ --profile|-P <profile-name> ]
        [ --projectId <project-ID> ]

.. include:: /includes/fact-command-line-help.rst

.. _dbuser-delete-options:

Options
-------

.. list-table::
   :header-rows: 1
   :widths: 20 10 60 10

   * - Option 
     - Type
     - Description
     - Required?

   * - ``--force``
     - 
     - Flag that indicates that the user can 
       be deleted without requiring confirmation. 
     - no

   * - ``--authDB``
     - string
     - Name of the authentication database for this user. Defaults to
       ``admin``.
     - no

   * - ``--profile``, ``-P``
     - string
     - Name of the profile where the public and private 
       keys for the project are saved. If omitted, uses the 
       ``default`` profile. To learn more about creating a 
       profile, see :ref:`mcli-configure`.
     - no

   * - ``--projectId``
     - string
     - Unique identifier of the project that contains the 
       cluster. If omitted, uses the project ID in the profile or 
       :ref:`environment variable <mcli-env-var>`.
     - no

   * - ``<username>``
     - string
     - Username for authenticating the user to MongoDB.
     - yes

.. _dbuser-delete-command-output:

Output 
------
The command prints a confirmation message similar to the 
following to the terminal if it succeeds. If the command 
prints an error, see :ref:`Troubleshooting <troubleshooting>` 
for recommended solutions.

.. code-block:: text
   :copyable: false

   DB user '<username>' deleted

where ``<username>`` is the username of the MongoDB user who was deleted.

.. _dbuser-delete-examples:

Example 
-------

The following examples use the ``mongocli atlas dbuser delete`` 
command to delete a user named ``janeDoe``.

Basic Example 
~~~~~~~~~~~~~

The following ``mongocli atlas dbuser delete`` command deletes 
a user named ``janeDoe`` after prompting for a confirmation. 
It uses the profile named ``egAtlasProfile`` to access the 
|service| cluster. 

.. code-block:: text
   :copyable: false

   ~ mongocli atlas dbuser delete janeDoe --profile egAtlasProfile 
   Are you sure you want to delete: janeDoe Yes

The command prints the following to the terminal:

.. code-block:: text
   :copyable: false

   DB user 'janeDoe' deleted

Force Example 
~~~~~~~~~~~~~

The following ``mongocli atlas dbuser delete`` command deletes 
a user named ``janeDoe`` without requiring confirmation. 
It uses the ``default`` profile for accessing the |service| 
cluster.

.. code-block:: text
   :copyable: false

   ~ mongocli atlas dbuser delete janeDoe --force

The command prints the following to the terminal:

.. code-block:: text
   :copyable: false

   DB user 'janeDoe' deleted
