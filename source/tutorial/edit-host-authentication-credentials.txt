===============================
Edit Authentication Credentials
===============================

.. default-domain:: mongodb


If your MongoDB deployment enforces access control, the |mms| agents must
authenticate to MongoDB as a user with the proper access.

Procedures
----------

+++++++++++++++++++++++++++++++++++++++++++++++

.. only:: onprem or classic
