Import Archive from S3
======================

.. default-domain:: mongodb

.. include:: /includes/fact-atlas-free-tier-limits.rst

You can restore data archived to |s3| buckets using |mongoimport|. This 
page has a sample procedure to import archived data using the |aws| CLI 
and the MongoDB |mongoimport| utility.

Prerequisites 
-------------

Before you begin, you must: 

- `Install 
  <https://docs.aws.amazon.com/cli/latest/userguide/cli-chap-install.html>`__ 
  the |aws| CLI
- `Configure 
  <https://docs.aws.amazon.com/cli/latest/userguide/cli-chap-configure.html#cli-quick-configuration>`__ 
  the |aws| CLI
- Install the :doc:`mongoimport </import/mongoimport>` tool
