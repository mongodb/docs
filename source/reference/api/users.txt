=====
Users
=====

.. only:: onprem

   The ``users`` resource allows you to create and update users and
   retrieve user information. This resource also provides an endpoint for
   creating the first user in a system and retrieving a :term:`Public API key`
   for use in future API calls. The endpoint for creating the first user is the
   only endpoint you can use without first having an API key.

.. only:: cloud

   The ``users`` resource allows you to create and update users and retrieve
   user information.

Endpoints
---------

The following endpoints are available for ``users``.

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``

     - :doc:`/api/public/v1.0/users/{USER-ID} </reference/api/user-get-by-id>`

     - Retrieve a user by their user id.

   * - ``GET``

     - :doc:`/api/public/v1.0/users/byName/{USER-NAME} </reference/api/user-get-by-name>`

     - Retrieve a user by their name.

   * - ``POST``

     - :doc:`/api/public/v1.0/users </reference/api/user-create>`

     - Create a new user.

   * - ``PATCH``

     - :doc:`/api/public/v1.0/users/{USER-ID} </reference/api/user-update>`

     - Update an existing user.

.. only:: onprem

   .. list-table::
      :widths: 5 55 40

      * - ``PATCH``

        - :doc:`/api/public/v1.0/unauth/users </reference/api/user-create-first>`

        - Create the first |mms| user.
