.. setting:: Daily Retention of Snapshots (in Days)

   *Type*: integer

   *Default*: 0

   
   Specifies how many days a daily snapshot is stored.
   
   Accepted values are:
   
   
   - **No daily retention**
   - One of the following integers: **3**, **4**, **5**, **6**, **7**,
     **15**, **30**, **60**, **90**, **120**, **180**, or **360**
   
   
   Corresponds to :setting:`brs.snapshotSchedule.retention.daily`.
