Endpoints
---------

The following lists the endpoints available for :doc:`alert configuration
</configure-alerts>`. An :doc:`alert configuration </configure-alerts>`
defines the conditions that trigger an alert and the methods of
notification.

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :doc:`/api/atlas/v1.0/alertConfigs/matchers/fieldNames </reference/api/alert-configurations-get-matchers-field-names>`
     - Get all field names that the ``matchers.fieldName`` parameter 
       accepts when you 
       :doc:`create </reference/api/alert-configurations-create-config/>` 
       or :doc:`update </reference/api/alert-configurations-update-config/>` 
       an Alert Configuration.

   * - ``GET``

     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/alertConfigs </reference/api/alert-configurations-get-all-configs>`

     - Get all alert configurations for the project associated to 
       ``{GROUP-ID}``.

   * - ``POST``

     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/alertConfigs </reference/api/alert-configurations-create-config>`

     - Create an alert configuration for the project associated to 
       ``{GROUP-ID}``.

   * - ``GET``

     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/alertConfigs/{ALERT-CONFIG-ID} </reference/api/alert-configurations-get-config>`

     - Get the alert configuration specified to ``{ALERT-CONFIG-ID}`` 
       for the project associated to ``{GROUP-ID}``.

   * - ``PUT``

     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/alertConfigs/{ALERT-CONFIG-ID} </reference/api/alert-configurations-update-config>`

     - Update the alert configuration specified to ``{ALERT-CONFIG-ID}``
       for the project associated to ``{GROUP-ID}``.

   * - ``PATCH``

     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/alertConfigs/{ALERT-CONFIG-ID} </reference/api/alert-configurations-enable-disable-config>`

     - Enable/disable the alert configuration specified to 
       ``{ALERT-CONFIG-ID}`` for the project associated to ``{GROUP-ID}``.

   * - ``DELETE``

     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/alertConfigs/{ALERT-CONFIG-ID} </reference/api/alert-configurations-delete-config>`

     - Delete the alert configuration specified to 
       ``{ALERT-CONFIG-ID}`` for the project associated to 
       ``{GROUP-ID}``.

   * - ``GET``

     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/alertConfigs/{ALERT-CONFIG-ID}/alerts </reference/api/alert-configurations-get-open-alerts>`

     - Get all open alerts for the alert configuration 
       specified to ``{ALERT-CONFIG-ID}`` for the project associated
       to ``{GROUP-ID}``.
