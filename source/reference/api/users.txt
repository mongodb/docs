=====
Users
=====

.. default-domain:: mongodb

.. include:: /includes/fact-groups-projects-synonyms.rst

.. include:: /includes/api/base-api-uri.rst

.. only:: onprem

   The ``users`` resource allows you to create and update users and
   retrieve user information. This resource also provides an endpoint for
   creating the first user in a system and retrieving a :term:`Public API key`
   for use in future API calls. The endpoint for creating the first user is the
   only endpoint you can use without first having an API key.

.. only:: cloud

   The ``users`` resource allows you to create and update users and retrieve
   user information.

Endpoints
---------

The following endpoints are available for ``users``.

.. list-table::
   :header-rows: 1
   :widths: 10 50 40

   * - Method
     - Endpoint
     - Description

   * - ``GET``

     - :doc:`/users/{USER-ID} </reference/api/user-get-by-id>`

     - Retrieve one user by their user ID.

   * - ``GET``

     - :doc:`/users/byName/{USER-NAME} </reference/api/user-get-by-name>`

     - Retrieve one user by their name.

   * - ``POST``

     - :doc:`/users </reference/api/user-create>`

     - Create one new user.

   * - ``PATCH``

     - :doc:`/users/{USER-ID} </reference/api/user-update>`

     - Update one existing user.

.. only:: onprem

   .. list-table::
      :widths: 10 50 40

      * - ``POST``

        - :doc:`/unauth/users </reference/api/user-create-first>`

        - Create the first |onprem| user.

      * - ``DELETE``

        - :doc:`/users/{USER-ID} </reference/api/users/delete-one-user>`

        - Delete one |onprem| user.
