100.7.1 Changelog
-----------------

*Released 2023-05-24*

This release fixes a few bugs and adds downloads for the following platforms:

- macOS 11 on ARM
- RedHat Enterprise Linux 9 (x86 and ARM)
- Amazon Linux 2023 (x86 and ARM)

The tools were compiled with Go 1.19.9.
