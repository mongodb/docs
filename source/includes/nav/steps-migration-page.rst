.. step:: In |service|, go to the :guilabel:`Live Migration to Atlas` page for your project.

   .. include:: /includes/nav/fact-nav-preview.rst

   a. If it's not already displayed, select the 
      organization that contains your project from the
      |ui-org-menu| in the navigation bar.

   #. If it's not already displayed, select your project 
      from the :guilabel:`Projects` menu in the navigation bar.

   #. In the sidebar, click :guilabel:`Migration` under 
      the :guilabel:`Services` heading.

      The `Live Migration to Atlas <https://cloud.mongodb.com/go?l=https%3A%2F%2Fcloud.mongodb.com%2Fv2%2F%3Cproject%3E%23%2Fmigration>`__ page displays.
