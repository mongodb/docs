=================================================
Deploy a {+FDI+} Data Store
=================================================

.. meta::
   :keywords: atlas cli, atlas ui
   :description: Deploy a federated database instance data store on AWS S3, Azure Blob Storage, GCP, HTTP, Atlas clusters, or online archives using the Atlas UI or CLI.

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol


A {+fdi+} data store connects |service| to a cloud storage
source so you can run federated queries across your data. You
can deploy a data store to |aws| |s3|, {+az-bs+}, {+gcs+},
|http| or |https| endpoints, an |service| cluster, or an
{+Online-Archive+}.

To deploy a {+fdi+} Data Store, select your cloud provider from the
dropdown menu:
