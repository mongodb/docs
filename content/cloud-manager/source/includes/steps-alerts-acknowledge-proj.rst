.. procedure::
   :style: normal

   .. include:: /includes/nav/steps-project-alerts.rst
      
   .. step:: Navigate to the :guilabel:`Open Alerts` tab.

   .. step:: Acknowledge alerts
      
      To acknowledge a single alert, on the line item for the alert, click 
      :guilabel:`Acknowledge`.
      
      To acknowledge multiple, but not all, alerts:
      
      a. Check the checkbox to the left of each alert to acknowledge.
      
      #. Click :guilabel:`Mark Acknowledge` above the table.
      
      To acknowledge all alerts: 
      
      a. Select the checkbox in the table header to select every alert.
      
      #. Click :guilabel:`Mark Acknowledge` above the table.
      
   .. step:: Select the time period for which to acknowledge the alert.

      Click the time frame for which you no longer wish to receive alerts.
      
      |mms| sends no further alert messages for the period of time you
      select.
      
   .. step:: If all Alerts were checked, select :guilabel:`All Visible` or :guilabel:`All Open Alerts`.
      
      If all alerts are checked, then another set of radio buttons appear:
      
      - Click :guilabel:`Acknowledge all visible checked alerts` to 
        acknowledge all alerts that were loaded onto the page.
      
      - Click :guilabel:`Acknowledge all open alerts` to acknowledge all 
        alerts: checked, unchecked, visible and not visible on the current 
        page.
      
   .. step:: Click :guilabel:`Acknowledge` to confirm.
