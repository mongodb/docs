==========================
Set up a Local Environment
==========================

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

To get started with Grove, you need to set up a local environment. This involves
installing dependencies, and optionally, configuring environment variables.

For intelligent IDE hints about deprecated Driver APIs and code correctness,
open the relevant code example test project as the root of your IDE.
