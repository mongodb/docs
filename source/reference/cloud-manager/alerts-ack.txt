=========================================
mongocli cloud-manager alerts acknowledge
=========================================


.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol


The ``alerts acknowledge`` command acknowledges one alert for the
specified |cloud-short| project.

.. _cm-alerts-ack-syntax:

Syntax
------

.. code-block:: sh

   mongocli cloud-manager|cm alert|alerts acknowledge|ack <alertID>
        [ --until <acknowledgement time> ]
        [ --forever ]
        [ --comment <optional comment> ]
        [ --output|-o json ]
        [ --profile|-P <profile-name> ]
        [ --projectId <id-of-project> ]

Examples
--------

Acknowledge for 1 Month
~~~~~~~~~~~~~~~~~~~~~~~

The following example uses the ``mongocli cm acknowledge`` command to
acknowledge one alert that occurred for the specified project for one
month.

.. code-block:: sh

   mongocli cm alerts acknowledge {ALERT-ID} \
            --projectId {PROJECT-ID} \
            --until 2020-06-28T17:09:36Z

The previous command returns the following |json| document. To learn
more about these fields, see :ref:`cm-alerts-ack-output`.

.. code-block:: json
   :copyable: false
   :linenos:

   {
     "acknowledgedUntil": "2020-06-04T20:26:26Z",
     "acknowledgementComment": "acknowledging for one month",
     "acknowledgingUsername": "mfnrwcia",
     "alertConfigId": "{ALERT-CONFIG-ID}",
     "created": "2020-05-04T20:23:01Z",
     "groupId": "{PROJECT-ID}",
     "id": "{ALERT-ID}",
     "lastNotified": "2020-05-04T20:23:09Z",
     "status": "OPEN",
     "updated": "2020-05-04T20:26:26Z",
     "eventTypeName": "USERS_WITHOUT_MULTI_FACTOR_AUTH",
   }


Acknowledge Forever
~~~~~~~~~~~~~~~~~~~

The following example uses the ``mongocli cm acknowledge`` command to
acknowledge one alert that occurred for the specified project forever.

.. code-block:: sh

   mongocli cm alerts acknowledge {ALERT-ID} \
            --projectId {PROJECT-ID} \
            --forever

The previous command returns the following |json| document. To learn
more about these fields, see :ref:`cm-alerts-ack-output`.

   :copyable: false
   :linenos:

     "acknowledgedUntil": "2120-06-04T20:26:26Z",
     "acknowledgementComment": "acknowledging forever",
     "acknowledgingUsername": "mfnrwcia",
     "alertConfigId": "{ALERT-CONFIG-ID}",
     "created": "2020-05-04T20:23:01Z",
     "eventTypeName": "USERS_WITHOUT_MULTI_FACTOR_AUTH",
