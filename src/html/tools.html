MongoDB Database Tools

mongodump

Creates binary export of the contents of a MongoDB deployment.

mongorestore

Restore data from the binary dump created by mongodump

mongoimport

Import Extended JSON, CSV, or TSV data to a MongoDB deployment.

mongoexport

Export data from a MongoDB deployment to Extended JSON or CSV.

mongostat

Displays an overview of the status of a mongod or mongos instance.

mongotop

Tracks the amount of time a mongod instance spends reading and writing data.

mongofiles

Read, write, delete, or update files in GridFS

bsondump

Display BSON files in a human-readable format.
