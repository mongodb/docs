================================
Tools, Migrators, and Connectors
================================

.. meta::
   :description: Find MongoDB tools and connectors by goal, including migrating from a relational database, syncing clusters, connecting BI tools, and managing self-managed deployments.

MongoDB has tools and integrations for development and management, including
database management, integration, migration, and data visualization. See the
following section to find the right tool for your use case. To browse tools
by category, see the reference table later on this page.

Choose the Tool For Your Use Case
---------------------------------

The following table maps common tasks to their corresponding tools:

.. list-table::
   :header-rows: 1
   :widths: 40 60

   * - Task
     - Tools

   * - Migrate from a relational database
     - `Relational Migrator <https://www.mongodb.com/docs/relational-migrator/>`__

   * - Sync data between clusters
     - `Mongosync <https://www.mongodb.com/docs/mongosync/current/>`__

   * - Connect BI and visualization tools
     - `BI Connector <https://www.mongodb.com/docs/bi-connector/current/>`__,
       `SQL Interface <https://www.mongodb.com/docs/sql-interface/>`__,
       `Atlas Charts <https://www.mongodb.com/docs/charts/>`__

   * - Work from your IDE or AI assistant
     - `VS Code Extension <https://www.mongodb.com/docs/mongodb-vscode/>`__,
       `IntelliJ Plugin <https://www.mongodb.com/docs/mongodb-intellij/>`__,
       `MongoDB MCP Server <https://www.mongodb.com/docs/mcp-server/>`__

   * - Manage a deployment
     - `Atlas CLI <https://www.mongodb.com/docs/atlas/cli/current/>`__,
       `Cloud Manager <https://www.mongodb.com/docs/cloud-manager/>`__,
       `Ops Manager <https://www.mongodb.com/docs/ops-manager/current/>`__,
       `MongoDB CLI <https://www.mongodb.com/docs/mongocli/current/>`__

   * - Explore your data
     - `Compass <https://www.mongodb.com/docs/compass/current/>`__,
       `MongoDB Shell <https://www.mongodb.com/docs/mongodb-shell/>`__
