===========================
atlas api teams addTeamUser
===========================

.. default-domain:: mongodb

.. contents:: On this page

`Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Adds one or more MongoDB Cloud users from the specified organization to the specified team.

Teams enable you to grant project access roles to MongoDB Cloud users. You can assign up to 250 MongoDB Cloud users from one organization to one team. To use this resource, the requesting API Key must have the Organization Owner role.


Note: This endpoint is deprecated. Use Add One MongoDB Cloud User to One Team to add an active or pending user to a team.

This command is autogenerated and corresponds 1:1 with the Atlas API endpoint https://www.mongodb.com/docs/atlas/reference/api-resources-spec/v2/#tag/Teams/operation/addTeamUser.

For more information and examples, see the referenced API documentation linked above.
