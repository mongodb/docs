{+avs+} filters the documents based on the ``year`` field value that
ranges between 1980 and 2020. It generated embeddings for the query 
text using the ``voyage-4`` embedding model specified in the query and 
considered up to ``100`` nearest neighbors before returning the top ten 
results. It returns documents that summarize fantasy journey of a hero 
in the plot for movies released between 1980 and 2020.
