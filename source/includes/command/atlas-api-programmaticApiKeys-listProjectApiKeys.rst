`experimental <https://www.mongodb.com/docs/atlas/cli/current/command/atlas-api/>`_: Returns all organization API keys that you assigned to the specified project.

Users with the Project Owner role in the project associated with the API key can use the organization API key to access the resources. To use this resource, the requesting API Key must have the Project Owner role.

This command is autogenerated and corresponds 1:1 with the Atlas API endpoint https://www.mongodb.com/docs/atlas/reference/api-resources-spec/v2/#tag/Programmatic-API-Keys/operation/listProjectApiKeys.

For more information and examples, see the referenced API documentation linked above.
