Create an |fts| Index using the {+mcli+}
============================================

This part of the tutorial guides you through creating an |fts| index in the
{+mcli+} for |fts|.

.. tip::

   You can also create |fts| indexes using the |service| :ref:`UI <fts-tutorial-index-creation-ui>` 
   and |fts| :ref:`API <fts-tutorial-index-creation-api>`.

This tutorial uses the ``sample_mflix`` database and the ``movies``
collection from the :ref:`sample-mflix`.

To learn more about {+mcli+} |service| clusters search indexes, see `{+mcli+}
Atlas clusters search <https://docs.mongodb.com/mongocli/stable/command/mongocli-atlas-clusters-search-indexes/>`__.

Prerequisites
-------------

To create an |fts| index using the {+mcli+}, you must: 

- Have an API Key assigned the :authrole:`Project Data Access Admin` or
  :authrole:`Project Owner` role for your |service| project.

- Ensure that your IP address that appears in the :doc:`access list for 
  your API Key </reference/api/api-access-list/create-api-access-entries>`.

- :ref:`Install the {+mcli+} <mcli-install>`.

- :ref:`Configure the {+mcli+} <mcli-configure>`.

- Choose :ref:`static or dynamic index mapping <static-dynamic-mappings>`.
