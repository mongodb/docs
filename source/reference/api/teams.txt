=====
Teams
=====

.. default-domain:: mongodb

The ``teams`` resource provides access to retrieve or create |service|
:ref:`teams <manage-teams>`.

.. include:: /includes/api-base-url-new.rst

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 40 50

   * - Method
     - Endpoint
     - Description

   * - ``GET``

     - :doc:`/orgs/{ORG-ID}/teams </reference/api/teams-get-all>`

     - Get all teams in an organization associated with ``{ORG-ID}`` to
       which the authenticated user has access.

   * - ``GET``

     - :doc:`/orgs/{ORG-ID}/teams/{TEAM-ID} </reference/api/teams-get-one-by-id>`

     - Get the team with ID ``{TEAM-ID}`` in the organization
       associated with ``{ORG-ID}``.

   * - ``GET``

     - :doc:`/orgs/{ORG-ID}/teams/byName/{TEAM-NAME} </reference/api/teams-get-one-by-name>`

     - Get the team with name ``{TEAM-NAME}`` in the organization
       associated with ``{ORG-ID}``.

   * - ``GET``

     - :doc:`/orgs/{ORG-ID}/teams/{TEAM-ID}/users </reference/api/teams-get-all-users>`

     - Get all users assigned to the team with ID ``{TEAM-ID}`` in the
       organization associated with ``{ORG-ID}``.

   * - ``POST``

     - :doc:`/orgs/{ORG-ID}/teams </reference/api/teams-create-one>`

     - Create one team in the organization associated with
       ``{ORG-ID}}``.

   * - ``POST``

     - :doc:`/orgs/{ORG-ID}/teams/{TEAM-ID}/users </reference/api/teams-add-user>`

     - Add one user from the organization associated with ``{ORG-ID}``
       to the team with ID ``{TEAM-ID}``.

   * - ``POST``

     - :doc:`/{GROUP-ID}/teams/ </reference/api/project-add-team>`

     - Assign one |service| :ref:`team <manage-teams>` to the
       project associated with ``{GROUP-ID}``.

   * - ``PATCH``

     - :doc:`/orgs/{ORG-ID}/teams/{TEAM-ID} </reference/api/teams-rename-one>`

     - Rename one team in one |service| organization.

   * - ``PATCH``

     - :doc:`/groups/{PROJECT-ID}/teams/{TEAM-ID} </reference/api/teams-update-roles>`

     - Update the roles of one team in one |service| project.

   * - ``DELETE``

     - :doc:`/orgs/{ORG-ID}/teams/{TEAM-ID}/users/{USER-ID} </reference/api/teams-remove-user>`

     - Remove the specified user from the specified team.

   * - ``DELETE``

     - :doc:`/groups/{GROUP-ID}/teams/{TEAM-ID} </reference/api/teams-remove-from-project>`

     - Removes the specified team from the specified project.

   * - ``DELETE``

     - :doc:`/orgs/{ORG-ID}/teams/{TEAM-ID} </reference/api/teams-delete-one>`

     - Delete the team with ID ``{TEAM-ID}`` from the organization
       specified to ``{ORG-ID}``.

.. note::

   The API endpoints to :doc:`/reference/api/project-add-team/` and
   :doc:`/reference/api/project-get-teams/` are documented in the
   :doc:`/reference/api/projects/` section.
