======================
Download a Single File
======================

.. meta::
   :description: Generate and download individual JSON schema and sample document files for your MongoDB applications using Relational Migrator.

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

{+rel-mig+} can automatically generate JSON Schema and sample document files for 
your applications connected to MongoDB. This page details how to generate and 
download individual JSON schema and sample document files.

Steps
-----

You can view, download, or copy a single collection's JSON schema or sample document file.

#. Navigate to the :guilabel:`JSON Schema` tab.
#. Select a :guilabel:`Template`.
#. (Optional) Enter a keyword in the :guilabel:`Filter` text box to 
   filter the MongoDB collections.
#. Click the MongoDB collection name.
#. Click the :guilabel:`Copy` :icon-lg:`Copy` or :guilabel:`Download` :icon-lg:`Download` 
   buttons in the target code file to download or copy that file's contents.
