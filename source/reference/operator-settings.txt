==============================
|k8s-op| Installation Settings
==============================

.. meta::
   :description: Explore optional settings for deploying the Kubernetes Operator using kubectl, oc, or Helm.

.. default-domain:: mongodb

When you install the |k8s-op-short|, you can provide optional settings
that affect your deployment.  How you provide these settings depends on
the environment to which you deploy the |k8s-op-short|.

  Review the settings that you can use when you install the
  |k8s-op-short| using Helm.
