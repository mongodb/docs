Resources
---------

.. include:: /includes/api-base-url.rst

The following lists the endpoints available for managing 
:doc:`Private Endpoints </security-private-endpoint>`.

.. list-table::
   :header-rows: 1
   :widths: 10 40 50

   * - Method
     - Endpoint
     - Description

   * - GET
     - :doc:`/groups/{GROUP-ID}/privateEndpoint </reference/api/private-endpoint-get-all-private-endpoint-connections>`
     - Retrieve details for all private endpoint connections in an 
       |service| project.

   * - POST
     - :doc:`/groups/{GROUP-ID}/privateEndpoint </reference/api/private-endpoint-create-one-private-endpoint-connection>`
     - Create one private endpoint connection in an |service| project.             

   * - GET
     - :doc:`/groups/{GROUP-ID}/privateEndpoint/{privateLinkId} </reference/api/private-endpoint-get-one-private-endpoint-connection>`
     - Retrieve details for one private endpoint connection by ID in an 
       |service| project.

   * - DELETE
     - :doc:`/groups/{GROUP-ID}/privateEndpoint/{privateLinkId} </reference/api/private-endpoint-delete-one-private-endpoint-connection>`
     - Remove one private endpoint connection in an |service| project.    

   * - POST
     - :doc:`/groups/{GROUP-ID}/privateEndpoint/{privateLinkId}/interfaceEndpoints </reference/api/private-endpoint-create-one-interface-endpoint>`
     - Add one interface endpoint to a private endpoint connection in an
       |service| project.       

   * - GET
     - :doc:`/groups/{GROUP-ID}/privateEndpoint/{privateLinkId}/interfaceEndpoints/{interfaceEndpointId} </reference/api/private-endpoint-get-one-interface-endpoint>`
     - Retrieve one interface endpoint in a private endpoint connection 
       in an |service| project.  

   * - DELETE
     - :doc:`/groups/{GROUP-ID}/privateEndpoint/{privateLinkId}/interfaceEndpoints/{interfaceEndpointId} </reference/api/private-endpoint-delete-one-interface-endpoint>`
     -  Remove one interface endpoint from a private endpoint connection 
        in an |service| project.
