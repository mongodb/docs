========================================
Upgrade |onprem| with an ``rpm`` Package
========================================

.. default-domain:: mongodb

Overview
--------

This tutorial describes how to upgrade an existing |onprem| installation
using an ``rpm`` package. |onprem| supports direct upgrades from version
1.5 and later.
