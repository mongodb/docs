========================================================================
Install a {+qe+} Compatible Driver and Dependencies
========================================================================

.. meta::
   :description: Install a compatible driver and dependencies to enable Queryable Encryption in your development environment.

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

Overview
--------

To enable {+qe+} in your development environment, start by installing
a compatible driver and dependencies. Some drivers are packaged with the
``libmongocrypt`` library, while for others you must install it separately.

.. _qe-quick-start-driver-dependencies:

Steps
-----

Select your operating system and driver language for installation steps.
