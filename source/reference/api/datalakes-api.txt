.. _adl-api:

============
{+adl+} APIs
============

.. default-domain:: mongodb

.. meta::
   :keywords: |data-lake|

The ``dataLakes`` resource provides access to the project's 
{+data-lake+} configuration. The resource lets you view, create,
modify, and delete {+data-lake+}s. The resource requires your
:atlas:`Project ID </api/#project-id>`.

.. include:: /includes/fact-projects-groups-synonyms.rst

.. important::

   Changes to {+data-lake+} configurations can affect costs. Before 
   making changes, please see :atlas:`Billing </billing>`.

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/dataLakes </reference/api/dataLakes-get-all-tenants>`
     - Get all {+data-lake+}s for the specified project.

   * - ``GET``
     - :ref:`/api/atlas/v1.0/groups/{GROUP-ID}/privateNetworkSettings/endpointIds <api-pvt-link-get-all>` 
     - Get all the private endpoints for the {+dl+}s in the specified 
       project.

   * - ``GET``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/dataLakes/{NAME} </reference/api/dataLakes-get-one-tenant>`
     - Get an {+data-lake+} in the specified project.

   * - ``GET``
     - :ref:`/api/atlas/v1.0/groups/{GROUP-ID}/privateNetworkSettings/endpointIds/{ENDPOINT-ID} <api-pvt-link-get-one>`
     - Get the specified private endpoint in the specified project.

   * - ``GET``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/dataLakes/{DATA-LAKE-NAME}/queryLogs.gz </reference/api/dataLakes-download-query-logs>`
     - Download query log for the specified {+data-lake+}.

   * - ``POST``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/dataLakes </reference/api/dataLakes-create-one-tenant>`
     - Create an {+data-lake+} in the specified project.

   * - ``POST``
     - :ref:`/api/atlas/v1.0/groups/{GROUP-ID}/privateNetworkSettings/endpointIds <api-pvt-link-create-one>`
     - Add one |aws| private endpoint for the {+data-lake+}s in the 
       specified project.

   * - ``PATCH``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/dataLakes/{NAME} </reference/api/dataLakes-update-one-tenant>`
     - Update an {+data-lake+} in the specified project.

   * - ``DELETE``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/dataLakes/{NAME} </reference/api/dataLakes-delete-one-tenant>`
     - Delete an {+data-lake+} in the specified project.

   * - ``DELETE``
     - :ref:`/api/atlas/v1.0/groups/{GROUP-ID}/privateNetworkSettings/endpointIds/{ENDPOINT-ID} <api-pvt-link-delete-one>`
     - Delete one |aws| private endpoint in the specified project.
