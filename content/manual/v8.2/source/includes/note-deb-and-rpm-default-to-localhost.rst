.. .. |mongodb-package| should be replaced with the binary name in other source
   files (mongod or mongos)

|mongodb-package| installed from official :ref:`Debian
<install-mdb-community-debian>` and :ref:`Red Hat or CentOS
<install-mdb-community-redhat-centos>` packages have the ``bind_ip``
configuration set to ``127.0.0.1`` by default.
