atlas api queryShapeInsights getClusterQueryShape
=================================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none

Returns the details for a single query shape.

The atlas api sub-command is automatically generated from the MongoDB Atlas Admin API and offers full coverage of the Admin API.
Admin API capabilities have their own release lifecycle, which you can check via the provided API endpoint documentation link.

This endpoint only returns query shapes with REJECTED status. If the specified query shape hash does not correspond to a rejected query shape, a 404 Not Found error is returned.

This command is autogenerated and corresponds 1:1 with the Atlas API endpoint https://www.mongodb.com/docs/api/doc/atlas-admin-api-v2/operation/operation-getgroupclusterqueryshape.

For more information and examples, see the referenced API documentation linked above.
