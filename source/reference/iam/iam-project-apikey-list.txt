.. _mcli-iam-project-apikey-list:

================================
mongocli iam project apikey list
================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``iam project apikey list`` command lists the  
|api| keys for the specified project.

You must have ``GROUP_USER_ADMIN`` privileges to list 
|api| keys for the project.

.. _mcli-iam-project-apikey-list-syntax:

Syntax 
------

.. code-block:: text 

   mongocli iam project|projects apiKey|apiKeys|apikeys list|ls
        [ --projectId <project-id> ]
        [ --output|-o <output-format> ]
        [ --profile|-P <profile-name> ]

.. _mcli-iam-project-apikey-list-output:

Output
------

If the command succeeds, it returns the following output in the default
format. If the command returns errors, see
:ref:`Troubleshooting <troubleshooting>` for recommended solutions.

.. code-block:: sh
   :copyable: false

   ID             PUBLIC KEY           DESCRIPTION
   <key-id-1>     <public-key-1>       <key-description-1>
   <key-id-2>     <public-key-2>       <key-description-2>

.. _mcli-iam-project-apikey-list-examples:

Examples
--------

.. tabs::

   .. tab:: Default Output
      :tabid: default-output

      The following command retrieves the |api| keys for a project
      using the {+default-profile+}, which contains credentials and the
      project ID. The output is returned in the default format.

      .. code-block:: sh

         mongocli iam project apikey list

      The previous command prints the following to the terminal.

      .. code-block:: sh
         :copyable: false

         ID                         PUBLIC KEY   DESCRIPTION
         5f4578684b4afb5e8cd44ee9   mkoztppy     Project Owner
         5f469ea0f2c5937c5cdb1081   hdlvegpx     External Read-Only

   .. tab:: JSON Output
      :tabid: json-output

      The following command retrieves the |api| keys for a 
      project using the ``myOM`` profile, which contains 
      credentials and specifies the |onprem| service. The output is 
      returned in |json| format.

      .. code-block:: sh

         mongocli iam project apikey list --projectId 5e399b9a7a3e5a1772948493 --profile myOM --output json
