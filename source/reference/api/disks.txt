=====
Disks
=====

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: twocols

Retrieves the disks and disk partitions on which MongoDB runs.

Endpoints
---------

Get All Disk Partitions
~~~~~~~~~~~~~~~~~~~~~~~

.. code-block:: none

   GET /api/public/v1.0/groups/GROUP-ID/hosts/HOST-ID/disks

Get a Disk Partition
~~~~~~~~~~~~~~~~~~~~

.. code-block:: none

   GET /api/public/v1.0/groups/GROUP-ID/hosts/HOST-ID/disks/PARTITION-NAME

Sample Entity
-------------

.. code-block:: javascript

   {
     "partitionName" : "xvdf",
     "links" : [ ... ]
  }

Entity Fields
-------------

.. list-table::
   :widths: 10 10 80
   :header-rows: 1

   * - Name
     - Type
     - Description

   * - partitionName
     - string
     - The name of a partition on the disk.

Links
-----

.. list-table::
   :header-rows: 1

   * - Relation
     - Description

   * - ``self``
     - Me

   * - ``http://mms.mongodb.com/group``
     - The group the disk belongs to.

   * - ``http://mms.mongodb.com/host``
     - The MongoDB processes running on the disk.

   * - ``http://mms.mongodb.com/measurements/disk/partition``
     - Disk partition measurements.

Examples
--------

Get All Disk Partitions
~~~~~~~~~~~~~~~~~~~~~~~

Request
```````

.. only:: cloud

   .. code-block:: none

      curl -i -u "username:apiKey" --digest "https://cloud.mongodb.com/api/public/v1.0/groups/533c5895b910306f21033a/hosts/814e70da8167883b9939608a12a/disks"

.. only:: onprem

   .. code-block:: none

      curl -i -u "username:apiKey" --digest "https://<ops-manager-host>/api/public/v1.0/groups/533c5895b910306f21033a/hosts/814e70da8167883b9939608a12a/disks"

Response
````````

.. code-block:: none

   HTTP/1.1 200 OK

   {
     "results" : [ {
       "links" : [ ... ],
       "partitionName" : "xvdf"
     }, ... ],
     "links" : [ ... ],
     "totalCount" : 4
   }
