.. tabs-platforms::

   .. tab::
      :tabid: windows

      The location of the {+mdbagent+} configuration directory is
      ``C:\MMSData\Automation\``.

      .. note::

         The {+mdbagent+} configuration directory is named
         ``Automation`` to simplify upgrades
         for those using legacy agents.

   .. tab::
      :tabid: macos

      The location of the {+mdbagent+} configuration file is
      ``/path/to/install/``.

   .. tab::
      :tabid: linux

      The location of the {+mdbagent+} configuration file is
      ``/path/to/install/``.

   .. tab::
      :tabid: rhel

      The location of the {+mdbagent+} configuration file is
      ``/etc/mongodb-mms/``.


   .. tab::
      :tabid: debian

      The location of the {+mdbagent+} configuration file is
      ``/etc/mongodb-mms/``.
