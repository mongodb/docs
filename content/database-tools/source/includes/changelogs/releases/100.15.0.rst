100.15.0 Changelog
------------------

*Released 2026-03-11*

We are pleased to announce version 100.15.0 of the MongoDB Database Tools.

This release includes dependency and Golang updates for security fixes and bug fixes.

Bug:

- :issue:`TOOLS-4105` - ``mongorestore`` crashes when applying a ``collMod`` 
  ``oplog`` event with ``forceNonUnique``.
- :issue:`TOOLS-4068` - QA test fails with MongoDB Server 7.0.28, but passed with 7.0.26.
- :issue:`TOOLS-4070` - ``mongorestore`` ``--bypassDocumentValidation`` does not 
  bypass document validation during oplog restore.
- :issue:`TOOLS-4071` - ``mongorestore`` crashes when applying a ``collMod`` 
  ``oplog`` event which does not specify ``expireAfterSeconds`` or ``hidden``.
- :issue:`TOOLS-4075` - ``mongorestore`` crashes when applying a ``collMod`` 
  ``oplog`` event that converts an index to a TTL index.

Task:

- :issue:`TOOLS-3968` - Update tools to driver v2.
- :issue:`TOOLS-4073` - Add ``mongorestore`` support for restoring dumps from 
  clusters with ``recordIdsReplicated`` enabled into clusters without this feature enabled.
- :issue:`TOOLS-4074` - Add a manual test program to test dump or restore between 
  clusters with and without replicated record IDs.
- :issue:`TOOLS-4091` - Fix unsafe file extraction in the release code.
- :issue:`TOOLS-4118` - Update to Go 1.25.7 to address some CVEs in 1.25.0.
