100.14.0 Changelog
------------------

*Released 2025-12-16*

We are pleased to announce version 100.14.0 of the MongoDB Database Tools.

This version adds important fixes to upstream dependencies.
It also adds new safety restrictions to ``mongofiles`` that help protect your data.

Vulnerability

- :issue:`TOOLS-4010` - Security Finding: Update package crypto@v0.38.0

Bug

- :issue:`TOOLS-4019` - Fix path traversal in mongofiles

Task

- :issue:`TOOLS-3983` - Make ``mongorestore`` warn when restoring archive from a 
  different server version
