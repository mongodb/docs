Overview
--------

These step-by-step tutorials cover common aggregation tasks. They are
adapted from examples in the book `Practical MongoDB Aggregations
<https://www.practical-mongodb-aggregations.com/>`__ by Paul Done.

Each tutorial includes examples for each of the :driver:`official
MongoDB drivers </>`. Use the :guilabel:`Select your language`
drop-down menu in the upper right of each tutorial to set the
programming language.
