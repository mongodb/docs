Common Tasks
------------

After you install the {+dbtools-short+}, use the following tools to
work with your MongoDB data:

- To back up a database, see :ref:`mongodump`.
- To restore data from a backup, see :ref:`mongorestore`.
- To import data into a collection, see :ref:`mongoimport`.
- To export data from a collection, see :ref:`mongoexport`.
