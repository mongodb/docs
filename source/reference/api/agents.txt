Endpoints
---------

The following endpoints are available for agents:

.. list-table::
   :header-rows: 1
   :widths: 10 65 25

   * - Method
     - Endpoint
     - Description

   * - ``GET``

     - :doc:`/api/public/v1.0/groups/{GROUP-ID}/agents </reference/api/agents-get-all>`

     - Get links to Monitoring, Backup, and Automation agent resources
       for a group.

   * - ``GET``

     - :doc:`/api/public/v1.0/groups/{GROUP-ID}/agents/{TYPE} </reference/api/agents-get-by-type>`

     - Get agents of a specified type (Monitoring, Backup, or Automation) for a group.
