Resource
--------

.. code-block:: http

   PATCH /groups/{GROUP-ID}

Request Parameters
------------------

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :widths: 10 10 80
   :header-rows: 1
   :stub-columns: 1

   * - Name
     - Type
     - Description

   * - GROUP-ID
     - string
     - *(Required.)* The unique identifier for the group.

Request Query Parameters
~~~~~~~~~~~~~~~~~~~~~~~~

.. include:: /includes/api/no-query-single-parameters.rst

Request Body Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :widths: 10 10 80
   :header-rows: 1
   :stub-columns: 1

   * - Name
     - Type
     - Description

   * - name
     - string
     - The display name for the group.

   * - tags
     - string array
     - The tags assigned to the group for use in programmatically
       identifying the group.

       - To view tags you must have either the
         :ref:`Global Read Only <global-read-only-role>` or
         :ref:`Global Owner <global-owner-role>`
         :doc:`role </reference/user-roles>`.
       - To create or edit tags, you must be a
         :ref:`Global Owner <global-owner-role>`.

       .. include:: /includes/requirements-tag-names.rst

   * - ldapGroupMappings
     - object array
     - For LDAP-backed |onprem|, the mappings of
       :doc:`LDAP groups </tutorial/enable-ldap-authentication-for-group>`
       to |onprem| roles. Only present for LDAP-backed |onprem|.

.. include:: /includes/extracts/group-elements.rst

Example Request
---------------

.. only:: onprem

   .. cssclass:: copyable-code
   .. code-block:: sh
      :emphasize-lines: 6-12

      curl --user "{username}:{apiKey}" --digest \
       --header "Accept: application/json" \
       --header "Content-Type: application/json" \
       --include \
       --request PATCH "https://{opsManagerHost}:{port}/api/public/v1.0/groups/{GROUP-ID}?pretty=true" \
       --data '
         {
           "ldapGroupMappings" : [ {
             "roleName": "GROUP_OWNER",
             "ldapGroups": [ {LDAP-Group} ]
            }]
         }'
