=====================================
Choose the Aggregation Pipeline Stage 
=====================================

.. default-domain:: mongodb

.. facet::
   :name: genre
   :values: reference

.. meta::
   :keywords: atlas search, $search aggregation pipeline stage, full text search on specified fields, $search fields, $search behavior, $searchMeta aggregation pipeline stage, return metadata result documents, $searchMeta fields, $searchMeta behavior
   :description: Learn how to use the $search aggregation pipeline stage to perform a full text search, and the $searchMeta aggregation pipeline stage to return metadata result documents.

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

|fts| queries take the form of an :manual:`aggregation pipeline stage 
</aggregation>`. Based on the pipeline stage that you choose, your 
query returns either the search results of a full-text search or metadata 
about your search results:

.. list-table::
   :header-rows: 1
   :widths: 40 60
   
   * - Aggregation Pipeline Stage
     - Purpose

   * - :pipeline:`$search`
     - Return the search results of a full-text search.

   * - :pipeline:`$searchMeta`
     - Return metadata about your search results.
