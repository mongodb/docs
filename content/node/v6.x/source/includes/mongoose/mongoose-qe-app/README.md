# Mongoose Queryable Encryption Tutorial

This project demonstrates an example implementation of Queryable Encryption
that uses the Mongoose ODM. To learn more about Queryable Encryption, see
[Queryable Encryption](https://www.mongodb.com/docs/manual/core/queryable-encryption/)
in the MongoDB Server manual.

## Instructions

To view instructions for running this application, see
[Tutorial: Queryable Encryption with Mongoose](https://www.mongodb.com/docs/drivers/node/current/integrations/mongoose/mongoose-qe/)
in the Node.js Driver documentation.
