``podSpec.persistence``
  *Type*: collection

  Only available in ``clusterSpecItem`` objects passed to 
  ``spec.configSrv.clusterSpecList`` and ``spec.shard.clusterSpecList``.
  Overrides the existing :ref:`persistence <pod-spec-persistence>` configuration 
  for a given cluster.
