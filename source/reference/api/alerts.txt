The ``alerts`` resource allows you to retrieve and acknowledge
:doc:`alerts </configure-alerts>`.

Endpoints
---------

The following lists the endpoints available for :doc:`alerts
</configure-alerts>`.

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``

     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/alerts </reference/api/alerts-get-all-alerts>`

     - Get all alerts for a group.

   * - ``GET``

     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/alerts/{ALERT-ID} </reference/api/alerts-get-alert>`

     - Get the specified alert for a group.

   * - ``PATCH``

     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/alerts/{ALERT-ID} </reference/api/alerts-acknowledge-alert>`

     - Acknowledge or unacknowledge an alert.
