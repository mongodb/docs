=====================
Deploy a BI Connector
=====================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

An instance of the MongoDB Connector for Business Intelligence
translates SQL queries from data analysis tools to MongoDB aggregation
pipelines on MongoDB database. The BI Connector allows you to query a
MongoDB database using SQL commands.

.. note::
   The BI Connector only allows ``SELECT`` queries of data served in 
   MongoDB databases. To create, update, or delete data you must use 
   the MongoDB shell or any appropriate driver.

.. seealso::

   For specifics on the BI Connector, see the 
   `BI Connector manual <https://docs.mongodb.com/bi-connector/>`_.

Use the following procedure to deploy a new MongoDB BI Connector 
for |mms| to manage.
