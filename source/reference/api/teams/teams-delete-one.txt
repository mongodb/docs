Syntax
------

.. code-block:: none

   DELETE /orgs/{ORG-ID}/teams/{TEAM-ID}

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :widths: 15 10 75

   * - Path Element
     - Required/Optional
     - Description

   * - ``ORG-ID``

     - Required.

     - The unique identifier for the 
       :ref:`organization <organizations>`
       associated with the team.

   * - ``TEAM-ID``

     - Required

     - The unique identifier of the :ref:`team <teams>` you want
       to delete.
