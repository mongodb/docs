.. step:: In |cloud|, go to the :guilabel:`Deleted Hosts` page for your project.

   a. If it is not already displayed, select the organization that
      contains your desired project from the |ui-org-menu| in the
      navigation bar.

   #. If it's not already displayed, select your desired project
      from the :guilabel:`Projects` menu in the navigation bar.

   #. In the sidebar, click :guilabel:`Deleted Hosts` under the 
      :guilabel:`Admin` heading.

   The `Deleted Hosts <https://cloud.mongodb.com/go?l=https%3A%2F%2Fcloud.mongodb.com%2Fv2%2F%3Cproject%3E%23%2FdeploymentAdmin%2FdeletedHosts>`__ 
   page displays.
