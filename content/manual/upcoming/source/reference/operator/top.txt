==========================
$top (expression operator)
==========================

.. meta::
   :description: Use the `$top` operator in MongoDB to return the top value of an array.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Definition
----------

.. expression:: $top 

   .. versionadded:: 8.3

   Returns the top element within an array according to the specified 
   sort order. 

.. note:: Disambiguation

   This page describes the ``$top`` expression operator. For the ``$top``
   accumulator operator, see :ref:`$top (accumulator operator) <top_accumulator_operator>`. 

Syntax
------

When used as an expression operator, ``$top`` has the following syntax:

.. code-block:: none
   :copyable: false

   {
      $top:
         {
            sortBy: <expression>,
            input: <expression>
         }
   }

.. list-table::
   :header-rows: 1
   :widths: 15 15 70

   * - Field
     - Necessity
     - Description

   * - sortBy

     - Required

     - Specifies the order of results. See :ref:`Sort Behavior <top_sort_behavior>` for more
       information on ``sortBy`` values. 

   * - input

     - Required

     - The array that ``$top`` evaluates.

Input Values
~~~~~~~~~~~~

The ``input`` field must resolve to an array. If you specify an ``input`` that is not
an array, MongoDB errors.
