========================
atlas teams users delete
========================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

Delete user from a team.

Syntax
------

.. code-block::

   atlas teams users delete <userId> [options]

.. Code end marker, please don't delete this comment
