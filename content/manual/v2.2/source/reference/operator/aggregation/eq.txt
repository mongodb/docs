=================
$eq (aggregation)
=================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

.. expression:: $eq

   Takes two values in an array and returns a boolean. The returned
   value is:

   - ``true`` when the values are equivalent.

   - ``false`` when the values are **not** equivalent.
