=========================================
mongocli ops-manager alerts unacknowledge
=========================================


.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol


The ``alerts unacknowledge`` command un-unacknowledges one alert for
the specified |onprem| project.

.. _om-alerts-unack-syntax:

Syntax
------

.. code-block:: sh

   mongocli ops-manager|om alert|alerts unacknowledge|unack <alertID>
        [ --until <unacknowledgement time> ]
        [ --comment <optional comment> ]
        [ --output|-o <output-format> ]
        [ --profile|-P <profile-name> ]
        [ --projectId <id-of-project> ]

Examples
--------

Acknowledge for 1 Month
~~~~~~~~~~~~~~~~~~~~~~~

The following example uses the ``mongocli om unacknowledge`` command to
unacknowledge one alert that occurred for the specified project for one
month.

.. code-block:: sh

   mongocli om alerts unacknowledge {ALERT-ID} \
            --projectId {PROJECT-ID} \
            --until 2020-06-04T20:24:26Z \
            --output json

The previous command returns the following |json| document. To learn
more about these fields, see :ref:`om-alerts-unack-output`.

.. code-block:: json
   :copyable: false
   :linenos:

     "acknowledgedUntil": "2020-06-04T20:24:26Z",
     "acknowledgementComment": "acknowledging for one month",
