===============
Access Tracking
===============

.. default-domain:: mongodb

.. include:: /includes/fact-access-tracking-auth-roles.rst

Use the following resources to view the :doc:`database access history
</access-tracking>` settings. Each resource requires at minimum your
:ref:`Group ID <group-id>`.

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :doc:`/groups/{GROUP-ID}/dbAccessHistory/clusters/{CLUSTER-NAME} </reference/api/access-tracking-get-database-history-clustername>`
     - Retrieve the access logs of a cluster by cluster name

   * - ``GET``
     - :doc:`/groups/{GROUP-ID}/dbAccessHistory/processes/{HOSTNAME} </reference/api/access-tracking-get-database-history-hostname>`
     - Retrieve the access logs of a process by hostname
