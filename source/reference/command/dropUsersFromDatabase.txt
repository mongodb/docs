=====================
dropUsersFromDatabase
=====================

.. default-domain:: mongodb

Definition
----------

.. dbcommand:: dropUsersFromDatabase

   Deletes all the users on the database where you run the command. To
   run the command you must have the :authaction:`dropUser` action on
   the database.

   .. warning::

      The :dbcommand:`dropUsersFromDatabase` removes all users from the database.

   The :dbcommand:`dropUsersFromDatabase` command uses the following
   syntax:

   .. code-block:: javascript

      { dropUsersFromDatabase: 1,
        writeConcern: { <write concern> }
      }

   The command has the following fields:

   .. include:: /reference/command/dropUsersFromDatabase-field.rst

Example
-------

The following sequence of commands in the :program:`mongo` shell drops
every user from the ``products`` database:

.. code-block:: javascript

   use products
   db.runCommand( { dropUsersFromDatabase: 1, writeConcern: { w: "majority" } } )

The ``n`` field in the results document shows that ``12`` users were
dropped:

.. code-block:: javascript

   { "n" : 12, "ok" : 1 }
