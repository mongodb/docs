.. option:: --minimumTLSVersion <TLS1_0|TLS1_1|TLS1_2>

   *Default*: TLS1_1

   Specifies the minimum required TLS version for
   :binary:`~bin.mongodrdl` to connect to a :binary:`~bin.mongod` or
   :binary:`~bin.mongos` instance. The default value ``TLS1_1``
   corresponds to TLS version 1.1.
