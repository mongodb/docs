=========================
Available Sample Datasets
=========================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

This page shows the available sample datasets in |service|. Click a
sample dataset to lean more about it.

.. list-table::
   :header-rows: 1
   :widths: 50 50

   * - Dataset Name
     - Description

   * - :ref:`sample-airbnb`
     -  Contains details on
        `AirBnB <https://www.airbnb.com>`__ listings.

   * - :ref:`sample-geospatial`
     - Contains shipwreck data.

   * - :ref:`sample-mflix`
     - Contains movie data.

   * - :ref:`sample-weather`
     - Contains detailed weather reports.

For instructions on loading this sample data into your cluster, see
:ref:`load-sample-data`.
