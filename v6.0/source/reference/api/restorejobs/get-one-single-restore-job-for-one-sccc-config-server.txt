=========================================================
Get One Restore Job for One Legacy Mirrored Config Server
=========================================================

.. contents:: On this page

.. include:: /includes/api/base-api-uri.rst

Resource
--------

.. code-block:: http

   GET /groups/{PROJECT-ID}/hosts/{HOST-ID}/restoreJobs/{JOB-ID}

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :widths: 20 20 60
   :header-rows: 1
   :stub-columns: 1

   * - Name
     - Type
     - Description

   * - PROJECT-ID
     - string
     - Unique identifier of the :opsmgr:`project </reference/glossary/#std-term-project>` that owns the restore
       job.

   * - HOST-ID
     - string
     - Unique identifier of the mirrored :manual:`config server </reference/glossary/#std-term-config-server>`
       (:abbr:`SCCC (Sync Cluster Connection Configuration)`) that the
       restore job represents.

   * - JOB-ID
     - string
     - Unique identifier of the restore job.

Example Request
---------------

.. code-block:: sh

   curl --user "{PUBLIC-KEY}:{PRIVATE-KEY}" --digest \
    --header "Accept: application/json" \
    --include \
    --request GET "https://{OPSMANAGER-HOST}:{PORT}/api/public/v1.0/groups/{PROJECT-ID}/hosts/{HOST-ID}/restoreJobs/{JOB-ID}?pretty=true"
