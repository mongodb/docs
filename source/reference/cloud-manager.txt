.. _mcli-reference-cm:

===================================
Cloud Manager ``mongocli`` commands
===================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

``cluster``
-----------

The ``cluster`` command provides access to your cluster configurations.

- :ref:`List clusters <mcli-cm-cluster-list-command>`
- :ref:`Describe a cluster <mcli-cm-cluster-describe-command>`
- :ref:`Create a cluster <mcli-cm-cluster-create-command>`
- :ref:`Modify a cluster <mcli-cm-cluster-update-command>`
- :ref:`Create or modify a cluster <mcli-cm-cluster-apply-command>`
- :ref:`Start mongod processes for a cluster <mcli-cm-cluster-startup-command>`
- :ref:`Stop mongod processes for a cluster <mcli-cm-cluster-shutdown-command>`

``alerts``
----------

The ``alerts configs list`` command allows you to list |service|
:atlas:`alerts </configure-alerts>`.

- :ref:`List alerts <mcli-cm-list-alert-config-command>`

``automation``
--------------

The ``automation status`` command retrieves the latest plan for MongoDB
processes in the specified |cloud-short| project.

- :ref:`Retrieve automation status <mcli-cm-automation-status-command>`

``backup``
----------

The ``backup`` command allows you to manage restore jobs.

- :ref:`Start a backup restore job <mcli-cm-backup-restore-start-cmd>`
- :ref:`List snapshots <mcli-cm-backup-snapshots-list-cmd>`
- :ref:`List checkpoints <mcli-cm-backup-checkpoints-list-cmd>`

``dbuser``
----------

The ``dbuser`` command allows you to list, create, modify, and
delete MongoDB database users.

- :ref:`List database users <mcli-cm-dbuser-list-command>`
- :ref:`Create a database user <mcli-cm-dbuser-create-command>`
- :ref:`Delete a database user <mcli-cm-dbuser-delete-command>`

``security``
------------

The ``security enable`` command enables an :cloudmgr:`authentication
mechanism </tutorial/enable-mongodbcr-authentication-for-group/>`
for the specified |cloud-short| clusters in the given project.

- :ref:`Enable security for a cluster <mcli-cm-security-enable-command>`

``servers``
-----------

The ``servers list`` command retrieves all hosts that are running an
Automation Agent in the specified |cloud-short| project.

- :ref:`List servers for a project <mcli-cm-servers-list-command>`

``process list``
----------------

Retrieve information about MongoDB processes running on a specified
|cloud| project.

- :ref:`List processes <mcli-cm-process-list-cmd>`

``measurements``
----------------

The ``measurements`` command allows you to list measurements for MongoDB
processes on a specified host.

- :ref:`List process measurements <mcli-cm-measurements-process-cmd>`
