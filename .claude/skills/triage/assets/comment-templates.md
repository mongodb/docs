# Requester Comment Templates

After triaging, add a comment to the ticket using the appropriate template:

**Assigned to Backlog:**
- *CET/Cloud:* > "Thanks for filing this! We've triaged this ticket and added it to our backlog. A writer will pick it up in a future sprint."
- *Server / Drivers/DBX:* > "Thanks for filing this! We've triaged this ticket and added it to our backlog. A writer will pick it up when it's prioritized."

**Moved to Ready for Work:**
> "Thanks for filing this! We've triaged this ticket and it's now ready for a writer to pick up soon."

**Closed:**
> "Thanks for filing this! After review, we're closing this ticket as [reason]. [If filed by a MongoDB employee: explain the close reason clearly.] If you think this decision was made in error, please reopen and add more context."
