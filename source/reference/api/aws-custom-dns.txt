=================================================
Custom DNS for |service| Clusters Deployed to AWS
=================================================

.. default-domain:: mongodb

.. meta::
   :canonical: https://www.mongodb.com/docs/atlas/reference/api-resources-spec/#tag/AWS-Clusters-DNS

The ``awsCustomDNS`` resource lets you retrieve and update the custom 
DNS configuration of an |service| project's clusters deployed to |aws|. 
The resource requires your :ref:`project-id`.

Enable custom DNS if you use the :ref:`VPC peering <vpc-peering>` on 
|aws| and you use your own |dns| servers instead of Amazon Route 53.
