======================================
Get Started with MongoDB Cloud Manager 
======================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

{+mcli+} provides a command-line interface for managing your |cloud| 
projects and clusters. 

This tutorial demonstrates how to: 

- Create clusters in your |cloud-short| project
- Retrieve details about the new cluster 

The reference pages for the commands mentioned in this tutorial are  
available in the :ref:`{+mcli+} Commands <mongocli>`.

.. _quick-start-cm-reqs:

Prerequisites 
-------------

Before you begin, you must complete the following: 

- :ref:`quick-start-install` 
- :ref:`quick-start-config-access-keys`
- :ref:`quick-start-config-mcli`

.. _quick-start-cm-retrieve-cluster:

Retrieve the Cluster 
--------------------

You can check whether your cluster was successfully created by 
retrieving the cluster details using the 
:ref:`mongocli-cloud-manager-clusters-describe` command.

.. code-block:: sh 

   mongocli cloud-manager cluster describe getStarted

.. _quick-start-cm-watch-automation-config:

Watch Your Automation Configuration Status
------------------------------------------

Run the :ref:`mongocli-cloud-manager-automation-watch` command to determine if 
the cluster is ready for use. The following command uses the default 
profile to access the |cloud-short| project:

.. code-block:: sh 

   mongocli cm automation watch
