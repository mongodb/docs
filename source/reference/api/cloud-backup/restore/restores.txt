====================================
{+Cloud-Backup+} Restore Jobs
====================================

.. default-domain:: mongodb

.. meta::
   :canonical: https://www.mongodb.com/docs/atlas/reference/api-resources-spec/#tag/Cloud-Backups

.. include:: /includes/fact-projects-groups-synonyms.rst

.. important:: {+Cloud-Backup+}s Only

   This resource only applies to clusters using
   :ref:`backup-cloud-provider`. 

The ``/backup/restoreJobs`` resource enables you to create or view
restore jobs for an |service| cluster with :ref:`backup-cloud-provider`
enabled.

Endpoints
---------

.. include:: /includes/api-base-url.rst

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/backup/restoreJobs </reference/api/cloud-backup/restore/get-all-restore-jobs>`
     - Get all {+Cloud-Backup+} restore jobs for the specified cluster.

   * - ``GET``
     - :doc:`/groups/{groupId}/serverless/{instanceName}/backup/restoreJobs </reference/api/cloud-backup/restore/return-all-restore-jobs-for-one-serverless-instance>`
     - Return all restore jobs of one serverless instance.

   * - ``GET``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/backup/restoreJobs/{JOB-ID} </reference/api/cloud-backup/restore/get-one-restore-job>`
     - Get one {+Cloud-Backup+} restore jobs for the specified cluster.

   * - ``GET``
     - :doc:`/groups/{groupId}/serverless/{instanceName}/backup/restoreJobs/{restoreJobId} </reference/api/cloud-backup/restore/return-one-restore-job-for-one-serverless-instance>`
     - Return one restore job of one serverless instance.

   * - ``POST``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/backup/restoreJobs </reference/api/cloud-backup/restore/create-one-restore-job>`
     - Restore one snapshot for the specified cluster.

       .. include:: /includes/api/facts/api-access-list-required.rst

   * - ``POST``
     - :doc:`/groups/{groupId}/serverless/{instanceName}/backup/restoreJobs </reference/api/cloud-backup/restore/restore-one-snapshot-of-one-serverless-instance>`
     - Restore one snapshot for the specified serverless instance.

       .. include:: /includes/api/facts/api-access-list-required.rst

   * - ``DELETE``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/backup/restoreJobs/{JOB-ID} </reference/api/cloud-backup/restore/delete-one-restore-job>`
     - Cancels the {+Cloud-Backup+} manual download restore job
       associated to ``{JOB-ID}``.

       .. include:: /includes/api/facts/api-access-list-required.rst
