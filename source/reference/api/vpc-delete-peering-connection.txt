=========================================
Delete an Existing VPC Peering Connection
=========================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

.. include:: /includes/fact-projects-groups-synonyms.rst

.. include:: /includes/api/base-api-uri.rst

Deletes an |aws| or |gcp| |vpc| peering connection associated with the
specified project.

.. important::

   If you delete the last |vpc| peering connection associated with a
   project, |service| also removes any |aws| security groups or |gcp|
   Compute Engine firewall rules from the
   :doc:`whitelist </security-whitelist>` for the project.

Syntax
------

.. code-block:: sh

   DELETE /groups/{GROUP-ID}/peers/{PEER-ID}

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :widths: 15 10 10 65

   * - Parameter
     - Type
     - Necessity
     - Description

   * - ``GROUP-ID``
     - string
     - Required
     - Unique identifier for the project.

   * - ``PEER-ID``
     - string
     - Required
     - |service| assigned :data:`unique ID <id>` for the peering
       connection.

       .. include:: /includes/api/fact-api-peer-id-not-vpc-id.rst
