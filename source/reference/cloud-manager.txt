.. _mcli-reference-cm:

===================================
Cloud Manager ``mongocli`` commands
===================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

``agent``
----------

The ``agents`` command allows you to manage agents installed to
your |cloud-short| deployments.

- :ref:`Upgrade agents to their latest version <mcli-cm-agents-upgrade-command>`

``alert``
----------

Lists |cloud-short| :cloudmgr:`alerts and alert configurations
</tutorial/nav/alerts/>`.

- :ref:`List alert configurations <mcli-cm-list-alert-config-command>`
- :ref:`Create one alert configuration <mcli-cm-create-alerts-config-command>`
- :ref:`Update one alert configuration <mcli-cm-update-alerts-config-command>`
- :ref:`Delete one alert configuration <mcli-cm-alerts-configs-delete-cmd>`
- :ref:`List alert configuration matcher fields <mcli-cm-alert-config-fields-type-command>`
- :ref:`List alerts <mcli-cm-alert-list-command>`
- :ref:`Describe an alert <mcli-cm-alert-describe-command>`
- :ref:`Acknowledge an alert <mcli-cm-alert-ack-command>`

``automation``
--------------

Retrieves the latest plan for MongoDB processes in the specified
|cloud-short| project.

- :ref:`Retrieve automation status <mcli-cm-automation-status-command>`

``backup``
----------

Manages backup and restore jobs.

- :ref:`Start a backup restore job <mcli-cm-backup-restore-start-cmd>`
- :ref:`List snapshots <mcli-cm-backup-snapshots-list-cmd>`
- :ref:`List checkpoints <mcli-cm-backup-checkpoints-list-cmd>`
- :ref:`List backup restore jobs <mcli-cm-backup-restore-list-cmd>`

``cluster``
-----------

Provides access to your cluster configurations.

- :ref:`List clusters <mcli-cm-cluster-list-command>`
- :ref:`Describe a cluster <mcli-cm-cluster-describe-command>`
- :ref:`Create a cluster <mcli-cm-cluster-create-command>`
- :ref:`Modify a cluster <mcli-cm-cluster-update-command>`
- :ref:`Create or modify a cluster <mcli-cm-cluster-apply-command>`
- :ref:`Start mongod processes for a cluster <mcli-cm-cluster-startup-command>`
- :ref:`Stop mongod processes for a cluster <mcli-cm-cluster-shutdown-command>`
- :ref:`Create a rolling index <mcli-cm-clusters-indexes-create-cmd>`

``dbuser``
----------

Manages MongoDB database users.

- :ref:`List database users <mcli-cm-dbuser-list-command>`
- :ref:`Create a database user <mcli-cm-dbuser-create-command>`
- :ref:`Delete a database user <mcli-cm-dbuser-delete-command>`

``event``
----------

Lists events for a specified project or organization.

- :ref:`List events <mcli-cm-events-list-cmd>`

``measurement``
----------------

Lists measurements for MongoDB processes on a specified host.

- :ref:`List process measurements <mcli-cm-measurements-process-cmd>`
- :ref:`Describe available measurements for a specified partition
  <mcli-cm-measurements-disks-describe-cmd>`

``process``
-----------

Retrieves information about MongoDB processes running on a specified
|cloud| project.

- :ref:`List processes <mcli-cm-process-list-cmd>`

``security``
------------

Enables an :cloudmgr:`authentication mechanism
</tutorial/enable-mongodbcr-authentication-for-group/>` for the
specified |cloud-short| clusters in the given project.

- :ref:`Enable security for a cluster <mcli-cm-security-enable-command>`

``server``
-----------

Retrieves all hosts that are running an Automation Agent in the
specified |cloud-short| project.

- :ref:`List servers for a project <mcli-cm-servers-list-command>`
