================
``mongorestore``
================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The :binary:`~bin.mongorestore` program loads data from either a binary
database dump created by :binary:`~bin.mongodump` or the standard input
into a :binary:`~bin.mongod` or :binary:`~bin.mongos` instance.

.. note:: ``mongorestore`` documentation migrated to `MongoDB Database Tools <https://docs.mongodb.com/database-tools>`__


   Starting in MongoDB 4.4, the documentation for ``mongorestore``
