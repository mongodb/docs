.. _adl-tutorials:

=========
Tutorials
=========

.. default-domain:: mongodb

.. meta::
   :keywords: |data-lake|

This section contains tutorials that describe how to perform the 
following actions:

- `Get MongoDB Data Into Parquet 
  <https://www.mongodb.com/developer/how-to/mongodb-data-parquet/>`__ - 
  This tutorial shows you how to use {+adl+} to get data in any format 
  from your |service| Cluster or |s3| Bucket in to Apache Parquet, 
  which allows you to query data in the cloud very fast.

- `Query From Multiple MongoDB Databases <https://www.mongodb.com/developer/how-to/query-multiple-databases-with-atlas-data-lake/>`__ - 
  This tutorial shows you how to use {+adl+} to query data across 
  multiple collections, databases, and |service| clusters.

- `Run Federated Queries and Write to an AWS S3 Bucket Using $out <https://www.mongodb.com/developer/how-to/atlas-data-lake-federated-queries-out-aws-s3/>`__ - 
  This tutorial shows you how to query data from multiple sources and 
  write the results of those queries to an |aws| |s3| bucket using 
  {+adl+} :ref:`adl-out-stage` syntax.

- `Copy Data Automatically From MongoDB to an AWS S3 Bucket <https://www.mongodb.com/developer/how-to/automated-continuous-data-copying-from-mongodb-to-s3/>`__ - 
  This tutorial shows you how to use |service| :atlas:`triggers 
  </triggers/>` and {+adl+} to write data from your |service| Cluster 
  to your |aws| |s3| bucket continuously on a schedule.

- `Archive Data to Cloud Object Storage Using Online Archive <https://www.mongodb.com/developer/how-to/atlas-data-lake-online-archive/>`__ - 
  This tutorial shows you how to use |service-fullname| Online Archive 
  to automatically archive data off of your |service| cluster to 
  fully-managed cloud object storage.
