io.realm.mongodb.functions
==========================


 | `java.lang.Object <https://docs.oracle.com/javase/7/docs/api/java/lang/Object.html>`__ 
 | 		io.realm.mongodb.functions.Functions


A *Functions* manager to call remote Realm functions for the associated Realm App.Arguments and results are encoded/decoded with the *Functions'* codec registry either inherited from the :ref:`AppConfiguration.getDefaultCodecRegistry() <io_realm_mongodb_AppConfiguration_getDefaultCodecRegistry__>`  or set explicitly when creating the 

*Functions*-instance through :ref:`User.getFunctions(CodecRegistry) <io_realm_mongodb_User_getFunctions_CodecRegistry_>`  or through the individual calls to :ref:`callFunction(String, List, Class, CodecRegistry) <io_realm_mongodb_functions_Functions_callFunction_String__List__Class__CodecRegistry_>` .
