Hedged reads are supported for the following operations:

.. hlist::
   :columns: 2

   - :dbcommand:`collStats`
   - :dbcommand:`count`
   - :dbcommand:`dataSize`
   - :dbcommand:`dbStats`
   - :dbcommand:`distinct`
   - :dbcommand:`filemd5`
   - :dbcommand:`find`
   - :dbcommand:`listCollections`
   - :dbcommand:`listIndexes`
   - :dbcommand:`planCacheListFilters`
