.. procedure::
   :style: normal

   .. step:: Click :guilabel:`Edit Document`

      At the top of this document, click :guilabel:`Edit Document`.
      |vsce| opens it as an editable |ejson| document titled ``<database>.<collection>:"<_id>"``.

      Make any necessary edits to the dcument.

   .. step:: Save the edited document

      Press :kbd:`Ctrl + S` (Windows/Linux) or :kbd:`Cmd + S` to save the
      edited document to the MongoDB database.

      If the update succeeds, |vsce| confirms that the database has
      stored the change. If the update results in an error, |vsce| displays 
      the error.
