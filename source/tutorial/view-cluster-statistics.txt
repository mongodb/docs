=============================
Aggregated Cluster Statistics
=============================

.. default-domain:: mongodb

Overview
--------

Cluster statistics provide an interface to view data for an entire
cluster at once. You can compare components dynamically across the
cluster and view host-specific and aggregated data, as well as
pinpoint moments in time and isolate data to specific components.
