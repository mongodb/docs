
a. MongoDB needs a data directory to store your data. By
   default, it uses ``C:\data\db``, but you may specify a
   different location in your config file. You can create the
   data directory using the Windows Command Prompt: 

   .. code-block:: bat 
        
      mkdir c:\data\db

#. If you specified a log path in the configuration file, create
   the log directory in the same way.
