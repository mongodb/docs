Introduction
------------

This tutorial explores features and capabilities of |charts| with
a :doc:`data source </data-sources>` containing the ``Supply Store``
:atlas:`sample data provided by Atlas </sample-data/sample-supplies>`.
This sample data contains sales order data from a mock office supply
company.

This tutorial demonstrates how to create a
:doc:`dashboard </dashboards>` containing charts of various types and
how to effectively use each chart's respective properties to obtain
powerful visualizations. You will also learn how to use multi-series
charts to gain further insight into the data.

After completing the tutorial, you will have a dashboard which looks
similar to the following:

.. figure:: /images/charts/order-tutorial-dashboard-final.png
   :figwidth: 750px
   :alt: Finished Tutorial Dashboard
