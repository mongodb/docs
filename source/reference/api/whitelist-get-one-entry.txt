Syntax
------

.. code-block:: none

   GET /groups/{GROUP-ID}/whitelist/{WHITELIST-ENTRY}

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :widths: 15 10 75

   * - Path Element
     - Required/Optional
     - Description

   * - ``GROUP-ID``

     - Required.

     - The unique identifier for the :ref:`project <group-id>` from
       which you want to retrieve a whitelist entry.

   * - ``WHITELIST-ENTRY``

     - Required.

     - The IP or :abbr:`CIDR (Classless Inter-Domain Routing)` address.
       If the entry includes a subnet mask, use the URL-encoded value
       ``%2F`` for the forward slash ``/``.

Example Request
---------------

.. code-block:: shell

   curl --user "{USERNAME}:{APIKEY}" --digest --include \
     --header "Accept: application/json" \
     --header "Content-Type: application/json" \
     --request GET "https://cloud.mongodb.com/api/atlas/v1.0/groups/5aec7afbf7d068e35a39809b/whitelist/203.0.113.0%2F24?pretty=true"
