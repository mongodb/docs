========================
Cloud Provider Snapshots
========================

.. default-domain:: mongodb

.. include:: /includes/fact-projects-groups-synonyms.rst

.. admonition:: Cloud Provider Snapshots Only
   :class: important

   This resource only applies to clusters using
   :ref:`backup-cloud-provider`. 

The ``/backup/snapshots`` resource enables you to view and delete
Cloud Provider Snapshots.

Endpoints
---------

.. include:: /includes/api-base-url.rst

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/backup/snapshots </reference/api/cloud-provider-snapshot-get-all>`
     - Get all cloud provider snapshots for the specified cluster.

   * - ``GET``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/backup/snapshots/{SNAPSHOT-ID} </reference/api/cloud-provider-snapshot-get-one>`
     - Get the snapshot associated to ``{SNAPSHOT-ID}``

   * - ``DELETE``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/backup/snapshots/{SHAPSHOT-ID} </reference/api/cloud-provider-snapshot-delete-one>`
     - Delete the snapshot associated to ``{SNAPSHOT-ID}``
