# Entity Framework

**Add for:** major, minor, and patch releases

| Variable | Value |
|---|---|
| `GITHUB_REPO` | `mongodb/mongo-efcore-provider` |
| `RELEASE_NOTES_FILE` | `content/entity-framework/upcoming/source/reference/release-notes.txt` |
| `CHANGELOG_LINK` | `mongodb/mongo-efcore-provider/releases/tag/v{FULL_VERSION}` |
| `UPGRADE_REF` | `entity-framework-breaking-changes-v{VERSION}` |

- Breaking changes: replace `{+driver-short+} v{VERSION}` with `The {VERSION} provider` in the admonition template.
- Upgrade guide section heading: `Version {FULL_VERSION} Breaking Changes`
  (not `Version {VERSION} Breaking Changes`).
