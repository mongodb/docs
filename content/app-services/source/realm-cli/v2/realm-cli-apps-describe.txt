realm-cli apps describe

Displays information about your App

View all of the aspects of your App to see what is configured and enabled
(e.g. services, functions, etc.). If you have more than one App, you will
be prompted to select an App to view.

Syntax
------

.. code-block::

   realm-cli apps describe [options]

Options
-------

.. list-table::
   :header-rows: 1
   :widths: 20 10 10 60

   * - Name
     - Type
     - Required
     - Description
   * - -a, --app
     - string
     - false
     - Specify the name or ID of an App to describe
   * - -h, --help
     - 
     - false
     - help for describe
