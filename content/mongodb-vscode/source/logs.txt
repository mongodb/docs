.. _mdb-vsce-logs:

==================================
View MongoDB VSCode Extension Logs
==================================

.. meta::
   :description: View logs for the MongoDB VS Code extension to help troubleshoot issues.

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

You can view logs for |vsce-full| to troubleshoot extension issues.

Steps
-----

.. procedure::
   :style: normal

   .. step:: Open the Command Palette (Ctrl+Shift+P or
      Cmd+Shift+P)

   .. step:: Run the :guilabel:`Developer: Open Log...`
      command

   .. step:: Select :guilabel:`MongoDB Extension` from the list of log
      files

Examples
--------
