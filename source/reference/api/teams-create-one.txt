Create a Team
=============

Syntax
------

.. code-block:: none

   POST /orgs/{ORG-ID}/teams

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :widths: 15 10 75

   * - Path Element
     - Required/Optional
     - Description

   * - ``ORG-ID``

     - Required.

     - The unique identifier for the 
       :doc:`organization </tutorial/manage-organizations/>`
       you want to associate the team with.

Request Query Parameters
~~~~~~~~~~~~~~~~~~~~~~~~

.. include:: /includes/api/no-query-single-parameters.rst

Request Body Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :widths: 25 35 50
   :header-rows: 1

   * - Name
     - Type
     - Description

   * - ``name``
     - string
     - The name of the team you want to create.

   * - ``usernames``
     - array of strings
     - Valid email addresses of users to add to the new team. |service|
       checks whether the user's email belongs to the organization,
       so that |service| can properly associate the users to the teams.
       The command fails and issues an error if you provide invalid
       email addresses to it.

Response
--------

.. list-table::
   :header-rows: 1
   :widths: 25 25 60

   * - Name
     - Type
     - Description

   * - ``id``
     - string
     - The unique identifier for the team.

   * - ``name``
     - string
     - The name of the organization associated with the team.

   * - ``usernames``
     - array of strings
     - Usernames added to the new team. 

   * - ``links``
     - object array
     - .. include:: /includes/links-explanation.rst

Example
-------

Request
~~~~~~~


.. code-block:: none

   curl -u "{PUBLIC-KEY}:{PRIVATE-KEY}" --digest \
    --header "Accept: application/json" \
    --header "Content-Type: application/json" \
    --request POST "https://cloud.mongodb.com/api/atlas/v1.0/orgs/{ORG-ID}/teams/" \
    --data '{ "name" : "myNewTeam", "usernames": ["jane.a.smith@example.com", "jane.b.smith@example.com", "jane.c.smith@example.com"] }'

Response
~~~~~~~~

.. code-block:: none

   {
    "id": "{TEAM-ID}",
    "links": [
        {
            "href": "https://cloud.mongodb.com/api/atlas/v1.0/orgs/{ORG-ID}/teams/{TEAM-ID}",
            "rel": "self"
        }
    ],
    "name": "myNewTeam",
    "usernames": ["jane.a.smith@example.com", "jane.b.smith@example.com", "jane.c.smith@example.com"]
   }
