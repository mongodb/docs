=======================================
Upgrade |onprem| with a ``deb`` Package
=======================================

.. default-domain:: mongodb

Overview
--------

This tutorial describes how to upgrade an existing |onprem| installation
using a ``deb`` package. |onprem| supports direct upgrades from version 1.6
and later.
