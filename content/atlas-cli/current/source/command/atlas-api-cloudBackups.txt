======================
atlas api cloudBackups
======================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

`Public Preview: please provide feedback at <https://feedback.mongodb.com/forums/930808-atlas-cli>`_: Manages Cloud Backup snapshots, snapshot export buckets, restore jobs, and schedules.

This resource applies only to clusters that use Cloud Backups.
