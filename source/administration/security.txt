Security Tutorials
==================

.. contents:: On this page

The following tutorials provide instructions for enabling and using
the security features available in MongoDB.

Before enabling role based access control, you should first consider the
users of the system. Once the users have been identified, determine the
roles required by the users. Roles may inherit from other roles to
provide a hierarchy.
