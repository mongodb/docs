==========================
AWS Self-Serve Marketplace
==========================

.. meta::
   :description: Link or unlink your AWS Billing Account to Atlas for flexible payment options via AWS Marketplace.

.. default-domain:: mongodb

.. facet::
   :name: genre
   :values: tutorial

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

|service| supports paying for usage via the AWS Marketplace (AWS MP)
without any upfront commitment. You can link an |service| account 
to an AWS Billing Account, as well as delete an existing 
subscription.

The following procedures cover |aws| Marketplace Subscriptions and
not |aws| Marketplace Contracts.
