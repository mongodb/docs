===================
MongoDB with Kotlin
===================

.. meta::
   :description: Develop server-side Kotlin applications using Atlas with coroutine-based and synchronous MongoDB drivers.

.. include:: /includes/docs-toc.rst

.. introduction::
      
   Develop server-side Kotlin applications with MongoDB's :atlas:`Atlas cloud database </>`. With support for 
   both coroutine-based and synchronous development, the MongoDB Kotlin drivers
   support developers in building server-side applications in Kotlin that are interoperable with the JVM ecosystem.

   .. button:: Get started with the MongoDB Kotlin Drivers
      :uri: https://www.mongodb.com/docs/drivers/kotlin-drivers/

.. image:: /languages/images/kotlin.svg
   :alt: Kotlin landing page hero image

Drivers
-------

MongoDB provides drivers for both coroutine-based and synchronous
applications.

.. card-group::
   :columns: 2

   .. card::
      :headline: Kotlin Coroutine Driver
      :url: https://www.mongodb.com/docs/drivers/kotlin/coroutine/current/
      :icon: /images/icons/kotlin.svg
      :icon-alt: Driver card icon

      Provides an API to connect to MongoDB in server-side Kotlin applications that use coroutines.

   .. card::
      :headline: Kotlin Sync Driver
      :url: https://www.mongodb.com/docs/drivers/kotlin-sync/
      :icon: /images/icons/kotlin.svg
      :icon-alt: Driver card icon

      Provides an API to connect to MongoDB in synchronous Kotlin
      applications. The Kotlin Sync Driver is built on the Kotlin Coroutine Driver.
