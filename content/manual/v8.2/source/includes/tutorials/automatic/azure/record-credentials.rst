.. important:: Record your Credentials

    Ensure you record the following credentials:

    - **Tenant ID**
    - **Client ID**
    - **Client secret**

    You need these credentials to construct your ``kmsProviders``
    object later in this tutorial. You can also authenticate by using an
    access token or automatic credential fetching. To learn about
    all available credential forms for Azure, see
    :ref:`qe-kms-provider-object-azure`.
