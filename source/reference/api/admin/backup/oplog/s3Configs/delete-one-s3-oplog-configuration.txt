Delete One S3 Oplog Configuration

.. default-domain:: mongodb

.. contents:: On this page

Deletes the configuration of one s3 oplog store.

.. include:: /includes/api/base-admin-backup-api-uri.rst

Resource
--------

.. code-block:: http

   DELETE /oplog/s3Configs/{S3-OPLOG-CONFIG-ID}

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :widths: 20 20 60
   :header-rows: 1
   :stub-columns: 1

   * - Name
     - Type
     - Description

   * - S3-OPLOG-CONFIG-ID
     - string
     - The ID that represents the S3 Oplog configuration.
