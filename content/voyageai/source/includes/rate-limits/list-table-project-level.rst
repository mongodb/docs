.. list-table::
   :header-rows: 1 
   :widths: 30 70 

   * - Column Name 
     - Column Description 
     
   * - :guilabel:`Model`
     - List of {+voyageai+} embedding models.

   * - :guilabel:`Tokens Per Minute (TPM)`
     - Number of tokens that you can request within a minute from the 
       {+voyageai+} {+voyage-api+} endpoints.

   * - :guilabel:`Requests Per Min (RPM)`
     - Number of requests you can send within a minute to the {+voyageai+}
       {+voyage-api+} endpoints.

   * - :guilabel:`Actions`
     - Actions you can take. You can: 
     
       - Reduce the number of tokens and requests per minute for the
         project.
       - Undo custom number of tokens and requests per minute while
         setting it.
