===========================
Connect via ``mongo`` Shell
===========================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol


The :guilabel:`Connect` dialog for a cluster provides the details to
connect to a cluster via the :program:`mongo <bin.mongo>` shell. To
connect using the :program:`mongo <bin.mongo>`, run the 3.2.6 or later
version with TLS/SSL support. If you do not have the latest 3.2 version
of the :program:`mongo <bin.mongo>` shell, you can download it from the
:guilabel:`Connect` dialog.

You can also download the Server package, which includes the
:program:`mongo <bin.mongo>` shell from the `Download Center
<https://www.mongodb.com/download-center>`_. If downloading the version
3.2.6 Server package for OSX, do not use homebrew to install the the
package for use with |service|; instead, download the ``*.tgz`` or
``*.zip`` file.

Connect to Your Cluster
-----------------------

Use the displayed connection details from the :guilabel:`Connect`
dialog to connect your :program:`mongo <bin.mongo>` shell to the
cluster. Specify the password for your user.

For example:

.. code-block:: sh

   mongo "mongodb://mycluster0-shard-00-00-wpeiv.mongodb.net:27017,mycluster0-shard-00-01-wpeiv.mongodb.net:27017,mycluster0-shard-00-02-wpeiv.mongodb.net:27017/admin?replicaSet=Mycluster0-shard-0" --ssl --username kay --password myRealPassword

.. include:: /includes/extracts/connection-details-shell.rst

For example, to connect to the ``test`` database and authenticate to
``admin``:

.. code-block:: sh

   mongo "mongodb://mycluster0-shard-00-00-wpeiv.mongodb.net:27017,mycluster0-shard-00-01-wpeiv.mongodb.net:27017,mycluster0-shard-00-02-wpeiv.mongodb.net:27017/test?replicaSet=Mycluster0-shard-0" --ssl --username kay --password myRealPassword --authenticationDatabase admin
