===============================================
mongocli atlas privateendpoint interface create
===============================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``privateendpoint interface create`` command adds one interface
endpoint to one private endpoint connection for a given |service|
project.

You must first :aws:`create the endpoint 
</AWSEC2/latest/APIReference/API_CreateVpcEndpoint.html>` in |aws| with
the following information:

- The endpoint service name that |service| creates with 
  :ref:`mcli-atlas-privateendpoint-create-cmd`.

  Retrieve the endpoint service name with 
  :ref:`mcli-atlas-privateendpoint-describe-cmd`.
- The unique identifier of the peer |aws| |vpc|.
- The unique identifiers of the subnets your |aws| VPC uses.

If the attempt to add an interface endpoint fails, :ref:`delete it 
<mcli-atlas-privateendpoint-describe-cmd>`, then
try to add a new one.

Syntax 
------

.. code-block:: text 

   mongocli atlas privateendpoint(s)|privateEndpoint(s) interface(s) create <interface-endpoint-id>
        --privateEndpointId <private-endpoint-id>
        [ --output|-o <output-format> ]
        [ --profile|-P <profile-name> ]
        [ --projectId <project-ID> ]

.. _atlas-privateendpoint-create-arguments:

Arguments  
---------

.. list-table::
   :header-rows: 1
   :widths: 20 10 60 10

   * - Argument 
     - Type 
     - Description 
     - Required?

   * - ``<interface-endpoint-id>``
     - string
     - Unique identifier of the interface endpoint that you
       want to add to the private endpoint connection.
     - yes

.. _atlas-privateendpoint-interface-create-options:

Options 
-------

.. list-table::
   :widths: 20 10 60 10
   :header-rows: 1

   * - Option 
     - Type
     - Description
     - Required?

   * - ``--privateEndpointId``
     - string
     - Unique identifier of the private endpoint connection for which
       you want to add the interface endpoint.
     - yes

   * - ``--output``, ``-o``
     - string 
     - .. include:: /includes/extracts/fact-basic-options-output.rst
     - no

   * - ``--profile``, ``-P``
     - string
     - Name of the profile where the public and private 
       keys for the project are saved. If omitted, uses the 
       {+default-profile+}. To learn more about creating a 
       profile, see :ref:`mcli-configure`.
     - no

   * - ``--projectId``
     - string
     - Unique identifier of the project for which you want to add
       the interface endpoint. If omitted, uses the
       project ID in the profile or :ref:`environment variable
       <mcli-env-var>`.
     - no

.. _atlas-privateendpoint-interface-create-output:

Output
------

If the command succeeds, it returns the following output in the default
format. If the command returns errors, see
:ref:`Troubleshooting <troubleshooting>` for recommended solutions.

.. code-block:: sh
   :copyable: false

   Interface endpoint '<interface-endpoint-id>' created.

.. include:: /includes/fact-default-output.rst

- :atlas:`Atlas API </reference/api/private-endpoint-create-one-interface-endpoint/#response-elements>`

Example
-------

.. tabs::

   .. tab:: Default Output
      :tabid: default-output

      The following command adds one interface endpoint to one private
      endpoint connection for a project using the {+default-profile+},
      which contains credentials and the project ID. The output is
      returned in the default format.

      .. code-block:: sh

         atlas privateendpoint interface create vpce-05ad08ff76fceb65b \
         --privateEndpointId 5f59398a33b5595e0cad7f0b

      The previous command prints the following to the terminal.

      .. code-block:: sh
         :copyable: false

         Interface endpoint 'vpce-05ad08ff76fceb65b' created.

   .. tab:: JSON Output
      :tabid: json-output

      The following command adds one interface endpoint to one private
      endpoint connection for a project using the {+default-profile+},
      which contains credentials and the project ID. The output is
      returned in |json| format.

      .. code-block:: sh

         atlas privateendpoint interface create vpce-05ad08ff76fceb65b \
         --privateEndpointId 5f59398a33b5595e0cad7f0b --output json

      The previous command prints the following to the terminal in
      |json| format.

      .. code-block:: json
         :copyable: false

         {
           "interfaceEndpointId": "vpce-05ad08ff76fceb65b",
           "deleteRequested": false,
           "connectionStatus": "PENDING_ACCEPTANCE"
         }
