=====
Disks
=====

.. default-domain:: mongodb

Retrieves the disks and disk partitions on which MongoDB runs.

.. only:: cloud

    .. include:: /includes/api-cm-base-url.rst

.. only:: onprem

    .. include:: /includes/api-ops-base-url.rst


Endpoints
---------

The following endpoints are available for ``hosts``.

.. list-table::
   :header-rows: 1
   :widths: 10 70 20

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :doc:`/api/public/v1.0/groups/{GROUP-ID}/hosts/{HOST-ID}/disks </reference/api/disks-get-all>`
     - Retrieves all disk partitions on the specified host.

   * - ``GET``
     - :doc:`/api/public/v1.0/groups/GROUP-ID/hosts/HOST-ID/disks/{PARTITION-NAME} </reference/api/disk-get-one>`
     - Retrieves a single disk parition.
