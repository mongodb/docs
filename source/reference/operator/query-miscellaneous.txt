   * - Name

   * - :query:`$rand`
     - Generates a random float between 0 and 1.
   * - :operator:`$natural`
     - A special hint that can be provided via the
       :method:`~cursor.sort()` or :method:`~cursor.hint()` methods
       that can be used to force either a forward or reverse collection
       scan.
