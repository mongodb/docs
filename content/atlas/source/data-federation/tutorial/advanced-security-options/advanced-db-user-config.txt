==========================================
Advanced User Configuration
==========================================

.. meta::
   :description: Learn how to configure advanced database user authentication and authorization settings in Atlas Data Federation using the Atlas CLI, Admin API, or Atlas UI.

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

Overview
--------

Learn about advanced authentication and authorization settings for
database users in {+adf+}, including available authentication methods.
You can manage users with the {+atlas-cli+}, the {+atlas-admin-api+},
and the {+atlas-ui+}.

.. note::

   .. include:: /includes/fact-database-user-limits.rst

Required Access
---------------

To use the advanced database user configurations, you must have
:authrole:`Organization Owner`, :authrole:`Project Owner`, or
:authrole:`Project Database Access Admin` access in |service|.

Database Actions
----------------

Use the {+atlas-cli+}, {+atlas-admin-api+}, or {+atlas-ui+} to
manage database users for {+adf+}.

.. composable-tutorial::
   :options: interface-default-atlas-cli
   :defaults: atlas-cli

   Authenticate Database Users
   ~~~~~~~~~~~~~~~~~~~~~~~~~~~

   {+adf+} offers the following forms of authentication
   for database users:

   .. include:: /includes/atlas-auth-methods.rst

   .. _adf-add-mongodb-users:

   Add New Database Users
   ~~~~~~~~~~~~~~~~~~~~~~

   A project can have users with different authentication methods.

   A user's authentication method is fixed at creation. To use a
   different authentication method, create a new user.

   .. Attention writers: If you make changes to the following
   .. procedure please check the related procedures in the following
   .. Server docs page(s) to see if they require changes as well:
   .. www.mongodb.com/docs/manual/tutorial/create-users/

   .. selected-content::
      :selections: atlas-cli

      The {+atlas-cli+} uses the following commands to create new
      database users and X.509 certificates. The options you specify
      determine the authentication method.

      .. include:: /includes/extracts/atlas-dbusers-create-and-certs-create.rst

   .. selected-content::
      :selections: admin-api

      You can add database users through the {+atlas-admin-api+}. The
      options you specify determine the authentication method. To learn
      more, see :oas-atlas-op:`Create One Database User
      </createDatabaseUser>`.

   .. selected-content::
      :selections: atlas-ui

      Select an authentication mechanism and follow the steps to create
      a new database user using the {+atlas-ui+}.

      .. include:: /includes/create-db-user-ui.rst
