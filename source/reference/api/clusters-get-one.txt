Syntax
------

.. code-block:: none

   GET /api/atlas/v1.0/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :header-rows: 1
   :widths: 15 10 75

   * - Path Element
     - Required/Optional
     - Description

   * - ``GROUP-ID``

     - Required.

     - The unique identifier for the :ref:`group <group-id>` containing the
       cluster you want to retrieve.

   * - ``CLUSTER-NAME``

     - Required

     - The name of the cluster to retrieve.

~~~~~~~~~~~~~~~~~~~~~~

Request
~~~~~~~

.. code-block:: none

   curl -i -u "username:apiKey" --digest "https://cloud.mongodb.com/api/atlas/v1.0/groups/5356823b3794de37132bb7b/clusters/LogData"
