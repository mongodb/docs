The {+sql-interface+} supports multiple authentication mechanisms
including :manual:`SCRAM-SHA-1, SCRAM-SHA-256 </core/security-scram/>`,
:ref:`MONGODB-X509 <security-auth-x509>`, :ref:`LDAP <security-auth-ldap>`,
:ref:`Kerberos (GSSAPI) <security-auth-kerberos>`,
:ref:`MongoDB Passwordless Authentication with AWS <set-up-pwdless-auth>`,
and :manual:`OpenID Connect (OIDC) </core/security-oidc/>`.

For comprehensive authentication information and component compatibility
details, see :ref:`sql-authentication`.
