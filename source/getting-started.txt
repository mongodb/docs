===============
Getting Started
===============

.. meta::
   :description: Create an AtlasGov account, organization, project, cluster, and database user with this getting started guide for MongoDB Atlas for Government.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

|cloudgov| is a fully-managed cloud database for the U.S. government 
(federal agencies and state, local, and education (SLED) entities).

This tutorial guides you through creating:

- An |cloudgov-short| account
- An organization
- A project
- An accessible cluster
- A database user for your cluster
