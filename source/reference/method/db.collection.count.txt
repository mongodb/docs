=====================
db.collection.count()
=====================

.. default-domain:: mongodb

Definition
----------

.. method:: db.collection.count( )

   Returns the count of documents that would match a :method:`find()
   <db.collection.find()>` query. The :method:`db.collection.count()`
   method does not perform the :method:`find() <db.collection.find()>`
   operation but instead counts the results that would be returned by
   the :method:`find() <db.collection.find()>`.

   The :method:`db.collection.count()` method takes the following form:

   .. code-block:: javascript

      db.collection.find(<query>).count();

   The :method:`db.collection.count()` method has the following parameter:

   .. include:: /reference/method/db.collection.count-param.rst

Examples
--------

The following are examples of the :method:`db.collection.count()` method.

.. example:: Count the number of all documents in the ``orders`` collection:

   .. code-block:: javascript

      db.orders.count()

   The query is equivalent to the following:

   .. code-block:: javascript

      db.orders.find().count()

.. example:: Count the number of the documents in the ``orders``
   collection with the field ``ord_dt`` greater than ``new
   Date('01/01/2012')``:

   .. code-block:: javascript

      db.orders.count( { ord_dt: { $gt: new Date('01/01/2012') } } )

   The query is equivalent to the following:

   .. code-block:: javascript

      db.orders.find( { ord_dt: { $gt: new Date('01/01/2012') } } ).count()

.. seealso:: :method:`cursor.count()`
