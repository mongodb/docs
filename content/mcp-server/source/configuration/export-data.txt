Export Directory Path
---------------------

The following table shows the default directory paths where the MCP
Server temporarily stores the exported data files:

.. list-table::
   :header-rows: 1

   * - Operating System
     - Default Directory Path

   * - Windows
     - ``%LOCALAPPDATA%\mongodb\mongodb-mcp\exports``

   * - macOS and Linux
     - ``~/.mongodb/mongodb-mcp/exports``

To change the export directory path, set one of these:

- ``exportsPath`` option in a JSON configuration file or command line
- ``MDB_MCP_EXPORTS_PATH`` environment variable

To view the JSON configuration file options, see
:ref:`mcp-server-configuration-options`. To view JSON configuration file
examples, see :ref:`mcp-get-started`.

The ``exportTimeoutMs`` configuration option specifies the time after
which the exported data is eligible for deletion.

The ``exportCleanupIntervalMs`` configuration option specifies the time
interval between runs of the cleanup process that deletes expired
export files.

For an export example, see :ref:`mcp-server-example-export-data`.
