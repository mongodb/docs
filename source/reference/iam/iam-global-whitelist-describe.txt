.. _mcli-iam-global-whitelist-describe:

=====================================
mongocli iam globalWhitelist describe
=====================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``iam globalWhitelist describe`` command provides details about
the specified global whitelist entry. You can also retrieve details 
about a global whitelist entry using the :opsmgr:`API 
</reference/api/api-keys/global/get-one-global-whitelist/>`.

You must have credentials with the :opsmgr:`Global Read Only 
</reference/user-roles/#Global-Read-Only>` role to retrieve details 
about a global whitelist entry for an |onprem| instance. 

.. _mcli-iam-global-whitelist-describe-syntax:

Syntax 
------

.. code-block:: text 

   mongocli iam globalWhitelist|ipwhitelist describe 
        <whitelist-entry-id>
        [ --output|-o <output-format> ]
        [ --profile|-P <profile-name> ]

.. include:: /includes/fact-command-line-help.rst

.. _mcli-iam-global-whitelist-describe-arguments:

Arguments  
---------

.. list-table::
   :header-rows: 1
   :widths: 20 10 60 10

   * - Argument 
     - Type 
     - Description 
     - Required?

   * - ``<whitelist-entry-id>``
     - string
     - Unique identifier of the global whitelist entry to retrieve.
     - yes

.. _mcli-iam-global-whitelist-describe-options:

Options 
-------

.. list-table:: 
   :header-rows: 1
   :widths: 20 10 60 10

   * - Option
     - Type 
     - Description 
     - Required? 

   * - ``--output``, ``-o``
     - string 
     - .. include:: /includes/extracts/fact-basic-options-output.rst
     - no

   * - ``--profile``, ``-P`` 
     - string 
     - Name of the profile where your credentials are saved. If omitted, 
       uses the {+default-profile+}. To learn more about creating a profile, 
       see :ref:`mcli-configure`.
       
       You must have credentials with the :opsmgr:`Global Read Only 
       </reference/user-roles/#Global-Read-Only>` role to
       retrieve details about a global whitelist entry for an |onprem| 
       instance. 
     - no

.. _mcli-iam-global-whitelist-describe-output:

Output
------

If the command succeeds, it returns the following output to the terminal 
in the default format. If the command returns errors, see :ref:`Troubleshooting 
<troubleshooting>` for recommended solutions.

.. code-block:: sh
   :copyable: false

   ID                    CIDR BLOCK          CREATED AT
   <whitelist-entry-id>  <cidr-block-1>      <date>

The default output is a subset of the fields returned by this command. 
For the complete list of |json| fields returned by the command, see 
the :opsmgr:`API reference 
</reference/api/api-keys/global/get-one-global-whitelist/#response>`. 

.. _mcli-iam-global-whitelist-describe-examples:

Examples
--------

.. tabs::

   .. tab:: Default Output
      :tabid: default-output

      The following command describes the specified global whitelist entry 
      using the {+default-profile+}, which contains credentials for accessing 
      the |onprem| instance. The output is returned in the default format.

      .. code-block:: sh

         mongocli iam globalWhitelists describe 5f5bb07e57aef32b04ed0770

      The previous command prints the following to the terminal.

      .. code-block:: json
         :copyable: false

         ID                         CIDR BLOCK          CREATED AT
         5f5bb07e57aef32b04ed0770   192.168.0.1/32      2020-09-11T17:59:04Z

   .. tab:: JSON Output
      :tabid: json-output

      The following command retrieves details about the specified global 
      whitelist entry using the ``myOM`` profile, which contains credentials 
      for accessing the |onprem| instance. The output is returned in
      |json| format.
      
      .. code-block:: sh

         mongocli iam globalWhitelists describe 5f5bb07e57aef32b04ed0770 -P myOM --output json

      The previous command prints the following to the terminal in |json| 
      format.

      .. code-block:: json
         :copyable: false

         {
           "id": "5f5bb07e57aef32b04ed0770",
           "cidrBlock": "192.168.0.1/32",
           "created": "2020-09-11T18:01:01Z",
           "description": "My Global IP",
           "type": "GLOBAL_ROLE",
           "updated": "2020-09-11T18:01:01Z"
         }
