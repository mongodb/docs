Downgrade the ``featureCompatibilityVersion`` to |oldfcv|.

.. important::

   After you modify the ``featureCompatibilityVersion`` and enable new
   features, you cannot downgrade the binary version without
   assistance from support. Make sure to evaluate your upgrade path
   before proceeding.

.. code-block:: javascript

   db.adminCommand( { setFeatureCompatibilityVersion: "8.2", confirm: true } )

The :dbcommand:`setFeatureCompatibilityVersion` command performs writes
to an internal system collection and is idempotent. If the command does
not complete successfully, retry the command on the |target|.
