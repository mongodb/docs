=================================================================
Generate a Wildcard Collection for a {+FDI+}
=================================================================

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

You can dynamically generate collection names that map to data to your 
|service| cluster or |aws| |s3| Bucket. To dynamically generate collection 
names, specify the wildcard (*), as the value for the collection name setting
in your {+fdi+} storage configuration.

You can use the storageSetConfig command to configure the settings for 
generating wildcard (*) collections.

To learn how to generate a wildcard collection in your Data Store,
Select your {+fdi+} Cloud provider from the dropdown menu:
