Facet or list of facets to exclude from count recalculation on the
basis of this query. Value must be one or more names of a facet
defined in ``$search.facets`` or ``$searchMeta.facets``. The operator
uses the specified facets for filtering documents to match this query,
but doesn't recalculate counts for them. For more information, see
:ref:`Multi-Select Facets <fts-facet-multi-select>`.
