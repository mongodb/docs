.. _mcli-cm-feature-policies-list-command:

===========================================
mongocli cloud-manager featurePolicies list
===========================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``featurePolicies list`` command retrieves all
|cloud-short| :cloudmgr:`feature control policies </reference/api/feature-control-policies>`
for the specified project. You can also retrieve all
feature policies through the |cloud-short| :cloudmgr:`API 
</reference/api/controlled-features/get-all-feature-control-policies/>`.

.. _mcli-cm-feature-policies-list-syntax:

Syntax
------

.. code-block:: text 

   mongocli cloud-manager|cm featurePolicies list|ls
        [ --limit <number-of-items-per-page> ]
        [ --output|-o <output-format> ]
        [ --page <page-num> ]
        [ --profile|-P <profile-name> ]
          --projectId <id-of-project>

Output

.. include:: /includes/command-output-intro.rst



.. code-block:: none
   :copyable: false

   NAME                     SYSTEM ID              POLICY
   <external-system-label>  <external-system-id>   <policy-name>

If you request output in |json| format, the command returns the
fields described in the API :cloudmgr:`reference </reference/api/controlled-features/get-all-feature-control-policies/#response>`.

Example
-------

The following example uses the ``mongocli featurePolicies cm
list`` command to retrieve all feature control policies
for the project specified in the profile called ``myCm``.

.. tabs::

   .. tab:: Default
      :tabid: default

      .. code-block:: none
   
         mongocli cm featurePolicies list -P myCm

      The previous command produces the following output. The output lists
      the ``EXTERNALLY_MANAGED_LOCK`` policy that is specified for this project.

      .. code-block:: none
         :copyable: false

         NAME                 SYSTEM ID    POLICY
         my-external-system                EXTERNALLY_MANAGED_LOCK

   .. tab:: JSON
      :tabid: json

      .. code-block:: none
   
         mongocli cm featurePolicies list -P myCm --output json

      The previous command produces the following output. The output lists
      the ``EXTERNALLY_MANAGED_LOCK`` policy that is specified for this project.

      .. code-block:: json
         :copyable: false

         {
          "created": "2021-01-22T15:10:39Z",
          "updated": "2021-01-22T15:11:32Z",
          "externalManagementSystem": {
            "name": "my-external-system"
          },
          "policies": [
            {
              "policy": "EXTERNALLY_MANAGED_LOCK"
            }
           ]
         }
