.. _api-online-archive:

==============
Online Archive
==============

.. default-domain:: mongodb

The ``onlineArchives`` resource provides access to create, edit, 
delete, pause, and resume an :ref:`online archive <online-archive-overview>` 
for a collection. You can also :ref:`configure and manage online archives 
<online-archive-overview>` from the {+atlas-ui+}. 

.. include:: /includes/fact-projects-groups-synonyms.rst

.. include:: /includes/api-base-url-new.rst

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :ref:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/onlineArchives <api-online-archive-get-all-for-clstr>` 
     - Return all the online archives for a cluster.

   * - ``GET``
     - :ref:`/groups/{GROUP-ID}/privateNetworkSettings/endpointIds <api-online-archive-pvt-link-get-all>` 
     - Return all the private endpoints for the online archives.

   * - ``GET``
     - :ref:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/onlineArchives/{ARCHIVE-ID} <api-online-archive-get-one>` 
     - Return the online archive specified by its ID.

   * - ``GET``
     - :ref:`/groups/{GROUP-ID}/privateNetworkSettings/endpointIds/{ENDPOINT-ID} <api-online-archive-pvt-link-get-one>` 
     - Return one private endpoint for the online archives.

   * - ``GET``
     - :ref:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/onlineArchives/queryLogs.gz <api-online-archive-download-query-logs>` 
     - Download query logs for an online archive.

   * - ``POST`` 
     - :ref:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/onlineArchives <api-online-archive-create-one>`
     - Create an online archive for a collection.

   * - ``POST`` 
     - :ref:`/groups/{GROUP-ID}/privateNetworkSettings/endpointIds <api-online-archive-pvt-link-create-one>`
     - Add one |aws| private endpoint for the online archives in the 
       specified project.

   * - ``PATCH``
     - :ref:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/onlineArchives/{ARCHIVE-ID} <api-online-archive-update-one>`
     - Update, pause, or resume an online archive by its ID.

   * - ``DELETE``
     - :ref:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/onlineArchives/{ARCHIVE-ID} <api-online-archive-delete-one>`
     - Delete an online archive by its ID.

   * - ``DELETE``
     - :ref:`/groups/{GROUP-ID}/privateNetworkSettings/endpointIds/{ENDPOINT-ID} <api-online-archive-pvt-link-delete-one>`
     - Delete one |aws| private endpoint for the online archives in the 
       specified project.
