=================
Snapshot Schedule
=================

.. default-domain:: mongodb

.. include:: /includes/fact-projects-groups-synonyms.rst

.. admonition:: Continuous Backups Only
   :class: important

   This resource only applies to clusters using
   :ref:`backup-continuous`.

This resource enables you to view and edit the :ref:`backup snapshot
schedule <modify-backup-schedule>` for a given cluster.

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/snapshotSchedule </reference/api/snapshot-schedule-get>`
     - Get the snapshot schedule for the specified cluster.

   * - ``PATCH``
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/snapshotSchedule </reference/api/snapshot-schedule-patch>`
     - Update the snapshot schedule for the specified cluster.
