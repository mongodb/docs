.. _private-endpoints-api:

=================
Private Endpoints
=================

.. default-domain:: mongodb

.. include:: /includes/fact-projects-groups-synonyms.rst

Resources
---------

.. include:: /includes/api-base-url.rst

The following lists the endpoints available for managing 
private endpoints on |aws| and |azure|:

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - GET
     - :doc:`/groups/{GROUP-ID}/privateEndpoint/{CLOUD-PROVIDER}/endpointService </reference/api/private-endpoints-service-get-all>`
     - Retrieve details for all private endpoint services for |aws| or
       |azure| in one |service| project.

   * - GET
     - :doc:`/groups/{GROUP-ID}/privateEndpoint/{CLOUD-PROVIDER}/endpointService/{ENDPOINT-SERVICE-ID} </reference/api/private-endpoints-service-get-one>`
     - Retrieve details for one private endpoint service for |aws| or
       |azure| in an |service| project.

   * - POST
     - :doc:`/groups/{GROUP-ID}/privateEndpoint/endpointService/ </reference/api/private-endpoints-service-create-one>`
     - Create one private endpoint service for |aws| or |azure| in an
       |service| project.

   * - DELETE
     - :doc:`/groups/{GROUP-ID}/privateEndpoint/{CLOUD-PROVIDER}/endpointService/{ENDPOINT-SERVICE-ID} </reference/api/private-endpoints-service-delete-one>`
     - Delete one private endpoint service for |aws| or |azure| in an
       |service| project.

   * - GET
     - :doc:`/groups/{GROUP-ID}/privateEndpoint/{CLOUD-PROVIDER}/endpointService/{ENDPOINT-SERVICE-ID}/endpoint/{ENDPOINT-ID} </reference/api/private-endpoints-endpoint-get-one>`
     - Retrieve details for one private endpoint for |aws| or |azure| in
       an |service| project.

   * - POST
     - :doc:`/groups/{GROUP-ID}/privateEndpoint/{CLOUD-PROVIDER}/endpointService/{ENDPOINT-SERVICE-ID}/endpoint </reference/api/private-endpoints-endpoint-create-one>`
     - Create one private endpoint for |aws| or |azure| in an |service|
       project.

   * - DELETE
     - :doc:`/groups/{GROUP-ID}/privateEndpoint/{CLOUD-PROVIDER}/endpointService/{ENDPOINT-SERVICE-ID}/endpoint/{ENDPOINT-ID} </reference/api/private-endpoints-endpoint-delete-one>`
     - Delete one private endpoint for |aws| or |azure| from an
       |service| project.

   * - GET
     - :doc:`/groups/{GROUP-ID}/privateEndpoint/regionalMode </reference/api/private-endpoints-get-regional-mode>`
     - Retrieve whether the regionalized private endpoint feature is 
       enabled for one |service| project.

   * - PATCH
     - :doc:`/groups/{GROUP-ID}/privateEndpoint/regionalMode </reference/api/private-endpoints-update-regional-mode>`
     - Enable or disable the regionalized private endpoint feature for
       one |service| project.
