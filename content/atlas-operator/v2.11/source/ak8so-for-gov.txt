==================================
|ak8so| for Government Quick Start
==================================

.. meta::
   :description: Learn how to configure and use the Atlas Kubernetes Operator to manage resources, deployments, and users in your MongoDB Atlas for Government environments.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 0
   :class: singlecol

You can use the |ak8so| to manage resources in |service| for Government.
The |ak8so| only supports managing |service| for Government on |aws| and |gcp|. It
doesn't support {+adf+}, {+Serverless-instances+}, or {+Flex-clusters+}.
For more information, see :atlas:`Supported Features
</government/overview/supported-features/>` in the |service-fullname|
for Government documentation. 

This page describes how to configure the |ak8so| to manage resources in
|service| for Government.

Prerequisites
-------------

You must install the |ak8so| using {+atlas-cli+} with the ``--gov``
flag. To learn more about installing the |ak8so| using the
{+atlas-cli+}, see :ref:`ak8so-quick-start-ref`.

Procedure 
---------

After you install the |ak8so|, do the following:

.. procedure:: 
   :style: normal 

   .. step:: Deploy |ak8so|. 

      Follow the procedure in the :ref:`ak8so-quick-start-ref` to deploy
      |ak8so|. If you install the |ak8so| using the {+atlas-cli+}, you must
      append the ``--atlasGov`` flag to the installation command.

   .. step:: Configure the |ak8so| for the |service| for Government URL.

      Configure the |ak8so| to connect to the |service| for Government
      API URL (``https://cloud.mongodbgov.com/``):

      - If you deploy using Helm, set ``atlasURI`` to
        ``https://cloud.mongodbgov.com/`` in your ``values.yaml`` file.
      - If you deploy the operator directly, pass the
        ``--atlas-domain=https://cloud.mongodbgov.com/`` flag when
        starting the operator.

      If you deployed the |ak8so| using the {+atlas-cli+} with the
      ``--atlasGov`` flag in the previous step, the operator is already
      configured to use the correct URL.

   .. step:: Set the cloud provider region for |service| for Government.

      Use the ``spec.regionUsageRestrictions`` parameter in the
      ``AtlasProject`` custom resource to set the cloud provider region for
      |service| for Government. If you omit the
      ``spec.regionUsageRestrictions`` parameter,
      this setting defaults to ``COMMERCIAL_FEDRAMP_REGIONS_ONLY``. To learn more, see
      :setting:`spec.regionUsageRestrictions`.
