========================================
Install or Uninstall ``mongot`` on Linux 
========================================

.. meta::
   :description: Learn about installing or uninstalling ``mongot`` on Linux.

.. facet::
   :name: genre
   :values: tutorial

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

You can install the {+fts+} and {+avs+} process, ``mongot``, in MongoDB 
Community Edition. The ``mongot`` process is available for deployment for all Linux 
distributions such as Red Hat, Ubuntu, Debian, SUSE, and Amazon Linux as a 
tarball with the ``.tgz`` extension.

.. note::

   To onboard quickly with a deployment for experimentation, see the 
   following tutorials for Self-Managed deployments:

   - :ref:`Local Development Quickstart <mongot-deployment-using-atlas-local>`
   - :ref:`MongoDB Search Quick Start <fts-tutorial-ref>`  
   - :ref:`MongoDB Vector Search Quick Start <vector-search-quick-start>`
