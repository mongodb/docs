.. dbcommand:: getnonce

   .. deprecated:: 4.0 

   Client libraries use :dbcommand:`getnonce` to generate a one-time
   password for authentication.
   
   Applications should not invoke :dbcommand:`getnonce` directly.

   This command will be removed in a future release.
