.. _mcli-atlas-cluster-watch-cmd:

============================
mongocli atlas cluster watch
============================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

.. include:: /includes/styles/corrections.rst

The ``cluster watch`` command monitors a running cluster in 
the specified project until it becomes available, at which time it 
displays a confirmation message. You can also view the 
availability of a cluster with the |service| 
:atlas:`UI </tutorial/view-all-clusters/>`.

.. _atlas-cluster-watch-syntax:

Syntax
------

.. code-block:: text

   mongocli atlas cluster watch <cluster-name>
        [ --output|-o <output-format> ]
        [ --projectId <project-ID> ]
        [ --profile|-P <profile-name> ]

.. include:: /includes/fact-command-line-help.rst

.. _atlas-cluster-watch-options:

Options
-------

.. list-table::
   :header-rows: 1
   :widths: 20 10 60 10

   * - Option 
     - Type 
     - Description 
     - Required? 

   * - ``<cluster-name>``
     - string
     - Name of the cluster you want to watch.
     - yes

   * - ``--output``, ``-o``
     - string 
     - .. include:: /includes/extracts/fact-basic-options-output.rst
     - no

   * - ``--profile``, ``-P``
     - string
     - Name of the profile where the public and private 
       keys for the project are set. If omitted, uses the 
       ``default`` profile. To learn more about creating a 
       profile, see :ref:`mcli-configure`.
     - no

   * - ``--projectId``
     - string
     - Unique identifier of the project that contains the cluster you
       want to watch. If omitted, uses the project ID in 
       the profile or :ref:`environment variable <mcli-env-var>`.
     - no

.. _atlas-cluster-watch-output-fields:

Output
------

The ``cluster watch`` command continuously polls a cluster until the 
cluster reaches an available state. While the command is polling, it 
periodically prints dots (``.``) to the terminal.

When the cluster reaches an available state, or if it is already in an 
available state, the command prints a status message and list of hosts 
for that cluster.

If the command prints an error, see 
:ref:`Troubleshooting <mcli-troubleshooting>` for recommended solutions.

.. _atlas-cluster-watch-examples: 

Example
-------

The following ``mongocli atlas cluster watch`` command monitors a sample
cluster named ``myCluster``. The command uses the default profile. 

.. code-block:: text
   :copyable: false

   mongocli atlas cluster watch myCluster --projectId abcdefghij0fab66c7d4072e

In this example, ``myCluster`` is instantiated just before running the 
command. The command prints dots while the cluster is instantiating, 
then prints a status message and list of hosts:

.. code-block:: text
   :copyable: false

   .......................
   Cluster available at: mongodb://cluster0-shard-00-00.111xx.mongodb.net:27017,cluster0-shard-00-01.111xx.mongodb.net:27017,cluster0-shard-00-02.111xx.mongodb.net:27017/?ssl=true&authSource=admin&replicaSet=atlas-123abc-shard-0
