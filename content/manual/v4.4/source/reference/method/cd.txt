Definition

.. method:: cd(path)


   .. list-table::
      :header-rows: 1
      :widths: 20 20 80
   
      * - Parameter
   
        - Type
   
        - Description
   
      * - ``path``
   
        - string
   
        - A path on the file system local to the :binary:`~bin.mongo` shell context.
          
          
   


   :method:`cd()` changes the directory context of the
   :binary:`~bin.mongo` shell and has no effect on the MongoDB server.
