=========================
Configure SSL for MongoDB
=========================

.. default-domain:: mongodb

Overview
--------

If your MongoDB deployment uses SSL, then you must configure the
:guilabel:`Use SSL` setting for the deployment, as described here.

.. only:: onprem

   You also must configure the agents' SSL settings, as described in
   :doc:`/tutorial/configure-monitoring-agent-for-ssl` and
   :doc:`/tutorial/configure-backup-agent-for-ssl`.

Procedures
----------

Edit Host SSL Use for Monitoring
++++++++++++++++++++++++++++++++

Before editing these credentials, update the agent's configuration file to
use SSL. :doc:`/tutorial/configure-monitoring-agent-for-ssl`.

To specify SSL use:
