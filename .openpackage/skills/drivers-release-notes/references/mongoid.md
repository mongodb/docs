# Mongoid

**Add for:** major, minor, and patch releases

| Variable | Value |
|---|---|
| `GITHUB_REPO` | `mongodb/mongoid` |
| `RELEASE_NOTES_FILE` | `content/mongoid/upcoming/source/reference/release-notes.txt` |
| `CHANGELOG_LINK` | `mongodb/mongoid/releases/tag/v{FULL_VERSION}` |
| `UPGRADE_REF` | n/a — no upgrade guide |

- Breaking changes: no upgrade guide. Use the **Mongoid: Breaking Changes
  Admonition** template from `references/docs-templates.md` immediately
  following the `What's New in {VERSION}` header.
