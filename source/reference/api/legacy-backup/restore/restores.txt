==============================
Restore {+Old-Backup+} Jobs
==============================

.. default-domain:: mongodb

.. include:: /includes/admonitions/deprecated-continuous-backup.rst

.. include:: /includes/fact-projects-groups-synonyms.rst

Create and retrieve restores jobs for a MongoDB cluster. A restore
job restores a MongoDB cluster to its state from an existing snapshot.

.. include:: /includes/fact-continuous-backup-only-resource.rst

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - GET
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/restoreJobs </reference/api/legacy-backup/restore/get-all-restore-jobs>`
     - Retrieve all restore jobs for the specified cluster.

   * - GET
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/restoreJobs/{JOB-ID} </reference/api/legacy-backup/restore/get-one-restore-job>`
     - Retrieve one restore job for the specified cluster.

   * - POST
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/restoreJobs </reference/api/legacy-backup/restore/create-one-restore-job>`
     - Create one restore job for the specified cluster.
