===================================
Continuous Backup Snapshot Schedule
===================================

.. default-domain:: mongodb

.. include:: /includes/admonitions/deprecated-continuous-backup.rst

.. include:: /includes/fact-projects-groups-synonyms.rst

This resource enables you to view and edit the :ref:`backup snapshot
schedule <modify-backup-schedule>` for a given cluster.

.. include:: /includes/fact-continuous-backup-only-resource.rst

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/snapshotSchedule </reference/api/snapshot-schedule-get>`
     - Get the snapshot schedule for the specified cluster.

   * - ``PATCH``
     - :doc:`/groups/{GROUP-ID}/clusters/{CLUSTER-NAME}/snapshotSchedule </reference/api/snapshot-schedule-patch>`
     - Update the snapshot schedule for the specified cluster.
