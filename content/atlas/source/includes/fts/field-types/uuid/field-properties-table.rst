Configure |fts-field-type| Field Properties 
-------------------------------------------

The |fts| |fts-field-type| type takes the following parameters:

.. list-table::
   :widths: 20 10 10 40
   :header-rows: 1

   * - Option
     - Type
     - Necessity
     - Description

   * - ``type``
     -  ``uuid``
     - Required
     - Human-readable label that identifies this field type.
       Value must be ``uuid``.
