.. _live-migration-api:

=====================
Live Migration (Push)
=====================

.. default-domain:: mongodb

.. meta::
   :canonical: https://www.mongodb.com/docs/atlas/reference/api-resources-spec/#tag/Cloud-Migration-Service

Considerations
--------------

- Source organizations, projects, and MongoDB clusters reside on |com|.

- Destination organizations, projects, and MongoDB clusters reside on
  |service|.

- Source databases can't use any authentication except SCRAM-SHA.


Endpoints
---------

.. include:: /includes/api-base-url-new.rst

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``POST``
     - :doc:`/orgs/{orgId}/liveMigrations/linkTokens </reference/api/live-migration/create-one-link-token>`
     - Create one new link-token for push live migrations.
   * - ``DELETE``
     - :doc:`/orgs/{orgId}/liveMigrations/linkTokens </reference/api/live-migration/delete-one-link-token>`
     - Delete one link-token.
   * - ``POST``
     - :doc:`/groups/{groupId}/liveMigrations/validate </reference/api/live-migration/validate-one-migration-request>`
     - Create one new validation request for push live migration.
   * - ``GET``
     - :doc:`/groups/{groupId}/liveMigrations/validate/{validationId} </reference/api/live-migration/return-one-specific-validation-job>`
     - Return status for one validation job for push live migration.
   * - ``POST``
     - :doc:`/groups/{groupId}/liveMigrations </reference/api/live-migration/create-one-migration>`
     - Create one new push live migration job.
   * - ``GET``
     - :doc:`/groups/{groupId}/liveMigrations/{liveMigrationId} </reference/api/live-migration/return-one-specific-migration>`
     - Return status of one push live migration job and the list of
       configured migration hosts.
   * - ``PUT``
     - :doc:`/groups/{groupId}/liveMigrations/{liveMigrationId}/cutover </reference/api/live-migration/start-the-migration-cutover>`
     - Start one push live migration process.
