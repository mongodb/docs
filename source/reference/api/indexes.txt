=============
Rolling Index
=============

.. default-domain:: mongodb

The ``index`` resource lets you create a :manual:`rolling index
</tutorial/build-indexes-on-replica-sets/>` for a cluster. Rolling
indexes are created on one cluster node at a time, reducing system stress.
