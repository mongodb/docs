=============================
|monitoring| Service Tutorial
=============================

.. only:: hosted

   To begin using |monitoring|, :doc:`install MMS On-Prem
   </installation>`, then install the agent as
   outlined below. 

To monitor MongoDB systems with MMS you must install a lightweight agent
that can connect to all MongoDB instances to gather information. See
the :doc:`/monitoring/tutorial/set-up-mms` document for complete information.
