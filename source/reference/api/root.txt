====
Root
====

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

The **root** resource is the starting point for the |mms| API. From
here, you can traverse the ``links`` to reach all other API resources.

.. include:: /includes/api/base-api-uri.rst

Resource
--------

.. code-block:: http

   GET /

Response
--------

The response includes:

.. list-table::
   :widths: 15 15 70
   :header-rows: 1
   :stub-columns: 1

   * - Name
     - Type
     - Description

   * - appName
     - string
     - .. only:: cloud

          MongoDB Cloud

       .. only:: onprem

          MongoDB Ops Manager

   * - build
     - string
     - The SHA that corresponds to the GitHub commit for the |mms|.

   * - links
     - object array
     - .. include:: /includes/api/links-explanation.rst

   * - throttling
     - boolean
     - The field is obsolete and has the value ``false``.

Example Request
---------------

.. only:: cloud

   .. class:: copyable-code

   .. code-block:: sh

      curl --user '{USERNAME}:{APIKEY}' --digest \ 
       --header 'Accept: application/json' \ 
       --include \ 
       --request GET "https://cloud.mongodb.com/api/public/v1.0"

.. only:: onprem

   .. class:: copyable-code

   .. code-block:: sh

      curl --user '{USERNAME}:{APIKEY}' --digest \ 
       --header 'Accept: application/json' \ 
       --include \ 
       --request GET "<ops-manager-host>:<port>/api/public/v1.0"
