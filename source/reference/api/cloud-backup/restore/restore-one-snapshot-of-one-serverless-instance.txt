===============================================
Restore One Snapshot Of One Serverless Instance
===============================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

Restore one snapshot for the specified serverless instance.

Required Roles
--------------

Your API Key must have one of the following roles to
successfully call this resource:

- :authrole:`Organization Owner`
- :authrole:`Project Owner`

.. include:: /includes/api/facts/api-access-list-required.rst

Request
-------

.. include:: /includes/api/base-api-uri.rst

.. code-block:: http

   POST /groups/{groupId}/serverless/{instanceName}/backup/restoreJobs

Request Path Parameters
~~~~~~~~~~~~~~~~~~~~~~~

.. list-table::
   :widths: 20 14 11 55
   :header-rows: 1
   :stub-columns: 1

   * - Name
     - Type
     - Necessity
     - Description

   * - groupId
     - string
     - Required
     - Unique 24-hexadecimal digit string that identifies your project.

   * - instanceName
     - string
     - Required
     - Human-readable label that identifies your serverless instance.

Example Request
---------------

.. code-block:: sh
   :linenos:
   :emphasize-lines: 6-11

   curl --user '{PUBLIC-KEY}:{PRIVATE-KEY}' --digest \ 
        --header 'Accept: application/json' \ 
        --header 'Content-Type: application/json' \ 
        --include \ 
        --request POST 'https://cloud.mongodb.com/api/atlas/v1.0/groups/{groupId}/serverless/{instanceName}/backup/restoreJobs?pretty=true' \ 
        --data '{
               "snapshotId": "{snapshotId}",
               "deliveryType": "automated",
               "targetGroupId": "{targetGroupId}",
               "targetClusterName": "{clusterName}"
            }'
