=================
Group Server Pool
=================

.. default-domain:: mongodb

.. include:: /includes/fact-server-pools-deprecated.rst

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

.. include:: /includes/fact-groups-projects-synonyms.rst

The ``groups/{PROJECT-ID}/serverPool`` resource returns the state of |mms|
:term:`server pool`. It is the root resource of the group's ``serverPool``
resource set and is a resource subset of ``groups``.

For detailed information on Server pools, see
:doc:`/tutorial/server-pool`.

For a list of all resources in the server pool resource set, see
:ref:`server-pool-api-resources`.

.. include:: /includes/api/base-api-uri.rst

Resource
--------

.. code-block:: http

   GET /groups/{PROJECT-ID}/serverPool

Response
--------

.. only:: onprem

  .. list-table::
     :widths: 10 10 80
     :header-rows: 1

     * - Name
       - Type
       - Description

     * - enabled
       - boolean
       - If ``true``, the server pool is enabled.

     * - links
       -
       -
         .. list-table::
            :header-rows: 1

            * - Relation
              - Description

            * - self
              - Me

            * - ``https://{opsManagerHost}:{port}/serverPoolServers``
              - The servers in the pool.

            * - ``https://{opsManagerHost}:{port}/serverPoolProperties``
              - The properties used to identify servers in the server  pool.

            * - ``https://{opsManagerHost}:{port}/serverPoolRequests``
              - All requests for servers from the server pool.
