=====================
Test a New Deployment
=====================

.. default-domain:: mongodb

Overview
--------

The following procedure deploys a MongoDB :term:`replica set` with random data
for use in testing an MMS deployment. Deploy the replica set to a separate
machine from the MMS deployment.

These instructions install the replica set on a server running **RHEL 6+** or
**Amazon Linux**. The procedure deploys all the members to one server.
