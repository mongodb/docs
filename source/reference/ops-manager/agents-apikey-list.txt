.. _mcli-om-agents-apikey-list-command:

=======================================
mongocli ops-manager agents apikey list
=======================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``agents apikey list`` command retrieves all agent API keys for
the specified |onprem| project.

You must have at least ``GROUP_READ_ONLY`` privileges for a project to
retrieve agent |api| keys for that project.

.. _om-agents-apikey-list-syntax:

Syntax 
------

.. code-block:: text 

   mongocli ops-manager|om agents apikey|apikeys list|ls
        [ --output|-o <output-format> ]
        [ --profile|-P <profile-name> ]
        [ --projectId <id-of-project> ]

.. _om-agents-apikey-list-options:

Options 
-------

.. list-table::
   :widths: 20 10 60 10
   :header-rows: 1

   * - Option 
     - Type
     - Description
     - Required?

   * - ``--output``, ``-o``
     - string 
     - .. include:: /includes/extracts/fact-basic-options-output.rst
     - no

   * - ``--profile``, ``-P``
     - string
     - Name of the profile where the public and private 
       keys for the project are saved. If omitted, uses the 
       {+default-profile+}. To learn more about creating a 
       profile, see :ref:`mcli-configure`.
     - no

   * - ``--projectId``
     - string
     - .. include:: /includes/extracts/fact-basic-options-project-id.rst
     - no

Example
-------

.. tabs::

   .. tab:: Default Output
      :tabid: default-output

      The following command retrieves agent API keys for a project
      using the {+default-profile+}, which contains credentials and the
      project ID. The output is returned in the default format.

      .. code-block:: sh

         mongocli om agents apikey list

      The previous command prints the following to the terminal.

      .. include:: /includes/admonitions/only-last-4-agent-api-key.rst

      .. code-block:: sh
         :copyable: false

         ID                         KEY                                DESCRIPTION                                CREATED AT
         5f1f3a04c902201990f5388a   ****************************39fe   Initial API Key                            1595881988159
         5f1f39ffc902201990f5387d   ****************************8b87   Public API generated group agent api key   1595881983276
