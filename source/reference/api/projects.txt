========
Projects
========

.. default-domain:: mongodb

.. include:: /includes/fact-projects-groups-synonyms.rst

The ``groups`` resource provides access to retrieve or create |service|
:ref:`projects <projects>`.

Endpoints
---------

.. list-table::
   :header-rows: 1
   :widths: 10 60 30

   * - Method
     - Endpoint
     - Description

   * - ``GET``

     - :doc:`/api/atlas/v1.0/groups </reference/api/project-get-all>`

     - Get all projects the authenticated user has access to.

   * - ``GET``

     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID} </reference/api/project-get-one>`

     - Get information about the project associated with ``{GROUP-ID}``.

   * - ``GET``

     - :doc:`/api/atlas/v1.0/groups/byName/{GROUP-NAME} </reference/api/project-get-one-by-name>`

     - Get information about the project associated with ``{GROUP-NAME}``.

   * - ``POST``

     - :doc:`/api/atlas/v1.0/groups </reference/api/project-create-one>`

     - Create a project.

   * - ``GET``

     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/teams/ </reference/api/project-get-teams>`

     - Get information about the teams assigned to the project 
       associated with ``{GROUP-ID}``.

   * - ``POST``
     
     - :doc:`/api/atlas/v1.0/groups/{GROUP-ID}/teams/ </reference/api/project-add-team>`
     
     - Assign an |service| :ref:`team <manage-teams>` to the 
       project associated with ``{GROUP-ID}``.
