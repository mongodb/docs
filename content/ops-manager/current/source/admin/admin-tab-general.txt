====================================
Administration Console : General Tab
====================================

.. meta::
   :description: Access system topology, users, API keys, audit events, MongoDB version manifest, and configuration settings through the Ops Manager Administration Console General tab.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

The :guilabel:`General` tab gives access to system topology, users,
messages, and other information. Use the sub-pages to configure
|onprem|, manage users and |api| keys, review audit events, and
view MongoDB version usage across your deployments.
