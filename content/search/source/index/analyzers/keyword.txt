================
Keyword Analyzer
================

.. default-domain:: mongodb

.. meta::
   :keywords: term search, keyword search, full-text search, text analyzer, single term index, exact match search, index strings, single string search, code example
   :description: Use the MongoDB Search keyword analyzer to index multiple terms in a string field as a single searchable term. Only exact matches on the field are returned.

The ``keyword`` analyzer accepts a string or array of strings as a
parameter and indexes them as a single term (token). Only exact matches
on the field are returned. It leaves all text in its original
letter case.

.. tip::

   For exact matching, instead of the ``keyword`` analyzer, index
   the field as the |fts| :ref:`token <bson-data-types-token>` type.
   Use the :ref:`equals <equals-ref>` operator to search the field.

|fts| returned the document because it matched the query term ``Class
Action`` to the single token ``Class Action`` that it creates for the
text in the field using the ``lucene.keyword`` analyzer. By contrast,
|fts| doesn't return any results for the following query:

.. code-block:: json 
   :copyable: true 

   db.cases.aggregate([
     {
       "$search": {
         "text": {
           "query": "action",
           "path": "title"
         }
       }
     }
   ])

Many documents in the collection contain the string ``action``.
The ``keyword`` analyzer returns no results for this query. It
only matches documents where the search term matches the entire
field contents. If you index the field using the
:ref:`ref-standard-analyzer` or :ref:`ref-simple-analyzer`, |fts|
returns many results. The results include the document with the
title ``Class Action``. Those analyzers create tokens similar to
the following and match them to the query term:

   :header-rows: 1

   * - Analyzer 
     - Output Tokens
     - Matches ``action`` 
     - Matches ``Class Action``

   * - Keyword Analyzer Tokens 
     - ``Class Action``
     - X
     - √
     
   * - Standard Analyzer Tokens 
     - ``class``, ``action``
     - √
     - √

   * - Simple Analyzer Tokens
     - ``class``, ``action``
