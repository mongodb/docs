Prerequisites
-------------

To delete a |service| cluster, you must have:

1. :doc:`The Atlas Open Service Broker installed </installation>` and
#. :doc:`At least one database deployed </deploy>`.

Procedure
---------

Pass ``metadata.name`` from the ``ServiceInstance`` |yaml| file of
the replica set or sharded cluster that you want to delete into the
following command: 

.. code-block:: sh

   kubectl delete serviceinstances <METADATA.NAME> -n <METADATA.NAMESPACE>
