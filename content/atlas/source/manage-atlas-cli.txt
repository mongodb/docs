Generate an Autocompletion Script
---------------------------------

.. tabs::

   .. tab:: Bash
      :tabid: bash

      To learn how to generate the autocompletion script for Bash,
      see :atlascli:`atlas completion bash 
      </command/atlas-completion-bash/>`.

   .. tab:: fish
      :tabid: fish

      To learn how to generate the autocompletion script for fish,
      see :atlascli:`atlas completion fish
      </command/atlas-completion-fish/>`.

   .. tab:: PowerShell
      :tabid: powershell

      To learn how to generate the autocompletion script for PowerShell,
      see :atlascli:`atlas completion powershell
      </command/atlas-completion-powershell/>`.

   .. tab:: zsh
      :tabid: zsh

      To learn how to generate the autocompletion script for zsh,
      see :atlascli:`atlas completion zsh
      </command/atlas-completion-zsh/>`.
