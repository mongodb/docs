atlas api cloudProviderAccess authorizeCloudProviderAccessRole
==============================================================

`experimental <https://www.mongodb.com/docs/atlas/cli/current/command/atlas-api/>`_: Grants access to the specified project for the specified access role.

To use this resource, the requesting API Key must have the Project Owner role. This API endpoint is one step in a procedure to create unified access for MongoDB Cloud services. This is not required for GCP service account access.

This command is autogenerated and corresponds 1:1 with the Atlas API endpoint https://www.mongodb.com/docs/atlas/reference/api-resources-spec/v2/#tag/Cloud-Provider-Access/operation/authorizeCloudProviderAccessRole.

For more information and examples, see the referenced API documentation linked above.
