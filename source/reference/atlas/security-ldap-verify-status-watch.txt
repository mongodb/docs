.. _mcli-atlas-security-ldap-verify-status-watch-cmd:

================================================
mongocli atlas security ldap verify status watch
================================================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

The ``security ldap verify status watch`` command watches 
for the |ldap| configuration request to complete. 

.. _mcli-atlas-security-ldap-verify-status-watch-syntax: 

Syntax 
------

.. code-block:: shell 
   
   mongocli atlas security ldap verify status watch <request-ID>
        [ --profile|-P <name-of-profile> ]
        [ --projectId <id-of-project> ]

.. include:: /includes/fact-command-line-help.rst

.. _mcli-atlas-security-ldap-verify-status-watch-args: 

Arguments 
---------

.. list-table::
   :header-rows: 1
   :widths: 25 10 55 10

   * - Argument 
     - Type 
     - Description 
     - Required?

   * - ``<request-ID>``
     - string
     - Unique identifier of the most recent request to verify 
       the |ldap| over |tls|/|ssl| configuration. Run the 
       :ref:`mcli-atlas-security-ldap-verify-cmd` command 
       for the request ID.
     - yes

.. _mcli-atlas-security-ldap-verify-status-watch-options:

Options 
-------

.. list-table::
   :header-rows: 1
   :widths: 25 10 55 10

   * - Option 
     - Type 
     - Description 
     - Required?

   * - ``--profile``, ``-P``
     - string 
     - Name of the profile to use for accessing the |service| 
       project. If omitted, uses the {+default-profile+}.
     - no

   * - ``--projectId``
     - string
     - Unique identifier of the |service| project. 
     - yes

.. _mcli-atlas-security-ldap-verify-status-watch-output: 

Output 
------

The command prints the following message to the terminal when the 
|ldap| configuration request completes. If the command returns 
errors, see :ref:`Troubleshooting<mcli-troubleshooting>` for 
recommended solutions.

.. code-block:: shell 
   :copyable: false 

   LDAP Configuration request completed.

.. _mcli-atlas-security-ldap-verify-status-watch-egs: 

Example 
-------

The following example uses the ``mongocli atlas security ldap verify 
status watch`` command watches the |ldap| configuration request 
specified by the verify status request ID. The command uses the 
{+default-profile+} to access the |service| project.

.. code-block:: shell 
   
   mongocli atlas security ldap verify status watch 5fa1ca4dcf78f219ebca052e 

The previous command prints the following to the terminal when the 
|ldap| configuration request completes: 

.. code-block:: shell 
   :copyable: false 

   LDAP Configuration request completed.
