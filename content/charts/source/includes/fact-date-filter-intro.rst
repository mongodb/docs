When you drag a date field to the filter panel, you can
filter based on a specified date range. This range can
either be:

- A relative date range, which specifies a range
  relative to the time the chart is rendered (for example,
  the last six months).

- A date range based on the current time when the
  chart is rendered, with a choice of periods.

- An absolute date range, which is a range between
  specific start and end dates.
