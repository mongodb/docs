===================
Custom Role Actions
===================

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 1
   :class: singlecol

This document lists the possible
:manual:`privilege actions </reference/privilege-actions/>`
you can assign to custom roles via the |service| |api|.
Specify one of these actions in the ``actions.action`` request body
parameter when you :doc:`create </reference/api/custom-roles-create-a-role>`
or :doc:`update </reference/api/custom-roles-create-a-role>` a
custom role.

.. note::

   The privilege actions available to the Custom Roles
   API resource represent a subset of the privilege actions
   available in the |service| Custom Roles UI.

Query and Write Actions
-----------------------

- :authaction:`FIND <find>`

- :authaction:`INSERT <insert>`

- :authaction:`REMOVE <remove>`

- :authaction:`UPDATE <update>`

- :authaction:`BYPASS_DOCUMENT_VALIDATION <bypassDocumentValidation>`

- :authaction:`USE_UUID <useUUID>`
