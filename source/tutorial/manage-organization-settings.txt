Require IP Access List for Public API
-------------------------------------

You can configure |service| to require API access lists at the
organization level. When enabled, all API calls within
that organization must originate from an entry on each respective
{+atlas-admin-api+} key access list.

.. seealso::
   :ref:`API Access Lists <atlas-admin-api-access>`
