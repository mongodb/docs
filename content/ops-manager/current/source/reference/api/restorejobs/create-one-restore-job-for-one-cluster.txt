======================================
Create One Restore Job for One Cluster
======================================

.. meta::
   :description: Create a restore job for a cluster in Ops Manager by specifying the project ID, cluster ID, delivery method, and restore type for HTTP or automated restores.

.. default-domain:: mongodb

.. contents:: On this page
   :local:
   :backlinks: none
   :depth: 2
   :class: singlecol

Use this endpoint to create a restore job for a cluster. Select your
restore method to view the appropriate request body parameters,
response elements, and examples.

.. include:: /includes/api/base-api-uri.rst

.. composable-tutorial::
   :options: restore-method, restore-type
   :defaults: http, None

   .. selected-content::
      :selections: http, None

      Resource
      --------

      .. code-block:: http

         POST /groups/{PROJECT-ID}/clusters/{CLUSTER-ID}/restoreJobs

      Request Path Parameters
      ~~~~~~~~~~~~~~~~~~~~~~~

      .. list-table::
         :widths: 20 20 60
         :header-rows: 1
         :stub-columns: 1

         * - Name
           - Type
           - Description

         * - ``PROJECT-ID``
           - string
           - Unique identifier of the project that owns the job.

         * - ``CLUSTER-ID``
           - string
           - Unique identifier of the cluster that the job
             represents.
